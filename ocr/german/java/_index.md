## Frequently Asked Questions

**Q: Kann ich Aspose.OCR für Java in einer kommerziellen Anwendung verwenden?**  
A: Ja. Eine gültige Aspose‑Lizenz entfernt alle Evaluations‑Einschränkungen und ermöglicht unbegrenzte Bereitstellung in kommerziellen Produkten.

**Q: Welche Bildformate werden für die Textextraktion unterstützt?**  
A: JPEG, PNG, BMP, TIFF, GIF und bildbasierte PDF‑Seiten werden alle sofort unterstützt.

**Q: Wie kann ich die Genauigkeit für mehrsprachige Dokumente verbessern?**  
A: Setzen Sie die `Language`‑Eigenschaft auf den entsprechenden ISO‑Sprachcode (z. B. `en`, `fr`, `zh`), bevor Sie die OCR‑Engine aufrufen.

**Q: Gibt es eine Möglichkeit, nur einen bestimmten Bildbereich zu verarbeiten?**  
A: Absolut. Verwenden Sie `DetectAreasMode` oder definieren Sie manuell ein Rechteck, um die OCR auf den gewünschten Bereich zu beschränken.

**Q: Was soll ich tun, wenn der extrahierte Text viele Rechtschreibfehler enthält?**  
A: Nachbearbeiten Sie das Ergebnis mit einer Rechtschreib‑Prüfbibliothek oder nutzen Sie Asposes integrierte `CorrectSpelling`‑Methode (in neueren Versionen verfügbar).
