# Εκτέλεση OCR σε Εικόνα με Java – Πλήρης Οδηγός

Έχετε αναρωτηθεί ποτέ πώς να **εκτελέσετε OCR σε εικόνα** απευθείας από κώδικα Java; Ίσως έχετε μια συλλογή από ταϊλανδικές αποδείξεις, σαρωμένα έγγραφα ή στιγμιότυπα οθόνης και χρειάζεστε το κείμενο χωρίς να το πληκτρολογήσετε χειροκίνητα. Αυτό είναι ένα κοινό πρόβλημα, ειδικά όταν η πηγή είναι PNG και η γλώσσα δεν είναι λατινική.

Σε αυτόν τον οδηγό θα σας δείξουμε ακριβώς πώς να **εκτελέσετε OCR σε εικόνα** χρησιμοποιώντας τη βιβλιοθήκη Aspose OCR, να μετατρέψετε την εικόνα σε απλό κείμενο και να εξάγετε αξιόπιστα χαρακτήρες ταϊλανδικής. Στο τέλος θα μπορείτε να **μετατρέψετε εικόνα σε κείμενο**, **εξάγετε κείμενο από PNG** και ακόμη να **αναγνωρίσετε ταϊλανδικό κείμενο** με λίγες μόνο γραμμές κώδικα.

## Τι Καλύπτει Αυτός ο Οδηγός

* Ρύθμιση της εξάρτησης Aspose OCR σε έργο Maven/Gradle.  
* Αρχικοποίηση του `OcrEngine` και ρύθμιση του για ταϊλανδική γλώσσα.  
* Εκτέλεση OCR σε αρχείο PNG και διαχείριση πιθανών σφαλμάτων.  
* Εκτύπωση του αποτελέσματος στην κονσόλα και επαλήθευση της εξόδου.  

Δεν απαιτείται προηγούμενη εμπειρία με το Aspose — αρκεί ένα βασικό IDE Java και ένα αρχείο εικόνας που θέλετε να επεξεργαστείτε.

## Προαπαιτούμενα

* Java 8 ή νεότερη εγκατεστημένη (η βιβλιοθήκη λειτουργεί επίσης με Java 11+).  
* Εργαλείο κατασκευής — Maven ή Gradle (θα δείξουμε το απόσπασμα Maven).  
* Άδεια Aspose OCR (η δωρεάν δοκιμή λειτουργεί για δοκιμές, αλλά μια άδεια αφαιρεί τους περιορισμούς αξιολόγησης).  
* Ένα PNG που περιέχει ταϊλανδικό κείμενο, π.χ. `input.png` τοποθετημένο στο φάκελο resources του έργου σας.

## Βήμα 1: Προσθήκη Aspose OCR στο Έργο Σας

### Maven

```xml
<dependency>
    <groupId>com.aspose</groupId>
    <artifactId>aspose-ocr</artifactId>
    <version>23.9</version> <!-- Use the latest version -->
</dependency>
```

### Gradle (Kotlin DSL)

```kotlin
implementation("com.aspose:aspose-ocr:23.9")
```

> **Συμβουλή:** Κρατήστε την έκδοση της βιβλιοθήκης σε συγχρονισμό με τις επίσημες σημειώσεις κυκλοφορίας του Aspose· οι νεότερες εκδόσεις προσθέτουν πακέτα γλωσσών και βελτιώσεις απόδοσης.

Μόλις λυθεί η εξάρτηση, το IDE σας θα πρέπει να εισάγει αυτόματα τις απαιτούμενες κλάσεις.

## Βήμα 2: Αρχικοποίηση της Μηχανής OCR

Η μηχανή είναι η καρδιά της διαδικασίας – σκεφτείτε την ως το «εγκέφαλο» που αναλύει μοτίβα εικονοστοιχείων. Θα της πούμε επίσης ότι μας ενδιαφέρουν μόνο οι ταϊλανδικοί χαρακτήρες.

```java
import com.aspose.ocr.*;

public class ThaiOcrDemo {
    public static void main(String[] args) throws Exception {
        // Step 2: Create the OCR engine instance
        OcrEngine ocrEngine = new OcrEngine();

        // Tell the engine to look for Thai language patterns
        ocrEngine.getRecognitionSettings().setLanguage("th");
```

Γιατί να ορίσουμε ρητά τη γλώσσα; Επειδή η καθορισμένη τιμή `"th"` περιορίζει το σύνολο χαρακτήρων, επιταχύνοντας την αναγνώριση και μειώνοντας τα λανθασμένα ανάγνωσμα παρόμοιων γλυφών.

## Βήμα 3: Εκτέλεση OCR στο Αρχείο PNG

Τώρα τροφοδοτούμε τη μηχανή με την εικόνα που θέλουμε να αποκωδικοποιήσουμε. Η μέθοδος `recognizeImage` επιστρέφει ένα αντικείμενο `OcrResult` που περιέχει το εξαγόμενο κείμενο και τις βαθμολογίες εμπιστοσύνης.

```java
        // Step 3: Perform OCR on the target image
        String imagePath = "src/main/resources/input.png";
        OcrResult ocrResult = ocrEngine.recognizeImage(imagePath);
```

Αν το αρχείο δεν βρεθεί, το Aspose ρίχνει ένα `FileNotFoundException`. Είναι καλή ιδέα να τυλίξετε την κλήση σε μπλοκ `try‑catch` για κώδικα παραγωγής, αλλά για αυτόν τον οδηγό θα το κρατήσουμε απλό.

## Βήμα 4: Εξαγωγή του Αναγνωρισμένου Κειμένου

Τέλος, εκτυπώνουμε το αποτέλεσμα. Η μέθοδος `getText()` επιστρέφει ένα απλό `String` Java που μπορείτε να αποθηκεύσετε, να στείλετε μέσω δικτύου ή να γράψετε σε αρχείο.

```java
        // Step 4: Display the extracted Thai text
        System.out.println("=== Recognized Thai Text ===");
        System.out.println(ocrResult.getText());
    }
}
```

Όταν εκτελέσετε το πρόγραμμα, θα πρέπει να δείτε κάτι όπως:

```
=== Recognized Thai Text ===
สวัสดีครับ นี่คือข้อความจากรูปภาพ
```

Αν η έξοδος φαίνεται κατεστραμμένη, ελέγξτε ξανά ότι το PNG προέλευσης είναι υψηλής ανάλυσης (τουλάχιστον 300 dpi) και ότι ο κωδικός γλώσσας `"th"` ταιριάζει με τη γλώσσα-στόχο σας.

### Πλήρης Λίστα

Παρακάτω βρίσκεται το πλήρες, έτοιμο‑για‑εκτέλεση αρχείο Java. Αντιγράψτε‑και‑επικολλήστε το στο IDE σας, αντικαταστήστε τη διαδρομή της εικόνας αν χρειάζεται, και πατήστε **Run**.

```java
import com.aspose.ocr.*;

public class ThaiOcrDemo {
    public static void main(String[] args) throws Exception {
        // Initialize the OCR engine
        OcrEngine ocrEngine = new OcrEngine();

        // Select Thai language for recognition
        ocrEngine.getRecognitionSettings().setLanguage("th");

        // Run OCR on the PNG image
        String imagePath = "src/main/resources/input.png";
        OcrResult ocrResult = ocrEngine.recognizeImage(imagePath);

        // Output the recognized text
        System.out.println("=== Recognized Thai Text ===");
        System.out.println(ocrResult.getText());
    }
}
```

## Βήμα 5: Συχνές Παραλλαγές & Ακραίες Περιπτώσεις

### Μετατροπή Εικόνας σε Κείμενο σε Άλλες Μορφές

Αν χρειάζεται να **μετατρέψετε εικόνα σε κείμενο** για JPEG ή BMP, απλώς αλλάξτε την επέκταση αρχείου στο `imagePath`. Το Aspose OCR υποστηρίζει PNG, JPEG, BMP, TIFF και ακόμη σελίδες PDF.

### Εξαγωγή Κειμένου από PNG με Πολλαπλές Γλώσσες

Μπορείτε να ζητήσετε από τη μηχανή να ανιχνεύσει πολλές γλώσσες ταυτόχρονα:

```java
ocrEngine.getRecognitionSettings().setLanguage("th,en");
```

Το αποτέλεσμα θα περιέχει ένα μείγμα ταϊλανδικών και αγγλικών χαρακτήρων, χρήσιμο για δίγλωσσες αποδείξεις.

### Διαχείριση Εικόνων Χαμηλής Ποιότητας

Για θολές σαρώσεις, εξετάστε την ενεργοποίηση προεπεξεργασίας:

```java
ocrEngine.getRecognitionSettings().setPreprocessMode(PreprocessMode.Auto);
```

Αυτό ενεργοποιεί ενσωματωμένους αλγόριθμους αποθορυβοποίησης και βελτίωσης αντίθεσης, βελτιώνοντας το βήμα **εξαγωγής κειμένου από PNG**.

### Ενεργοποίηση Άδειας

Χωρίς άδεια, το Aspose προσθέτει μια γραμμή υδατογράφησης στην έξοδο μετά από 100 σελίδες. Ενεργοποιήστε την άδειά σας νωρίς:

```java
License license = new License();
license.setLicense("Aspose.OCR.lic");
```

Τοποθετήστε το αρχείο `.lic` δίπλα στο JAR ή αναφέρετέ το μέσω απόλυτης διαδρομής.
