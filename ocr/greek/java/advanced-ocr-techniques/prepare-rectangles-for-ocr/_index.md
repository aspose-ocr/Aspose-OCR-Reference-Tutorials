# Προετοιμασία Ορθογωνίων για Αναγνώριση Κειμένου OCR στο Aspose.OCR

## Εισαγωγή

Στον σημερινό κόσμο που βασίζεται στα δεδομένα, η **ocr text recognition** αποτελεί θεμέλιο για τη μετατροπή σαρωμένων εγγράφων, στιγμιοτύπων οθόνης και φωτογραφιών σε περιεχόμενο που μπορεί να αναζητηθεί και να επεξεργαστεί. Το Aspose.OCR for Java κάνει αυτή τη διαδικασία γρήγορη και αξιόπιστη, ειδικά όταν χρειάζεται να εστιάσετε σε συγκεκριμένες περιοχές μιας εικόνας. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από όλα τα απαραίτητα βήματα για την προετοιμασία ορθογωνίων που περιορίζουν το OCR στις περιοχές που σας ενδιαφέρουν, προσφέροντας ακριβή έλεγχο και καλύτερη απόδοση.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη διαχειρίζεται την αναγνώριση κειμένου OCR σε Java;** Aspose.OCR for Java.  
- **Χρειάζομαι άδεια για χρήση σε παραγωγή;** Ναι – μια έγκυρη άδεια Aspose.OCR ξεκλειδώνει τη πλήρη λειτουργικότητα.  
- **Μπορώ να περιορίσω το OCR σε συγκεκριμένα μέρη μιας εικόνας;** Απόλυτα· ορίζετε ορθογώνια που περιορίζουν τις επιθυμητές ζώνες.  
- **Ποια είναι τα κύρια προαπαιτούμενα;** JDK 17+, Aspose.OCR for Java, και ένα Java IDE.  
- **Είναι αυτή η προσέγγιση κατάλληλη για εξαγωγή κειμένου από εικόνες;** Ναι, είναι ένας αποδοτικός τρόπος για **extract text image java** projects.

## Τι είναι η Αναγνώριση Κειμένου OCR;
Η OCR (Optical Character Recognition) μετατρέπει εικόνες βασισμένες σε pixel σε χαρακτήρες που μπορεί να διαβάσει η μηχανή. Σας επιτρέπει να αναζητάτε, να επεξεργάζεστε και να αναλύετε περιεχόμενο που αρχικά υπήρχε μόνο ως εικόνες.

## Γιατί να Προετοιμάσετε Ορθογώνια για Αναγνώριση Κειμένου OCR;
Ορίζοντας ορθογώνια εστιάζετε τη μηχανή στις περιοχές που πραγματικά περιέχουν κείμενο, κάτι που:
* Μειώνει το χρόνο επεξεργασίας.
* Βελτιώνει την ακρίβεια αγνοώντας θορυβώδεις φόντους.
* Σας επιτρέπει να εξάγετε μόνο τα δεδομένα που χρειάζεστε — ιδανικό για φόρμες, τιμολόγια και αποδείξεις.

- **Περιβάλλον Ανάπτυξης** – Οποιοδήποτε Java IDE (IntelliJ IDEA, Eclipse, VS Code, κ.λπ.) είναι αποδεκτό.

## Εισαγωγή Πακέτων

Στο αρχείο πηγαίου κώδικα Java, εισάγετε τις απαιτούμενες κλάσεις Aspose.OCR και τις τυπικές βοηθητικές κλάσεις Java:

```java
package com.aspose.ocr.examples.OcrFeatures;

import com.aspose.ocr.AsposeOCR;
import com.aspose.ocr.examples.License.SetLicense;
import com.aspose.ocr.examples.Utils;

import java.awt.*;
import java.io.IOException;
import java.util.ArrayList;
```

> *Εισάγουμε το `java.awt.Rectangle` επειδή το OCR API αναμένει ορθογώνια που ορίζουν τις περιοχές σάρωσης.*

## Βήμα 1: Ρύθμιση Άδειας

```java
SetLicense.main(null);
```

Η κλήση του `SetLicense` ενεργοποιεί την άδεια Aspose.OCR, αφαιρεί τους περιορισμούς αξιολόγησης και ενεργοποιεί την πλήρη λειτουργία της **ocr text recognition**.

## Βήμα 2: Ορισμός Καταλόγου Εγγράφου και Διαδρομής Εικόνας

```java
String dataDir = "Your Document Directory";
String imagePath = dataDir + "p.png";
```

Αντικαταστήστε το `"Your Document Directory"` με την απόλυτη διαδρομή όπου βρίσκεται η εικόνα σας (`p.png`). Αυτή είναι η εικόνα που θα υποβληθεί σε επεξεργασία.

## Βήμα 3: Δημιουργία Αντικειμένου Aspose.OCR

```java
AsposeOCR api = new AsposeOCR();
```

Η δημιουργία του `AsposeOCR` σας δίνει πρόσβαση στη μέθοδο `RecognizePage`, η οποία εκτελεί το πραγματικό OCR.

## Βήμα 4: Προετοιμασία Ορθογωνίων με Κείμενα

```java
ArrayList<Rectangle> rectArray = new ArrayList<Rectangle>();
rectArray.add(new Rectangle(138, 352, 2033, 537));
rectArray.add(new Rectangle(147, 890, 2033, 1157));
rectArray.add(new Rectangle(923, 2045, 465, 102));
rectArray.add(new Rectangle(104, 2147, 2076, 819));
```

Κάθε `Rectangle(x, y, width, height)` λέει στο Aspose.OCR ακριβώς πού να ψάξει για κείμενο. Προσαρμόστε τις συντεταγμένες ώστε να ταιριάζουν με τη διάταξη της πηγαίας εικόνας.

## Βήμα 5: Εκτέλεση Αναγνώρισης OCR

```java
try {
    String result = api.RecognizePage(imagePath, rectArray);
    System.out.println("Result with rect: " + result);
} catch (IOException e) {
    e.printStackTrace();
}
```

Η κλήση `RecognizePage` επεξεργάζεται μόνο τα ορισμένα ορθογώνια και επιστρέφει το εξαγόμενο κείμενο. Η έξοδος στην κονσόλα σας επιτρέπει να επαληθεύσετε άμεσα το αποτέλεσμα της **ocr text recognition**.

## Κοινά Προβλήματα και Συμβουλές

| Πρόβλημα | Αιτία | Λύση |
|----------|-------|------|
| **Καμία έξοδος** | Λάθος συντεταγμένες ορθογωνίου ή διαδρομή εικόνας | Ελέγξτε ξανά την τιμή του `dataDir` και βεβαιωθείτε ότι τα ορθογώνια καλύπτουν περιοχές κειμένου. |
| **Ακατάλληλοι χαρακτήρες** | Χαμηλή ανάλυση εικόνας ή μη υποστηριζόμενη γραμματοσειρά | Χρησιμοποιήστε εικόνα υψηλότερης ανάλυσης ή εφαρμόστε προεπεξεργασία (π.χ., δυαδικοποίηση). |
| **Η άδεια δεν εφαρμόστηκε** | Η `SetLicense` δεν κλήθηκε πριν το OCR | Βεβαιωθείτε ότι εκτελείται `SetLicense.main(null);` πριν από οποιαδήποτε κλήση API. |
| **Καθυστέρηση απόδοσης** | Πάρα πολλά ή μεγάλα ορθογώνια | Περιορίστε τον αριθμό των ορθογωνίων και κρατήστε τα όσο πιο στενά γίνεται γύρω από το κείμενο. |
