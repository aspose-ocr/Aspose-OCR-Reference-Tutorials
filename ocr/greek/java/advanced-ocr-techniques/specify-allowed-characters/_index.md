# Εξαγωγή Κειμένου από Εικόνες Χρησιμοποιώντας Aspose.OCR – Επιτρεπόμενοι Χαρακτήρες

## Εισαγωγή

Η εξαγωγή κειμένου από εικόνες είναι μια συχνή απαίτηση στις σύγχρονες εφαρμογές—είτε επεξεργάζεστε τιμολόγια, σαρώστε αποδείξεις ή ψηφιοποιείτε έντυπα έγγραφα. Σε αυτό το tutorial θα περάσουμε από ένα πλήρες **aspose ocr java tutorial** που δείχνει πώς να **εξάγετε κείμενο από εικόνες** με το Aspose.OCR για Java, να ορίσετε επιτρεπόμενους χαρακτήρες και να εφαρμόσετε προσωρινή άδεια όταν δοκιμάζετε τη βιβλιοθήκη.

## Γρήγορες Απαντήσεις
- **Τι κάνει το Aspose.OCR;** Εξάγει κείμενο από εικόνες με υψηλή ακρίβεια και υποστηρίζει προσαρμοσμένα σύνολα χαρακτήρων.  
- **Χρειάζομαι άδεια;** Απαιτείται προσωρινή ή μόνιμη άδεια για χρήση σε παραγωγή.  
- **Ποια έκδοση του JDK υποστηρίζεται;** Οι τελευταίες εκδόσεις του JDK είναι πλήρως συμβατές.  
- **Μπορώ να περιορίσω τους αναγνωρισμένους χαρακτήρες;** Ναι—χρησιμοποιήστε το API `setAllowedCharacters` για να περιορίσετε την έξοδο.  
- **Πόσο διαρκεί η εγκατάσταση;** Περίπου 10‑15 λεπτά για μια βασική υλοποίηση.

## Τι είναι η “εξαγωγή κειμένου από εικόνες”;
Η εξαγωγή κειμένου από εικόνες αναφέρεται στη διαδικασία μετατροπής οπτικού κειμένου (π.χ. τυπωμένου ή χειρόγραφου) σε μηχανικά αναγνώσιμες συμβολοσειρές. Αυτό επιτρέπει επόμενες εργασίες όπως η αναζήτηση, η ευρετηρίαση ή η ανάλυση δεδομένων.

## Γιατί να Χρησιμοποιήσετε το Aspose.OCR για Java;
- **Υψηλή ακρίβεια** σε πολλές γλώσσες και γραμματοσειρές.  
- **Απλό API** που ενσωματώνεται σε οποιοδήποτε έργο Java.  
- **Προσαρμόσιμο** σύνολο χαρακτήρων, πακέτα γλωσσών και προεπεξεργασία εικόνας.  
- **Χωρίς εξωτερικές εξαρτήσεις**—η βιβλιοθήκη είναι αυτόνομη.

## Συμβουλές προεπεξεργασίας εικόνας OCR

Καλή ποιότητα εικόνας βελτιώνει δραστικά τα αποτελέσματα αναγνώρισης. Πριν καλέσετε τη μηχανή OCR, σκεφτείτε:

- Μετατροπή της εικόνας σε κλίμακα του γκρι.
- Αύξηση της αντίθεσης ώστε οι χαρακτήρες να ξεχωρίζουν.
- Αφαίρεση θορύβου φόντου με φίλτρο δυαδικοποίησης.
- Αλλαγή μεγέθους εικόνων χαμηλής ανάλυσης σε τουλάχιστον 300 dpi.

Αυτά τα βήματα αποτελούν μέρος της **ocr image preprocessing** και μπορούν να εκτελεστούν με οποιαδήποτε τυπική βιβλιοθήκη επεξεργασίας εικόνας Java πριν την κλήση του Aspose.OCR.

## Εισαγωγή Πακέτων

Μόλις τα προαπαιτούμενα είναι έτοιμα, εισάγετε τα απαραίτητα πακέτα στο έργο Java σας:

```java
package com.aspose.ocr.examples.OcrFeatures;

import com.aspose.ocr.AsposeOCR;
import com.aspose.ocr.examples.Utils;

import java.io.IOException;
```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Ορίστε τον Κατάλογο Εγγράφων σας

Ορίστε έναν φάκελο όπου θα αποθηκεύετε τα αποτελέσματα επεξεργασίας OCR. Αυτή η διαδρομή θα χρησιμοποιηθεί αργότερα για τον εντοπισμό του αρχείου εικόνας.

```java
String dataDir = "Your Document Directory";
```

### Βήμα 2: Καθορίστε τη Διαδρομή της Εικόνας

Κατευθύνετε το API στην εικόνα που θέλετε να αναλύσετε.

```java
String imagePath = dataDir + "0001460985.Jpeg";
```

### Βήμα 3: Δημιουργήστε μια Περίπτωση Aspose.OCR

Δημιουργήστε μια παρουσία του κινητήρα OCR με το κλειδί άδειας σας. Το κλειδί μπορεί να είναι μια προσωρινή ή μόνιμη συμβολοσειρά άδειας.

```java
AsposeOCR api = new AsposeOCR("YourLicenseKey");
```

### Βήμα 4: Εκτελέστε Αναγνώριση OCR

Καλέστε τη μέθοδο `RecognizeLine` για να εξάγετε μια γραμμή κειμένου από την εικόνα. Το αποτέλεσμα είναι μια απλή συμβολοσειρά που μπορείτε να επεξεργαστείτε ή να αποθηκεύσετε περαιτέρω.

```java
try {
    String result = api.RecognizeLine(imagePath);
    System.out.println("File: " + imagePath);
    System.out.println("Result line: " + result);
} catch (IOException e) {
    e.printStackTrace();
}
```

> **Συμβουλή:** Αν χρειάζεται να περιορίσετε την έξοδο σε ένα συγκεκριμένο σύνολο χαρακτήρων (π.χ. μόνο ψηφία), χρησιμοποιήστε τη μέθοδο `setAllowedCharacters` στην παρουσία `AsposeOCR` πριν καλέσετε το `RecognizeLine`. Αυτό εξασφαλίζει ότι η μηχανή αγνοεί οποιονδήποτε χαρακτήρα εκτός του ορισμένου συνόλου.
