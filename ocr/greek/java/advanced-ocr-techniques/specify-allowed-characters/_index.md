# Εξαγωγή Κειμένου από Εικόνες με Aspose.OCR – Επιτρεπόμενοι Χαρακτήρες

## Εισαγωγή

Η εξαγωγή κειμένου από εικόνες είναι μια κοινή απαίτηση σε σύγχρονες εφαρμογές—είτε επεξεργάζεστε τιμολόγια, σκανάρετε αποδείξεις ή ψηφιοποιείτε έντυπα έγγραφα. Το **Aspose.OCR for Java** κάνει αυτή τη δουλειά απλή, προσφέροντας υψηλή ακρίβεια αναγνώρισης και ευέλικτες επιλογές διαμόρφωσης, όπως ο καθορισμός επιτρεπόμενων χαρακτήρων. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από ένα πλήρες **aspose ocr java tutorial** που δείχνει πώς να ρυθμίσετε τη βιβλιοθήκη, να εκτελέσετε OCR και να περιορίσετε το σύνολο χαρακτήρων σύμφωνα με τις ανάγκες σας.

## Γρήγορες Απαντήσεις
- **Τι κάνει το Aspose.OCR;** Εξάγει κείμενο από εικόνες με υψηλή ακρίβεια και υποστηρίζει προσαρμοσμένα σύνολα χαρακτήρων.  
- **Χρειάζομαι άδεια;** Απαιτείται προσωρινή ή μόνιμη άδεια για χρήση σε παραγωγή.  
- **Ποια έκδοση JDK υποστηρίζεται;** Οι τελευταίες εκδόσεις JDK είναι πλήρως συμβατές.  
- **Μπορώ να περιορίσω τους αναγνωρισμένους χαρακτήρες;** Ναι—χρησιμοποιήστε το API επιτρεπόμενων χαρακτήρων για να περιορίσετε την έξοδο.  
- **Πόσο χρόνο διαρκεί η εγκατάσταση;** Περίπου 10‑15 λεπτά για μια βασική υλοποίηση.

## Εισαγωγή Πακέτων

Μόλις τα προαπαιτούμενα είναι έτοιμα, εισάγετε τα απαραίτητα πακέτα στο έργο Java σας:

```java
package com.aspose.ocr.examples.OcrFeatures;

import com.aspose.ocr.AsposeOCR;
import com.aspose.ocr.examples.Utils;

import java.io.IOException;
```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Ορίστε τον Κατάλογο Εγγράφων σας

Ορίστε έναν φάκελο όπου θα αποθηκεύετε τα αποτελέσματα επεξεργασίας OCR. Αυτή η διαδρομή θα χρησιμοποιηθεί αργότερα για την εύρεση του αρχείου εικόνας.

```java
String dataDir = "Your Document Directory";
```

### Βήμα 2: Καθορίστε τη Διαδρομή της Εικόνας

Κατευθύνετε το API στην εικόνα που θέλετε να αναλύσετε.

```java
String imagePath = dataDir + "0001460985.Jpeg";
```

### Βήμα 3: Δημιουργήστε ένα Παράδειγμα Aspose.OCR

Δημιουργήστε την μηχανή OCR με το κλειδί άδειας σας. Το κλειδί μπορεί να είναι προσωρινό ή μόνιμο.

```java
AsposeOCR api = new AsposeOCR("YourLicenseKey");
```

### Βήμα 4: Εκτελέστε την Αναγνώριση OCR

Καλέστε τη μέθοδο `RecognizeLine` για να εξάγετε μια γραμμή κειμένου από την εικόνα. Το αποτέλεσμα είναι μια απλή συμβολοσειρά που μπορείτε να επεξεργαστείτε ή να αποθηκεύσετε περαιτέρω.

```java
try {
    String result = api.RecognizeLine(imagePath);
    System.out.println("File: " + imagePath);
    System.out.println("Result line: " + result);
} catch (IOException e) {
    e.printStackTrace();
}
```

> **Συμβουλή επαγγελματία:** Εάν χρειάζεται να περιορίσετε την έξοδο σε ένα συγκεκριμένο σύνολο χαρακτήρων (π.χ. μόνο ψηφία), χρησιμοποιήστε τη μέθοδο `setAllowedCharacters` στο αντικείμενο `AsposeOCR` πριν καλέσετε το `RecognizeLine`. Αυτό εξασφαλίζει ότι η μηχανή αγνοεί οποιουσδήποτε χαρακτήρες εκτός του καθορισμένου συνόλου.
