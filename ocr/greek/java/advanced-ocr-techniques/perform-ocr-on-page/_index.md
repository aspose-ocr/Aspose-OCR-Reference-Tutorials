Είτε δημιουργείτε μια γραμμή επεξεργασίας τιμολογίων είτε ένα σύστημα αρχειοθέτησης εγγράφων, η εξαγωγή μόνο της σχετικής σελίδας εξοικονομεί χρόνο και πόρους. ## Βήμα 1: Ρύθμιση Άδειας (process image OCR)

Πριν χρησιμοποιήσετε το Aspose.OCR, πρέπει να φορτώσετε μια έγκυρη άδεια. Απενεργοποιήστε το σχόλιο στη γραμμή `SetLicense.main(null)` στον κώδικά σας και δείξτε το αρχείο `.lic`. Αυτό το βήμα ξεκλειδώνει τις πλήρεις δυνατότητες OCR και αφαιρεί τα υδατογραφήματα αξιολόγησης.

## Βήμα 2: Καθορισμός Καταλόγου Εγγράφου και Διαδρομής Εικόνας (extract text image java)

Ορίστε πού βρίσκονται τα αρχεία πηγής σας και δείξτε στην ακριβή εικόνα από την οποία θέλετε να **convert image to text**. Ενημερώστε τις μεταβλητές `dataDir` και `imagePath` ώστε να ταιριάζουν με το περιβάλλον σας.

```java
String dataDir = "Your Document Directory";
String imagePath = dataDir + "p3.png";
```

## Βήμα 3: Δημιουργία Αντικειμένου AsposeOCR

Δημιουργήστε ένα στιγμιότυπο της κλάσης `AsposeOCR`. Αυτό το αντικείμενο σας δίνει πρόσβαση σε όλες τις λειτουργίες OCR, συμπεριλαμβανομένης της αναγνώρισης σε επίπεδο σελίδας που χρειαζόμαστε.

```java
AsposeOCR api = new AsposeOCR();
```

## Βήμα 4: Αναγνώριση της Καθορισμένης Σελίδας

Καλέστε τη μέθοδο `RecognizePage` για να εκτελέσετε την λειτουργία OCR στην επιλεγμένη εικόνα. Η μέθοδος επιστρέφει ένα `String` που περιέχει το εξαγόμενο κείμενο, το οποίο μπορείτε στη συνέχεια να επεξεργαστείτε ή να αποθηκεύσετε όπως χρειάζεται.

```java
try {
    String result = api.RecognizePage(imagePath);
    System.out.println("Result: " + result);
} catch (IOException e) {
    e.printStackTrace();
}
```
