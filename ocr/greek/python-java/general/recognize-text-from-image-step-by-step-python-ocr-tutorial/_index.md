# αναγνώριση κειμένου από εικόνα – Πλήρης Οδηγός Python OCR

Κάποτε χρειάστηκε να **αναγνωρίσετε κείμενο από εικόνα** αλλά δεν ήξερατε από πού να ξεκινήσετε; Ίσως έχετε μια σαρωμένη φόρμα, μια απόδειξη ή ένα στιγμιότυπο οθόνης και θέλετε μόνο τις λέξεις μέσα σε ένα συγκεκριμένο πλαίσιο. Τα καλά νέα είναι ότι με λίγες γραμμές Python μπορείτε να φορτώσετε την εικόνα για OCR, να εστιάσετε σε μία περιοχή και να εξάγετε το ακριβές κείμενο που χρειάζεστε—χωρίς χειροκίνητη αντιγραφή.

Σε αυτό το tutorial θα περάσουμε από όλη τη διαδικασία: φόρτωση της εικόνας, ορισμός περιοχής ενδιαφέροντος (ROI), εκτέλεση του κινητήρα OCR και εκτύπωση του αποτελέσματος. Στο τέλος θα μπορείτε να ενσωματώσετε αυτό το απόσπασμα σε οποιοδήποτε έργο που χρειάζεται εξαγωγή κειμένου από συγκεκριμένο τμήμα εικόνας. Χωρίς βαριά pipelines επεξεργασίας εικόνας, μόνο καθαρός, αναγνώσιμος κώδικας που λειτουργεί σήμερα.

## Προαπαιτούμενα

- Python 3.8+ εγκατεστημένο  
- Το πακέτο `ocr` (ή οποιαδήποτε συμβατή βιβλιοθήκη OCR) – εγκαταστήστε το με `pip install ocr-lib` (αντικαταστήστε με το πραγματικό όνομα του πακέτου που χρησιμοποιείτε)  
- Μια εικόνα PNG/JPEG που περιέχει τη φόρμα που θέλετε να διαβάσετε  
- Βασική εξοικείωση με συναρτήσεις και κλάσεις Python  

Αν είστε ήδη εξοικειωμένοι με αυτά, υπέροχα—μπορείτε να προχωρήσετε. Διαφορετικά, πάρτε έναν γρήγορο καφέ και βεβαιωθείτε ότι τα παραπάνω στοιχεία είναι έτοιμα· τα επόμενα βήματα υποθέτουν ότι είναι στη θέση τους.

## Βήμα 1: Δημιουργία Αντικειμένου OCR Engine – Πυρήνας “αναγνώριση κειμένου από εικόνα”

Το πρώτο που χρειαζόμαστε είναι ένα αντικείμενο που ξέρει πώς να επικοινωνεί με τον κινητήρα OCR. Σκεφτείτε το ως τον εγκέφαλο που θα **αναγνωρίσει κείμενο από εικόνα** αργότερα.

```python
import ocr  # Import the OCR library

# Initialize the engine
ocr_engine = ocr.OcrEngine()
```

> **Γιατί είναι σημαντικό:** Η αρχικοποίηση του κινητήρα μία φορά σας επιτρέπει να επαναχρησιμοποιήσετε ρυθμίσεις (όπως πακέτα γλώσσας) σε πολλαπλές εικόνες, βελτιώνοντας την απόδοση και διατηρώντας τον κώδικα καθαρό.

## Βήμα 2: Φόρτωση Εικόνας για OCR – Φέρνοντας την Εικόνα στη Μνήμη

Τώρα φορτώνουμε πραγματικά την **εικόνα για OCR**. Η βιβλιοθήκη παρέχει μια βολική στατική μέθοδο που διαβάζει το αρχείο και επιστρέφει ένα αντικείμενο που μπορεί να καταλάβει ο κινητήρας.

```python
# Load the source image (replace the path with your own)
image_path = "YOUR_DIRECTORY/form.png"
ocr_engine.set_image(ocr.Imaging.Image.load(image_path))
```

> **Συμβουλή:** Αν η εικόνα σας είναι μεγάλη, σκεφτείτε να την αλλάξετε μέγεθος πριν τη φόρτωση. Οι μικρότερες εικόνες επιταχύνουν το OCR χωρίς να θυσιάζουν την ακρίβεια για τα περισσότερα τυπωμένα κείμενα.

## Βήμα 3: Ορισμός Περιοχής Ενδιαφέροντος OCR (ROI)

Συχνά δεν χρειάζεστε ολόκληρη τη σελίδα—μόνο ένα συγκεκριμένο πλαίσιο όπου ο χρήστης έγραψε δεδομένα. Ο ορισμός μιας **περιοχής ενδιαφέροντος OCR** λέει στον κινητήρα να αγνοήσει τα υπόλοιπα, μειώνοντας το θόρυβο και επιταχύνοντας την επεξεργασία.

```python
# Define ROI: (left, top, width, height) in pixels
region_of_interest = ocr.Rectangle(120, 340, 560, 90)

# Register the ROI with the engine
ocr_engine.add_region_of_interest(region_of_interest)
```

> **Γιατί να εστιάσετε στην ROI;**  
> - **Ταχύτητα:** Ο κινητήρας σαρώνει λιγότερα pixel.  
> - **Ακρίβεια:** Τα γραφικά φόντου εκτός της ROI μπορούν να μπερδέψουν την αναγνώριση χαρακτήρων.  
> - **Απλότητα:** Λαμβάνετε μια καθαρή συμβολοσειρά που αντιστοιχεί ακριβώς στο πεδίο που σας ενδιαφέρει.

## Βήμα 4: Εκτέλεση της Διαδικασίας OCR – Η Στιγμή της Αλήθειας

Με όλα έτοιμα, τελικά **αναγνωρίζουμε κείμενο από εικόνα** μέσα στην καθορισμένη ROI.

```python
# Execute OCR on the specified region
ocr_result = ocr_engine.recognize()
```

Αν ο κινητήρας υποστηρίζει πολλαπλές περιοχές, το `ocr_result` θα περιέχει μια λίστα· στην περίπτωση μιας μόνο ROI είναι ένα απλό αντικείμενο με μέθοδο `get_text()`.

## Βήμα 5: Εξαγωγή και Εκτύπωση του Κειμένου – Λήψη του Τελικού Αποτελέσματος

Τώρα εξάγουμε τη απλή συμβολοσειρά από το αντικείμενο αποτελέσματος και την εμφανίζουμε. Εδώ μπορείτε να ενσωματώσετε το αποτέλεσμα σε βάση δεδομένων, αρχείο CSV ή οποιαδήποτε λογική επεξεργασίας.

```python
# Retrieve the recognized text
extracted_text = ocr_result.get_text()

# Show the result
print("Text inside ROI:\n", extracted_text)
```

**Αναμενόμενο αποτέλεσμα** (παράδειγμα για πεδίο ονόματος που έχει συμπληρωθεί):

```
Text inside ROI:
 John Doe
```

Αν ο κινητήρας OCR επιστρέψει επιπλέον κενά ή αλλαγές γραμμής, μπορείτε να τα καθαρίσετε με `.strip()` ή μια κανονική έκφραση.

## Διαχείριση Συνηθισμένων Περιπτώσεων Άκρων

| Situation                              | What to Do                                                               |
|----------------------------------------|--------------------------------------------------------------------------|
| **Low‑resolution image**               | Αυξήστε την ανάλυση με `Pillow` (`Image.resize`) πριν τη φόρτωση.                  |
| **Skewed or rotated text**             | Εφαρμόστε διόρθωση περιστροφής (`ocr.Imaging.Image.rotate`).               |
| **Multiple languages**                | Ορίστε το πακέτο γλώσσας στον κινητήρα: `ocr_engine.set_language('eng+spa')`. |
| **No ROI defined**                     | Παραλείψτε το `add_region_of_interest`; ο κινητήρας θα επεξεργαστεί ολόκληρη την εικόνα. |
| **Unexpected characters (e.g., commas)** | Μετα‑επεξεργασία της συμβολοσειράς: `extracted_text.replace(',', '')`.            |

Αυτές οι συμβουλές διατηρούν το pipeline **load image for OCR** ανθεκτικό ακόμη και όταν το υλικό προέλευσης δεν είναι τέλειο.

## Πλήρες Παράδειγμα – Αντιγραφή, Επικόλληση, Εκτέλεση

Παρακάτω είναι το πλήρες script που μπορείτε να τοποθετήσετε σε ένα αρχείο `.py` και να το εκτελέσετε. Περιλαμβάνει όλες τις εισαγωγές, διαχείριση σφαλμάτων και έναν μικρό βοηθό που ελέγχει αν η εικόνα υπάρχει.

```python
import os
import ocr

def recognize_text_from_image(image_path: str,
                              roi: tuple = (120, 340, 560, 90)):
    """
    Recognize text from a specific region of an image.

    Parameters
    ----------
    image_path : str
        Full path to the image file.
    roi : tuple
        (left, top, width, height) defining the region of interest.

    Returns
    -------
    str
        The extracted text, stripped of surrounding whitespace.
    """
    if not os.path.isfile(image_path):
        raise FileNotFoundError(f"Image not found: {image_path}")

    # Step 1 – create engine
    engine = ocr.OcrEngine()

    # Step 2 – load image for OCR
    engine.set_image(ocr.Imaging.Image.load(image_path))

    # Step 3 – define ROI
    left, top, width, height = roi
    engine.add_region_of_interest(ocr.Rectangle(left, top, width, height))

    # Step 4 – run OCR
    result = engine.recognize()

    # Step 5 – extract text
    text = result.get_text().strip()
    return text


if __name__ == "__main__":
    # Adjust the path to point at your form image
    img_path = "YOUR_DIRECTORY/form.png"

    try:
        roi_text = recognize_text_from_image(img_path)
        print("Text inside ROI:\n", roi_text)
    except Exception as e:
        print("OCR failed:", e)
```

Εκτελώντας αυτό το script εκτυπώνει τη καθαρισμένη συμβολοσειρά από την ROI, παρέχοντάς σας ένα έτοιμο προς χρήση κομμάτι δεδομένων.
