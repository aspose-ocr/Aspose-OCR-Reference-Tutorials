## Βήμα 1: Ρύθμιση του περιβάλλοντος για αναγνώριση κειμένου png

Το πρώτο που πρέπει να κάνουμε είναι να εισάγουμε τα απαιτούμενα modules του Aspose και να διαμορφώσουμε τη μηχανή OCR. Αυτό το βήμα εξασφαλίζει ότι η μηχανή γνωρίζει ότι πρέπει να **αναγνωρίσει κείμενο png** αρχεία και να εντοπίσει αυτόματα το σύστημα γραφής (Κυριλλικό στην περίπτωσή μας).

```python
# Step 1: Import required Aspose modules
import aspose.ocr as aocr
import aspose.storage as storage

# Create the OCR engine instance
ocr_engine = aocr.OcrEngine()

# Let the engine auto‑detect the language/script
ocr_engine.language = aocr.Language.AUTO
```

**Γιατί είναι σημαντικό:**  
Η ρύθμιση `Language.AUTO` λέει στο Aspose να σαρώσει την εικόνα για οποιοδήποτε υποστηριζόμενο σύστημα γραφής, κάτι που είναι απαραίτητο όταν θέλετε να **εντοπίσετε κυριλλικούς χαρακτήρες** χωρίς να καθορίσετε σκληρά τη γλώσσα. Εάν γνωρίζετε εκ των προτέρων το σύστημα γραφής, μπορείτε να ορίσετε `aocr.Language.CYRILLIC`, το οποίο μπορεί να βελτιώσει ελαφρώς την ταχύτητα.

## Βήμα 2: Εντοπισμός Κυριλλικών χαρακτήρων στην εικόνα

Τώρα φορτώνουμε το PNG που περιέχει το κυριλλικό κείμενο. Το Aspose Storage διευκολύνει την ανάγνωση μιας εικόνας από δίσκο ή ακόμη και από cloud bucket.

```python
# Step 2: Load the image containing Cyrillic text
image_path = "YOUR_DIRECTORY/cyrillic_sample.png"
image = storage.Image.load(image_path)
```

> **Τι γίνεται αν το αρχείο δεν είναι PNG;**  
> Το Aspose OCR υποστηρίζει JPEG, BMP, TIFF και άλλα. Απλώς αλλάξτε την επέκταση του αρχείου· η ίδια κλήση `Image.load` θα το διαχειριστεί.

---

## Βήμα 3: Εξαγωγή κειμένου από εικόνα χρησιμοποιώντας Aspose OCR

Με την εικόνα στα χέρια, ζητάμε από τη μηχανή OCR να κάνει τη μαγεία της. Η μέθοδος `recognize` επιστρέφει ένα αντικείμενο `OcrResult` που περιέχει τη ανιχνευμένη συμβολοσειρά και τις βαθμολογίες εμπιστοσύνης.

```python
# Step 3: Perform OCR on the loaded image
result = ocr_engine.recognize(image)

# The recognized text is stored in result.text
recognized_text = result.text
```

**Γιατί να χρησιμοποιήσετε `recognize` αντί για `recognize_async`;**  
Για ένα μόνο αρχείο PNG, η συγχρονική κλήση είναι πιο απλή και αποφεύγει το επιπλέον boilerplate της διαχείρισης futures. Εάν χρειάζεται να επεξεργαστείτε δεκάδες εικόνες σε παρτίδες, η ασύγχρονη έκδοση μπορεί να προσφέρει καλύτερη απόδοση.

## Βήμα 4: Επαλήθευση του αποτελέσματος και ανάγνωση κυριλλικών γραμμάτων

Τέλος, εκτυπώνουμε το αποτέλεσμα στην κονσόλα. Εδώ μπορείτε να επιβεβαιώσετε ότι η μηχανή OCR διάβασε επιτυχώς **κυριλλικά γράμματα** όπως “Ҙ”, “Ў”, και “ӱ”.

```python
# Step 4: Output the recognized text
print("Detected text:")
print(recognized_text)   # Expected to include characters like “Ҙ”, “Ў”, “ӱ”

# Simple verification: check if any Cyrillic Unicode block is present
cyrillic_range = (0x0400, 0x04FF)  # Unicode range for Cyrillic
has_cyrillic = any(cyrillic_range[0] <= ord(ch) <= cyrillic_range[1] for ch in recognized_text)

print("\nDid we detect Cyrillic characters? ", "Yes ✅" if has_cyrillic else "No ❌")
```

**Αναμενόμενη έξοδος κονσόλας (παράδειγμα):**

```
Detected text:
Привет мир! Это тестовый текст с символами: Ҙ, Ў, ӱ.

Did we detect Cyrillic characters?  Yes ✅
```

Αν ο έλεγχος εμφανίσει “No ❌”, ελέγξτε ξανά ότι η εικόνα είναι καθαρή και ότι χρησιμοποιείτε την πιο πρόσφατη έκδοση του Aspose OCR (στην ώρα της συγγραφής, έκδοση 23.12). Θολές εικόνες ή χαμηλή αντίθεση μπορούν να μπερδέψουν τη μηχανή, οπότε ίσως χρειαστεί να προεπεξεργαστείτε το PNG (π.χ., να αυξήσετε την αντίθεση) πριν το χρησιμοποιήσετε.

## Βήμα 5: Bonus – Επεξεργασία πολλαπλών PNG σε φάκελο (προαιρετικό)

Συχνά θα χρειαστεί να **εξάγετε κείμενο από εικόνα** αρχεία μαζικά. Το παρακάτω απόσπασμα διατρέχει όλα τα αρχεία PNG σε έναν φάκελο, εκτελεί την ίδια διαδικασία OCR και γράφει κάθε αποτέλεσμα σε αρχείο `.txt`.

```python
import os

def ocr_folder(folder_path: str, output_folder: str):
    os.makedirs(output_folder, exist_ok=True)
    for filename in os.listdir(folder_path):
        if filename.lower().endswith(".png"):
            img_path = os.path.join(folder_path, filename)
            img = storage.Image.load(img_path)
            txt = ocr_engine.recognize(img).text

            out_path = os.path.join(output_folder, f"{os.path.splitext(filename)[0]}.txt")
            with open(out_path, "w", encoding="utf-8") as f:
                f.write(txt)

            print(f"Processed {filename} → {out_path}")

# Example usage:
# ocr_folder("YOUR_DIRECTORY/pngs", "YOUR_DIRECTORY/ocr_output")
```

**Γιατί αυτό βοηθά:**  
Η επεξεργασία σε παρτίδες είναι ένα κοινό σενάριο στον πραγματικό κόσμο όταν χρειάζεται να **εξάγετε κείμενο από εικόνα** για pipelines εισαγωγής δεδομένων. Η παραπάνω συνάρτηση διατηρεί τον κώδικα καθαρό και επαναχρησιμοποιεί την ίδια παρουσία μηχανής OCR, κάτι που είναι πιο αποδοτικό από το να τη δημιουργείτε ξανά για κάθε αρχείο.

## Συχνές Ερωτήσεις & Ειδικές Περιπτώσεις

- **Τι γίνεται αν το OCR επιστρέφει ακατάληπτους χαρακτήρες;**  
  Δοκιμάστε να αυξήσετε την ανάλυση της εικόνας τουλάχιστον στα 300 dpi, ή χρησιμοποιήστε `ocr_engine.image_preprocessing = aocr.ImagePreprocessing.AUTO` για να αφήσετε το Aspose να βελτιώσει αυτόματα την εικόνα.  

- **Μπορώ να περιορίσω την ανίχνευση μόνο σε Κυριλλικά;**  
  Ναι—ορίστε `ocr_engine.language = aocr.Language.CYRILLIC`. Αυτό μειώνει τα ψευδώς θετικά από λατινικούς χαρακτήρες.  

- **Υπάρχει τρόπος να λάβω βαθμολογίες εμπιστοσύνης για κάθε λέξη;**  
  Το αντικείμενο `OcrResult` εκθέτει επίσης `result.words`, το καθένα με ιδιότητα `confidence`. Επανάληψη μέσω αυτών αν χρειάζεστε λεπτομερή επαλήθευση.  

- **Χρειάζομαι πληρωμένη άδεια για παραγωγή;**  
  Η έκδοση αξιολόγησης λειτουργεί για ανάπτυξη και μικρές δοκιμές, αλλά μια εμπορική άδεια αφαιρεί το υδατογράφημα αξιολόγησης και αφαιρεί τους περιορισμούς χρήσης.
