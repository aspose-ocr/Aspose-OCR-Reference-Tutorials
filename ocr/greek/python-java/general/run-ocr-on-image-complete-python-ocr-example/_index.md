# Εκτέλεση OCR σε Εικόνα – Πλήρες Παράδειγμα Python OCR

Κάποτε χρειάστηκε να **εκτελέσετε OCR σε αρχεία εικόνας** αλλά δεν ήξερατε από πού να ξεκινήσετε; Δεν είστε μόνοι· πολλοί προγραμματιστές αντιμετωπίζουν αυτό το εμπόδιο όταν προσπαθούν για πρώτη φορά να εξάγουν κείμενο από σαρωμένα έγγραφα. Σε αυτό το tutorial θα περάσουμε από ένα **παράδειγμα Python OCR** που σας επιτρέπει να **αναγνωρίσετε κείμενο από PNG** αρχεία, **φορτώσετε εικόνα για OCR**, και **εξάγετε λέξεις από την εικόνα** με βαθμούς εμπιστοσύνης—όλα σε λίγες γραμμές κώδικα.

Θα καλύψουμε όλα όσα χρειάζεστε: τη βιβλιοθήκη, πώς να ρυθμίσετε τη μηχανή, γιατί κάθε βήμα είναι σημαντικό, και πώς φαίνεται το αποτέλεσμα. Στο τέλος, θα μπορείτε να ενσωματώσετε αυτό το απόσπασμα στον δικό σας κώδικα και να αρχίσετε αμέσως να εξάγετε κείμενο από οποιαδήποτε εικόνα. Χωρίς περιττές εξηγήσεις, μόνο μια πρακτική, εκτελέσιμη λύση.

## Τι Θα Χρειαστείτε

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

- Python 3.8 ή νεότερη έκδοση εγκατεστημένη  
- Το πακέτο `ocrengine` (ή οποιαδήποτε βιβλιοθήκη που παρέχει κλάση `OcrEngine`). Μπορείτε να το εγκαταστήσετε με `pip install ocrengine` – προσαρμόστε το όνομα αν χρησιμοποιείτε διαφορετική βιβλιοθήκη OCR όπως `pytesseract`.  
- Ένα αρχείο εικόνας (PNG, JPG, κ.λπ.) που θέλετε να επεξεργαστείτε – για αυτό το tutorial θα χρησιμοποιήσουμε το `invoice.png`.  

Αυτό είναι όλο. Χωρίς βαριές εξαρτήσεις, χωρίς εξωτερικές υπηρεσίες, μόνο καθαρό Python.

## Βήμα 1 – Εγκατάσταση και Εισαγωγή της Βιβλιοθήκης OCR

Πρώτα απ' όλα, ας φέρουμε τη μηχανή OCR στο περιβάλλον μας και ας την εισάγουμε. Αν χρησιμοποιείτε το υποθετικό πακέτο `ocrengine`, η εισαγωγή είναι η εξής:

```python
# Install the package (run once in your terminal)
# pip install ocrengine

# Import the OCR engine class
from ocrengine import OcrEngine
```

**Γιατί είναι σημαντικό:** Η εισαγωγή της σωστής κλάσης σας δίνει πρόσβαση στις μεθόδους που θα καλέσουμε αργότερα, όπως `setImageFromFile` και `recognize`. Αν παραλείψετε αυτό το βήμα, η Python θα ρίξει `ModuleNotFoundError` και θα κολλήσετε πριν καν φορτώσετε μια εικόνα.

## Βήμα 2 – Δημιουργία Αντικειμένου Μηχανής OCR

Τώρα που η βιβλιοθήκη είναι έτοιμη, χρειαζόμαστε ένα αντικείμενο μηχανής που θα κρατά τις ρυθμίσεις και την κατάσταση της διαδικασίας αναγνώρισης.

```python
# Step 2: Create an OCR engine instance
ocr_engine = OcrEngine()
```

*Συμβουλή:* Κάποιες μηχανές OCR επιτρέπουν να ρυθμίσετε μοντέλα γλώσσας ή DPI σε αυτό το σημείο. Για ένα βασικό **python OCR example**, οι προεπιλογές λειτουργούν καλά, αλλά αν δουλεύετε με χαμηλής ανάλυσης σκαναρίσματα, σκεφτείτε να τα προσαρμόσετε εδώ.

## Βήμα 3 – Φόρτωση της Εικόνας που Θέλετε να Επεξεργαστείτε

Το επόμενο λογικό βήμα είναι να **φορτώσετε εικόνα για OCR**. Δείχνετε στη μηχανή το μονοπάτι του αρχείου PNG (ή οποιασδήποτε υποστηριζόμενης μορφής) που θέλετε να αναλύσετε.

```python
# Step 3: Load the image you want to process
ocr_engine.setImageFromFile("YOUR_DIRECTORY/invoice.png")
```

**Τι συμβαίνει στο παρασκήνιο;** Η μηχανή διαβάζει τα δεδομένα των εικονοστοιχείων, τα μετατρέπει σε μορφή που καταλαβαίνει ο αλγόριθμος αναγνώρισης και τα αποθηκεύει εσωτερικά. Αν το μονοπάτι είναι λανθασμένο, θα λάβετε `FileNotFoundError`, οπότε ελέγξτε ξανά ότι η εικόνα υπάρχει.

## Βήμα 4 – Εκτέλεση του Αλγορίθμου Αναγνώρισης

Με την εικόνα φορτωμένη, τελικά **εκτελούμε OCR σε εικόνα** για να εξάγουμε το κειμενικό περιεχόμενο.

```python
# Step 4: Run the recognition algorithm to obtain results
ocr_result = ocr_engine.recognize()
```

Σε αυτό το σημείο η μηχανή σαρώνει το bitmap, εφαρμόζει αντιστοίχιση προτύπων, και επιστρέφει ένα αντικείμενο που περιέχει κάθε ανιχνευμένη λέξη, γραμμή και μετρική εμπιστοσύνης.

## Βήμα 5 – Επανάληψη στις Αναγνωρισμένες Λέξεις και Εμφάνιση Εμπιστοσύνης

Το πιο χρήσιμο μέρος κάθε ροής OCR είναι η προβολή **της εμπιστοσύνης** για κάθε εξαγόμενο token. Σας δείχνει πόσο σίγουρη είναι η μηχανή για κάθε λέξη, επιτρέποντάς σας να φιλτράρετε τα αποτελέσματα χαμηλής εμπιστοσύνης αν χρειάζεται.

```python
# Step 5: Iterate over each recognized word and display its text with confidence
for recognized_word in ocr_result.getWords():
    word_text = recognized_word.getText()
    confidence = recognized_word.getConfidence()   # 0‑100%
    print(f"Word: '{word_text}' – confidence: {confidence}%")
```

**Αναμενόμενο αποτέλεσμα** (παράδειγμα):

```
Word: 'Invoice' – confidence: 98%
Word: 'Number' – confidence: 95%
Word: ':' – confidence: 92%
Word: '2023-07-15' – confidence: 97%
Word: 'Total' – confidence: 96%
Word: ':' – confidence: 93%
Word: '$' – confidence: 88%
Word: '1,250.00' – confidence: 94%
```

Τώρα μπορείτε να δείτε ακριβώς **τι λέξεις εξήχθησαν από την εικόνα** και πόσο αξιόπιστη είναι κάθε ανίχνευση. Αυτό είναι ο πυρήνας κάθε **extract words from image** pipeline.

## Διαχείριση Συνηθισμένων Edge Cases

### Τι γίνεται αν η εικόνα είναι σε κλίμακα του γκρι;

Κάποιες μηχανές OCR αποδίδουν καλύτερα με έγχρωμες εικόνες. Αν παρατηρήσετε χαμηλή εμπιστοσύνη γενικά, δοκιμάστε να μετατρέψετε το PNG σε μια υψηλής αντίθεσης ασπρόμαυρη έκδοση πριν το δώσετε στη μηχανή. Η Pillow μπορεί να βοηθήσει:

```python
from PIL import Image, ImageOps

img = Image.open("YOUR_DIRECTORY/invoice.png")
bw = ImageOps.grayscale(img).point(lambda x: 0 if x < 128 else 255, '1')
bw.save("YOUR_DIRECTORY/invoice_bw.png")
ocr_engine.setImageFromFile("YOUR_DIRECTORY/invoice_bw.png")
```

### Διαχείριση Πολλαπλών Γλωσσών

Αν το έγγραφό σας περιέχει τόσο Αγγλικά όσο και Ισπανικά, θα θέλετε να **αναγνωρίσετε κείμενο από PNG** χρησιμοποιώντας ένα πολυγλωσσικό μοντέλο. Οι περισσότερες μηχανές επιτρέπουν να ορίσετε τη λίστα γλωσσών κατά την αρχικοποίηση:

```python
ocr_engine = OcrEngine(languages=["eng", "spa"])
```

### Φιλτράρισμα Λέξεων Χαμηλής Εμπιστοσύνης

Μερικές φορές χρειάζεστε μόνο λέξεις με εμπιστοσύνη πάνω από, π.χ., 90 %. Ένα γρήγορο φίλτρο είναι το εξής:

```python
high_confidence_words = [
    w.getText()
    for w in ocr_result.getWords()
    if w.getConfidence() >= 90
]
print(high_confidence_words)
```

## Πλήρες, Έτοιμο‑για‑Εκτέλεση Script

Συνδυάζοντας όλα τα παραπάνω, εδώ είναι ένα ενιαίο script που μπορείτε να αντιγράψετε‑και‑επικολλήσετε και να τρέξετε αμέσως (απλώς αντικαταστήστε το μονοπάτι του PNG σας).

```python
# run_ocr_on_image.py
# -------------------------------------------------
# Complete Python OCR example: load image for OCR,
# run OCR on image, and extract words from image.
# -------------------------------------------------

# Install the library first:
# pip install ocrengine pillow   # pillow only needed for optional preprocessing

from ocrengine import OcrEngine

def main():
    # 1️⃣ Create the OCR engine
    ocr_engine = OcrEngine()

    # 2️⃣ Load the target PNG (replace with your own file)
    image_path = "YOUR_DIRECTORY/invoice.png"
    ocr_engine.setImageFromFile(image_path)

    # 3️⃣ Run recognition
    ocr_result = ocr_engine.recognize()

    # 4️⃣ Print each word with its confidence
    print("\n--- OCR Results ---")
    for word in ocr_result.getWords():
        text = word.getText()
        conf = word.getConfidence()
        print(f"Word: '{text}' – confidence: {conf}%")

if __name__ == "__main__":
    main()
```

Τρέξτε το με:

```bash
python run_ocr_on_image.py
```

Θα πρέπει να δείτε τη λίστα λέξεων και τα ποσοστά εμπιστοσύνης να εκτυπώνονται στην κονσόλα, ακριβώς όπως φαίνεται παραπάνω.

## Συχνές Ερωτήσεις

**Λειτουργεί αυτό με αρχεία JPG ή TIFF;**  
Απόλυτα. Η μέθοδος `setImageFromFile` δέχεται οποιαδήποτε μορφή μπορεί να αποκωδικοποιήσει η υποκείμενη βιβλιοθήκη, οπότε μπορείτε να **εκτελέσετε OCR σε εικόνα** αρχείων τύπου JPG, TIFF, BMP, κ.λπ.

**Μπορώ να επεξεργαστώ πολλές εικόνες σε βρόχο;**  
Βεβαίως. Τυλίξτε τα βήματα φόρτωσης και αναγνώρισης μέσα σε έναν `for` βρόχο πάνω σε μια λίστα μονοπατιών αρχείων. Απλώς θυμηθείτε να επανεκκινήσετε τη μηχανή αν η βιβλιοθήκη απαιτεί νέο αντικείμενο ανά εικόνα.

**Τι γίνεται αν χρειάζομαι το κείμενο ως μια ενιαία συμβολοσειρά αντί για ανά‑λέξη;**  
Οι περισσότερες αντικείμενα αποτελεσμάτων OCR εκθέτουν μια μέθοδο `getText()` που επιστρέφει ολόκληρο το έγγραφο. Παράδειγμα:

```python
full_text = ocr_result.getText()
print(full_text)
```

## Επόμενα Βήματα και Σχετικά Θέματα

Τώρα που ξέρετε πώς να **εκτελείτε OCR σε εικόνα**, εξετάστε:

- **Batch processing**: Συνδυάστε το script με `os.listdir()` για να διαχειριστείτε ολόκληρους φακέλους σαρωμένων εγγράφων.  
- **Εναλλακτικές βιβλιοθήκες**: Το `pytesseract` είναι μια δημοφιλής ανοιχτή επιλογή· η ροή εργασίας είναι παρόμοια—απλώς αντικαταστήστε τις κλήσεις στη μηχανή.  
- **Εξαγωγή αποτελεσμάτων**: Γράψτε τις εξαγόμενες λέξεις και τους βαθμούς εμπιστοσύνης σε CSV για περαιτέρω ανάλυση.
