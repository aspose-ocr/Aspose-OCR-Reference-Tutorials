# Δημιουργία ROI Polygon – Πλήρης Εκπαίδευση για OCR σε Επιλεγμένη Περιοχή

Έχετε ποτέ χρειαστεί να **create ROI polygon** ώστε να εκτελείτε OCR μόνο στο τμήμα μιας εικόνας που έχει σημασία; Ίσως σαρώνετε αποδείξεις και μόνο τα σύνολα έχουν σημασία, ή έχετε μια φόρμα όπου μόνο το πεδίο υπογραφής χρειάζεται ανάγνωση. Σε αυτές τις περιπτώσεις, **ocr on selected area** εξοικονομεί χρόνο και αυξάνει την ακρίβεια.  

Σε αυτόν τον οδηγό θα περάσουμε από όλα όσα χρειάζεστε: από τον ορισμό δύο πολυγώνων, την καταχώρισή τους σε μια μηχανή OCR, μέχρι την ανάκτηση του αναγνωρισμένου κειμένου με μία καθαρή κλήση. Στο τέλος θα έχετε ένα έτοιμο‑για‑εκτέλεση script και μια σαφή κατανόηση του γιατί η εστίαση σε περιοχές ενδιαφέροντος (ROI) είναι σημαντική.

## What You’ll Learn

- Πώς να **create ROI polygon** αντικείμενα χρησιμοποιώντας μια τυπική βιβλιοθήκη OCR.
- Η διαφορά μεταξύ ορισμού μιας μόνο περιοχής και πολλαπλών περιοχών.
- Πώς να καταχωρίσετε αυτές τις περιοχές στη μηχανή ώστε να εκτελείται `ocr on selected area`.
- Αναμενόμενο αποτέλεσμα και πώς να αντιμετωπίσετε κοινά προβλήματα (π.χ. επικαλυπτόμενα ROI, κενά αποτελέσματα).

### Prerequisites

- Python 3.8+ εγκατεστημένο.
- Μια βιβλιοθήκη OCR που εκθέτει τις μεθόδους `Polygon`, `Point` και `add_region_of_interest` (το παράδειγμα χρησιμοποιεί ένα φανταστικό module `ocr`; αντικαταστήστε το με το πραγματικό SDK σας, όπως Tesseract‑Python wrappers ή EasyOCR).
- Ένα δείγμα αρχείου εικόνας (`sample.png`) που περιέχει κείμενο στις συντεταγμένες που φαίνονται παρακάτω.

> **Pro tip:** Αν χρησιμοποιείτε πραγματική βιβλιοθήκη, βεβαιωθείτε ότι η εικόνα φορτώνεται στο ίδιο σύστημα συντεταγμένων (αρχή στην πάνω‑αριστερή γωνία, X αυξάνεται προς τα δεξιά, Y αυξάνεται προς τα κάτω).

## Step 1: Import the OCR Module and Load Your Image  

Πρώτα, φέρετε το πακέτο OCR στο scope και διαβάστε την εικόνα που θέλετε να επεξεργαστείτε.  

```python
import ocr  # Replace with your actual OCR library import
from pathlib import Path

# Load the image – adjust the path as needed
image_path = Path("sample.png")
ocr_engine = ocr.Engine(image_path)
```

*Why this matters:* Η μηχανή χρειάζεται τα δεδομένα της εικόνας πριν μπορέσετε να συνδέσετε οποιοδήποτε **ROI polygon**. Κάποιες βιβλιοθήκες επιτρέπουν επίσης τη μετάβαση της εικόνας αργότερα, αλλά η αρχική αρχικοποίηση κρατά τη ροή εργασίας τακτοποιημένη.

## Step 2: Define the First ROI Polygon  

Τώρα δημιουργούμε την πρώτη περιοχή ενδιαφέροντος. Σκεφτείτε το σαν να σχεδιάζετε ένα ορθογώνιο γύρω από την κεφαλίδα ενός πίνακα.  

```python
# Step 2: Define the first ROI polygon
first_roi = ocr.Polygon([
    ocr.Point(50, 20),   # top‑left
    ocr.Point(200, 20),  # top‑right
    ocr.Point(200, 80),  # bottom‑right
    ocr.Point(50, 80)    # bottom‑left
])
```

*Explanation:*  
- `ocr.Point(x, y)` χρησιμοποιεί συντεταγμένες pixel.  
- Η λίστα των σημείων είναι ταξινομημένη δεξιόστροφα, όπως απαιτούν οι περισσότερες μηχανές.  
- Μπορείτε να προσθέσετε όσες σημεία θέλετε για να δημιουργήσετε ακανόνιστα σχήματα· ένα ορθογώνιο είναι απλώς μια ειδική περίπτωση.

## Step 3: Define Additional ROI Polygons (Optional)  

Δεν χρειάζεται να σταματήσετε στο ένα. Εδώ είναι ένα δεύτερο πολύγωνο που καταγράφει ένα πεδίο υπογραφής πιο κάτω στη σελίδα.  

```python
# Step 3: Define the second ROI polygon
second_roi = ocr.Polygon([
    ocr.Point(300, 150),
    ocr.Point(480, 150),
    ocr.Point(480, 210),
    ocr.Point(300, 210)
])
```

*Why add more?* Η εκτέλεση **ocr on selected area** για πολλαπλές ζώνες σας επιτρέπει να εξάγετε διαφορετικά κομμάτια δεδομένων σε μία μόνο διεργασία, κάτι που είναι πολύ πιο γρήγορο από το κόψιμο και την επεξεργασία κάθε τμήματος ξεχωριστά.

## Step 4: Register the ROIs with the OCR Engine  

Με τα πολύγωνα έτοιμα, πείτε στη μηχανή ποιες περιοχές πρέπει να εξετάσει.  

```python
# Step 4: Register the ROIs
ocr_engine.add_region_of_interest(first_roi)
ocr_engine.add_region_of_interest(second_roi)
```

*Important note:* Κάποια SDK απαιτούν κλήση της μεθόδου `clear_regions()` πριν προσθέσετε νέες, εάν επαναχρησιμοποιείτε τη μηχανή για άλλη εικόνα.

## Step 5: Run OCR and Retrieve the Text  

Τέλος, εκκινήστε την αναγνώριση. Η μηχανή θα κοιτάξει μόνο μέσα στα πολύγωνα που προσθέσαμε.  

```python
# Step 5: Perform OCR on the defined regions
recognized_text = ocr_engine.recognize().get_text()
print("=== Recognized Text ===")
print(recognized_text)
```

**Expected output** (υποθέτοντας ότι το `sample.png` περιέχει τις λέξεις “Invoice Total: $123.45” στο πρώτο ROI και “Signature: John Doe” στο δεύτερο):

```
=== Recognized Text ===
Invoice Total: $123.45
Signature: John Doe
```

Αν το αποτέλεσμα είναι κενό, ελέγξτε ξανά ότι οι συντεταγμένες πραγματικά διασχίζουν κείμενο και ότι η ανάλυση της εικόνας ταιριάζει με το σύστημα συντεταγμένων.

## Edge Cases & Common Pitfalls  

| Situation                               | What to Watch For                               | Fix / Work‑around                              |
|----------------------------------------|--------------------------------------------------|-----------------------------------------------|
| **Overlapping ROIs**                    | Το κείμενο μπορεί να επιστραφεί δύο φορές.      | Κρατήστε τα πολύγωνα διακριτά ή αφαιρέστε διπλότυπα από το αποτέλεσμα. |
| **ROI outside image bounds**           | Η μηχανή μπορεί να ρίξει σφάλμα ή να μην επιστρέψει τίποτα. | Περιορίστε τις συντεταγμένες στο `0 ≤ x < width`, `0 ≤ y < height`. |
| **Very small ROI**                     | Η ακρίβεια του OCR πέφτει λόγω ανεπαρκών pixel. | Επεκτείνετε το πολύγωνο με μερικά pixel ή αυξήστε την ανάλυση της εικόνας πρώτα. |
| **Non‑rectangular shape**               | Κάποιες μηχανές υποστηρίζουν μόνο κυρτά πολύγωνα. | Διασπάστε πολύπλοκα σχήματα σε πολλαπλά κυρτά ROI. |
| **Changing image size**                | Οι σκληρά κωδικοποιημένες συντεταγμένες γίνονται άκυρες. | Υπολογίστε τις συντεταγμένες ROI σχετικά με τις διαστάσεις της εικόνας (π.χ. ποσοστά). |

## Full Working Example  

Παρακάτω είναι το πλήρες script που μπορείτε να αντιγράψετε‑και‑επικολλήσετε και να τρέξετε (αντικαταστήστε το `ocr` με τη δική σας βιβλιοθήκη).  

```python
import ocr                      # <- your OCR library
from pathlib import Path

# -------------------------------------------------
# Configuration
# -------------------------------------------------
IMAGE_PATH = Path("sample.png")

# -------------------------------------------------
# Initialize OCR engine with the image
# -------------------------------------------------
engine = ocr.Engine(IMAGE_PATH)

# -------------------------------------------------
# Define ROI polygons
# -------------------------------------------------
first_roi = ocr.Polygon([
    ocr.Point(50, 20), ocr.Point(200, 20),
    ocr.Point(200, 80), ocr.Point(50, 80)
])

second_roi = ocr.Polygon([
    ocr.Point(300, 150), ocr.Point(480, 150),
    ocr.Point(480, 210), ocr.Point(300, 210)
])

# -------------------------------------------------
# Register ROIs
# -------------------------------------------------
engine.add_region_of_interest(first_roi)
engine.add_region_of_interest(second_roi)

# -------------------------------------------------
# Run OCR on the selected areas
# -------------------------------------------------
result = engine.recognize().get_text()

print("=== Recognized Text ===")
print(result)
```

Τρέξτε το με `python ocr_roi_example.py` και θα δείτε τις εξαγόμενες συμβολοσειρές από τις δύο καθορισμένες ζώνες.
