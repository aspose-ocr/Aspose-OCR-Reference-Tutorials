# Πώς να Διορθώσετε την Κλίση μιας Εικόνας – Πλήρης Οδηγός με OCR

Έχετε αναρωτηθεί ποτέ **πώς να διορθώσετε την κλίση μιας εικόνας** που προήλθε από ένα φθηνό σαρωτή; Δεν είστε μόνοι. Μια στραβή σελίδα φαίνεται ακατάστατη, και πιο σημαντικό, η κλίση μπορεί να διαταράξει οποιονδήποτε μηχανισμό OCR που προσπαθεί να **αναγνωρίσει κείμενο από εικόνα**.  

Σε αυτό το σεμινάριο θα περάσουμε από μια πλήρη **διαδικασία προεπεξεργασίας εικόνας** που διορθώνει την κλίση, κάνει δυαδική απεικόνιση και αφαιρεί τον θόρυβο από ένα σκαν, και στη συνέχεια τη παραδίδει σε έναν μηχανισμό OCR ώστε να μπορείτε να **μετατρέψετε τη σαρωμένη εικόνα σε κείμενο** με ελάχιστη δυσκολία. Χωρίς μαγεία, μόνο απλό Python και μια μικρή βιβλιοθήκη που κάνει το σκληρό έργο.

## Τι Θα Χρειαστείτε

- Python 3.9+ (ο κώδικας λειτουργεί σε 3.10 και νεότερες εκδόσεις)
- Το πακέτο `ocr` (εγκαταστήστε το με `pip install ocr‑toolkit` – αντικαταστήστε το με το πραγματικό όνομα της βιβλιοθήκης σας)
- Μια σαρωμένη JPEG ή PNG που είναι εμφανώς στραβή (π.χ., `skewed_scan.jpg`)
- Μια μικρή δόση περιέργειας για το γιατί κάθε βήμα προεπεξεργασίας είναι σημαντικό

Αυτό είναι όλο. Δεν χρειάζονται βαριές εξαρτήσεις όπως το OpenCV, εκτός αν θέλετε να εμβαθύνετε αργότερα.

## Βήμα 1: Φόρτωση της Σαρωμένης Εικόνας

Το πρώτο βήμα είναι να φορτώσουμε το αρχείο στη μνήμη. Αυτό το βήμα είναι απλό αλλά κρίσιμο—αν η διαδρομή είναι λανθασμένη, όλα τα υπόλοιπα καταρρέουν.

```python
# Step 1: Load the scanned image
image_path = "YOUR_DIRECTORY/skewed_scan.jpg"
original_image = ocr.Imaging.Image.load(image_path)
```

> **Γιατί;**  
> Η φόρτωση της εικόνας μας δίνει ένα αντικείμενο που μπορεί να χειριστεί το `ocr.ImagePreprocessor`. Αν παραλείψετε αυτό το βήμα, η αλυσίδα παραμένει τυφλή στα πραγματικά δεδομένα pixel.

## Πώς να Διορθώσετε την Κλίση της Εικόνας και να Την Προετοιμάσετε για OCR

Τώρα που έχουμε την εικόνα, ας την ευθυγραμμίσουμε. Η μέθοδος `deskew()` εκτιμά τη γωνία κλίσης και περιστρέφει την εικόνα πίσω στην οριζόντια θέση.

```python
# Step 2: Create an image preprocessor and correct the orientation
preprocessor = ocr.ImagePreprocessor()
preprocessor.deskew()
```

> **Συμβουλή επαγγελματία:** Αν οι σαρώσεις σας είναι μόνο ελαφρώς λανθασμένες (≤ 3°), ο προεπιλεγμένος αλγόριθμος είναι συνήθως ακριβής. Για ακραίες γωνίες ίσως χρειαστεί να ρυθμίσετε τις εσωτερικές παραμέτρους—ελέγξτε την τεκμηρίωση της βιβλιοθήκης για το `max_angle`.

## Δυαδική Απεικόνιση της Εικόνας – Κάντε την Μαύρη‑και‑Άσπρη

Οι μηχανισμοί OCR αγαπούν εισόδους υψηλής αντίθεσης. Η μετατροπή της εικόνας σε δυαδική (μαύρη‑και‑άσπρη) αναπαράσταση αφαιρεί ήπιες αποχρώσεις που μπορούν να μπερδέψουν την αναγνώριση χαρακτήρων.

```python
# Step 3: Convert the image to a binary (black‑and‑white) representation
preprocessor.binarize(threshold=128)
```

> **Τι συμβαίνει;**  
> Τα pixel που είναι φωτεινότερα από 128 γίνονται λευκά· όλα τα υπόλοιπα γίνονται μαύρα. Ρυθμίστε το όριο αν η σάρωση σας είναι ασυνήθιστα σκοτεινή ή φωτεινή.

## Αφαίρεση Θορύβου από τη Σάρωση Πριν το OCR

Ακόμη και μια τέλεια ευθυγραμμισμένη εικόνα μπορεί να περιέχει στίγματα, σκόνη ή τεχνητά σφάλματα συμπίεσης. Αυτά τα μικρά στίγματα είναι η αδυναμία κάθε μηχανισμού OCR.

```python
# Step 4: Reduce noise to improve OCR accuracy
preprocessor.denoise(radius=1)
```

> **Γιατί να αφαιρέσουμε τον θόρυβο;**  
> Ο θόρυβος δημιουργεί ψευδείς άκρες που ο μηχανισμός OCR μπορεί να ερμηνεύσει ως χαρακτήρες. Μια μικρή ακτίνα λειτουργεί για τις περισσότερες σαρώσεις· αυξήστε την για πολύ σπογγώδη έγγραφα.

## Εφαρμογή της Διαδικασίας Προεπεξεργασίας Εικόνας

Όλες οι ρυθμίσεις είναι έτοιμες—τώρα τρέχουμε τη διαδικασία στην αρχική εικόνα.

```python
# Step 5: Apply the preprocessing pipeline to the loaded image
processed_image = preprocessor.apply(original_image)
```

> **Αποτέλεσμα:** Το `processed_image` είναι ένα καθαρό, ευθυγραμμισμένο, υψηλής αντίθεσης bitmap έτοιμο για εξαγωγή κειμένου.

## Αναγνώριση Κειμένου από Εικόνα με Μηχανισμό OCR

Ώρα να διαβάσουμε το κείμενο. Αρχικοποιούμε τον μηχανισμό OCR, του δίνουμε την επεξεργασμένη εικόνα και ζητάμε τη ακατέργαστη συμβολοσειρά.

```python
# Step 6: Initialise the OCR engine and provide the processed image
ocr_engine = ocr.OcrEngine()
ocr_engine.set_image(processed_image)

# Step 7: Recognise text and output the result
recognized_text = ocr_engine.recognize().get_text()
print(recognized_text)
```

> **Αναμενόμενο αποτέλεσμα** (παράδειγμα):

```
Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
```

Αν δείτε **ακατάλληλους χαρακτήρες**, ελέγξτε ξανά το βήμα διόρθωσης κλίσης ή δοκιμάστε διαφορετική τιμή `threshold`.

## Δημιουργία μιας Διαδικασίας Προεπεξεργασίας Εικόνας – Πλήρες Script

Συνδυάζοντας όλα τα παραπάνω, εδώ είναι ένα ενιαίο, εκτελέσιμο script που μπορείτε να αποθηκεύσετε σε αρχείο `.py` και να τρέξετε:

```python
import ocr  # Replace with the actual import path of your OCR library

# -------------------------------------------------
# 1️⃣ Load the scanned image
# -------------------------------------------------
image_path = "YOUR_DIRECTORY/skewed_scan.jpg"
original_image = ocr.Imaging.Image.load(image_path)

# -------------------------------------------------
# 2️⃣ Deskew – how to deskew image
# -------------------------------------------------
preprocessor = ocr.ImagePreprocessor()
preprocessor.deskew()

# -------------------------------------------------
# 3️⃣ Binarize – convert to black‑and‑white
# -------------------------------------------------
preprocessor.binarize(threshold=128)

# -------------------------------------------------
# 4️⃣ Denoise – remove noise from scan
# -------------------------------------------------
preprocessor.denoise(radius=1)

# -------------------------------------------------
# 5️⃣ Apply the pipeline
# -------------------------------------------------
processed_image = preprocessor.apply(original_image)

# -------------------------------------------------
# 6️⃣ OCR – recognize text from image
# -------------------------------------------------
ocr_engine = ocr.OcrEngine()
ocr_engine.set_image(processed_image)

# -------------------------------------------------
# 7️⃣ Output – convert scanned image to text
# -------------------------------------------------
recognized_text = ocr_engine.recognize().get_text()
print(recognized_text)
```

> **Συμβουλή:** Τυλίξτε όλο το κομμάτι σε μια συνάρτηση (`def ocr_from_file(path): …`) αν σκοπεύετε να επεξεργαστείτε πολλά αρχεία σε παρτίδα.

## Συχνές Ερωτήσεις & Ακραίες Περιπτώσεις

- **Τι γίνεται αν η εικόνα είναι ήδη όρθια;**  
  Η μέθοδος `deskew()` εντοπίζει μια γωνία σχεδόν μηδενική και αφήνει την εικόνα αμετάβλητη, οπότε μπορείτε να την εκτελείτε με ασφάλεια σε κάθε αρχείο.

- **Η σάρωση μου είναι έγχρωμη—να τη διατηρήσω;**  
  Το χρώμα δεν προσθέτει αξία στις περισσότερες εργασίες OCR. Η δυαδική απεικόνιση το αφαιρεί, επιταχύνοντας την επεξεργασία και μειώνοντας τη χρήση μνήμης.

- **Μπορώ να συνδυάσω πολλαπλά βήματα προεπεξεργασίας;**  
  Απόλυτα. Η κλάση `ImagePreprocessor` έχει σχεδιαστεί για pipelines· μπορείτε να προσθέσετε `sharpen()`, `contrast_enhance()` ή προσαρμοσμένα φίλτρα πριν καλέσετε `apply()`.

- **Η έξοδος του OCR έχει ανεπιθύμητες αλλαγές γραμμής—πώς να το διορθώσω;**  
  Μετα-επεξεργαστείτε τη συμβολοσειρά με `text.replace("\n", " ").strip()` ή χρησιμοποιήστε μια πιο προχωρημένη βιβλιοθήκη ανάλυσης διάταξης όπως τις `--psm` λειτουργίες του Tesseract.
