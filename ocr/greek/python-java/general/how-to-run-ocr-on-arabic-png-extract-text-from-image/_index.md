# Πώς να Εκτελέσετε OCR σε Αραβικό PNG – Εξαγωγή Κειμένου από Εικόνα

Έχετε αναρωτηθεί **πώς να εκτελέσετε OCR** σε μια εικόνα που περιέχει αραβικό κείμενο; Ίσως έχετε ένα σαρωμένο απόδειξη, ένα ιστορικό χειρόγραφο ή απλώς ένα στιγμιότυπο οθόνης από μια ανάρτηση στα κοινωνικά δίκτυα και χρειάζεστε το κείμενο σε μορφή αναζητήσιμη. Δεν είστε μόνοι—προγραμματιστές παγκοσμίως αντιμετωπίζουν αυτό το πρόβλημα όταν δουλεύουν με γλώσσες από δεξιά προς τα αριστερά.

Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από ένα πλήρες, εκτελέσιμο παράδειγμα που δείχνει **πώς να εκτελέσετε OCR** σε αρχείο PNG, να εξάγετε αραβικό κείμενο και να εκτυπώσετε το αποτέλεσμα στην κονσόλα. Χωρίς ασαφείς «δείτε την τεκμηρίωση» συνδέσμους· μόνο ο κώδικας που μπορείτε να αντιγράψετε‑επικολλήσετε, μαζί με εξηγήσεις για το γιατί κάθε γραμμή είναι σημαντική. Στο τέλος θα μπορείτε **να μετατρέψετε εικόνα σε κείμενο** αξιόπιστα, ακόμη και όταν η πηγή είναι ένα δύσκολο αραβικό PNG.

> **Τι θα μάθετε**
> - Ρύθμιση μιας μηχανής OCR σε Python για Αραβικά  
> - Φόρτωση PNG εικόνας και αντιμετώπιση κοινών προβλημάτων  
> - Αναγνώριση αραβικού κειμένου και επαλήθευση του αποτελέσματος  
> - Συμβουλές για **εξαγωγή αραβικού κειμένου** από εικόνες διαφορετικής ποιότητας  

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε εγκατεστημένο το Python 3.8+ και μια πρόσφατη έκδοση της βιβλιοθήκης `ocr` (αυτή που χρησιμοποιείται στο απόσπασμα). Αν χρησιμοποιείτε εικονικό περιβάλλον, ενεργοποιήστε το τώρα—αυτό κρατά τις εξαρτήσεις οργανωμένες.

## Προαπαιτούμενα

- Python 3.8 ή νεότερο  
- Πακέτο `ocr` (`pip install ocr‑engine` – αντικαταστήστε με το πραγματικό όνομα του πακέτου)  
- Μια αραβική PNG εικόνα (`arabic_doc.png`) τοποθετημένη κάπου που μπορείτε να αναφέρετε  
- Βασική εξοικείωση με συναρτήσεις και κλάσεις Python  

Αυτό είναι όλο. Χωρίς βαρύτατα frameworks, χωρίς Docker containers—απλώς καθαρό Python.

## Βήμα 1: Εγκατάσταση και Εισαγωγή της Βιβλιοθήκης OCR

Πρώτα απ’ όλα. Χρειαζόμαστε τη μηχανή OCR. Η βιβλιοθήκη που θα χρησιμοποιήσουμε εκθέτει μια κλάση `OcrEngine` με ένα απλό API.

```python
# Install the library (run once in your terminal)
# pip install ocr-engine   # <-- replace with the correct package name

# Import the necessary classes
import ocr
from ocr import Imaging
```

*Γιατί εισάγουμε το `Imaging` ξεχωριστά;* Το υπο-μοντέλο `Imaging` μας παρέχει μια βολική μέθοδο `Image.load` που καταλαβαίνει PNG, JPEG και TIFF από προεπιλογή. Η παράλειψη αυτού του βήματος θα σας ανάγκαζε να διαχειριστείτε ακατέργαστα bytes, κάτι περιττό για τις περισσότερες περιπτώσεις χρήσης.

## Βήμα 2: Δημιουργία του Αντικειμένου Μηχανής OCR

Τώρα δημιουργούμε ένα αντικείμενο της μηχανής. Σκεφτείτε το ως το «εγκέφαλο» που θα επεξεργαστεί την εικόνα.

```python
# Step 2: Create an OCR engine instance
ocr_engine = ocr.OcrEngine()
```

> **Pro tip:** Αν σκοπεύετε να επεξεργαστείτε πολλές εικόνες διαδοχικά, επαναχρησιμοποιήστε το ίδιο αντικείμενο `ocr_engine`. Κρατά στην μνήμη τα μοντέλα γλώσσας, κάτι που επιταχύνει τις επόμενες αναγνώσεις.

## Βήμα 3: Ορισμός της Γλώσσας σε Αραβικά

Τα Αραβικά δεν είναι Λατινικά· έχουν το δικό τους σύνολο χαρακτήρων, κατεύθυνση από δεξιά προς τα αριστερά και συμφραζόμενη διαμόρφωση. Γι’ αυτό πρέπει ρητά να πούμε στη μηχανή ποιο μοντέλο γλώσσας να φορτώσει.

```python
# Step 3: Set the language to Arabic (language code "ar")
ocr_engine.set_language("ar")
```

Αν παραλείψετε αυτή τη γραμμή, η μηχανή προεπιλέγει τα Αγγλικά και θα λάβετε ακατάλληλο αποτέλεσμα—κάτι που έχω δει να συμβαίνει πολύ συχνά.

## Βήμα 4: Φόρτωση της PNG Εικόνας σας

Εδώ αρχίζει πραγματικά το **convert image to text**. Θα φορτώσουμε το αρχείο PNG που περιέχει το αραβικό κείμενο.

```python
# Step 4: Load the image that contains the Arabic text
image_path = "YOUR_DIRECTORY/arabic_doc.png"   # replace with your actual path
ocr_engine.set_image(Imaging.Image.load(image_path))
```

### Συνηθισμένες Ακραίες Περιπτώσεις

| Πρόβλημα | Συμπτωμα | Διόρθωση |
|----------|----------|----------|
| Η εικόνα είναι πολύ σκοτεινή | Το αποτέλεσμα περιέχει πολλά κενά | Προεπεξεργασία με Pillow (`ImageEnhance.Brightness`) |
| Το PNG έχει κανάλι αλφα | Κάποιες μηχανές OCR διαβάζουν λανθασμένα διαφανή pixel | Μετατροπή σε RGB (`image.convert("RGB")`) πριν τη φόρτωση |
| Το κείμενο είναι περιστραμμένο | Το αναγνωρισμένο κείμενο είναι ανάποδο | Περιστροφή της εικόνας (`image.rotate(90, expand=True)`) πριν τη δώσετε στη μηχανή |

## Βήμα 5: Εκτέλεση της Διαδικασίας Αναγνώρισης

Με όλα έτοιμα, ζητάμε τελικά από τη μηχανή να κάνει τη δουλειά της.

```python
# Step 5: Run the recognition process
ocr_result = ocr_engine.recognize()
```

Η μέθοδος `recognize` επιστρέφει ένα αντικείμενο που περιέχει το ακατέργαστο Unicode string, βαθμούς εμπιστοσύνης και περιοριστικά πλαίσια. Για τους περισσότερους προγραμματιστές, το απλό κείμενο είναι ό,τι χρειάζονται.

## Βήμα 6: Εμφάνιση του Αναγνωρισμένου Αραβικού Κειμένου

Τώρα εκτυπώνουμε το αποτέλεσμα. Σε μια πραγματική εφαρμογή ίσως γράψετε σε αρχείο, βάση δεδομένων ή το στέλνετε σε API μετάφρασης.

```python
# Step 6: Output the recognized text
print(ocr_result.get_text())
```

Όταν τρέξετε το script, θα πρέπει να δείτε τους αραβικούς χαρακτήρες στην κονσόλα σας (βεβαιωθείτε ότι το τερματικό σας υποστηρίζει Unicode). Αν βλέπετε ερωτηματικά ή κενές συμβολοσειρές, ελέγξτε ξανά τη ρύθμιση γλώσσας και την ποιότητα της εικόνας.

### Αναμενόμενο Αποτέλεσμα

هذا نص عربي من صورة PNG تم استخراجها باستخدام OCR.

Αν το αποτέλεσμα μοιάζει με τη γραμμή παραπάνω, συγχαρητήρια—έχετε εξαγάγει **αραβικό κείμενο** από ένα PNG!

## Πλήρες Παράδειγμα Εργασίας

Παρακάτω είναι ολόκληρο το script, έτοιμο για αντιγραφή‑επικόλληση. Αντικαταστήστε το `YOUR_DIRECTORY/arabic_doc.png` με τη διαδρομή του δικού σας αρχείου.

```python
import ocr
from ocr import Imaging

def main():
    # Create OCR engine
    ocr_engine = ocr.OcrEngine()
    
    # Set Arabic language
    ocr_engine.set_language("ar")
    
    # Load the PNG image
    image_path = "YOUR_DIRECTORY/arabic_doc.png"
    ocr_engine.set_image(Imaging.Image.load(image_path))
    
    # Recognize text
    ocr_result = ocr_engine.recognize()
    
    # Print the extracted Arabic text
    print(ocr_result.get_text())

if __name__ == "__main__":
    main()
```

Τρέξτε το με `python run_ocr.py` (ή ό,τι όνομα δώσατε στο αρχείο). Αν όλα είναι εγκατεστημένα σωστά, θα δείτε την αραβική πρόταση να εμφανίζεται στην κονσόλα.

## Πώς να Εκτελέσετε OCR σε Διάφορες Μορφές Εικόνας

Ο ίδιος κώδικας λειτουργεί για JPEG, TIFF ή BMP—απλώς αλλάξτε την επέκταση του αρχείου. Η μέθοδος `Image.load` ανιχνεύει αυτόματα τη μορφή, ώστε να μπορείτε **να μετατρέψετε εικόνα σε κείμενο** χωρίς επιπλέον κώδικα.

```python
# Example for a JPEG file
ocr_engine.set_image(Imaging.Image.load("sample.jpg"))
```

Θυμηθείτε, η ποιότητα της πηγαίας εικόνας επηρεάζει σε μεγάλο βαθμό την ακρίβεια του βήματος **recognize Arabic text**. Εικόνες υψηλής ανάλυσης και χαμηλής συμπίεσης δίνουν τα καλύτερα αποτελέσματα.

## Εξαγωγή Κειμένου από PNG: Συμβουλές για Καλύτερη Ακρίβεια

1. **Το DPI Μετρά** – Στοχεύστε τουλάχιστον 300 dpi. Χαμηλότερο DPI συχνά οδηγεί σε χαμένα γράμματα.  
2. **Η Αντίθεση Είναι Βασική** – Χρησιμοποιήστε εργαλεία επεξεργασίας εικόνας (π.χ. OpenCV) για να αυξήσετε την αντίθεση πριν δώσετε την εικόνα στη μηχανή OCR.  
3. **Αφαίρεση Θορύβου** – Μικρά σπινθήρια μπορούν να μπερδέψουν το μοντέλο· η μεσαία φιλτράρισμα βοηθά.  

Ακολουθεί ένα γρήγορο απόσπασμα με Pillow για βελτίωση ενός PNG πριν το OCR:

```python
from PIL import Image, ImageEnhance, ImageFilter

def preprocess_png(path):
    img = Image.open(path).convert("L")          # grayscale
    img = ImageEnhance.Contrast(img).enhance(2)  # boost contrast
    img = img.filter(ImageFilter.MedianFilter()) # denoise
    return img

# Use the preprocessed image
prepped = preprocess_png("arabic_doc.png")
ocr_engine.set_image(Imaging.Image.from_pil(prepped))
```

## Συχνές Ερωτήσεις

**Ε: Λειτουργεί αυτό με γλώσσες από δεξιά προς τα αριστερά εκτός από τα Αραβικά;**  
Α: Απόλυτα. Απλώς αλλάξτε τον κωδικό γλώσσας (`"ar"` → `"he"` για τα Εβραϊκά, `"fa"` για τα Περσικά) και η μηχανή θα φορτώσει το αντίστοιχο μοντέλο.

**Ε: Τι γίνεται αν χρειαστεί να εξάγω κείμενο από πολλά PNG σε έναν φάκελο;**  
Α: Κάντε βρόχο πάνω στα αρχεία, επαναχρησιμοποιήστε το ίδιο αντικείμενο `ocr_engine` και συγκεντρώστε τα αποτελέσματα σε λίστα ή γράψτε το καθένα σε ξεχωριστό αρχείο `.txt`.

**Ε: Μπορώ να λάβω βαθμούς εμπιστοσύνης για κάθε λέξη;**  
Α: Ναι. Το `ocr_result` συνήθως εκθέτει μια μέθοδο `get_confidences()`. Συνδυάστε κάθε βαθμό εμπιστοσύνης με τη σχετική λέξη για έλεγχο ποιότητας.

## Επόμενα Βήματα

Τώρα που ξέρετε **πώς να εκτελέσετε OCR** σε αραβικά PNG, σκεφτείτε τις παρακάτω ιδέες:

- **Batch processing:** Συνδυάστε το script με `os.listdir` για **εξαγωγή αραβικού κειμένου** από ολόκληρο φάκελο.  
- **Post‑processing:** Χρησιμοποιήστε τις βιβλιοθήκες `arabic_reshaper` και `python-bidi` για σωστή εμφάνιση δεξιά‑προς‑αριστερά εξόδου σε PDF.  
- **Integration:** Ενσωματώστε το εξαγόμενο κείμενο σε ευρετήριο αναζήτησης (π.χ. Elasticsearch) ώστε τα σαρωμένα έγγραφα να γίνονται αναζητήσιμα.  

Κάθε ένα από αυτά τα θέματα βασίζεται στα θεμέλια που καλύψαμε εδώ και θα σας βοηθήσει να μετατρέψετε ένα απλό **convert image to text** script σε μια παραγωγική γραμμή εργασίας.
