# Πώς να Διορθώσετε την Κλίση Εικόνας – Πλήρης Οδηγός για Προεπεξεργασία OCR

Έχετε αναρωτηθεί ποτέ **πώς να διορθώσετε την κλίση εικόνας** σε αρχεία που φαίνονται σαν να έχουν ληφθεί από μια στραβή σαρωτική μηχανή; Δεν είστε μόνοι. Σε πολλά πραγματικά έργα, το πρώτο που πρέπει να κάνετε πριν μπορέσετε να **εξάγετε κείμενο από εικόνα** είναι να ευθυγραμμίσετε αυτήν την κλίση.

Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από ένα πρακτικό, ολοκληρωμένο παράδειγμα που **προετοιμάζει την εικόνα για OCR**, διορθώνει την περιστροφή και τελικά **μετατρέπει τη σάρωση σε κείμενο** χρησιμοποιώντας μια βιβλιοθήκη OCR για Python. Χωρίς ασαφείς αναφορές, μόνο ένα λειτουργικό script που μπορείτε να αντιγράψετε‑επικολλήσετε, μαζί με συμβουλές για κοινά προβλήματα.

## Τι Θα Επιτύχετε

* Φορτώστε οποιοδήποτε σαρωμένο JPEG ή PNG που είναι ελαφρώς κλινό.  
* Εφαρμόστε ένα φίλτρο διόρθωσης κλίσης και ένα βήμα δυαδικοποίησης για να αυξήσετε την ακρίβεια του OCR.  
* Εκτελέστε τη μηχανή OCR και **εξάγετε κείμενο από εικόνα** αξιόπιστα.  
* Κατανοήστε γιατί η **σωστή περιστροφή εικόνας** είναι σημαντική για την επακόλουθη εξαγωγή κειμένου.  

### Προαπαιτούμενα

* Python 3.9+ εγκατεστημένο στο σύστημά σας.  
* Ένα πακέτο OCR που μπορεί να εγκατασταθεί μέσω pip και μιμείται το namespace `ocr` που χρησιμοποιείται στο παράδειγμα (π.χ., μια ελαφριά επικάλυψη γύρω από το Tesseract).  
* Βασική εξοικείωση με συναρτήσεις Python και έννοιες επεξεργασίας εικόνας.  

Αν τα έχετε, ας ξεκινήσουμε.

## Βήμα 1: Ρύθμιση της Μηχανής OCR – Πώς να Διορθώσετε την Κλίση Εικόνας Χρησιμοποιώντας Python

Πρώτα απ' όλα: χρειάζεστε μια μηχανή OCR που να μπορεί να καταλάβει τη γλώσσα του εγγράφου σας. Το παρακάτω απόσπασμα δείχνει το ελάχιστο boilerplate για τη δημιουργία της μηχανής και την ενημέρωση ότι εργάζεστε με αγγλικό κείμενο.

```python
import ocr  # Assume this is a wrapper around your OCR backend

# Create an OCR engine instance and set the language
engine = ocr.OcrEngine()
engine.language = ocr.Language.ENGLISH
```

*Γιατί είναι σημαντικό:* Η ρύθμιση γλώσσας της μηχανής επηρεάζει το σύνολο χαρακτήρων και το λεξικό που χρησιμοποιεί. Η παράλειψη αυτού του βήματος μπορεί να κάνει το OCR να ερμηνεύσει λανθασμένα κοινές λέξεις, ειδικά μετά την **διόρθωση περιστροφής εικόνας**.

## Βήμα 2: Φόρτωση της Σαρωμένης Εικόνας που Θέλετε να Ευθυγραμμίσετε

Τώρα φέρνουμε το αρχείο στη μνήμη. Αντικαταστήστε το `"YOUR_DIRECTORY/skewed_scan.jpg"` με τη διαδρομή της δικής σας εικόνας.

```python
# Load the raw scanned image
raw_image = ocr.Image.load("YOUR_DIRECTORY/skewed_scan.jpg")
```

Αν η εικόνα είναι ήδη σε έναν πίνακα NumPy ή σε ένα OpenCV `Mat`, μπορείτε να προσαρμόσετε τον φορτωτή ανάλογα – το κλειδί είναι το αντικείμενο να εκθέτει τη μέθοδο `apply_filter` που χρησιμοποιείται αργότερα.

## Βήμα 3: Προεπεξεργασία Εικόνας για OCR – Διόρθωση Κλίσης και Δυαδικοποίηση

Εδώ συμβαίνει η μαγεία. Συνδέουμε δύο φίλτρα:

1. **Deskew** – εντοπίζει αυτόματα τη κυρίαρχη βάση κειμένου και περιστρέφει την εικόνα πίσω σε οριζόντια θέση.  
2. **Binarize (Otsu)** – μετατρέπει την εικόνα σε καθαρό μαύρο‑και‑λευκό, βελτιώνοντας δραματικά τα ποσοστά αναγνώρισης.

```python
# Preprocess: deskew then binarize
preprocessed_image = (
    raw_image
    .apply_filter(ocr.Filter.deskew())          # <-- how to deskew image
    .apply_filter(ocr.Filter.binarize_otsu())  # improves OCR reliability
)
```

*Συμβουλή:* Αν παρατηρήσετε ότι το κείμενο εξακολουθεί να φαίνεται θολό μετά τη δυαδικοποίηση, δοκιμάστε να ρυθμίσετε την αντίθεση ή να χρησιμοποιήσετε διαφορετική μέθοδο ορίου. Το module `ocr.Filter` συχνά περιλαμβάνει τη λειτουργία `adaptive_threshold()` για πιο δύσκολες περιπτώσεις.

## Βήμα 4: Εκτέλεση OCR – Εξαγωγή Κειμένου από Εικόνα

Με έναν καθαρό, ευθυγραμμισμένο καμβά, παραδίδουμε την εικόνα στη μηχανή. Το αντικείμενο αποτελέσματος περιέχει τη αναγνωρισμένη συμβολοσειρά, τις βαθμολογίες εμπιστοσύνης και ακόμη και τα πλαίσια οριοθέτησης αν τα χρειαστείτε αργότερα.

```python
# Perform recognition on the preprocessed image
recognition_result = engine.recognize(preprocessed_image)

# Print the raw text output
print(recognition_result.text)
```

Η τυπική έξοδος μοιάζει με:

```
Invoice #12345
Date: 2026-07-01
Total: $1,250.00
Thank you for your business!
```

Παρατηρήσατε πώς οι αλλαγές γραμμής ευθυγραμμίζονται τέλεια; Αυτό είναι το όφελος της **σωστής περιστροφής εικόνας** – το OCR δεν χρειάζεται πλέον να μαντεύει την κατεύθυνση των γραμμών.

## Βήμα 5: Συνδυάστε Όλα – Ένα Σενάριο σε Ένα Αρχείο για Μετατροπή Σάρωσης σε Κείμενο

Παρακάτω βρίσκεται το πλήρες, εκτελέσιμο σενάριο που συνδυάζει όλα τα μέρη που συζητήσαμε. Αποθηκεύστε το ως `deskew_ocr.py` και εκτελέστε `python deskew_ocr.py`.

```python
#!/usr/bin/env python3
"""
Complete example: how to deskew image, preprocess it for OCR,
and extract text from a scanned document.
"""

import ocr  # Replace with your actual OCR library import

def main():
    # 1️⃣ Initialize OCR engine
    engine = ocr.OcrEngine()
    engine.language = ocr.Language.ENGLISH

    # 2️⃣ Load the image (change path as needed)
    raw_image = ocr.Image.load("YOUR_DIRECTORY/skewed_scan.jpg")

    # 3️⃣ Preprocess – deskew then binarize
    preprocessed = (
        raw_image
        .apply_filter(ocr.Filter.deskew())          # how to deskew image
        .apply_filter(ocr.Filter.binarize_otsu())  # preprocess image for OCR
    )

    # 4️⃣ Recognize text
    result = engine.recognize(preprocessed)

    # 5️⃣ Output the extracted text
    print("=== Recognized Text ===")
    print(result.text)

if __name__ == "__main__":
    main()
```

### Γιατί Λειτουργεί Αυτό

* **Deskew πρώτα** – η περιστροφή της εικόνας πριν τη δυαδικοποίηση εξασφαλίζει ότι ο αλγόριθμος ορίου λειτουργεί σε οριζόντια βάση.  
* **Δυαδικοποίηση μετά το deskew** – η μέθοδος Otsu υποθέτει ένα δισυνοπτικό ιστόγραμμα· μια κλινή σελίδα θα σπάσει αυτήν την υπόθεση.  
* **Μοντέλο αγγλικής γλώσσας** – ενημερώνει το OCR για τους χαρακτήρες που πρέπει να περιμένει, μειώνοντας τα ψευδώς θετικά.  

Αν χρειάζεται να υποστηρίξετε άλλες γλώσσες, απλώς αντικαταστήστε το `ocr.Language.ENGLISH` με το αντίστοιχο enum.

## Συχνές Ερωτήσεις & Ακραίες Περιπτώσεις

| Ερώτηση | Απάντηση |
|----------|--------|
| *Τι γίνεται αν η σάρωση είναι ανάποδα;* | Το φίλτρο `deskew()` συνήθως εντοπίζει και περιστροφή 180°. Αν αποτύχει, καλέστε `apply_filter(ocr.Filter.rotate(180))` πριν το deskew. |
| *Το έγγραφό μου έχει χρωματιστά γραφικά – θα τα διαγράψει η δυαδικοποίηση;* | Ναι. Για μεικτό περιεχόμενο, σκεφτείτε να χρησιμοποιήσετε μόνο το `ocr.Filter.deskew()`, και στη συνέχεια να εκτελέσετε OCR στην έγχρωμη εικόνα. Μπορείτε ακόμα να εξάγετε κείμενο διατηρώντας τα γραφικά. |
| *Μπορώ να επεξεργαστώ μια δέσμη αρχείων;* | Τυλίξτε τη λογική σε έναν βρόχο, διαβάστε κάθε διαδρομή αρχείου από μια λίστα και αποθηκεύστε κάθε `result.text` σε ξεχωριστό αρχείο `.txt`. |
| *Πώς μπορώ να βελτιώσω την ακρίβεια σε σάρωσεις χαμηλής ανάλυσης;* | Αυξήστε την ανάλυση της εικόνας με ένα διπλό κυβικό φίλτρο **πριν** το deskew, έπειτα εφαρμόστε φίλτρο όξυνσης. Περισσότερα pixel δίνουν στη μηχανή OCR καλύτερα στοιχεία. |

## Επιπλέον: Οπτική Επαλήθευση του Deskew

Αν θέλετε να δείτε το πριν‑και‑μετά πλάι-πλάι, προσθέστε ένα γρήγορο απόσπασμα Matplotlib:

```python
import matplotlib.pyplot as plt

def show_comparison(original, processed):
    fig, axs = plt.subplots(1, 2, figsize=(10, 4))
    axs[0].imshow(original.to_numpy(), cmap='gray')
    axs[0].set_title('Original')
    axs[0].axis('off')

    axs[1].imshow(processed.to_numpy(), cmap='gray')
    axs[1].set_title('Deskewed & Binarized')
    axs[1].axis('off')
    plt.show()

show_comparison(raw_image, preprocessed)
```
