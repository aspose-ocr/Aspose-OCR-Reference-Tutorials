## Receipt Parsing OCR – Βήμα 1: Εγκατάσταση και Εισαγωγή Απαιτούμενων Modules

Πρώτα απ' όλα, ας ετοιμάσουμε το περιβάλλον Python. Θα εισάγουμε το `json` για σειριοποίηση και τις κλάσεις ειδικές για OCR.

```python
# Step 1: Import required modules
import json                     # Built‑in JSON handling
from ocr_library import OcrEngine, Image   # Replace with your actual library
```

*Γιατί είναι σημαντικό*: Η εισαγωγή στην αρχή διατηρεί το script καθαρό και εξασφαλίζει ότι η μηχανή OCR είναι διαθέσιμη σε όλη τη διάρκεια. Αν ξεχάσετε να εισάγετε το `json`, η κλήση `json.dumps` αργότερα θα προκαλέσει `NameError`.

**Pro tip**: Αν η βιβλιοθήκη OCR σας βρίσκεται κάτω από διαφορετικό namespace (π.χ., `easyocr` ή `pytesseract`), προσαρμόστε τη γραμμή εισαγωγής αναλόγως. Το υπόλοιπο του tutorial παραμένει το ίδιο.

## Πώς να Εξάγετε Δεδομένα Απόδειξης – Βήμα 2: Δημιουργία του Αντικειμένου OCR Engine

Τώρα ενεργοποιούμε τη μηχανή. Σκεφτείτε το `OcrEngine()` ως τον εγκέφαλο που θα διαβάσει την απόδειξη.

```python
# Step 2: Create an OCR engine instance
engine = OcrEngine()
```

Τα περισσότερα σύγχρονα OCR SDK επιτρέπουν τη διαμόρφωση πακέτων γλώσσας ή λειτουργιών ανίχνευσης εδώ. Για μια τυπική απόδειξη θα θέλετε Αγγλικά (ή τη γλώσσα της απόδειξης) και μια λειτουργία “structured” αν είναι διαθέσιμη.

> **Σημείωση**: Αν η βιβλιοθήκη σας απαιτεί κλειδί άδειας, περάστε το ως όρισμα: `engine = OcrEngine(api_key="YOUR_KEY")`.

## Φόρτωση Εικόνας OCR – Βήμα 3: Σημειώστε τη Μηχανή στην Απόδειξή Σας

Με τη μηχανή έτοιμη, πρέπει να της δώσουμε το αρχείο εικόνας. Εδώ έρχεται σε παιχνίδι το **load image OCR**.

```python
# Step 3: Load the image to be processed
engine.set_image(Image.from_file("YOUR_DIRECTORY/receipt.png"))
```

*Τι συμβαίνει*: Το `Image.from_file` διαβάζει το PNG (ή JPG, TIFF, κλπ.) και το τυλίγει σε μορφή που καταλαβαίνει η μηχανή OCR. Αν η απόδειξή σας είναι PDF, μετατρέψτε πρώτα την πρώτη σελίδα σε εικόνα—πολλές βιβλιοθήκες παρέχουν έναν βοηθό `pdf2image`.

**Edge case**: Οι αποδείξεις που σαρώθηκαν ανάποδα θα είναι ακόμα αναγνώσιμες αν τις περιστρέψετε:

```python
# Optional: auto‑rotate if needed
image = Image.from_file("YOUR_DIRECTORY/receipt.png")
image = image.rotate(180) if image.is_upside_down() else image
engine.set_image(image)
```

## Πώς να OCR Απόδειξη – Βήμα 4: Εκτέλεση Δομημένης Αναγνώρισης Κειμένου

Τώρα συμβαίνει η μαγεία. Ζητάμε από τη μηχανή να εκτελέσει μια δομημένη σάρωση, η οποία προσπαθεί να ομαδοποιήσει τα στοιχεία, τα σύνολα και τις ημερομηνίες.

```python
# Step 4: Perform structured text recognition
result = engine.recognize_structured()
```

Αν η βιβλιοθήκη OCR σας προσφέρει μόνο μια απλή μέθοδο `recognize()`, μπορείτε ακόμη να εξάγετε τα πεδία χειροκίνητα, αλλά το `recognize_structured()` συχνά επιστρέφει ένα dictionary με κλειδιά όπως `items`, `total` και `date`.

## Python OCR Example – Βήμα 5: Μετατροπή του Αποτελέσματος σε JSON

Το ακατέργαστο αντικείμενο αποτελέσματος είναι συνήθως μια προσαρμοσμένη κλάση. Η μετατροπή του σε απλό dict Python το καθιστά εύκολο για σειριοποίηση.

```python
# Step 5: Convert the recognition result to a nicely formatted JSON string
json_str = json.dumps(result.to_dict(), ensure_ascii=False, indent=2)
```

*Γιατί `ensure_ascii=False`?* Οι αποδείξεις μπορεί να περιέχουν σύμβολα νομισμάτων (€, £) ή χαρακτήρες με τόνους. Αυτή η σημαία τα διατηρεί αντί να τα μετατρέπει σε `\u00e9`.

## Πώς να Εξάγετε Απόδειξη – Βήμα 6: Έξοδος της Αναπαράστασης JSON

Τέλος, εκτυπώνουμε (ή γράφουμε) το JSON ώστε να το διοχετεύσετε σε μια βάση δεδομένων, ένα λογιστικό φύλλο ή ένα API.

```python
# Step 6: Output the JSON representation of the recognized data
print(json_str)
```

**Αναμενόμενη έξοδος** (μορφοποιημένη για ευανάγνωστη εμφάνιση):

```json
{
  "merchant": "Coffee House",
  "date": "2024-03-15",
  "items": [
    {"name": "Latte", "quantity": 1, "price": "3.50"},
    {"name": "Bagel", "quantity": 2, "price": "4.00"}
  ],
  "subtotal": "7.50",
  "tax": "0.60",
  "total": "8.10"
}
```

Αν δείτε ένα κενό dict ή λείπουν πεδία, ελέγξτε ξανά ότι η εικόνα της απόδειξης είναι καθαρή και ότι η μηχανή OCR είναι ρυθμισμένη στη σωστή γλώσσα.

## Pro Tips & Συνηθισμένα Πιθανά Προβλήματα (Τμήμα Bonus)

| Πρόβλημα | Γιατί συμβαίνει | Γρήγορη Διόρθωση |
|----------|----------------|-------------------|
| **Θολή ή χαμηλής αντίθεσης εικόνα** | Το OCR δυσκολεύεται με τον θόρυβο | Προεπεξεργασία με OpenCV: `cv2.threshold` ή `cv2.bilateralFilter` |
| **Περιστρεφόμενη απόδειξη** | Η μηχανή υποθέτει κείμενο σε όρθια θέση | Ανίχνευση προσανατολισμού με `engine.detect_orientation()` ή περιστροφή χειροκίνητα (δείτε το Βήμα 3) |
| **Μη λατινικοί χαρακτήρες** | Λάθος πακέτο γλώσσας | Αρχικοποιήστε τη μηχανή με `language="spa"` για Ισπανικά κ.λπ. |
| **Μεγάλες αποδείξεις προκαλούν σφάλμα μνήμης** | Ολόκληρη η εικόνα φορτώνεται ταυτόχρονα | Κόψτε στην περιοχή ενδιαφέροντος: `engine.set_image(image.crop((x, y, w, h)))` |
