# Πώς να Εκτελέσετε OCR σε Python – Εξαγωγή Κειμένου Γρήγορα

Έχετε αναρωτηθεί ποτέ **πώς να εκτελέσετε OCR** σε μια σαρωμένη απόδειξη ή ένα θολό PDF; Δεν είστε μόνοι. Σε πολλά έργα η ανάγκη να **εξάγετε κείμενο από εικόνα** εμφανίζεται νωρίτερα παρά αργότερα, και η συνηθισμένη προσέγγιση «πληκτρολογήστε τα πάντα με το χέρι» δεν είναι κλιμακώσιμη.  

Σε αυτό το tutorial θα δείτε ένα πλήρες, έτοιμο‑για‑εκτέλεση παράδειγμα που δείχνει **πώς να χρησιμοποιήσετε OCR** για να διαβάσετε κείμενο από σάρωση, να εξάγετε δεδομένα από ένα τιμολόγιο, και ακόμη να διαχειριστείτε αυτόματη διόρθωση κλίσης — όλα με λίγες μόνο γραμμές κώδικα Python.

## Τι Θα Μάθετε

* Οι ακριβείς εξαρτήσεις και εισαγωγές που απαιτούνται.
* Πώς να δημιουργήσετε και να διαμορφώσετε ένα αντικείμενο `OcrEngine`.
* Τρόποι για **εξαγωγή κειμένου από εικόνα**, **ανάγνωση κειμένου από σάρωση**, και **εξαγωγή κειμένου από τιμολόγιο** χρησιμοποιώντας την ίδια μηχανή.
* Συνηθισμένα προβλήματα (λάθος γλώσσα, ελλιπή αρχεία, μεγάλες εικόνες) και πώς να τα αποφύγετε.
* Αναμενόμενο αποτέλεσμα ώστε να μπορείτε να επαληθεύσετε ότι το OCR πέτυχε.

Δεν χρειάζονται εξωτερικοί σύνδεσμοι τεκμηρίωσης — όλα είναι αυτόνομα, ώστε να μπορείτε να αντιγράψετε‑και‑επικολλήσετε τον κώδικα και να δείτε τα αποτελέσματα αμέσως.

## Προαπαιτούμενα

Πριν προχωρήσουμε, βεβαιωθείτε ότι έχετε:

* Εγκατεστημένο Python 3.8+ (το πακέτο `ocr` λειτουργεί με οποιαδήποτε πρόσφατη έκδοση).
* Διαθέσιμη τη βιβλιοθήκη `ocr` (`pip install ocr‑engine` – αντικαταστήστε με το πραγματικό όνομα του πακέτου αν είναι διαφορετικό).
* Ένα αρχείο εικόνας που θέλετε να επεξεργαστείτε – για την επίδειξη θα χρησιμοποιήσουμε το `invoice.png` που βρίσκεται σε φάκελο με όνομα `YOUR_DIRECTORY`.

Αυτό είναι όλο. Αν έχετε ήδη αυτά, είστε έτοιμοι να ξεκινήσετε.

## Βήμα 1: Εγκατάσταση και Εισαγωγή του OCR Module

Πρώτα απ' όλα: χρειαζόμαστε τη βιβλιοθήκη OCR. Αν δεν την έχετε εγκαταστήσει ακόμη, εκτελέστε την παρακάτω εντολή στο τερματικό σας:

```bash
pip install ocr-engine
```

Τώρα εισάγουμε τη βιβλιοθήκη στο script μας.

```python
# Step 1: Import the OCR module
import ocr
```

> **Συμβουλή:** Διατηρήστε το εικονικό σας περιβάλλον καθαρό· αποτρέπει συγκρούσεις εκδόσεων όταν προσθέτετε αργότερα άλλα πακέτα επεξεργασίας εικόνας.

## Βήμα 2: Δημιουργία και Διαμόρφωση του OCR Engine

Η δημιουργία του engine είναι τόσο απλή όσο η κλήση του κατασκευαστή του, αλλά η πραγματική δύναμη βρίσκεται στη σωστή διαμόρφωση. Θα ορίσουμε τη γλώσσα στα Αγγλικά και θα ενεργοποιήσουμε την αυτόματη διόρθωση κλίσης, κάτι που είναι απαραίτητο όταν δουλεύετε με σαρωμένα τιμολόγια που δεν είναι τέλεια ευθυγραμμισμένα.

```python
# Step 2: Create an OCR engine instance
engine = ocr.OcrEngine()

# Step 3: Configure the engine – set language and enable automatic skew correction
engine.language = ocr.Language.English   # any supported language, e.g., Spanish, French
engine.auto_skew = True                 # fixes tilted scans automatically
```

Γιατί να ενεργοποιήσουμε το `auto_skew`; Πολλοί σαρωτές παράγουν εικόνες που είναι μερικές μοίρες εκτός. Χωρίς διόρθωση, η μηχανή μπορεί να χάσει χαρακτήρες, μετατρέποντας ένα πλήρως αναγνώσιμο τιμολόγιο σε ακατανόητο κείμενο.

## Βήμα 3: Εκτέλεση OCR στην Εικόνα-Στόχο σας

Τώρα τροφοδοτούμε το αρχείο εικόνας στη μηχανή. Η μέθοδος `recognize_image` επιστρέφει ένα αντικείμενο που περιέχει το ακατέργαστο κείμενο καθώς και βαθμολογίες εμπιστοσύνης (αν η βιβλιοθήκη τις παρέχει).

```python
# Step 4: Perform OCR on the target image
ocr_result = engine.recognize_image("YOUR_DIRECTORY/invoice.png")
```

Αν εργάζεστε με σενάριο **ανάγνωση κειμένου από σάρωση**, απλώς αντικαταστήστε τη διαδρομή με το σαρωμένο PDF που έχει μετατραπεί σε PNG ή JPEG. Η ίδια κλήση λειτουργεί για οποιαδήποτε μορφή εικόνας υποστηρίζει η υποκείμενη βιβλιοθήκη.

## Βήμα 4: Επιθεώρηση και Χρήση του Εξαγόμενου Κειμένου

Ας εκτυπώσουμε το ακατέργαστο αποτέλεσμα του OCR. Σε μια πραγματική γραμμή επεξεργασίας τιμολογίων πιθανότατα θα αναλύσετε αυτή τη συμβολοσειρά, θα εξάγετε γραμμές, σύνολα και ημερομηνίες, αλλά προς το παρόν μια γρήγορη ματιά θα επιβεβαιώσει ότι το OCR πέτυχε.

```python
# Step 5: Display the raw extracted text
print("Raw OCR text:\n", ocr_result.text)
```

**Αναμενόμενο αποτέλεσμα (συνοπτικό):**

```

 Widget B      5     12.00   60.00

 Subtotal:                     110.00
```

Αν το αποτέλεσμα φαίνεται ακατάληπτο, ελέγξτε ξανά ότι η εικόνα είναι καθαρή και ότι το `engine.language` ταιριάζει με τη γλώσσα του εγγράφου.

## Βήμα 5: Διαχείριση Συνηθισμένων Ακραίων Περιπτώσεων

### Μεγάλες Εικόνες

Η επεξεργασία μιας σάρωσης 5000 × 5000 pixel μπορεί να καταναλώνει πολύ μνήμη. Ένας γρήγορος τρόπος για να το μετριάσετε είναι να μειώσετε την ανάλυση της εικόνας πριν τη στείλετε στη μηχανή:

```python
from PIL import Image

def load_and_resize(path, max_dim=2000):
    img = Image.open(path)
    img.thumbnail((max_dim, max_dim), Image.ANTIALIAS)
    return img

scaled_image = load_and_resize("YOUR_DIRECTORY/invoice.png")
ocr_result = engine.recognize_image(scaled_image)
```

### Πολλαπλές Γλώσσες

Αν χρειάζεται να **εξάγετε κείμενο από εικόνα** που περιέχει τόσο Αγγλικά όσο και Ισπανικά, μπορείτε να ορίσετε μια λίστα γλωσσών:

```python
engine.language = [ocr.Language.English, ocr.Language.Spanish]
```

Η μηχανή θα προσπαθήσει να αναγνωρίσει χαρακτήρες και από τα δύο σύνολα.

### Διαχείριση Σφαλμάτων

Μην υποθέτετε ποτέ ότι το αρχείο υπάρχει. Τυλίξτε την κλήση σε μπλοκ try‑except για να δώσετε ένα φιλικό μήνυμα:

```python
try:
    ocr_result = engine.recognize_image("YOUR_DIRECTORY/invoice.png")
    print("OCR succeeded!")
except FileNotFoundError:
    print("Error: The image file was not found. Check the path and try again.")
except ocr.OcrError as e:
    print(f"OCR failed with error: {e}")
```

## Πλήρες, Εκτελέσιμο Παράδειγμα

Συνδυάζοντας όλα, εδώ είναι ένα ενιαίο script που μπορείτε να εκτελέσετε από τη γραμμή εντολών. Καλύπτει εγκατάσταση, διαμόρφωση, διαχείριση σφαλμάτων, και ένα απλό βήμα μεταεπεξεργασίας που γράφει το εξαγόμενο κείμενο σε αρχείο με όνομα `output.txt`.

```python
# full_ocr_demo.py
# -------------------------------------------------
# Demonstrates how to perform OCR, extract text from image,
# read text from scan, and extract text from invoice.
# -------------------------------------------------

import ocr
from pathlib import Path

def main(image_path: str, output_path: str = "output.txt"):
    # Create and configure the engine
    engine = ocr.OcrEngine()
    engine.language = ocr.Language.English
    engine.auto_skew = True

    # Verify the image exists
    if not Path(image_path).is_file():
        print(f"❌  Image not found: {image_path}")
        return

    # Run OCR
    try:
        result = engine.recognize_image(image_path)
    except ocr.OcrError as err:
        print(f"❌  OCR failed: {err}")
        return

    # Output the raw text
    print("✅  OCR completed. Raw text:")
    print(result.text)

    # Save to a text file for later processing
    Path(output_path).write_text(result.text, encoding="utf-8")
    print(f"💾  Text saved to {output_path}")

if __name__ == "__main__":
    # Replace with your actual file location
    main("YOUR_DIRECTORY/invoice.png")
```

Η εκτέλεση του `python full_ocr_demo.py` θα εκτυπώσει το εξαγόμενο κείμενο στην κονσόλα και θα το αποθηκεύσει στο `output.txt`. Από εκεί μπορείτε να εφαρμόσετε κανονικές εκφράσεις, CSV writers, ή οποιαδήποτε άλλη λογική χρειάζεστε για αυτοματοποίηση **εξαγωγής κειμένου από τιμολόγιο**.
