# Πώς να εμφανίσετε πληροφορίες – Λίστα τοπικών μοντέλων, Όνομα, Μέγεθος, Τελευταία χρήση

Έχετε αναρωτηθεί ποτέ **πώς να εμφανίσετε πληροφορίες** από την εγκατάσταση του AsposeAI χωρίς να σκάβετε στα αρχεία καταγραφής ή στο UI; Δεν είστε οι μόνοι. Σε πολλά data‑science pipelines το πρώτο που χρειάζεστε είναι μια γρήγορη ματιά στο ποια μοντέλα βρίσκονται στον υπολογιστή σας, πώς ονομάζονται, πόσο μεγάλα είναι και πότε τα χρησιμοποιήσατε τελευταία φορά.

Ακριβώς αυτό θα καλύψουμε: ένα σύντομο, ολοκληρωμένο απόσπασμα Python που **καταγράφει τα τοπικά μοντέλα**, στη συνέχεια **εμφανίζει το όνομα του μοντέλου**, **δείχνει το μέγεθος του μοντέλου** και **εμφανίζει τις χρονικές σήμανσεις τελευταίας χρήσης**. Χωρίς εξωτερικές βιβλιοθήκες, χωρίς κρυφή μαγεία — μόνο ο πελάτης AsposeAI που ήδη έχετε.

Στο τέλος αυτού του tutorial θα μπορείτε να ενσωματώσετε τον κώδικα σε οποιοδήποτε script, να τον εκτελέσετε και άμεσα να λάβετε έναν τακτοποιημένο πίνακα με τα τοπικά αποθηκευμένα μοντέλα σας. Είναι ιδανικό για έλεγχο υγείας του περιβάλλοντος, δημιουργία dashboards ή απλώς για να ικανοποιήσετε την περιέργειά σας για το τι κρύβεται στον δίσκο.

## Προαπαιτούμενα

- Python 3.8 ή νεότερο (το παράδειγμα χρησιμοποιεί f‑strings, οπότε απαιτείται 3.6+)
- Πακέτο `asposeai` εγκατεστημένο (`pip install asposeai`)
- Ένα ενεργό άδεια AsposeAI ή κλειδί δοκιμής (ο πελάτης θα αντλήσει τα διαπιστευτήρια από μεταβλητές περιβάλλοντος ή αρχείο ρυθμίσεων)

Αν έχετε ήδη ελέγξει αυτά τα σημεία, υπέροχα — ας ξεκινήσουμε.

## Βήμα 1: Πώς να εμφανίσετε πληροφορίες – Αρχικοποίηση του AsposeAI Client

Πριν μπορέσουμε να **καταγράψουμε τα τοπικά μοντέλα**, χρειαζόμαστε ένα αντικείμενο client που επικοινωνεί με το runtime του AsposeAI. Αυτό το βήμα είναι το θεμέλιο· χωρίς αυτό, ο υπόλοιπος κώδικας θα προκαλούσε `NameError`.

```python
# Step 1: Create an instance of the AsposeAI client
# The client automatically reads credentials from the environment
aspose_ai = AsposeAI()
```

*Γιατί είναι σημαντικό*: Η αρχικοποίηση του client δημιουργεί μια συνεδρία με τη τοπική μηχανή inference, φορτώνοντας τυχόν απαραίτητες native βιβλιοθήκες. Επίσης, επαληθεύει ότι η άδειά σας είναι ενεργή, αποτρέποντας ασαφείς σφάλματα αργότερα όταν προσπαθήσετε να ερωτήσετε μοντέλα.

> **Pro tip**: Αν τρέχετε αυτό σε CI server, ορίστε τη μεταβλητή `ASPOSEAI_LICENSE` στο περιβάλλον ώστε ο client να ξεκινά χωρίς διαδραστικά prompts.

## Βήμα 2: Λίστα τοπικών μοντέλων – Ανάκτηση διαθέσιμων μοντέλων

Τώρα που ο client είναι έτοιμος, μπορούμε να **καταγράψουμε τα τοπικά μοντέλα**. Η μέθοδος `list_local()` επιστρέφει μια συλλογή αντικειμένων, το καθένα εκθέτει ιδιότητες όπως `name`, `size_mb` και `last_used`.

```python
# Step 2: Retrieve the list of locally available models
local_models = aspose_ai.list_local()
```

*Τι συμβαίνει στο παρασκήνιο*: Η `list_local()` σαρώει τον φάκελο όπου το AsposeAI αποθηκεύει τα αρχεία μοντέλων (`~/.asposeai/models` εξ ορισμού) και δημιουργεί ελαφριά αντικείμενα μεταδεδομένων. Είναι γρήγορη επειδή δεν φορτώνει τα βάρη του μοντέλου — διαβάζει μόνο ένα μικρό JSON manifest.

Αν ποτέ αναρωτηθείτε αν ένα συγκεκριμένο μοντέλο είναι ήδη στην cache, αυτή η κλήση είναι ο πιο γρήγορος τρόπος να το επιβεβαιώσετε.

## Βήμα 3: Εμφάνιση ονόματος μοντέλου, εμφάνιση μεγέθους μοντέλου και εμφάνιση τελευταίας χρήσης

Με τα μοντέλα στα χέρια, τέλος **εμφανίζουμε πληροφορίες** επαναλαμβάνοντας τη συλλογή και εκτυπώνοντας κάθε χαρακτηριστικό. Εδώ **εμφανίζουμε το όνομα του μοντέλου**, **δείχνουμε το μέγεθος του μοντέλου** και **εμφανίζουμε την τελευταία χρήση** όλα σε μία καθαρή γραμμή.

```python
# Step 3: Print each model's name, size (in MB), and last‑used timestamp
print("Available local models:")
print("-" * 50)
for model_info in local_models:
    # Using an f‑string to format the output cleanly
    print(f"{model_info.name} – {model_info.size_mb} MB – last used {model_info.last_used}")
```

**Αναμενόμενη έξοδος** (οι χρονικές σήμανσεις σας θα διαφέρουν):

```
Available local models:
--------------------------------------------------
gpt‑tiny‑en – 120 MB – last used 2025-11-02 14:23:11
bert‑base‑fr – 420 MB – last used 2025-10-18 09:07:45
whisper‑large‑audio – 1580 MB – last used 2025-09-30 22:41:02
```

*Γιατί το μορφοποιούμε έτσι*: Η παύλα (`–`) χωρίζει τα πεδία για ευανάγνωστη παρουσίαση, και η γραμμή κεφαλίδας κάνει την έξοδο της κονσόλας πιο εύκολη στην ανάγνωση. Αν χρειαστείτε μορφή μηχανής αργότερα (CSV, JSON), μπορείτε εύκολα να αντικαταστήσετε την κλήση `print` με `writer.writerow` ή `json.dump`.

### Διαχείριση περιπτώσεων άκρων

- **Δεν υπάρχουν μοντέλα στην cache** – Η `list_local()` επιστρέφει κενή λίστα. Ο βρόχος θα παραλείψει απλώς, αφήνοντάς σας μόνο την κεφαλίδα. Μπορεί να θέλετε να προσθέσετε έναν έλεγχο:

  ```python
  if not local_models:
      print("No local models found. Use aspose_ai.download(...) to fetch one.")
  ```

- **Λείπουν ιδιότητες** – Σε σπάνιες περιπτώσεις το manifest μπορεί να είναι κατεστραμμένο. Η πρόσβαση στο `model_info.last_used` μπορεί να προκαλέσει `AttributeError`. Τυλίξτε την εκτύπωση σε `try/except` αν προβλέπετε τέτοια προβλήματα.

- **Μεγάλοι φάκελοι μοντέλων** – Αν έχετε εκατοντάδες μοντέλα, σκεφτείτε να σελίδωση την έξοδο ή να γράψετε σε αρχείο αντί για εκτύπωση στην κονσόλα.

## Πλήρες λειτουργικό σενάριο

Συνδυάζοντας όλα τα παραπάνω, εδώ είναι το πλήρες, έτοιμο‑για‑εκτέλεση script:

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-

"""
How to display info from AsposeAI:
List local models and show each model's name, size, and last‑used timestamp.
"""

from asposeai import AsposeAI

def main():
    # Initialize client
    aspose_ai = AsposeAI()

    # Retrieve local models
    local_models = aspose_ai.list_local()

    # Header
    print("Available local models:")
    print("-" * 50)

    if not local_models:
        print("No local models found. Use aspose_ai.download(...) to fetch one.")
        return

    # Display each model's details
    for model_info in local_models:
        try:
            print(f"{model_info.name} – {model_info.size_mb} MB – last used {model_info.last_used}")
        except AttributeError:
            # Fallback if any attribute is missing
            print(f"{model_info.name} – info incomplete")

if __name__ == "__main__":
    main()
```

Αποθηκεύστε το ως `list_models.py`, κάντε το εκτελέσιμο (`chmod +x list_models.py`) και τρέξτε:

```bash
./list_models.py
```

Θα δείτε τη τακτοποιημένη λίστα που παρουσιάστηκε νωρίτερα.
