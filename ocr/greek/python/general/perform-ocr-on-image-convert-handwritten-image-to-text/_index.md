# Εκτέλεση OCR σε Εικόνα – Πλήρης Ανάκτηση Χειρόγραφου Κειμένου

Έχετε χρειαστεί ποτέ να **εκτελέσετε OCR σε εικόνα** αρχεία αλλά δεν ήσασταν σίγουροι αν η μηχανή μπορεί να διαβάσει ακατάστατη χειρόγραφη γραφή; Δεν είστε μόνοι. Σε πολλές πραγματικές εφαρμογές—σκεφτείτε σαρωτές αποδείξεων εξόδων ή βοηθήματα λήψης σημειώσεων—θα συναντήσετε φωτογραφίες με χαρές που πρέπει να μετατραπούν σε απλό κείμενο.  

Σε αυτόν τον οδηγό θα σας δείξουμε πώς να **μετατρέψετε εικόνες με χειρόγραφο** αρχεία, **εξάγετε κείμενο από jpg**, και ακόμη **αναγνωρίσετε κείμενο από ροές φωτογραφιών** χρησιμοποιώντας μια μικρή βιβλιοθήκη σε στυλ Python που ονομάζεται `ocr`. Στο τέλος θα έχετε ένα έτοιμο‑για‑εκτέλεση script που εξάγει κάθε λέξη από μια χειρόγραφη σημείωση, ανεξάρτητα από το πόσο τρεμάτη ήταν η πένα.

## Τι Θα Χρειαστεί

- Python 3.8+ (ο κώδικας λειτουργεί σε οποιονδήποτε πρόσφατο διερμηνέα)
- Το υποθετικό πακέτο `ocr` – εγκαταστήστε το με `pip install ocr-lib` (αντικαταστήστε με το πραγματικό όνομα του πακέτου που χρησιμοποιείτε)
- Μια καθαρή φωτογραφία μιας χειρόγραφης σημείωσης αποθηκευμένη ως `note.jpg` (ή οποιαδήποτε άλλη μορφή εικόνας)
- Μια μέτρια δόση περιέργειας—δεν απαιτείται προχωρημένο υπόβαθρο ML

Αυτό είναι όλο. Χωρίς εξωτερικές υπηρεσίες, χωρίς κλειδιά API, μόνο μια τοπική μηχανή που μπορεί να **εκτελέσει OCR σε εικόνα** δεδομένα.

## Υλοποίηση Βήμα‑βήμα

Παρακάτω χωρίζουμε τη διαδικασία σε μικρά κομμάτια. Κάθε επικεφαλίδα περιλαμβάνει μια λέξη‑κλειδί ώστε να μπορείτε να περιηγηθείτε γρήγορα, και κάθε μπλοκ εξηγεί **γιατί** κάνουμε ό,τι κάνουμε—όχι μόνο **τι**.

### Βήμα 1: Εγκατάσταση και Επαλήθευση της Βιβλιοθήκης OCR

Πριν μπορέσετε να **εκτελέσετε OCR σε εικόνα** αρχεία, η βιβλιοθήκη πρέπει να είναι παρούσα στο περιβάλλον σας. Ανοίξτε ένα τερματικό και εκτελέστε:

```bash
pip install ocr-lib
```

> **Συμβουλή:** Εάν εργάζεστε σε εικονικό περιβάλλον (συνιστάται έντονα), ενεργοποιήστε το πρώτα. Αυτό διατηρεί τις εξαρτήσεις σας οργανωμένες και αποφεύγει συγκρούσεις εκδόσεων.

Μετά την εγκατάσταση, ας βεβαιωθούμε ότι η Python μπορεί να εισάγει το πακέτο:

```python
try:
    import ocr
    print("OCR library loaded successfully.")
except ImportError as e:
    raise SystemExit("Failed to import ocr library. Did you run pip install?") from e
```

Αν δείτε το μήνυμα επιτυχίας, είστε έτοιμοι να **μετατρέψετε εικόνα με χειρόγραφο** δεδομένα.

### Βήμα 2: Δημιουργία Αντικειμένου Μηχανής και Επιλογή Λειτουργίας Χειρόγραφου

Οι περισσότερες μηχανές OCR προεπιλογή είναι η αναγνώριση τυπωμένου κειμένου. Δεδομένου ότι θέλουμε να **εξάγουμε χειρόγραφο κείμενο**, πρέπει να αλλάξουμε τη λειτουργία ρητά. Αυτό το βήμα είναι κρίσιμο επειδή η χειρόγραφη γραφή συχνά απαιτεί διαφορετική προεπεξεργασία (όπως εξομάλυνση γραμμών).

```python
# Step 2: Initialize the OCR engine
engine = ocr.OcrEngine()

# Tell the engine we’re dealing with handwriting
engine.set_recognition_mode(ocr.RecognitionMode.HANDWRITTEN)

print("Engine configured for handwritten recognition.")
```

*Γιατί είναι σημαντικό:* Οι χειρόγραφοι χαρακτήρες μπορούν να διαφέρουν πολύ σε μέγεθος και κλίση. Ορίζοντας το `RecognitionMode.HANDWRITTEN`, η μηχανή εφαρμόζει ένα μοντέλο εκπαιδευμένο σε δείγματα καλλιγραφίας, βελτιώνοντας δραματικά την ακρίβεια.

### Βήμα 3: Φόρτωση της Φωτογραφίας που Θέλετε να Αναλύσετε

Τώρα πραγματικά **εκτελούμε OCR σε εικόνα** περιεχόμενο. Η μέθοδος `load_image` δέχεται μια διαδρομή ή ένα αντικείμενο‑αρχείο. Για επίδειξη, θα φορτώσουμε ένα JPEG, αλλά η ίδια κλήση λειτουργεί για PNG, BMP ή ακόμη και σελίδες PDF.

```python
# Step 3: Load the image file (replace the path with yours)
image_path = "YOUR_DIRECTORY/note.jpg"
engine.load_image(image_path)

print(f"Image '{image_path}' loaded into the OCR engine.")
```

Αν η εικόνα σας βρίσκεται σε cloud bucket, απλώς κατεβάστε την πρώτα ή περάστε ένα ρεύμα `BytesIO`—το `ocr` είναι αρκετά ευέλικτο για να διαχειριστεί και τα δύο.

### Βήμα 4: Εκτέλεση της Διαδικασίας Αναγνώρισης

Με τη μηχανή προετοιμασμένη και την εικόνα στη μνήμη, τελικά **εκτελούμε OCR σε εικόνα** και ανακτούμε το ακατέργαστο κείμενο.

```python
# Step 4: Execute recognition
extracted_text = engine.recognize()

print("=== Extracted Text ===")
print(extracted_text)
```

Η κλήση `recognize()` επιστρέφει μια απλή συμβολοσειρά Unicode. Για τις περισσότερες περιπτώσεις χρήσης μπορείτε να τη γράψετε απευθείας σε αρχείο `.txt`, να τη δώσετε σε pipeline φυσικής γλώσσας, ή να την εμφανίσετε σε GUI.

### Βήμα 5: Προαιρετικό – Καθαρισμός ή Μεταεπεξεργασία του Αποτελέσματος

Η χειρόγραφη OCR δεν είναι τέλεια· συχνά θα δείτε ανεπιθύμητες αλλαγές γραμμής ή λανθασμένους χαρακτήρες. Ένα γρήγορο βήμα καθαρισμού μπορεί να βελτιώσει τα αποτελέσματα σε επόμενα στάδια.

```python
# Step 5: Simple post‑processing (optional)
def tidy(text):
    # Collapse multiple spaces, strip leading/trailing whitespace
    return "\n".join(line.strip() for line in text.splitlines() if line.strip())

clean_text = tidy(extracted_text)

print("\n=== Cleaned Text ===")
print(clean_text)
```

Μη διστάσετε να ενσωματώσετε ελεγκτές ορθογραφίας, μοντέλα γλώσσας ή προσαρμοσμένες εκφράσεις regex ανάλογα με το πεδίο σας.

### Πλήρες Script – Έτοιμο για Αντιγραφή & Επικόλληση

Συνδυάζοντας όλα, εδώ είναι το πλήρες, εκτελέσιμο πρόγραμμα που **εξάγει χειρόγραφο κείμενο** από ένα JPEG και εκτυπώνει ένα τακτοποιημένο αποτέλεσμα.

```python
# -*- coding: utf-8 -*-
"""
Complete example: Perform OCR on image to extract handwritten text.
"""

# -------------------------------------------------
# Step 1: Import the OCR library and create an engine
# -------------------------------------------------
import ocr

engine = ocr.OcrEngine()

# -------------------------------------------------
# Step 2: Set the engine to recognize handwritten text
# -------------------------------------------------
engine.set_recognition_mode(ocr.RecognitionMode.HANDWRITTEN)

# -------------------------------------------------
# Step 3: Load the image you want to analyze
# -------------------------------------------------
image_path = "YOUR_DIRECTORY/note.jpg"   # <-- change this to your file
engine.load_image(image_path)

# -------------------------------------------------
# Step 4: Perform recognition and output the extracted text
# -------------------------------------------------
raw_text = engine.recognize()
print("=== Raw OCR Output ===")
print(raw_text)

# -------------------------------------------------
# Step 5: Optional cleanup for nicer display
# -------------------------------------------------
def tidy(text):
    return "\n".join(line.strip() for line in text.splitlines() if line.strip())

clean_text = tidy(raw_text)
print("\n=== Cleaned OCR Output ===")
print(clean_text)
```

**Αναμενόμενο αποτέλεσμα** (το πραγματικό κείμενό σας θα διαφέρει, φυσικά):

```
=== Raw OCR Output ===
Meeting notes:
- Discuss project timeline
- Assign tasks to John, Ana
- Review budget Q3

=== Cleaned OCR Output ===
Meeting notes:
- Discuss project timeline
- Assign tasks to John, Ana
- Review budget Q3
```

Αν δείτε ακαταλαβίστικο κείμενο, ελέγξτε ξανά την ποιότητα της εικόνας (καλό φωτισμό, ελάχιστη θολότητα) και βεβαιωθείτε ότι βρίσκεστε σε λειτουργία `HANDWRITTEN`. Αυτοί οι δύο παράγοντες εξηγούν τα περισσότερα σφάλματα αναγνώρισης.

## Συχνές Ερωτήσεις (FAQ)

| Ερώτηση | Απάντηση |
|----------|--------|
| **Μπορώ να το χρησιμοποιήσω για εξαγωγή κειμένου από PNG;** | Απολύτως. `engine.load_image("scan.png")` λειτουργεί με τον ίδιο τρόπο. |
| **Τι γίνεται αν η εικόνα μου είναι μια σελίδα PDF;** | Μετατρέψτε πρώτα τη σελίδα σε εικόνα (π.χ., με `pdf2image`) και μετά τη δώστε στη μηχανή. |
| **Είναι η βιβλιοθήκη ασφαλής για νήματα (thread‑safe);** | Ναι, μπορείτε να δημιουργήσετε ξεχωριστά αντικείμενα `OcrEngine` ανά νήμα. |
| **Πώς διαφέρει από το `pytesseract`;** | Το `ocr` αφαιρεί την ανάγκη για το εκτελέσιμο Tesseract και περιλαμβάνει ενσωματωμένο μοντέλο χειρόγραφου, οπότε δεν χρειάζεται να εγκαταστήσετε εξωτερικά εκτελέσιμα. |
| **Τι κάνω αν χρειάζεται να **εξάγω κείμενο από JPG** αρχεία μαζικά;** | Τυλίξτε το script σε βρόχο, ή χρησιμοποιήστε `engine.load_image` σε κάθε αρχείο και συλλέξτε τα αποτελέσματα σε λίστα ή CSV. |

## Ακραίες Περιπτώσεις & Καλές Πρακτικές

1. **Φωτογραφίες χαμηλής αντίθεσης** – Αυξήστε το αντίθεση προγραμματιστικά πριν τη φόρτωση, ή χρησιμοποιήστε `engine.apply_preprocessing('contrast', level=2)`.
2. **Μικτό τυπωμένο & χειρόγραφο** – Εκτελέστε δύο περάσματα: πρώτα με `HANDWRITTEN`, μετά με `PRINTED`, και συγχωνεύστε τα αποτελέσματα.
3. **Μεγάλες εικόνες** – Μειώστε το μέγεθος σε ~1500 px πλάτος· οι μηχανές OCR συνήθως λειτουργούν πιο γρήγορα με μικρότερα buffers χωρίς να χάνουν ακρίβεια.
4. **Χαρακτήρες Unicode** – Η βιβλιοθήκη επιστρέφει συμβολοσειρές UTF‑8, ώστε να μπορείτε να διαχειριστείτε emojis, γράμματα με τόνους ή μαθηματικά σύμβολα αμέσως.
