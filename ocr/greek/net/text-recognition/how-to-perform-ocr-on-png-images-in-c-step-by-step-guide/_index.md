# Πώς να Εκτελέσετε OCR σε Εικόνες PNG σε C# – Πλήρης Οδηγός

Έχετε χρειαστεί ποτέ να **εκτελέσετε OCR** σε ένα στιγμιότυπο οθόνης ή σε σαρωμένο έγγραφο αλλά δεν ήξερες από πού να ξεκινήσεις σε C#; Δεν είστε οι μόνοι. Οι προγραμματιστές ρωτούν συνεχώς: «Πώς διαβάζω κείμενο από αρχεία PNG χωρίς να τα στέλνω σε εξωτερική υπηρεσία;» Τα καλά νέα είναι ότι με το Aspose.OCR μπορείτε να **εξάγετε ρωσικό κείμενο**, **διαβάσετε κείμενο από png**, και να λάβετε μια καθαρή συμβολοσειρά με λίγες μόνο γραμμές κώδικα.

Σε αυτό το tutorial θα περάσουμε από όλα όσα χρειάζεστε: εγκατάσταση της βιβλιοθήκης, επιλογή του σωστού μοντέλου γλώσσας, εκτέλεση της αναγνώρισης και αντιμετώπιση κοινών προβλημάτων. Στο τέλος θα μπορείτε να **εξάγετε κείμενο** από οποιαδήποτε εικόνα PNG, είτε είναι Αγγλικά, Ρωσικά ή οποιαδήποτε από τις 70+ γλώσσες που υποστηρίζει το Aspose. Χωρίς περιττές πληροφορίες, μόνο ένα πρακτικό, εκτελέσιμο παράδειγμα που μπορείτε να ενσωματώσετε σε μια εφαρμογή κονσόλας αμέσως.

- Διαμόρφωση της μηχανής για **εξαγωγή ρωσικού κειμένου** χρησιμοποιώντας το μοντέλο Cyrillic.

## Βήμα 1 – Εγκατάσταση του Πακέτου Aspose.OCR

Για να ξεκινήσετε, προσθέστε τη βιβλιοθήκη Aspose.OCR στο έργο σας. Ανοίξτε ένα τερματικό στον φάκελο του έργου και εκτελέστε:

```bash
dotnet add package Aspose.OCR
```

Ή, αν προτιμάτε το UI του Visual Studio, κάντε δεξί‑κλικ στο **Dependencies → Manage NuGet Packages**, αναζητήστε το **Aspose.OCR** και κάντε κλικ στο **Install**.

> **Pro tip**: Το πακέτο είναι μόνο μερικά megabytes, και τα μοντέλα γλώσσας κατεβάζονται κατ' απαίτηση, οπότε δεν θα φορτώσετε την εφαρμογή σας με περιττά αρχεία.

---

## Βήμα 2 – Αρχικοποίηση της Μηχανής OCR (Κύρια Λέξη‑Κλειδί σε Δράση)

Η δημιουργία της μηχανής είναι απλή. Ο κατασκευαστής ενεργοποιεί αυτόματα τη *λειτουργία αυτόματης λήψης*, που σημαίνει ότι την πρώτη φορά που ζητάτε μια γλώσσα που δεν υπάρχει τοπικά, το Aspose θα την κατεβάσει για εσάς.

```csharp
using System;
using Aspose.OCR;

class Program
{
    static void Main()
    {
        // 1️⃣ Initialize the OCR engine – it will download missing models automatically.
        var ocrEngine = new OcrEngine();

        // 2️⃣ Choose the language model – Russian Cyrillic (downloads if missing).
        ocrEngine.Language = Language.RussianCyrillic;

        // 3️⃣ Run OCR on the PNG image.
        var ocrResult = ocrEngine.RecognizeImage("sample_russian.png");

        // 4️⃣ Output the recognized text.
        Console.WriteLine(ocrResult.Text);
    }
}
```

> **Γιατί είναι σημαντικό**: Χρησιμοποιώντας τη προεπιλεγμένη λειτουργία αυτόματης λήψης αποφεύγετε χειροκίνητη διαχείριση αρχείων. Αν αργότερα χρειαστεί να **διαβάσετε κείμενο από png** σε άλλη γλώσσα, απλώς αλλάξτε το `Language.RussianCyrillic` στην κατάλληλη τιμή enum.

## Βήμα 3 – Προετοιμασία της Εικόνας PNG

Βεβαιωθείτε ότι η εικόνα που θέλετε να επεξεργαστείτε είναι προσβάσιμη κατά το χρόνο εκτέλεσης. Τοποθετήστε το `sample_russian.png` στον ίδιο φάκελο με το μεταγλωττισμένο `.exe`, ή χρησιμοποιήστε απόλυτη διαδρομή αν προτιμάτε. Η εικόνα πρέπει να είναι σαφής σάρωση ή στιγμιότυπο· η ακρίβεια του OCR μειώνεται δραστικά σε θολές ή υπερβολικά συμπιεσμένες PNG.

**Κοινό σενάριο άκρης**: Αν η PNG περιέχει πολλές γλώσσες, μπορείτε να ορίσετε `ocrEngine.Language = Language.Multilingual;` ώστε η μηχανή ανιχνεύσει αυτόματα κάθε μπλοκ.

## Βήμα 5 – Προηγμένες Ρυθμίσεις για Καλύτερη Ακρίβεια

Αν και οι προεπιλεγμένες ρυθμίσεις λειτουργούν για τις περισσότερες περιπτώσεις, ίσως θελήσετε να κάνετε λεπτομερή ρύθμιση της μηχανής:

| Ρύθμιση | Τι κάνει | Πότε να τη χρησιμοποιήσετε |
|---------|----------|----------------------------|
| `ocrEngine.PreprocessOptions.Deskew = true;` | Διορθώνει ελαφριά κλίση | Σαρωμένα έγγραφα που δεν είναι τέλεια ευθυγραμμισμένα |
| `ocrEngine.PreprocessOptions.RemoveNoise = true;` | Φιλτράρει ψήγματα φόντου | PNG χαμηλής ποιότητας από κινητά τηλέφωνα |
| `ocrEngine.RecognitionOptions.CharWhitelist = "0123456789";` | Περιορίζει τους χαρακτήρες σε ψηφία | Εξαγωγή αριθμών από τιμολόγια |

Προσθέστε οποιοδήποτε από αυτά πριν καλέσετε το `RecognizeImage`:

```csharp
ocrEngine.PreprocessOptions.Deskew = true;
ocrEngine.PreprocessOptions.RemoveNoise = true;
```

## Συχνές Ερωτήσεις

**Ε: Λειτουργεί αυτό και με άλλες μορφές εικόνας όπως JPEG ή BMP;**  
Α: Απόλυτα. Η `RecognizeImage` δέχεται οποιαδήποτε μορφή υποστηρίζεται από τη βιβλιοθήκη .NET `System.Drawing`, συμπεριλαμβανομένων JPEG, BMP και TIFF.

**Ε: Τι γίνεται αν χρειαστεί να εξάγω αγγλικό κείμενο στην ίδια εκτέλεση;**  
Α: Δημιουργήστε μια δεύτερη παρουσία `OcrEngine` με `Language.English` ή αλλάξτε την ιδιότητα γλώσσας μεταξύ των κλήσεων.

**Ε: Μπορώ να τρέξω OCR σε web API χωρίς να μπλοκάρω το κύριο νήμα;**  
Α: Ναι. Τυλίξτε την κλήση αναγνώρισης σε `Task.Run` ή χρησιμοποιήστε την ασύγχρονη υπερφόρτωση `RecognizeImageAsync` (διαθέσιμη σε νεότερες εκδόσεις Aspose).

**Ε: Υπάρχει όριο στο μέγεθος της PNG;**  
Α: Η βιβλιοθήκη μπορεί να διαχειριστεί μεγάλες εικόνες, αλλά η χρήση μνήμης αυξάνεται με την ανάλυση. Αν αντιμετωπίσετε `OutOfMemoryException`, εξετάστε το ενδεχόμενο να μειώσετε την ανάλυση της εικόνας πρώτα.

## Πλήρες Παράδειγμα Εργασίας (Έτοιμο για Αντιγραφή‑Επικόλληση)

Παρακάτω βρίσκεται το πλήρες πρόγραμμα που μπορείτε να επικολλήσετε σε ένα νέο έργο κονσόλας (`dotnet new console`) και να τρέξετε αμέσως μετά την εγκατάσταση του πακέτου NuGet.

```csharp
// File: Program.cs
using System;
using Aspose.OCR;

namespace OcrDemo
{
    class Program
    {
        static void Main()
        {
            // Initialize OCR engine – auto‑download enabled.
            var ocrEngine = new OcrEngine();

            // Choose Russian Cyrillic language model.
            ocrEngine.Language = Language.RussianCyrillic;

            // Optional: improve accuracy for skewed or noisy images.
            ocrEngine.PreprocessOptions.Deskew = true;
            ocrEngine.PreprocessOptions.RemoveNoise = true;

            // Path to the PNG file you want to read.
            string imagePath = "sample_russian.png";

            // Perform OCR.
            OcrResult result = ocrEngine.RecognizeImage(imagePath);

            // Output the recognized text.
            Console.WriteLine("=== OCR Result ===");
            Console.WriteLine(result.Text);

            // Save to a text file for later use.
            System.IO.File.WriteAllText("ocr_output.txt", result.Text);
            Console.WriteLine("Result saved to ocr_output.txt");
        }
    }
}
```

**Αναμενόμενη έξοδος στην κονσόλα** (υποθέτοντας ότι το δείγμα περιέχει τη φράση «Привет, мир!»):

```
=== OCR Result ===
Привет, мир! Это пример текста на русском языке.
Result saved to ocr_output.txt
```
