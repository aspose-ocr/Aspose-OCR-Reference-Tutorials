# ocr document mode – Λειτουργία Ανίχνευσης Περιοχών στην OCR Αναγνώριση Εικόνας

## Εισαγωγή

Στη σύγχρονη ανάπτυξη .NET, το **ocr document mode** είναι η προτιμώμενη προσέγγιση όταν χρειάζεστε ακριβή έλεγχο του τρόπου ανίχνευσης κειμένου μέσα σε εικόνες. Το Aspose.OCR για .NET καθιστά εύκολη τη μετάβαση μεταξύ διαφορετικών στρατηγικών ανίχνευσης, επιτρέποντάς σας να **extract table text image** από σύνθετες διατάξεις όπως αποδείξεις, τιμολόγια ή έγγραφα πολλαπλών στηλών. Αυτό το **aspose ocr tutorial c#** θα σας καθοδηγήσει στη λειτουργία Detect Areas Mode, θα εξηγήσει πότε να χρησιμοποιήσετε κάθε λειτουργία και θα σας δείξει ένα έτοιμο για εκτέλεση δείγμα κώδικα.

## Γρήγορες Απαντήσεις
- **Τι είναι το ocr document mode?** Ένα σύνολο στρατηγικών ανίχνευσης (PHOTO, DOCUMENT, COMBINE) που λέει στο Aspose.OCR πώς να εντοπίζει περιοχές κειμένου.
- **Ποια λειτουργία είναι η καλύτερη για πίνακες;** Η λειτουργία `PHOTO` διαπρέπει στην εξαγωγή **extract table text image** και μικρών μπλοκ κειμένου.
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμαστική άδεια είναι επαρκής για δοκιμές· απαιτείται εμπορική άδεια για παραγωγή.
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6 και μεταγενέστερες.
- **Πόσο χρόνο παίρνει η εγκατάσταση;** Συνήθως κάτω από 10 λεπτά για ενσωμάτωση και εκτέλεση του δείγματος κώδικα.

## Τι είναι το ocr document mode;
`ocr document mode` αναφέρεται στη ρύθμιση που λέει στο Aspose.OCR πώς να διαχωρίζει μια εικόνα πριν από την αναγνώριση κειμένου. Οι τρεις ενσωματωμένες λειτουργίες είναι:

- **PHOTO** – Βελτιστοποιημένο για φωτογραφίες, αποδείξεις, τιμολόγια και μικρές περιοχές κειμένου (ιδανικό για **extract table text image**).
- **DOCUMENT** – Κατάλληλο για σελίδες εκτυπωμένες σε πολλαπλές στήλες και έγγραφα που περιέχουν ενσωματωμένα γραφικά.
- **COMBINE** – Συγχωνεύει τα αποτελέσματα των PHOTO και DOCUMENT για τη μεγαλύτερη κάλυψη.

## Γιατί να χρησιμοποιήσετε τη λειτουργία Detect Areas Mode;
Η επιλογή της κατάλληλης λειτουργίας ανίχνευσης μειώνει τα ψευδώς θετικά, επιταχύνει την επεξεργασία και βελτιώνει την ακρίβεια—ιδιαίτερα όταν εργάζεστε με δομημένα δεδομένα όπως πίνακες. Προσαρμόζοντας τη λειτουργία στον τύπο της εικόνας σας, μπορείτε να επιτύχετε αξιόπιστα αποτελέσματα OCR χωρίς μετα-επεξεργασία.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- **Aspose.OCR for .NET** – Κατεβάστε και εγκαταστήστε τη βιβλιοθήκη από την [Aspose.OCR for .NET documentation](https://reference.aspose.com/ocr/net/).
- **Document Directory** – Ένας φάκελος στον υπολογιστή σας που περιέχει τις εικόνες που θέλετε να επεξεργαστείτε (π.χ., `table.png`).

## Εισαγωγή Namespaces

Πρώτα, εισάγετε τα namespaces που απαιτούνται για εργασία με το Aspose.OCR στο έργο C# σας.

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

## Βήμα 1: Αρχικοποίηση Aspose.OCR

Δημιουργήστε μια παρουσία της μηχανής OCR και δείξτε τη στον φάκελο δεδομένων σας.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## Βήμα 2: Φόρτωση της Εικόνας και Επιλογή Detect Areas Mode

Φορτώστε την εικόνα-στόχο και καθορίστε τη στρατηγική ανίχνευσης που ταιριάζει στο σενάριό σας.

```csharp
// Recognize image
RecognitionResult result = api.RecognizeImage(dataDir + "table.png", new RecognitionSettings
{
    // Choose the Detect Areas Mode
    DetectAreasMode = DetectAreasMode.PHOTO
    // Other options: NONE, DOCUMENT, COMBINE
});
```

## Βήμα 3: Ανάκτηση και Εμφάνιση του Αναγνωρισμένου Κειμένου

Αφού ολοκληρωθεί το OCR, μπορείτε να έχετε πρόσβαση στο εξαγόμενο κείμενο—ιδανικό για περαιτέρω επεξεργασία ή αποθήκευση σε βάση δεδομένων.

```csharp
// Display the recognized text
Console.WriteLine(result.RecognitionText);

Console.WriteLine("OCRDetectAreasMode executed successfully");
```
