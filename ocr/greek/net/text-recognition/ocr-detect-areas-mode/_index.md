# ocr document mode – Λειτουργία Ανίχνευσης Περιοχών στην Αναγνώριση Εικόνας OCR

## Εισαγωγή

Στη σύγχρονη ανάπτυξη .NET, το **ocr document mode** είναι η προτιμώμενη προσέγγιση για **βελτίωση της ακρίβειας OCR** όταν χρειάζεστε ακριβή έλεγχο του τρόπου ανίχνευσης κειμένου μέσα σε εικόνες. Το Aspose.OCR για .NET καθιστά εύκολη τη μετάβαση μεταξύ διαφορετικών στρατηγικών ανίχνευσης, επιτρέποντάς σας να **extract table text image** από σύνθετες διατάξεις όπως αποδείξεις, τιμολόγια ή έγγραφα πολλαπλών στηλών. Αυτό το **aspose ocr tutorial c#** θα σας καθοδηγήσει μέσω της λειτουργίας Detect Areas Mode, θα εξηγήσει πότε να χρησιμοποιήσετε κάθε λειτουργία και θα σας δείξει ένα έτοιμο παράδειγμα κώδικα.

## Γρήγορες Απαντήσεις
- **Τι είναι το ocr document mode;** Ένα σύνολο στρατηγικών ανίχνευσης (PHOTO, DOCUMENT, COMBINE) που λέει στο Aspose.OCR πώς να εντοπίζει περιοχές κειμένου.
- **Ποια λειτουργία είναι η καλύτερη για πίνακες;** Η λειτουργία `PHOTO` διαπρέπει στην εξαγωγή table text image και μικρών τμημάτων κειμένου.
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμαστική άδεια είναι επαρκής για δοκιμές· απαιτείται εμπορική άδεια για παραγωγή.
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6 και μεταγενέστερες.
- **Πόσο διαρκεί η εγκατάσταση;** Συνήθως λιγότερο από 10 λεπτά για ενσωμάτωση και εκτέλεση του δείγματος κώδικα.

## Πώς να βελτιώσετε την ακρίβεια OCR με τη λειτουργία Detect Areas Mode;

Η επιλογή της σωστής **Detect Areas Mode** είναι ο πιο αποτελεσματικός τρόπος για να αυξήσετε την ακρίβεια OCR σε δομημένες εικόνες. Ενημερώνοντας τη μηχανή αν η εικόνα μοιάζει με φωτογραφία, έντυπο έγγραφο ή συνδυασμό και των δύο, μειώνετε τις ψευδείς ανιχνεύσεις, επιταχύνετε την επεξεργασία και λαμβάνετε καθαρότερο κείμενο—ιδιαίτερα για πίνακες, αποδείξεις και διατάξεις πολλαπλών στηλών.

## Τι είναι το ocr document mode;

`ocr document mode` αναφέρεται στη ρύθμιση που καθορίζει στο Aspose.OCR πώς θα τμηματοποιήσει μια εικόνα πριν από την αναγνώριση κειμένου. Οι τρεις ενσωματωμένες λειτουργίες είναι:

- **PHOTO** – Βελτιστοποιημένο για φωτογραφίες, αποδείξεις, τιμολόγια και μικρές περιοχές κειμένου (ιδανικό για **extract table text image**).
- **DOCUMENT** – Κατάλληλο για σελίδες πολλαπλών στηλών και έγγραφα που περιέχουν ενσωματωμένα γραφικά.
- **COMBINE** – Συγχωνεύει τα αποτελέσματα των PHOTO και DOCUMENT για την πιο πλήρη κάλυψη.

## Γιατί να χρησιμοποιήσετε τη λειτουργία Detect Areas Mode;

Η επιλογή της κατάλληλης λειτουργίας ανίχνευσης μειώνει τα ψευδώς θετικά, επιταχύνει την επεξεργασία και βελτιώνει την ακρίβεια—σημαντικούς παράγοντες όταν επιδιώκετε **βελτίωση της ακρίβειας OCR** σε δομημένα δεδομένα όπως πίνακες. Η προσαρμογή της λειτουργίας στον τύπο της εικόνας σας εξαλείφει την ανάγκη εκτενούς μετα-επεξεργασίας.

## Κοινές Περιπτώσεις Χρήσης

| Σενάριο | Προτεινόμενη Λειτουργία | Γιατί βοηθά |
|----------|------------------|--------------|
| Αποδείξεις ή τιμολόγια με πυκνούς πίνακες | **PHOTO** | Επικεντρώνεται σε μικρά τμήματα κειμένου και διατηρεί τη διάταξη του πίνακα |
| Περιοδικά ή εκθέσεις πολλαπλών στηλών | **DOCUMENT** | Διαχειρίζεται το διαχωρισμό στηλών και τα ενσωματωμένα γραφικά |
| Σαρωμένα έγγραφα που περιέχουν τόσο φωτογραφίες όσο και κείμενο | **COMBINE** | Αξιοποιεί τα πλεονεκτήματα και των δύο, PHOTO και DOCUMENT |

## Βήμα 1: Αρχικοποίηση Aspose.OCR

Δημιουργήστε μια παρουσία του μηχανήματος OCR και δείξτε του το φάκελο δεδομένων σας.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## Βήμα 2: Φόρτωση της Εικόνας και Επιλογή Detect Areas Mode

Φορτώστε την εικόνα-στόχο και ορίστε τη στρατηγική ανίχνευσης που ταιριάζει στο σενάριό σας.

```csharp
// Recognize image
RecognitionResult result = api.RecognizeImage(dataDir + "table.png", new RecognitionSettings
{
    // Choose the Detect Areas Mode
    DetectAreasMode = DetectAreasMode.PHOTO
    // Other options: NONE, DOCUMENT, COMBINE
});
```

## Βήμα 3: Ανάκτηση και Εμφάνιση του Αναγνωρισμένου Κειμένου

Αφού ολοκληρωθεί το OCR, μπορείτε να έχετε πρόσβαση στο εξαγόμενο κείμενο—ιδανικό για περαιτέρω επεξεργασία ή αποθήκευση σε βάση δεδομένων.

```csharp
// Display the recognized text
Console.WriteLine(result.RecognitionText);

Console.WriteLine("OCRDetectAreasMode executed successfully");
```
