# Πώς να Εκτελέσετε OCR σε C# – Εξαγωγή Αραβικού Κειμένου από Εικόνες

Έχετε αναρωτηθεί ποτέ **πώς να εκτελέσετε OCR** σε ένα αραβικό τιμολόγιο ή σε ένα σαρωμένο έγγραφο; Δεν είστε μόνοι—πολλοί προγραμματιστές συναντούν δυσκολίες όταν πρέπει να εξάγουν αραβικούς χαρακτήρες από ένα bitmap. Τα καλά νέα είναι ότι με λίγες γραμμές C# μπορείτε να αναγνωρίσετε κείμενο από αρχεία εικόνας, να μετατρέψετε την εικόνα σε κείμενο και τελικά **να εξάγετε αραβικό κείμενο** για επεξεργασία downstream.

Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από ένα πλήρες, εκτελέσιμο παράδειγμα που δείχνει ακριβώς **πώς να εκτελέσετε OCR**, γιατί κάθε βήμα είναι σημαντικό, και τι πρέπει να προσέξετε όταν δουλεύετε με σενάρια από δεξιά προς αριστερά. Στο τέλος θα μπορείτε να **εξάγετε κείμενο από εικόνα** σε αρχεία στα Αραβικά, Urdu, Κουρδικά ή οποιαδήποτε γλώσσα υποστηρίζεται από τη μηχανή OCR.

## Προαπαιτούμενα

- .NET 6.0 ή νεότερο (ο κώδικας μεταγλωττίζεται επίσης με .NET Core)  
- Μια αναφορά στη βιβλιοθήκη OCR που παρέχει `OcrEngine` (π.χ., `MyOcrSdk.dll`).  
- Ένα αρχείο εικόνας που περιέχει αραβικό κείμενο, όπως `invoice_arabic.png`.  
- Βασική εξοικείωση με εφαρμογές κονσόλας C#.

> **Συμβουλή:** Αν δεν έχετε διαθέσιμο OCR SDK, η δωρεάν έκδοση κοινότητας του *MyOcrSdk* λειτουργεί για δοκιμές και υποστηρίζει τις γλώσσες που θα χρησιμοποιήσουμε.

---

## Βήμα 1 – Ρύθμιση του Έργου και Εισαγωγή του Namespace OCR

Πριν μπορέσουμε να **αναγνωρίσουμε κείμενο από εικόνα** χρειάζεται ένα σκελετό έργου και τις σωστές οδηγίες `using`.

```csharp
using System;
using MyOcrSdk;          // <-- Replace with the actual namespace of your OCR library
using MyOcrSdk.IO;       // For ImageStream helper class

namespace ArabicOcrDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // The rest of the OCR workflow will go here
        }
    }
}
```

*Γιατί είναι σημαντικό:* Η εισαγωγή των σωστών namespaces σας δίνει πρόσβαση στα `OcrEngine`, `Language` και `ImageStream`. Η παράλειψη αυτού του βήματος οδηγεί σε σφάλματα μεταγλώττισης που μπορεί να απογοητεύσουν τους αρχάριους.

---

## Βήμα 2 – Δημιουργία Παραδείγματος OCR Engine (Περιλαμβάνει Κύρια Λέξη‑Κλειδί)

Τώρα πραγματικά **εκτελούμε OCR** δημιουργώντας το αντικείμενο μηχανής.

```csharp
// Step 2: Create an OCR engine instance
var ocrEngine = new OcrEngine();
```

Το αντικείμενο `OcrEngine` είναι η καρδιά της λειτουργίας· διατηρεί τις ρυθμίσεις, εκτελεί το βαρέως βάρους κομμάτι και επιστρέφει το αντικείμενο αποτελέσματος που περιέχει το εξαγόμενο string. Σκεφτείτε το ως το «εγκέφαλο» που θα **μετατρέψει την εικόνα σε κείμενο**.

---

## Βήμα 3 – Επιλογή Γλώσσας για Αναγνώριση

Αραβικά, Urdu, Κουρδικά… όλα μοιράζονται την ίδια οικογένεια γραφής, οπότε πρέπει να πούμε στη μηχανή ποια γλώσσα να περιμένει. Αυτό βελτιώνει την ακρίβεια δραματικά.

```csharp
// Step 3: Choose the language for recognition (Arabic, Urdu, Kurdish, etc.)
ocrEngine.Config.Language = Language.Arabic;   // You can switch to Language.Urdu, etc.
```

*Γιατί είναι σημαντικό:* Οι μηχανές OCR βασίζονται σε μοντέλα γλώσσας. Η επιλογή του σωστού μοντέλου μειώνει την λανθασμένη αναγνώριση παρόμοιων χαρακτήρων, ειδικά για σενάρια από δεξιά προς αριστερά.

---

## Βήμα 4 – Φόρτωση της Εικόνας που Περιέχει το Κείμενο

Χρειαζόμαστε ένα bitmap που η μηχανή μπορεί να αναλύσει. Η βοηθητική μέθοδος `ImageStream.FromFile` αφαιρεί τις λεπτομέρειες του file‑IO.

```csharp
// Step 4: Load the image that contains the text to be recognized
ocrEngine.Image = ImageStream.FromFile(@"YOUR_DIRECTORY/invoice_arabic.png");
```

Βεβαιωθείτε ότι η διαδρομή δείχνει σε μια **έγκυρη εικόνα**. Αν το αρχείο λείπει ή είναι κατεστραμμένο, η μηχανή θα ρίξει εξαίρεση και δεν θα μπορείτε να **εξάγετε κείμενο από εικόνα** επιτυχώς.

---

## Βήμα 5 – Εκτέλεση της Διαδικασίας OCR και Λήψη του Αποτελέσματος

Τέλος, καλούμε το `Recognize()` και εμφανίζουμε το εξαγόμενο string.

```csharp
// Step 5: Perform the OCR operation
var ocrResult = ocrEngine.Recognize();

// Step 6: Display the extracted text
Console.WriteLine("=== Extracted Arabic Text ===");
Console.WriteLine(ocrResult.Text);
```

Η ιδιότητα `ocrResult.Text` περιέχει την απλή κειμενική αναπαράσταση όλων όσων η μηχανή μπόρεσε να διαβάσει. Στις περισσότερες περιπτώσεις θα δείτε ένα μείγμα αραβικών χαρακτήρων και αριθμών, ιδανικό για περαιτέρω επεξεργασία όπως εισαγωγή σε βάση δεδομένων ή μετάφραση.

### Αναμενόμενη Έξοδος

```
=== Extracted Arabic Text ===
فاتورة رقم: 12345
التاريخ: 2026/03/15
المبلغ: 2500.00 ريال
...
```

Αν δείτε ακατάλληλους χαρακτήρες, ελέγξτε ξανά τη ρύθμιση γλώσσας και βεβαιωθείτε ότι η εικόνα έχει υψηλή ανάλυση (300 dpi ή περισσότερο είναι ιδανικό).

---

## Πλήρες Παράδειγμα Λειτουργίας

Παρακάτω βρίσκεται το **πλήρες, αυτόνομο πρόγραμμα** που μπορείτε να αντιγράψετε‑επικολλήσετε σε ένα νέο έργο κονσόλας και να τρέξετε αμέσως.

```csharp
// Full OCR demo – extracts Arabic text from an image
using System;
using MyOcrSdk;
using MyOcrSdk.IO;

namespace ArabicOcrDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // 1️⃣ Create the OCR engine
            var ocrEngine = new OcrEngine();

            // 2️⃣ Set language – Arabic (change to Urdu or Kurdish if needed)
            ocrEngine.Config.Language = Language.Arabic;

            // 3️⃣ Load the source image
            // Replace YOUR_DIRECTORY with the folder that holds your PNG/JPG
            string imagePath = @"YOUR_DIRECTORY/invoice_arabic.png";
            ocrEngine.Image = ImageStream.FromFile(imagePath);

            try
            {
                // 4️⃣ Run OCR
                var result = ocrEngine.Recognize();

                // 5️⃣ Output the text
                Console.WriteLine("=== Extracted Arabic Text ===");
                Console.WriteLine(result.Text);
            }
            catch (Exception ex)
            {
                // Graceful error handling – useful when you need to **extract text from image** in batch jobs
                Console.Error.WriteLine($"OCR failed: {ex.Message}");
            }
        }
    }
}
```

> **Σημείωση:** Αν το OCR SDK σας απαιτεί άδεια, βεβαιωθείτε ότι αρχικοποιείτε την άδεια πριν από το βήμα 1 (π.χ., `OcrEngine.SetLicense("YOUR_LICENSE_KEY");`). Αυτή η γραμμή παραλείπεται εδώ για συντομία.

## Διαχείριση Συνηθισμένων Περιπτώσεων Άκρων

| Κατάσταση | Γιατί Συμβαίνει | Γρήγορη Διόρθωση |
|-----------|----------------|-------------------|
| **Θολή ή χαμηλής ανάλυσης εικόνα** | Η ακρίβεια του OCR πέφτει κάτω από 70 % | Σαρώστε στα 300 dpi ή αυξήστε την ανάλυση με αλγόριθμο bicubic πριν τη μεταβίβαση στη μηχανή. |
| **Μικτές γλώσσες (Αραβικά + Αγγλικά)** | Η μηχανή μπορεί να σταματήσει μετά το πρώτο μπλοκ γλώσσας | Ενεργοποιήστε τη λειτουργία πολλαπλών γλωσσών: `ocrEngine.Config.Language = Language.Arabic \| Language.English;` |
| **Προβλήματα εμφάνισης από δεξιά προς αριστερά** | Η κονσόλα εκτυπώνει χαρακτήρες αριστερά‑προς‑δεξιά, κάνοντας τα Αραβικά να φαίνονται ανάποδα | Χρησιμοποιήστε `Console.OutputEncoding = System.Text.Encoding.UTF8;` και ένα τερματικό που υποστηρίζει σενάρια RTL. |
| **Μεγάλα PDFs χωρισμένα σε πολλές σελίδες** | Η κατανάλωση μνήμης αυξάνεται | Επεξεργαστείτε μία σελίδα τη φορά: φορτώστε κάθε σελίδα ως ξεχωριστή εικόνα και επαναλάβετε τη ροή OCR. |
| **Ειδικά σύμβολα (νόμισμα, ημερομηνίες)** | Ορισμένα μοντέλα OCR τα θεωρούν θόρυβο | Μετα-επεξεργαστείτε το `ocrResult.Text` με regex για να κανονικοποιήσετε γνωστά μοτίβα. |

## Επέκταση της Λύσης – Από OCR σε Εξαγωγή Δεδομένων

Τώρα που **ξέρετε πώς να εκτελέσετε OCR**, ίσως αναρωτιέστε: *Τι μπορώ να κάνω με το εξαγόμενο αραβικό κείμενο;* Εδώ είναι μερικές ιδέες που ακολουθούν φυσικά:

1. **Ανάλυση τιμολογίων** – Χρησιμοποιήστε κανονικές εκφράσεις για να εξάγετε αριθμούς τιμολογίου, ημερομηνίες και σύνολα.  
2. **Αποστολή σε API μετάφρασης** – Στείλτε το αραβικό string στο Azure Translator ή στο Google Cloud Translate.  
3. **Αποθήκευση σε βάση δεδομένων** – Εισάγετε το καθαρισμένο κείμενο σε πίνακα SQL για αναφορές.  
4. **Έναρξη ροών εργασίας** – Συνδυάστε με ουρά μηνυμάτων (π.χ., RabbitMQ) για να ξεκινήσετε downstream επεξεργασία.

Όλα αυτά τα σενάρια περιλαμβάνουν την ίδια βασική λειτουργία: **μετατρέψτε την εικόνα σε κείμενο**, έπειτα επεξεργαστείτε το αποτέλεσμα.
