# Πώς να Χρησιμοποιήσετε OCR σε C# – Εξαγωγή Αραβικού Κειμένου από JPG

Έχετε αναρωτηθεί ποτέ **πώς να χρησιμοποιήσετε OCR** όταν έχετε μια εικόνα με αραβικό κείμενο στον σκληρό σας δίσκο; Δεν είστε μόνοι. Πολλοί προγραμματιστές αντιμετωπίζουν το ίδιο πρόβλημα: έχουν ένα JPEG, χρειάζονται τις λέξεις μέσα, και δεν θέλουν να γράψουν από το μηδέν ένα νευρωνικό δίκτυο.  

Τα καλά νέα; Με μερικές γραμμές C# μπορείτε **να αναγνωρίσετε κείμενο από εικόνα** αρχεία, να εξάγετε κάθε αραβικό χαρακτήρα και να καταλήξετε με μια καθαρή συμβολοσειρά που μπορείτε να αποθηκεύσετε, να αναζητήσετε ή να εμφανίσετε. Σε αυτό το tutorial θα περάσουμε από όλη τη διαδικασία — εγκατάσταση της βιβλιοθήκης, ρύθμιση του μοντέλου γλώσσας, εκτέλεση της σάρωσης και τελικά εκτύπωση του αποτελέσματος. Στο τέλος θα μπορείτε **να μετατρέψετε JPG σε κείμενο** σε δευτερόλεπτα.

## Τι Θα Μάθετε

- Εγκατάσταση του πακέτου NuGet Aspose.OCR για .NET.  
- Αρχικοποίηση της μηχανής OCR σε λειτουργία CPU (ιδανική για μικρές εργασίες).  
- Φόρτωση του μοντέλου γλώσσας Αραβικά αυτόματα.  
- Εκτέλεση OCR σε εικόνα JPEG και ανάκτηση του αναγνωρισμένου κειμένου.  
- Διαχείριση κοινών προβλημάτων όπως μεγάλα αρχεία ή έλλειψη δεδομένων γλώσσας.

Δεν απαιτείται προηγούμενη εμπειρία με OCR· αρκεί μια βασική κατανόηση του C# και του Visual Studio. Έτοιμοι; Ας ξεκινήσουμε.

## Εγκατάσταση Aspose OCR για .NET

Πριν τρέξει οποιοσδήποτε κώδικας, χρειάζεστε τη βιβλιοθήκη Aspose.OCR. Διανέμεται ως πακέτο NuGet, οπότε η εγκατάσταση είναι μια εντολή:

```bash
dotnet add package Aspose.OCR
```

Ή, αν προτιμάτε το UI του Visual Studio, ανοίξτε **Tools → NuGet Package Manager → Manage NuGet Packages for Solution**, ψάξτε για **Aspose.OCR** και κάντε κλικ στο **Install**. Το πακέτο φέρνει όλα όσα χρειάζεστε, συμπεριλαμβανομένων των αρχείων δεδομένων γλώσσας που η μηχανή θα κατεβάσει κατά την πρώτη χρήση.

> **Pro tip:** Κρατήστε το έργο σας στο .NET 6 ή νεότερο· παλαιότερα πλαίσια λειτουργούν, αλλά θα χάσετε τις βελτιώσεις απόδοσης.

## Αρχικοποίηση της Μηχανής OCR

Τώρα που η βιβλιοθήκη είναι στη θέση της, μπορούμε να δημιουργήσουμε ένα στιγμιότυπο του `OcrEngine`. Για τις περισσότερες μικρής κλίμακας εργασίες η προεπιλεγμένη λειτουργία CPU είναι περισσότερο από αρκετή — χρησιμοποιεί τον επεξεργαστή του υπολογιστή και αποφεύγει την επιπλέον ρύθμιση που απαιτεί η επιτάχυνση GPU.

```csharp
using Aspose.OCR;
using Aspose.OCR.Settings;

// Initialise the OCR engine (CPU mode is sufficient for small jobs)
var ocrEngine = new OcrEngine();
```

Γιατί να δημιουργούμε μια νέα μηχανή κάθε φορά; Το αντικείμενο `OcrEngine` κρατά ρυθμίσεις όπως γλώσσα, DPI και μορφή εξόδου. Κρατώντας το περιορισμένο σε μία λειτουργία εξασφαλίζετε καθαρή κατάσταση και αποφεύγετε τυχαίες αλληλεπιδράσεις μεταξύ διαφορετικών μοντέλων γλώσσας.

## Φόρτωση του Μοντέλου Γλώσσας Αραβικά

Η Aspose παρέχει πακέτα γλώσσας κατόπιν ζήτησης. Την πρώτη φορά που ορίζετε `Language.Arabic`, η βιβλιοθήκη θα κατεβάσει περίπου 30 MB δεδομένων σε φάκελο cache στον υπολογιστή σας. Αυτή η εφάπαξ λήψη κάνει τις επόμενες εκτελέσεις αστραπιαία γρήγορες.

```csharp
// Choose the language model to load.
// The first assignment triggers an automatic download of the language data (~30 MB).
ocrEngine.Settings.Language = Language.Arabic;
```

Αν χρειαστεί να υποστηρίξετε επιπλέον γραφές (π.χ. Αγγλικά ή Χίντι), απλώς αλλάξτε την τιμή του enum — δεν απαιτείται επιπλέον κώδικας. Η μηχανή θα αποθηκεύσει στην cache κάθε γλώσσα ξεχωριστά.

## Εκτέλεση OCR σε Εικόνα JPG

Με τη μηχανή έτοιμη και το μοντέλο Αραβικών φορτωμένο, δείξτε την στην εικόνα που θέλετε να επεξεργαστείτε. Η διαδρομή μπορεί να είναι απόλυτη ή σχετική· απλώς βεβαιωθείτε ότι το αρχείο υπάρχει και είναι αναγνώσιμο.

```csharp
// Run OCR on the input image.
var recognitionResult = ocrEngine.Recognize(@"YOUR_DIRECTORY/arabic_doc.jpg");
```

**Περίπτωση άκρης:** Αν το JPEG σας είναι τεράστιο (πάνω από 5 MB) ίσως θελήσετε πρώτα να το μειώσετε σε μέγεθος. Οι μεγάλες εικόνες αυξάνουν τη χρήση μνήμης και μπορούν να επιβραδύνουν την αναγνώριση. Ένα γρήγορο προ‑αυξομείωση με `System.Drawing` ή `ImageSharp` μπορεί να μειώσει δραστικά τον χρόνο επεξεργασίας.

## Ανάκτηση και Εμφάνιση του Αναγνωρισμένου Κειμένου

Η μέθοδος `Recognize` επιστρέφει ένα αντικείμενο `OcrResult`. Η ιδιότητα `Text` του περιέχει την απλή κειμενική αναπαράσταση όλων όσων η μηχανή μπόρεσε να διαβάσει.

```csharp
// Retrieve the recognised text and output it.
string extractedText = recognitionResult.Text;
Console.WriteLine(extractedText);
```

Όταν τρέξετε το πρόγραμμα, θα δείτε τους αραβικούς χαρακτήρες να εκτυπώνονται στην κονσόλα, διατηρώντας τη σειρά και τις αλλαγές γραμμής. Αν χρειάζεστε το κείμενο σε αρχείο, απλώς γράψτε το με `File.WriteAllText`.

### Πλήρες Παράδειγμα Εργασίας

Συνδυάζοντας όλα τα παραπάνω, εδώ είναι μια πλήρης, έτοιμη για εκτέλεση εφαρμογή κονσόλας:

```csharp
using System;
using Aspose.OCR;
using Aspose.OCR.Settings;

class Program
{
    static void Main()
    {
        // 1️⃣ Initialise the OCR engine (CPU mode works fine for most cases)
        var ocrEngine = new OcrEngine();

        // 2️⃣ Load the Arabic language pack – triggers a one‑time download (~30 MB)
        ocrEngine.Settings.Language = Language.Arabic;

        // 3️⃣ Specify the image path – replace with your actual file location
        string imagePath = @"YOUR_DIRECTORY/arabic_doc.jpg";

        // 4️⃣ Run OCR and capture the result
        var result = ocrEngine.Recognize(imagePath);

        // 5️⃣ Extract the text and display it
        string extractedText = result.Text;
        Console.WriteLine("=== Extracted Arabic Text ===");
        Console.WriteLine(extractedText);
    }
}
```

**Αναμενόμενη έξοδος (παράδειγμα):**

```
=== Extracted Arabic Text ===
هذا نص تجريبي باللغة العربية
تم استخراج هذا النص بنجاح
```

Αν η έξοδος φαίνεται παραμορφωμένη, ελέγξτε ξανά ότι η εικόνα είναι καθαρή και ότι η γλώσσα έχει οριστεί σε `Arabic`. Θολές σάρωσες ή σελίδες με περιστροφή είναι συχνά λόγοι αποτυχίας του OCR.

## Συχνές Ερωτήσεις & Συμβουλές

- **Τι γίνεται αν η εικόνα περιέχει τόσο Αραβικά όσο και Αγγλικά;**  
  Ορίστε `ocrEngine.Settings.Language = Language.Multilingual;` για να επιτρέψετε στην Aspose να εντοπίσει αυτόματα πολλαπλές γραφές.

- **Μπορώ να επιταχύνω την επεξεργασία με GPU;**  
  Ναι — αντικαταστήστε τη προεπιλεγμένη μηχανή με `new OcrEngine(OcrEngineMode.Gpu)` εφόσον έχετε συμβατή κάρτα γραφικών και εγκατεστημένο το runtime CUDA.

- **Πώς διαχειρίζομαι την απόδοση δεξιά‑προς‑αριστερά (RTL) σε UI;**  
  Η ακατέργαστη συμβολοσειρά είναι Unicode· τα περισσότερα σύγχρονα UI frameworks (WinForms, WPF, ASP.NET) υποστηρίζουν αυτόματα τη διάταξη RTL όταν ορίσετε το κατάλληλο `FlowDirection`.

- **Υπάρχει όριο στο μέγεθος της εικόνας;**  
  Τεχνικά όχι, αλλά η κατανάλωση μνήμης αυξάνεται με την ανάλυση. Για τεράστιες σαρώσεις (π.χ. βιβλία), σκεφτείτε να επεξεργάζεστε μία σελίδα τη φορά.

- **Μετα-επεξεργασία:** Χρήση κανονικών εκφράσεων για καθαρισμό περιττών σημείων στίξης ή αλλαγών γραμμής.
