# Πώς να διαβάσετε Κυριλλικά: Μετατροπή εικόνας σε κείμενο με Aspose OCR

Έχετε αναρωτηθεί ποτέ **πώς να διαβάσετε Κυριλλικά** απευθείας από ένα στιγμιότυπο οθόνης ή σαρωμένο έγγραφο; Δεν είστε οι μόνοι—οι προγραμματιστές χρειάζονται συνεχώς να εξάγουν ρωσικό κείμενο από εικόνες για εισαγωγή δεδομένων, τοπικοποίηση ή εκπαίδευση chatbot. Τα καλά νέα; Με λίγες γραμμές C# και Aspose OCR μπορείτε **να μετατρέψετε εικόνα σε κείμενο** σε μια στιγμή.

Σε αυτό το tutorial θα περάσουμε από όλη τη διαδικασία: από την εγκατάσταση της βιβλιοθήκης, στη ρύθμιση της μηχανής για Ρωσικά (Κυριλλικά), μέχρι την πραγματική **εκτέλεση OCR σε εικόνα** και την εμφάνιση του αποτελέσματος. Στο τέλος θα μπορείτε **να εξάγετε ρωσικούς** χαρακτήρες χωρίς να φύγετε από το IDE σας, και θα δείτε πώς να **αναγνωρίζετε κυριλλικά από εικόνα** δεδομένα αξιόπιστα.

## Προαπαιτούμενα — Τι χρειάζεστε πριν ξεκινήσετε

- .NET 6.0 ή νεότερο (ο κώδικας λειτουργεί επίσης σε .NET Core 3.1, αλλά προτιμάται η νεότερη έκδοση)
- Visual Studio 2022 (ή οποιονδήποτε επεξεργαστή C# προτιμάτε)
- Ένα πακέτο NuGet Aspose OCR (`Aspose.OCR`) – εγκαταστήστε το μέσω του Package Manager Console:
  ```powershell
  Install-Package Aspose.OCR
  ```
- Μια δείγμα εικόνας που περιέχει ρωσικό κυριλλικό κείμενο, π.χ., `russian_sample.png`.  
  *(Αν δεν έχετε κάποια, απλώς τραβήξτε ένα στιγμιότυπο οθόνης από οποιαδήποτε ρωσική ιστοσελίδα.)*

Αυτό είναι όλο—χωρίς επιπλέον μηχανές OCR, χωρίς εγγενή DLLs για μεταγλώττιση. Το Aspose διαχειρίζεται αυτόματα τις λήψεις δεδομένων γλώσσας, γι' αυτό το παράδειγμα παραμένει ελαφρύ.

## Βήμα 1: Αρχικοποίηση της μηχανής OCR (Πώς να διαβάσετε Κυριλλικά αποδοτικά)

Το πρώτο που κάνουμε είναι να δημιουργήσουμε μια παρουσία `OcrEngine`. Από προεπιλογή θα κατεβάσει τα απαιτούμενα πακέτα γλώσσας αυτόματα, οπότε δεν χρειάζεται να διαχειριστείτε κανένα αρχείο.

```csharp
using Aspose.Ocr;
using System;

namespace CyrillicOcrDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // Step 1 – create the OCR engine with default options.
            var ocrEngine = new OcrEngine();   // auto‑download enabled
```

**Γιατί είναι σημαντικό:** Η αρχικοποίηση της μηχανής μία φορά και η επαναχρησιμοποίηση της σε πολλές εικόνες μειώνει το κόστος. Η λειτουργία αυτόματης λήψης εξασφαλίζει ότι τα δεδομένα της ρωσικής γλώσσας (`ru`) είναι διαθέσιμα, ώστε να μην λάβετε ποτέ σφάλμα “language not found”.

## Βήμα 2: Ορίστε στη μηχανή ποια γλώσσα να αναγνωρίσει

Το Aspose OCR υποστηρίζει δεκάδες γλώσσες, αλλά πρέπει να ορίσετε τον κωδικό γλώσσας ρητά. Για τα Ρωσικά (Κυριλλικά) ο κωδικός ISO‑639‑1 είναι `"ru"`.

```csharp
            // Step 2 – select Russian (Cyrillic) as the target language.
            ocrEngine.Language = "ru";
```

**Συμβουλή:** Αν χρειάζεται να διαχειριστείτε έγγραφα με πολλαπλές γλώσσες, μπορείτε να περάσετε μια λίστα χωρισμένη με κόμμα όπως `"ru,en"` και η μηχανή θα δοκιμάσει και τις δύο. Για καθαρά Κυριλλικά, το `"ru"` δίνει την καλύτερη ακρίβεια.

## Βήμα 3: Εκτέλεση OCR στο αρχείο εικόνας σας

Τώρα δίνουμε τη διαδρομή της εικόνας στη `RecognizeImage`. Η μέθοδος επιστρέφει ένα αντικείμενο `OcrResult` που περιέχει το εξαγόμενο κείμενο και τις βαθμολογίες εμπιστοσύνης.

```csharp
            // Step 3 – run OCR on the chosen image.
            var ocrResult = ocrEngine.RecognizeImage("YOUR_DIRECTORY/russian_sample.png");
```

**Ακρόατο σενάριο:** Αν η εικόνα είναι μεγάλη (πάνω από 5 MB) σκεφτείτε να την αλλάξετε μέγεθος πρώτα· η ακρίβεια του OCR μειώνεται όταν το αρχείο είναι τεράστιο και η μηχανή ξοδεύει επιπλέον χρόνο για τη φόρτωσή του.

## Βήμα 4: Εξαγωγή του αναγνωρισμένου Κυριλλικού κειμένου

Τέλος, εκτυπώστε το αποτέλεσμα στην κονσόλα. Μπορείτε επίσης να το γράψετε σε αρχείο, βάση δεδομένων ή να το περάσετε σε άλλη υπηρεσία.

```csharp
            // Step 4 – display the recognized Cyrillic text.
            Console.WriteLine("Recognized Cyrillic text:");
            Console.WriteLine(ocrResult.Text);
        }
    }
}
```

Όταν εκτελέσετε το πρόγραμμα, θα πρέπει να δείτε κάτι όπως:

```
Recognized Cyrillic text:
Привет, мир! Это пример текста на русском языке.
```

## Πώς να εκτελέσετε OCR σε αρχεία εικόνας σε πραγματικά έργα

Το παραπάνω απόσπασμα λειτουργεί για μία εικόνα, αλλά ο κώδικας παραγωγής συχνά χρειάζεται να επεξεργαστεί πολλά αρχεία. Εδώ είναι ένα γρήγορο μοτίβο που μπορείτε να αντιγράψετε‑επικολλήσετε:

```csharp
static void ProcessFolder(string folderPath)
{
    var engine = new OcrEngine { Language = "ru" };

    foreach (var file in Directory.GetFiles(folderPath, "*.png"))
    {
        var result = engine.RecognizeImage(file);
        File.WriteAllText(Path.ChangeExtension(file, ".txt"), result.Text);
        Console.WriteLine($"Processed {Path.GetFileName(file)}");
    }
}
```

**Γιατί να επαναχρησιμοποιήσετε τη μηχανή;** Η δημιουργία νέου `OcrEngine` για κάθε αρχείο θα κατέβαζε επανειλημμένα δεδομένα γλώσσας και θα σπαταλούσε κύκλους CPU. Η διατήρηση μιας ενεργής παρουσίας βελτιώνει δραματικά το ρυθμό επεξεργασίας.

## Συνηθισμένα προβλήματα & Πώς να εξάγετε Ρωσικά με ακρίβεια

| Συμπτωμα | Πιθανή Αιτία | Διόρθωση |
|----------|--------------|----------|
| Ακατάληπτοι χαρακτήρες (π.χ., `????`) | Λάθος κωδικός γλώσσας (`en` αντί για `ru`) | Ορίστε `ocrEngine.Language = "ru"` |
| Χαμηλές βαθμολογίες εμπιστοσύνης | Η εικόνα είναι θολή ή χαμηλής ανάλυσης | Προεπεξεργασία της εικόνας (αύξηση DPI, όξυνση) |
| Απουσία διακριτικών | Η γραμματοσειρά δεν υποστηρίζεται στο προεπιλεγμένο μοντέλο OCR | Χρησιμοποιήστε την πιο πρόσφατη έκδοση Aspose OCR (v23.12+ περιλαμβάνει καλύτερη υποστήριξη Κυριλλικών) |
| Εξαίρεση “Unable to download language data” | Δεν υπάρχει πρόσβαση στο διαδίκτυο στην πρώτη εκτέλεση | Κατεβάστε χειροκίνητα το πακέτο γλώσσας από το portal του Aspose και ορίστε το `OcrEngine.LanguageDataPath` |

Η αντιμετώπιση αυτών των ζητημάτων εξασφαλίζει ότι **αναγνωρίζετε κυριλλικά από εικόνα** πηγές με υψηλή αξιοπιστία.

## Επέκταση του παραδείγματος – Από κονσόλα σε Web API

Αν δημιουργείτε μια web υπηρεσία που δέχεται ανεβασμένες εικόνες, χρειάζεται μόνο να αντικαταστήσετε το τμήμα ανάγνωσης αρχείου:

```csharp
[HttpPost("ocr")]
public async Task<IActionResult> Upload(IFormFile image)
{
    using var stream = image.OpenReadStream();
    var engine = new OcrEngine { Language = "ru" };
    var result = engine.RecognizeImage(stream);
    return Ok(new { text = result.Text });
}
```

Τώρα οποιοσδήποτε πελάτης μπορεί **να εκτελέσει OCR σε εικόνα** φορτία και να λάβει ρωσικό κείμενο άμεσα—ιδανικό για αγωγούς μετάφρασης ή εργαλεία διαχείρισης περιεχομένου.

## Επόμενα βήματα & Σχετικά θέματα

- Δοκιμάστε το `ocrEngine.DetectOrientation` για αυτόματη περιστροφή παραμορφωμένων σαρώσεων.
- Συνδυάστε OCR με APIs μετάφρασης (Google Translate, Azure Translator) για **μετατροπή εικόνας σε κείμενο** και στη συνέχεια στα Αγγλικά σε μία ροή.
- Εξερευνήστε τη δυνατότητα `OcrRegion` του Aspose αν χρειάζεστε μόνο να **αναγνωρίσετε κυριλλικά από εικόνα** τμήματα (π.χ., πεδίο φόρμας).

Μη διστάσετε να αλλάξετε τον κωδικό γλώσσας σε `"uk"` για ουκρανικά ή `"bg"` για βουλγαρικά—το Aspose OCR υποστηρίζει ολόκληρη την οικογένεια των Κυριλλικών.
