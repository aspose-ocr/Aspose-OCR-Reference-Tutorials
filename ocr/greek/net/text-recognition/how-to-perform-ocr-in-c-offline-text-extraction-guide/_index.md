# Πώς να Εκτελέσετε OCR σε C# – Οδηγός Εξαγωγής Κειμένου Εκτός Σύνδεσης

Έχετε αναρωτηθεί ποτέ **πώς να εκτελέσετε OCR** σε μια εφαρμογή C# χωρίς να στέλνετε δεδομένα στο σύννεφο; Δεν είστε μόνοι. Πολλοί προγραμματιστές χρειάζονται έναν αξιόπιστο τρόπο να *εξάγουν κείμενο από εικόνα* αρχεία διατηρώντας τα πάντα εντός της υποδομής—ιδιαίτερα όταν εργάζονται με ευαίσθητα έγγραφα.

Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα από ένα πλήρες, εκτελέσιμο παράδειγμα που δείχνει πώς να **φορτώσετε εικόνα για OCR**, να ρυθμίσετε τη μηχανή Aspose OCR για χρήση εκτός σύνδεσης και, τέλος, να **επεξεργαστείτε εικόνα με OCR** ώστε να λάβετε καθαρό, αναζητήσιμο κείμενο. Χωρίς εξωτερικές υπηρεσίες, χωρίς κρυφές κλήσεις δικτύου—απλός κώδικας C# που μπορείτε να ενσωματώσετε σε οποιοδήποτε έργο .NET.

> **Τι θα πάρετε:** ένα αυτόνομο πρόγραμμα που διαβάζει ένα PNG, εκτελεί αναγνώριση γαλλικής γλώσσας και εκτυπώνει το αποτέλεσμα στην κονσόλα. Θα καλύψουμε επίσης κοινές παγίδες, προαιρετικές βελτιώσεις και ιδέες για τα επόμενα βήματα, ώστε να προσαρμόσετε τη λύση σε οποιαδήποτε γλώσσα ή σενάριο.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα εξής:

- **.NET 6.0** (ή οποιαδήποτε πρόσφατη έκδοση του .NET). Παλαιότερες εκδόσεις λειτουργούν, αλλά η σύνταξη που εμφανίζεται ταιριάζει με το τρέχον SDK.
- **Aspose.OCR for .NET** πακέτο NuGet. Εγκαταστήστε το με `dotnet add package Aspose.OCR`.
- Ένας φάκελος με όνομα `OCRResources` που περιέχει τα πακέτα γλώσσας που χρειάζεστε (διαθέσιμα για λήψη από τον ιστότοπο της Aspose).  
- Ένα αρχείο εικόνας (`offline_test.png`) που θέλετε να αναγνωρίσετε.  
- Ένα βασικό IDE όπως το Visual Studio, VS Code ή Rider.

Αν σας λείπει κάποιο από αυτά, αποκτήστε το τώρα—διαφορετικά ο κώδικας δεν θα μεταγλωττιστεί.

## Βήμα 1: Ρύθμιση του Offline OCR Engine (Primary Keyword in Action)

Το πρώτο που πρέπει να κάνουμε είναι **πώς να εκτελέσετε OCR** χωρίς σύνδεση στο διαδίκτυο. Αυτό σημαίνει ότι θα κατευθύνουμε το `OcrEngine` σε έναν τοπικό φάκελο πόρων και θα απενεργοποιήσουμε τυχόν αυτόματες λήψεις.

```csharp
using Aspose.OCR;
using Aspose.OCR.Models;

class OfflineDemo
{
    static void Main()
    {
        // 1️⃣ Create and configure the OCR engine for offline use
        var ocrEngine = new OcrEngine
        {
            // Tell the engine where the language files live
            ResourcePath = @"YOUR_DIRECTORY\OCRResources",
            // Prevent the SDK from trying to fetch missing files online
            AllowOnlineDownload = false
        };
```

**Γιατί είναι σημαντικό:** Ορίζοντας το `AllowOnlineDownload` σε `false`, εξασφαλίζετε ότι η διαδικασία παραμένει εντελώς τοπική. Αυτό είναι κρίσιμο για περιβάλλοντα με αυστηρούς κανονισμούς (υγεία, χρηματοοικονομικά κ.λπ.) όπου τα δεδομένα δεν πρέπει ποτέ να φύγουν από την υποδομή.

## Βήμα 2: Φόρτωση της Εικόνας για OCR

Τώρα που η μηχανή είναι έτοιμη, πρέπει να **φορτώσετε εικόνα για OCR**. Η Aspose παρέχει μια βολική static μέθοδο που διαβάζει κοινές μορφές (PNG, JPEG, TIFF) απευθείας σε ένα αντικείμενο `OcrImage`.

```csharp
        // 2️⃣ Load the image you want to recognize
        var ocrImage = OcrImage.FromFile(@"YOUR_DIRECTORY\offline_test.png");
```

> **Pro tip:** Αν η εικόνα σας βρίσκεται σε stream (π.χ. προέρχεται από βάση δεδομένων), χρησιμοποιήστε `OcrImage.FromStream(yourStream)` αντί αυτού. Αυτό αποφεύγει προσωρινά αρχεία και μπορεί να βελτιώσει την απόδοση.

---

## Βήμα 3: Επιλογή Γλώσσας και Επεξεργασία Εικόνας με OCR

Με την εικόνα στη μνήμη, τελικά **επεξεργαζόμαστε εικόνα με OCR**. Η μέθοδος `Recognize` δέχεται τόσο την εικόνα όσο και μια τιμή του enum `Language`. Στο παράδειγμά μας επιλέγουμε τη γαλλική, αλλά μπορείτε να την αντικαταστήσετε με οποιαδήποτε γλώσσα έχετε κατεβάσει.

```csharp
        // 3️⃣ Perform OCR using the desired language (French in this case)
        var ocrResult = ocrEngine.Recognize(ocrImage, Language.French);
```

**Τι συμβαίνει στο παρασκήνιο;** Η μηχανή εκτελεί μια σειρά βημάτων προεπεξεργασίας—δυαδικοποίηση, αφαίρεση θορύβου, ανάλυση διάταξης—πριν περάσει τα δεδομένα pixel στο νευρωνικό δίκτυο OCR. Το αντικείμενο αποτελέσματος περιέχει το καθαρό κείμενο, τα σκορ εμπιστοσύνης και ακόμη και τα πλαίσια οριοθέτησης αν τα χρειάζεστε αργότερα.

## Βήμα 4: Εξαγωγή Κειμένου από Εικόνα και Εμφάνιση του

Το τελευταίο κομμάτι του παζλ είναι να **εξάγετε κείμενο από εικόνα** και να κάνετε κάτι χρήσιμο με αυτό. Στο demo αυτό απλώς γράφουμε το κείμενο στην κονσόλα, αλλά μπορείτε να το αποθηκεύσετε σε βάση δεδομένων, να το στείλετε σε ευρετήριο αναζήτησης ή να το περάσετε σε άλλη υπηρεσία.

```csharp
        // 4️⃣ Output the recognized text
        System.Console.WriteLine("=== OCR Result ===");
        System.Console.WriteLine(ocrResult.Text);
    }
}
```

Όταν εκτελέσετε το πρόγραμμα, θα πρέπει να δείτε κάτι σαν:

```
=== OCR Result ===
Bonjour, ceci est un test d'OCR hors ligne.
```

Αν η έξοδος εμφανίζεται παραμορφωμένη, ελέγξτε ξανά ότι το σωστό πακέτο γλώσσας υπάρχει στο `OCRResources`. Η έλλειψη χαρακτήρων συχνά υποδεικνύει απουσία ή μη αντιστοίχιση του αρχείου πόρου.

## Πλήρες Παράδειγμα Εργασίας (Έτοιμο για Αντιγραφή‑Επικόλληση)

Παρακάτω βρίσκεται ολόκληρο το πρόγραμμα, έτοιμο για μεταγλώττιση. Αντικαταστήστε τις διαδρομές placeholder με τις πραγματικές σας.

```csharp
using Aspose.OCR;
using Aspose.OCR.Models;

class OfflineDemo
{
    static void Main()
    {
        // Step 1 – Configure the offline OCR engine
        var ocrEngine = new OcrEngine
        {
            ResourcePath = @"C:\MyProject\OCRResources", // <-- adjust this
            AllowOnlineDownload = false
        };

        // Step 2 – Load the image you want to recognize
        var ocrImage = OcrImage.FromFile(@"C:\MyProject\offline_test.png"); // <-- adjust this

        // Step 3 – Run OCR (choose the language you need)
        var ocrResult = ocrEngine.Recognize(ocrImage, Language.French);

        // Step 4 – Display the extracted text
        System.Console.WriteLine("=== OCR Result ===");
        System.Console.WriteLine(ocrResult.Text);
    }
}
```

> **Αναμενόμενη έξοδος:** Η κονσόλα εκτυπώνει το ακριβές κείμενο που εμφανίζεται στο `offline_test.png`. Αν η εικόνα περιέχει αγγλικά, αλλάξτε το `Language.French` σε `Language.English`.

## Συχνές Ερωτήσεις & Ακραίες Περιπτώσεις

| Ερώτηση | Απάντηση |
|----------|--------|
| *Τι γίνεται αν χρειάζομαι πολλαπλές γλώσσες σε μία εικόνα;* | Καλέστε το `Recognize` δύο φορές—μία για κάθε γλώσσα—ή χρησιμοποιήστε το `Language.AutoDetect` (αν ενεργοποιήσετε διαδικτυακούς πόρους). |
| *Η εικόνα μου είναι multi‑page TIFF· μπορώ να επεξεργαστώ όλες τις σελίδες;* | Ναι. Κάντε βρόχο σε κάθε σελίδα με `OcrImage.FromMultiPageFile` και περάστε κάθε τμήμα στο `Recognize`. |
| *Πώς βελτιώνω την ακρίβεια σε σαρώσεις χαμηλής ποιότητας;* | Προεπεξεργαστείτε το bitmap μόνοι σας (π.χ. αυξήστε την αντίθεση, διορθώστε την κλίση) πριν το περάσετε στο `OcrImage`. |
| *Μπορώ να τρέξω αυτό σε Docker container;* | Απόλυτα. Απλώς αντιγράψτε το φάκελο `OCRResources` στην εικόνα του container και ορίστε το `ResourcePath` αναλόγως. |
| *Υπάρχει τρόπος να λάβω τα σκορ εμπιστοσύνης;* | Το αντικείμενο `OcrResult` εκθέτει την ιδιότητα `Confidence` ανά χαρακτήρα· επαναλάβετε πάνω στο `ocrResult.Characters` αν χρειάζεστε λεπτομερή δεδομένα. |

## Pro Tips για OCR Έτοιμο για Παραγωγή

1. **Cache τη μηχανή** – Η δημιουργία νέου `OcrEngine` ανά αίτηση προσθέτει επιβάρυνση. Διατηρήστε μια singleton παρουσία αν η εφαρμογή σας επεξεργάζεται πολλές εικόνες.  
2. **Επικυρώστε το μέγεθος εισόδου** – Πολύ μεγάλες εικόνες μπορούν να προκαλέσουν εξαιρέσεις OutOfMemory. Αλλάξτε το μέγεθος σε λογικό DPI (300 dpi είναι καλή ισορροπία).  
3. **Ασφάλεια νήματος** – Η μηχανή είναι thread‑safe, αλλά τα υποκείμενα αρχεία πόρων είναι μόνο‑ανάγνωση, οπότε μπορείτε να παραλληλοποιήσετε τις κλήσεις με ασφάλεια.  
4. **Καταγραφή (Logging)** – Καταγράψτε το `ocrResult.Text` και τυχόν σφάλματα σε δομημένο log· αυτό βοηθά όταν χρειάζεται να ελέγξετε τα αποτελέσματα OCR για συμμόρφωση.
