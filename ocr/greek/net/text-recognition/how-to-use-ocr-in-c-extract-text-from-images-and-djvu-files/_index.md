Στο τέλος θα έχετε μια έτοιμη εφαρμογή κονσόλας που εκτυπώνει το αναγνωρισμένο κείμενο απευθείας στην κονσόλα. Χωρίς μυστικά, χωρίς επιπλέον εξαρτήσεις, μόνο καθαρό C# και Aspose.

## Πώς να Χρησιμοποιήσετε OCR με το Aspose σε C#

Αυτό είναι το βασικό βήμα όπου απαντάμε στην ερώτηση **πώς να χρησιμοποιήσετε OCR** σε ένα έργο C#. Ο κώδικας παρακάτω είναι ένα πλήρες πρόγραμμα· μπορείτε να τον τοποθετήσετε σε μια νέα εφαρμογή κονσόλας και να πατήσετε F5.

```csharp
using System;
using Aspose.OCR;

namespace AsposeOcrDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // Step 1: Create the OCR engine instance.
            var ocrEngine = new OcrEngine();

            // Step 2: Load the image (or DJVU page) you want to analyze.
            // Replace the path with your own file location.
            ocrEngine.Image = ImageStream.FromFile("YOUR_DIRECTORY/book-page.djvu");

            // Step 3: Run the recognition process.
            OcrResult ocrResult = ocrEngine.Recognize();

            // Step 4: Output the extracted text.
            Console.WriteLine("=== OCR Result ===");
            Console.WriteLine(ocrResult.Text);
        }
    }
}
```

**Γιατί λειτουργεί:**  
- `OcrEngine` είναι το σημείο εισόδου· αφαιρεί το βάρος της προεπεξεργασίας εικόνας, της ανίχνευσης γλώσσας και της ταξινόμησης χαρακτήρων.  
- `ImageStream.FromFile` μπορεί να διαχειριστεί πολλές μορφές, συμπεριλαμβανομένου του DJVU, πράγμα που σημαίνει ότι μπορείτε να **μετατρέψετε DJVU σε κείμενο** χωρίς επιπλέον βήμα μετατροπής.  
- `Recognize()` επιστρέφει ένα αντικείμενο `OcrResult` που περιέχει το καθαρό κείμενο, τις βαθμολογίες εμπιστοσύνης και ακόμη τα πλαίσια περιορισμού αν τα χρειαστείτε αργότερα.

Η εκτέλεση του προγράμματος εκτυπώνει κάτι όπως:

```
=== OCR Result ===
In the beginning God created the heavens and the earth.
```

Αυτό είναι—η πρώτη σας επιτυχημένη **εξαγωγή κειμένου από DJVU**.

## Φόρτωση Εικόνας για OCR

Πριν η μηχανή μπορέσει να διαβάσει οτιδήποτε, πρέπει να **φορτώσετε την εικόνα για OCR** σωστά. Το Aspose υποστηρίζει τις περισσότερες μορφές raster αμέσως, αλλά μερικές συμβουλές μπορούν να κάνουν την αναγνώριση πιο ομαλή:

- **Αλλάξτε το μέγεθος μεγάλων εικόνων** σε μέγιστο 2000 px στην μεγαλύτερη πλευρά. Πολύ μεγάλα αρχεία αυξάνουν τη χρήση μνήμης και μπορεί να επιβραδύνουν τη μηχανή.
- **Μετατρέψτε χρωματικές εικόνες σε γκρι κλίμακα** αν παρατηρήσετε θορυβώδη φόντο. Χρησιμοποιήστε `ocrEngine.Image = ImageStream.FromFile(path).ConvertToGrayscale();`.
- **Ορίστε DPI χειροκίνητα** για σαρώσεις χαμηλής ανάλυσης (`ocrEngine.Image.DpiX = 300; ocrEngine.Image.DpiY = 300;`).

Αυτές οι ρυθμίσεις είναι προαιρετικές αλλά συχνά βελτιώνουν την ακρίβεια όταν **εξάγετε κείμενο από εικόνα** όπως σαρωμένες αποδείξεις.

## Εξαγωγή Κειμένου από Εικόνα – Καλές Πρακτικές

Όταν εργάζεστε με τυπικές μορφές εικόνας (PNG, JPEG, BMP), τα βήματα παραμένουν τα ίδια, αλλά μπορείτε να ρυθμίσετε τη μηχανή:

```csharp
ocrEngine.Image = ImageStream.FromFile("sample.png");

// Optional: enable language packs for better accuracy.
ocrEngine.Language = Language.English;

// Optional: set recognition mode to auto.
ocrEngine.RecognitionMode = RecognitionMode.Auto;
```

- **Η επιλογή γλώσσας** έχει σημασία. Αν επεξεργάζεστε πολυγλωσσικά έγγραφα, ορίστε `ocrEngine.Language = Language.Multilingual;`.
- **RecognitionMode** μπορεί να είναι `Auto`, `Fast` ή `Accurate`. Το `Accurate` προσφέρει υψηλότερη ποιότητα με κόστος την ταχύτητα—χρησιμοποιήστε το για αρχειοθετημένες εργασίες.

## Μετατροπή DJVU σε Κείμενο – Διαχείριση Αρχείων Πολλών Σελίδων

Τα αρχεία DJVU συχνά περιέχουν πολλές σελίδες. Το Aspose αντιμετωπίζει κάθε σελίδα ως ξεχωριστό ρεύμα εικόνας, έτσι μπορείτε να τις επαναλάβετε:

```csharp
using Aspose.OCR;
using System.Collections.Generic;

var ocrEngine = new OcrEngine();
List<string> allPagesText = new List<string>();

int pageCount = ImageStream.GetPageCount("book.djvu");
for (int i = 1; i <= pageCount; i++)
{
    ocrEngine.Image = ImageStream.FromFile("book.djvu", i); // Load page i
    OcrResult result = ocrEngine.Recognize();
    allPagesText.Add(result.Text);
}

// Combine all pages into a single string.
string fullText = string.Join("\n--- Page Break ---\n", allPagesText);
Console.WriteLine(fullText);
```

**Γιατί επανάληψη;**  
Τα αρχεία DJVU είναι ουσιαστικά μια στοίβα εικόνων. Επαναλαμβάνοντας κάθε σελίδα, **εξάγετε κείμενο από DJVU** με τη σωστή σειρά, διατηρώντας τη ροή του αρχικού εγγράφου.

## Συνηθισμένες Παγίδες και Επαγγελματικές Συμβουλές

| Πρόβλημα | Γιατί συμβαίνει | Διόρθωση |
|------|----------------|-----|
| **Ακατάλληλοι χαρακτήρες** | Χαμηλό DPI ή έντονη συμπίεση | Ανεβάστε την ανάλυση, ορίστε `ocrEngine.Image.DpiX/Y = 300` |
| **Αργή επεξεργασία** | Χρήση λειτουργίας `Accurate` σε τεράστια αρχεία | Μεταβείτε σε λειτουργία `Fast` για μαζικές εργασίες |
| **Έλλειψη υποστήριξης γλώσσας** | Η προεπιλεγμένη γλώσσα είναι μόνο Αγγλικά | Φορτώστε επιπλέον πακέτα γλώσσας (`ocrEngine.Language = Language.Spanish;`) |
| **Σφάλματα έλλειψης μνήμης** | Φόρτωση πολλών υψηλής ανάλυσης σελίδων ταυτόχρονα | Επεξεργαστείτε τις σελίδες μία‑μία και ελευθερώστε πόρους (`ocrEngine.Dispose();`) |

Μια γρήγορη **επαγγελματική συμβουλή**: πάντα καλέστε `ocrEngine.Dispose()` μετά το τέλος μιας σελίδας. Απελευθερώνει εγγενείς πόρους και αποτρέπει διακριτές διαρροές μνήμης που μπορούν να καταρρεύσουν υπηρεσίες μακράς διάρκειας.

## Δοκιμή του OCR Pipeline Σας

Για να επαληθεύσετε ότι όλα λειτουργούν, δοκιμάστε αυτούς τους απλούς ελέγχους:

1. **Εκτυπώστε το μήκος** της αναγνωρισμένης συμβολοσειράς: `Console.WriteLine($"Characters recognized: {ocrResult.Text.Length}");`
2. **Συγκρίνετε με γνωστό κείμενο** χρησιμοποιώντας μια απλή βιβλιοθήκη diff αν έχετε πραγματικά δεδομένα.
3. **Καταγράψτε τις βαθμολογίες εμπιστοσύνης** (`ocrResult.Confidence`) για να εντοπίσετε αυτόματα σελίδες χαμηλής ποιότητας.

Αν η εμπιστοσύνη είναι συνεχώς κάτω από 0,7, επανεξετάστε τα βήματα προεπεξεργασίας εικόνας που αναφέρθηκαν νωρίτερα.
