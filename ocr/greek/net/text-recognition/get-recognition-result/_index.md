## Εισαγωγή Namespaces

Στην εφαρμογή .NET, ξεκινήστε εισάγοντας τα απαιτούμενα namespaces:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Καθορίστε το φάκελο που περιέχει την εικόνα που θέλετε να επεξεργαστείτε:

```csharp
string dataDir = "Your Document Directory";
```

## Βήμα 2: Αρχικοποιήστε το Aspose.OCR

Δημιουργήστε μια παρουσία της μηχανής OCR:

```csharp
AsposeOcr api = new AsposeOcr();
```

## Βήμα 3: Καθορίστε τη Διαδρομή της Εικόνας

Δείξτε στο ακριβές αρχείο εικόνας που θέλετε να αναγνωρίσετε:

```csharp
string fullPath = dataDir + "sample.png";
```

## Βήμα 4: Διαμορφώστε τις Ρυθμίσεις Αναγνώρισης

Ρυθμίστε τις παραμέτρους ώστε να ταιριάζουν με το σενάριό σας—είτε χρειάζεστε προεπιλεγμένη συμπεριφορά είτε προσαρμοσμένες επιλογές όπως η επιλογή γλώσσας για πολυγλωσσική αναγνώριση κειμένου:

```csharp
RecognitionSettings settings = new RecognitionSettings
{
    // Specify your recognition settings here
    // Example: Language = Language.English | Language.Spanish
};
```

## Βήμα 5: Εκτελέστε την Αναγνώριση Εικόνας

Τρέξτε τη διαδικασία OCR και καταγράψτε το αποτέλεσμα:

```csharp
RecognitionResult result = api.RecognizeImage(fullPath, settings);
```

## Βήμα 6: Εκτυπώστε το Αποτέλεσμα Αναγνώρισης

Εμφανίστε την πλήρη έξοδο αναγνώρισης, η οποία περιλαμβάνει το εξαγόμενο κείμενο, πληροφορίες διάταξης, αναπαράσταση JSON και τυχόν προειδοποιήσεις:

```csharp
PrintRecognitionResult(result);
```

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **Δεν επιστράφηκε κείμενο** | Λάθος διαδρομή εικόνας ή μη υποστηριζόμενη μορφή | Επαληθεύστε το `fullPath` και βεβαιωθείτε ότι η εικόνα είναι υποστηριζόμενου τύπου (PNG, JPEG, BMP). |
| **Λανθασμένη ανίχνευση γλώσσας** | Οι προεπιλεγμένες ρυθμίσεις γλώσσας μπορεί να μην ταιριάζουν με την εικόνα | Ορίστε `settings.Language` στις κατάλληλες γλώσσες για καλύτερη ακρίβεια. |
| **Μείωση απόδοσης σε μεγάλες εικόνες** | Οι εικόνες υψηλής ανάλυσης αυξάνουν το χρόνο επεξεργασίας | Αλλάξτε το μέγεθος της εικόνας πριν από την αναγνώριση ή ρυθμίστε το `settings.Dpi` σε χαμηλότερη τιμή. |
| **Χαμηλή ακρίβεια σε σαρωμένα έγγραφα** | Οι σαρωμένες εικόνες μπορεί να περιέχουν θόρυβο | Χρησιμοποιήστε βήματα προεπεξεργασίας όπως η δυαδικοποίηση ή εφαρμόστε `settings.Preprocess = true` για **improve ocr accuracy**. |
| **Απαιτείται διαχείριση σαρωμένου PDF** | Το PDF πρέπει πρώτα να μετατραπεί σε εικόνες | **Convert scanned image** σελίδες σε PNG/JPEG χρησιμοποιώντας μια βιβλιοθήκη PDF‑to‑image, έπειτα τροφοδοτήστε κάθε εικόνα στο Aspose.OCR. |
