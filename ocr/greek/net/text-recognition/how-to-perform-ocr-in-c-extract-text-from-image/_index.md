# Πώς να Εκτελέσετε OCR σε C# – Εξαγωγή Κειμένου από Εικόνα

Έχετε αναρωτηθεί ποτέ **πώς να εκτελέσετε OCR** σε μια φωτογραφία χειρόγραφης σημείωσης; Ίσως έχετε ένα σαρωμένο απόδειξη, μια σήμανση σε ξένη γλώσσα ή μια σελίδα PDF που δεν επιτρέπει αντιγραφή‑επικόλληση. Τα καλά νέα; Με λίγες γραμμές C# και τη βιβλιοθήκη Aspose OCR μπορείτε να μετατρέψετε αυτήν την εικόνα σε επεξεργάσιμο κείμενο σε μια στιγμή.  

Σε αυτόν τον οδηγό δεν θα σας δείξουμε μόνο **πώς να εκτελέσετε OCR**, αλλά θα περάσουμε επίσης από **εξαγωγή κειμένου από εικόνα**, **μετατροπή εικόνας σε κείμενο**, και ακόμη **πώς να ελέγξετε την ορθογραφία** του αποτελέσματος όταν εργάζεστε με έγγραφα στη ρωσική γλώσσα. Ακούγεται πολύ; Μείνετε μαζί μας – θα εξηγήσουμε τα πάντα βήμα προς βήμα.

## Τι Θα Μάθετε

* Ρυθμίστε το Aspose OCR για υποστήριξη της ρωσικής γλώσσας.  
* Φορτώστε ένα αρχείο εικόνας και εκτελέστε οπτική αναγνώριση χαρακτήρων για **εξαγωγή κειμένου από εικόνα**.  
* Χρησιμοποιήστε τον ενσωματωμένο ορθογραφικό έλεγχο για αυτόματη διόρθωση λανθασμένων λέξεων – η τέλεια απάντηση στο “**πώς να ελέγξετε την ορθογραφία**” των αποτελεσμάτων OCR.  
* Εκτυπώστε τη διορθωμένη συμβολοσειρά στην κονσόλα, έτοιμη για περαιτέρω επεξεργασία ή αποθήκευση.

**Απαιτούμενα** – θα χρειαστείτε:

* .NET 6.0 ή νεότερο (ο κώδικας λειτουργεί επίσης με .NET Framework 4.8).  
* Ένα έγκυρο άδεια Aspose OCR ή ένα προσωρινό κλειδί αξιολόγησης.  
* Ένα αρχείο εικόνας που περιέχει ρωσικό κείμενο (για την επίδειξη θα το ονομάσουμε `russian_note.jpg`).  

Αν κάτι από αυτά σας φαίνεται άγνωστο, μην ανησυχείτε. Τα παρακάτω βήματα περιλαμβάνουν την ακριβή εντολή NuGet για την προσθήκη του Aspose OCR, και ο κώδικας είναι πλήρως αυτόνομος.

## Βήμα 1 – Εγκατάσταση Aspose OCR και Προσθήκη Namespaces

Πρώτα απ' όλα, χρειάζεστε τη βιβλιοθήκη. Ανοίξτε ένα τερματικό στον φάκελο του έργου σας και εκτελέστε:

```bash
dotnet add package Aspose.OCR
```

Αυτή η εντολή κατεβάζει την πιο πρόσφατη σταθερή έκδοση (ως Απρίλιο 2026 είναι 22.9). Μετά την επαναφορά του πακέτου, προσθέστε τις απαιτούμενες οδηγίες `using` στην αρχή του αρχείου C# σας:

```csharp
using Aspose.OCR;
using Aspose.OCR.SpellCheck;
using System.Drawing;
```

*Συμβουλή:* Εάν χρησιμοποιείτε Visual Studio, το IDE θα προτείνει την αυτόματη προσθήκη αυτών μόλις πληκτρολογήσετε το πρώτο όνομα κλάσης.

## Βήμα 2 – Αρχικοποίηση του OCR Engine για τη Ρωσική Γλώσσα

Το μέρος **πώς να εκτελέσετε OCR** ξεκινά με τη δημιουργία μιας παρουσίας `OcrEngine`. Καθορίζοντας `OcrLanguage.Russian` λέμε στο engine να φορτώσει το κυριλλικό σύνολο χαρακτήρων και τις γλωσσικές‑συγκεκριμένες ευρετικές.

```csharp
// Step 2: Initialise OCR engine for Russian
OcrEngine ocrEngine = new OcrEngine
{
    Language = OcrLanguage.Russian   // Enables Russian language support
};
```

Γιατί είναι σημαντικό αυτό; Χωρίς τον καθορισμό της γλώσσας, το engine προεπιλέγει τα Αγγλικά και θα ερμηνεύσει λανθασμένα πολλούς κυριλλικούς χαρακτήρες, οδηγώντας σε ακατάληπτο αποτέλεσμα. Η ρητή ρύθμιση της γλώσσας βελτιώνει δραματικά την ακρίβεια.

## Βήμα 3 – Φόρτωση της Εικόνας και **Μετατροπή Εικόνας σε Κείμενο**

Τώρα φέρνουμε την εικόνα στη μνήμη. Η μέθοδος `Image.FromFile` λειτουργεί με τις πιο κοινές μορφές (JPG, PNG, BMP). Μετά τη φόρτωση, καλούμε το `Recognize` για **μετατροπή εικόνας σε κείμενο**.

```csharp
// Step 3: Load the image that contains Russian text
Image sourceImage = Image.FromFile(@"YOUR_DIRECTORY/russian_note.jpg");

// Step 4: Perform OCR – this is the core “how to perform OCR” call
string rawText = ocrEngine.Recognize(sourceImage);
```

Η μεταβλητή `rawText` τώρα περιέχει το ακατέργαστο αποτέλεσμα OCR, το οποίο μπορεί ακόμη να περιέχει τυπογραφικά λάθη ή λανθασμένα αναγνωρισμένους χαρακτήρες. Μπορείτε να το εκτυπώσετε για να δείτε τι κατέγραψε το engine:

```csharp
Console.WriteLine("Raw OCR output:");
Console.WriteLine(rawText);
```

## Βήμα 4 – **Πώς να Ελέγξετε την Ορθογραφία** του Αποτελέσματος OCR

Το ρωσικό OCR μπορεί να είναι θορυβώδες, ειδικά με σαρώσεις χαμηλής ανάλυσης. Η Aspose παρέχει μια κλάση `SpellChecker` που κατανοεί τα ρωσικά λεξικά αμέσως. Να πώς την καλείτε:

```csharp
// Step 5: Initialise the spell checker
SpellChecker spellChecker = new SpellChecker();

// Step 6: Correct spelling mistakes in the recognized text
string correctedText = spellChecker.CheckAndCorrect(rawText, OcrLanguage.Russian);
```

Η μέθοδος `CheckAndCorrect` επιστρέφει μια νέα συμβολοσειρά όπου οι λανθασμένες λέξεις αντικαθίστανται με τις πιο πιθανές σωστές εναλλακτικές. Σεβαστεί επίσης την στίξη, ώστε να μην καταλήξετε με ένα τείχος κειμένου.

*Περίπτωση άκρης:* Εάν το αποτέλεσμα OCR είναι κενό (π.χ., η εικόνα είναι εντελώς λευκή), το `CheckAndCorrect` θα επιστρέψει απλώς μια κενή συμβολοσειρά. Είναι καλή ιδέα να το ελέγχετε αν σκοπεύετε να γράψετε το αποτέλεσμα σε αρχείο.

## Βήμα 5 – Εμφάνιση του Καθαρισμένου Αποτελέσματος

Τέλος, εμφανίζουμε τη διορθωμένη συμβολοσειρά. Σε μια πραγματική εφαρμογή μπορεί να τη γράψετε σε βάση δεδομένων, σε API JSON ή σε έγγραφο Word. Για αυτήν την επίδειξη, μια εκτύπωση στην κονσόλα αρκεί:

```csharp
// Step 7: Show the corrected result
Console.WriteLine("\nCorrected text:");
Console.WriteLine(correctedText);
```

Όταν εκτελέσετε το πρόγραμμα, θα πρέπει να δείτε κάτι σαν:

```
Raw OCR output:
Привeт мир! Этo тeкcт c русcкoй оcнoвнoй.

Corrected text:
Привет мир! Это текст с русской основой.
```

Παρατηρήστε πώς ο ορθογραφικός έλεγχος μετατρέπει το “Привeт” (μικτή Λατινική ‘e’) στο σωστό κυριλλικό “Привет”. Αυτή είναι η μαγεία του **πώς να ελέγξετε την ορθογραφία** των αποτελεσμάτων OCR.

## Πλήρες Παράδειγμα Λειτουργίας

Παρακάτω βρίσκεται το πλήρες, εκτελέσιμο πρόγραμμα που ενώνει όλα τα βήματα. Αντιγράψτε‑επικολλήστε το σε ένα νέο έργο κονσόλας και πατήστε **F5**.

```csharp
using Aspose.OCR;
using Aspose.OCR.SpellCheck;
using System.Drawing;

class SpellCheckDemo
{
    static void Main()
    {
        // Step 1: Create an OCR engine configured for Russian language
        OcrEngine ocrEngine = new OcrEngine { Language = OcrLanguage.Russian };

        // Step 2: Load the image that contains the text to be recognized
        Image sourceImage = Image.FromFile(@"YOUR_DIRECTORY/russian_note.jpg");

        // Step 3: Perform OCR to extract raw text from the image
        string rawText = ocrEngine.Recognize(sourceImage);

        // Optional: Show raw OCR output
        System.Console.WriteLine("Raw OCR output:");
        System.Console.WriteLine(rawText);

        // Step 4: Initialise the spell checker
        SpellChecker spellChecker = new SpellChecker();

        // Step 5: Correct spelling mistakes in the recognized text
        string correctedText = spellChecker.CheckAndCorrect(rawText, OcrLanguage.Russian);

        // Step 6: Display the corrected result
        System.Console.WriteLine("\nCorrected text:");
        System.Console.WriteLine(correctedText);
    }
}
```

### Αναμενόμενο Αποτέλεσμα

Η εκτέλεση του προγράμματος με μια καθαρή, υψηλής ανάλυσης φωτογραφία ρωσικής χειρογραφίας συνήθως δίνει μια καθαρή, αναγνώσιμη πρόταση. Εάν η εικόνα είναι θολή, μπορεί να λάβετε μερικές σωστές λέξεις, αλλά ο ορθογραφικός έλεγχος θα εξομαλύνει τα περισσότερα προφανή σφάλματα.

## Συνηθισμένα Προβλήματα & Συμβουλές

| Πρόβλημα | Γιατί Συμβαίνει | Πώς να Διορθώσετε / Αποφύγετε |
|----------|----------------|------------------------------|
| **Αχρείαστοι χαρακτήρες** | Χαμηλό DPI ή θορυβώδες φόντο | Προεπεξεργασία της εικόνας (αύξηση αντίθεσης, αλλαγή μεγέθους σε ≥300 dpi) πριν την περάσετε στο `Recognize`. |
| **Κενό αποτέλεσμα** | Λάθος διαδρομή αρχείου ή μη υποστηριζόμενη μορφή | Επαληθεύστε τη διαδρομή και χρησιμοποιήστε `Image.FromFile` μέσα σε μπλοκ `try/catch` για να εμφανιστούν τα σφάλματα. |
| **Ο ορθογραφικός έλεγχος παραλείπει σφάλματα** | Σπάνιες λέξεις που δεν υπάρχουν στο λεξικό | Επεκτείνετε το λεξικό φορτώνοντας μια προσαρμοσμένη λίστα λέξεων μέσω `spellChecker.AddUserDictionary("mywords.txt")`. |
| **Καθυστέρηση απόδοσης σε μεγάλες παρτίδες** | Το OCR είναι εντατικό σε CPU | Εκτελέστε OCR σε νήμα παρασκηνίου ή χρησιμοποιήστε `Parallel.ForEach` για πολλαπλές εικόνες. |
| **Απόρριψη άδειας** | Χρήση της έκδοσης αξιολόγησης πέραν της δοκιμαστικής περιόδου | Αγοράστε άδεια και καλέστε `License license = new License(); license.SetLicense("Aspose.Total.lic");` πριν δημιουργήσετε το `OcrEngine`. |
