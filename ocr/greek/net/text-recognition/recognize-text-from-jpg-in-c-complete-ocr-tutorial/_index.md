# Αναγνώριση Κειμένου από JPG σε C# – Πλήρης Οδηγός OCR

Ποτέ χρειάστηκε να **αναγνωρίσετε κείμενο από αρχεία JPG** αλλά δεν ήξερες ποια βιβλιοθήκη να επιλέξεις; Δεν είσαι μόνος/η. Πολλοί προγραμματιστές αντιμετωπίζουν το ίδιο πρόβλημα όταν προσπαθούν για πρώτη φορά να εξάγουν κείμενο από εικόνες, ειδικά όταν η πηγή είναι JPEG.  

Σε αυτόν τον οδηγό θα περάσουμε βήμα‑βήμα από ένα **παράδειγμα C# OCR** που φορτώνει ένα JPG, εκτελεί οπτική αναγνώριση χαρακτήρων και εκτυπώνει το αποτέλεσμα στην κονσόλα. Στο τέλος θα μπορείτε να **εξάγετε κείμενο από εικόνα**, **μετατρέψετε εικόνα σε κείμενο**, και ακόμη να προσαρμόσετε τον κώδικα για άλλες μορφές. Χωρίς περιττές εξηγήσεις—απλώς μια λειτουργική λύση που μπορείτε να αντιγράψετε‑και‑επικολλήσετε.

## Τι Θα Μάθετε

- Πώς να ενεργοποιήσετε τη λειτουργία δοκιμής για Aspose.OCR (ή να φορτώσετε κλειδί άδειας)
- Τα ακριβή βήματα για **φόρτωση εικόνας για OCR** σε έργο C#
- Πώς να καλέσετε τη μηχανή OCR και να ανακτήσετε το αναγνωρισμένο κείμενο
- Συμβουλές για την αντιμετώπιση κοινών προβλημάτων όπως χαμηλής ανάλυσης JPG ή διαρροές μνήμης
- Πού να πάτε μετά αν χρειάζεστε πολυ‑σελίδες PDF ή λεξικά συγκεκριμένων γλωσσών

**Προαπαιτούμενα**  
Θα χρειαστείτε .NET 6+ (ή .NET Framework 4.6+), Visual Studio 2022 (ή το αγαπημένο σας IDE) και το πακέτο NuGet Aspose.OCR. Αν δεν έχετε εγκαταστήσει ακόμη το πακέτο, εκτελέστε:

```bash
dotnet add package Aspose.OCR
```

Τώρα που έχουμε θέσει τα θεμέλια, ας βουτήξουμε στον κώδικα.

![recognize text from jpg example](/images/recognize-text-from-jpg.png "Screenshot showing C# console output after recognizing text from a JPG file")

## Βήμα 1 – Ενεργοποίηση Λειτουργίας Δοκιμής (ή Εφαρμογή Άδειας)

Πριν η μηχανή OCR μπορέσει να κάνει οτιδήποτε, η Aspose απαιτεί να ενεργοποιήσετε τη λειτουργία δοκιμής ή να φορτώσετε ένα έγκυρο αρχείο άδειας. Η παράλειψη αυτού του βήματος θα προκαλέσει εξαίρεση κατά την εκτέλεση.

```csharp
using Aspose.OCR;

// Enable the free trial – remove this line once you have a license
OcrEngine.EnableTrialMode();
```

*Γιατί είναι σημαντικό*: Η λειτουργία δοκιμής αφαιρεί το υδατογράφημα “evaluation” και ξεκλειδώνει το πλήρες σύνολο λειτουργιών για περιορισμένο χρονικό διάστημα. Αν αργότερα προσθέσετε άδεια, απλώς αντικαταστήστε την κλήση `EnableTrialMode` με `OcrEngine.SetLicense("YourLicenseFile.lic");`.

## Βήμα 2 – Δημιουργία Αντικειμένου OCR Engine

Η κλάση `OcrEngine` είναι η καρδιά της βιβλιοθήκης. Η δημιουργία ενός αντικειμένου ανά εφαρμογή είναι συνήθως επαρκής, αλλά μπορείτε να δημιουργήσετε πολλαπλά αν χρειάζεστε διαφορετικές ρυθμίσεις γλώσσας.

```csharp
// Initialize the OCR engine
var ocrEngine = new OcrEngine();
```

*Συμβουλή*: Αν σκοπεύετε να επεξεργάζεστε πολλές εικόνες σε βρόχο, επαναχρησιμοποιήστε το ίδιο αντικείμενο `ocrEngine`. Μειώνει το κόστος και επιταχύνει την επεξεργασία παρτίδας.

## Βήμα 3 – Φόρτωση της Εικόνας JPG που Θέλετε να Επεξεργαστείτε

Εδώ **φορτώνουμε εικόνα για OCR**. Η Aspose.OCR λειτουργεί με την κλάση `Image` από το ίδιο namespace, οπότε δεν χρειάζεται το System.Drawing.

```csharp
// Replace the path with your actual JPG location
var imagePath = @"C:\Images\sample.jpg";
var image = Image.Load(imagePath);
```

*Τι γίνεται αν το αρχείο δεν είναι JPG;*  
Η Aspose μπορεί να χειριστεί PNG, BMP, TIFF και ακόμη σελίδες PDF. Απλώς αλλάξτε την επέκταση του αρχείου και η κλήση `Image.Load` θα κάνει το υπόλοιπο.

## Βήμα 4 – Αναγνώριση Κειμένου από τη Φορτωμένη Εικόνα

Τώρα καλούμε τη μέθοδο `Recognize`. Επιστρέφει ένα αντικείμενο `OcrResult` που περιέχει το εξαγόμενο κείμενο, τις βαθμολογίες εμπιστοσύνης και πληροφορίες διάταξης.

```csharp
// Run OCR on the loaded image
var ocrResult = ocrEngine.Recognize(image);
```

*Γιατί χρησιμοποιούμε ξεχωριστή μεταβλητή*: Η αποθήκευση του αποτελέσματος σας επιτρέπει να ελέγξετε το `ocrResult.Confidence` ή τα `ocrResult.TextBlocks` αργότερα, κάτι χρήσιμο για εντοπισμό σφαλμάτων ή μετα‑επεξεργασία.

## Βήμα 5 – Εμφάνιση (ή Αποθήκευση) του Αναγνωρισμένου Κειμένου

Τέλος, εκτυπώνουμε το αναγνωρισμένο κείμενο στην κονσόλα. Σε πραγματική εφαρμογή μπορεί να το γράψετε σε βάση δεδομένων, αρχείο ή να το στείλετε μέσω API.

```csharp
// Print the extracted text
Console.WriteLine("=== Recognized Text ===");
Console.WriteLine(ocrResult.Text);
```

**Αναμενόμενο αποτέλεσμα**

```
=== Recognized Text ===
Hello, world!
This is a sample JPG image.
```

Αν το αποτέλεσμα φαίνεται ακατάληπτο, δοκιμάστε να αυξήσετε την ανάλυση της εικόνας ή να εφαρμόσετε φίλτρο προεπεξεργασίας (π.χ. ενίσχυση ή δυαδικοποίηση). Η Aspose.OCR προσφέρει επίσης `ImagePreprocessor` για πιο προχωρημένες ρυθμίσεις.

## Πλήρες Παράδειγμα Λειτουργίας

Συνδυάζοντας όλα τα παραπάνω, εδώ είναι ένα αυτόνομο πρόγραμμα που μπορείτε να μεταγλωττίσετε και να τρέξετε αμέσως:

```csharp
using System;
using Aspose.OCR;

class Program
{
    static void Main()
    {
        // 1️⃣ Enable trial mode (remove when you have a license)
        OcrEngine.EnableTrialMode();

        // 2️⃣ Create OCR engine
        var ocrEngine = new OcrEngine();

        // 3️⃣ Load the JPG image
        var imagePath = @"C:\Images\sample.jpg"; // 👉 Change to your file
        var image = Image.Load(imagePath);

        // 4️⃣ Recognize the text
        var ocrResult = ocrEngine.Recognize(image);

        // 5️⃣ Show the result
        Console.WriteLine("=== Recognized Text ===");
        Console.WriteLine(ocrResult.Text);
    }
}
```

Αντιγράψτε τον κώδικα σε ένα νέο έργο Console App, προσαρμόστε το `imagePath` και πατήστε **F5**. Θα πρέπει να δείτε το εξαγόμενο κείμενο στην κονσόλα.

## Συνηθισμένα Προβλήματα & Πώς να Τα Διορθώσετε

| Πρόβλημα | Γιατί Συμβαίνει | Γρήγορη Λύση |
|----------|----------------|--------------|
| **Ακατάλληλοι χαρακτήρες** | Χαμηλής ανάλυσης JPG ή έντονη συμπίεση | Χρησιμοποιήστε πηγή υψηλότερης ανάλυσης ή καλέστε `image = ImagePreprocessor.Binarize(image);` πριν την αναγνώριση |
| **Εξαίρεση Out‑of‑memory** | Επεξεργασία πολλών μεγάλων εικόνων σε βρόχο χωρίς απελευθέρωση | Τυλίξτε `Image.Load` και `ocrEngine` σε `using` ή καλέστε `image.Dispose();` μετά από κάθε επανάληψη |
| **Λάθος γλώσσα** | Η προεπιλεγμένη γλώσσα είναι Αγγλικά· η εικόνα περιέχει άλλη γλώσσα | Ορίστε `ocrEngine.Language = OcrLanguage.French;` (ή οποιαδήποτε υποστηριζόμενη γλώσσα) πριν το `Recognize` |
| **Αργή απόδοση** | Μονονηματική επεξεργασία πολλών αρχείων | Παράλληλη εκτέλεση με `Parallel.ForEach` και επαναχρησιμοποίηση ενός `ocrEngine` ανά νήμα |

## Επέκταση του Παραδείγματος

- **Επεξεργασία παρτίδας**: Διατρέξτε έναν φάκελο με JPG, συλλέξτε κάθε `ocrResult.Text` και γράψτε τα σε αρχείο CSV.
- **Μετατροπή σε PDF**: Μετά την εξαγωγή του κειμένου, μπορείτε να το περάσετε σε βιβλιοθήκη PDF (π.χ. Aspose.PDF) για δημιουργία αναζητήσιμων PDF.
- **Ανίχνευση γλώσσας**: Συνδυάστε Aspose.OCR με βιβλιοθήκη ανίχνευσης γλώσσας για αυτόματη επιλογή της κατάλληλης γλώσσας OCR.
