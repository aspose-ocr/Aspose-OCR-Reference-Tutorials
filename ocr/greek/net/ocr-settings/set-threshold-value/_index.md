## How to extract text using OCR with a custom threshold

Τώρα, ας αναλύσουμε τη διαδικασία ορισμού της τιμής κατωφλίου στην αναγνώριση εικόνας OCR σε εύκολα ακολουθήσιμα βήματα.

### Step 1: Define Your Document Directory

Βήμα 1: Ορίστε τον φάκελο εγγράφων σας

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";
```

### Step 2: Initialize Aspose.OCR

Βήμα 2: Αρχικοποίηση Aspose.OCR

```csharp
// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

### Step 3: Recognize Image with Custom Threshold

Βήμα 3: Αναγνώριση εικόνας με προσαρμοσμένο κατώφλι

```csharp
// Recognize image with a specific threshold value (e.g., 230)
RecognitionResult result = api.RecognizeImage(dataDir + "sample.png", new RecognitionSettings
{
    ThresholdValue = 230
});
```

### Step 4: Display Recognized Text

Βήμα 4: Εμφάνιση αναγνωρισμένου κειμένου

```csharp
// Display the recognized text
Console.WriteLine(result.RecognitionText);
```

### Step 5: Confirm Successful Execution

Βήμα 5: Επιβεβαίωση επιτυχούς εκτέλεσης

```csharp
Console.WriteLine("SetThresholdValue executed successfully");
```

Τώρα που έχετε ορίσει επιτυχώς την τιμή κατωφλίου στην αναγνώριση εικόνας OCR χρησιμοποιώντας το Aspose.OCR για .NET, μπορείτε να ενσωματώσετε αυτή τη λειτουργία στις εφαρμογές σας για βελτιωμένη εξαγωγή κειμένου.

## Why set the OCR threshold value?

Γιατί να ορίσετε την τιμή κατωφλίου του OCR;

Η ρύθμιση του κατωφλίου σας δίνει άμεσο έλεγχο στο πώς η μηχανή OCR ερμηνεύει την ένταση των pixel. Αυτό είναι ιδιαίτερα χρήσιμο όταν εργάζεστε με σαρωμένα έγγραφα, αποδείξεις ή φωτογραφίες που λήφθηκαν σε φτωχό φωτισμό. Με τη λεπτομερή ρύθμιση του κατωφλίου, μπορείτε να **εξάγετε κείμενο με OCR** πιο αξιόπιστα και να μειώσετε την ανάγκη για μετα‑επεξεργασία.

## Additional Frequently Asked Questions

Πρόσθετες Συχνές Ερωτήσεις

**Q: Επηρεάζει η αλλαγή του κατωφλίου την απόδοση;**  
A: Η επίδραση είναι ελάχιστη· η μηχανή OCR εξακολουθεί να επεξεργάζεται τον ίδιο αριθμό pixel, αλλά ακραίες τιμές μπορεί να αυξήσουν ελαφρώς το χρόνο προεπεξεργασίας.

**Q: Μπορώ να ορίσω το κατώφλι προγραμματιστικά για κάθε εικόνα;**  
A: Ναι—απλώς περάστε μια διαφορετική `ThresholdValue` στο αντικείμενο `RecognitionSettings` για κάθε κλήση της `RecognizeImage`.

**Q: Υπάρχει τρόπος να αφήσω το Aspose.OCR να καθορίσει αυτόματα το καλύτερο κατώφλι;**  
A: Παραλείποντας το `ThresholdValue`, η βιβλιοθήκη χρησιμοποιεί τον εσωτερικό προσαρμοστικό αλγόριθμο της, που λειτουργεί καλά στις περισσότερες περιπτώσεις.

**Q: Ποιοι τύποι εικόνας υποστηρίζονται;**  
A: Κοινές μορφές όπως PNG, JPEG, BMP, TIFF και GIF υποστηρίζονται πλήρως.

**Q: Πώς να διαχειριστώ PDF πολλαπλών σελίδων;**  
A: Μετατρέψτε κάθε σελίδα PDF σε εικόνα (π.χ., χρησιμοποιώντας Aspose.PDF) και στη συνέχεια εφαρμόστε τη διαδικασία OCR με το επιθυμητό κατώφλι.
