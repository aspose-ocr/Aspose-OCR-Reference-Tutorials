# Πώς να αποθηκεύσετε το αποτέλεσμα OCR ως έγγραφο

## Εισαγωγή

Σε αυτό το σεμινάριο θα ανακαλύψετε **πώς να αποθηκεύσετε το OCR** χρησιμοποιώντας το Aspose.OCR για .NET. Θα περάσουμε από την αναγνώριση κειμένου σε μια εικόνα, στη συνέχεια τη μετατροπή αυτού του κειμένου σε δημοφιλείς μορφές εγγράφων όπως DOCX, TXT, PDF, και XLSX. Στο τέλος, θα μπορείτε να αυτοματοποιήσετε την εξαγωγή δεδομένων από εικόνες και να τα αποθηκεύετε ως αναζητήσιμα, επεξεργάσιμα αρχεία — ιδανικά για αρχειοθέτηση, αναφορές ή επεξεργασία downstream.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “πώς να αποθηκεύσετε ocr”;** Αναφέρεται στην αποθήκευση του κειμένου που αναγνωρίστηκε από μια εικόνα σε μορφή αρχείου όπως DOCX, PDF, κλπ.  
- **Σε ποιες μορφές μπορώ να εξάγω;** Το DOCX, TXT, PDF και XLSX υποστηρίζονται αμέσως.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγική χρήση.  
- **Μπορώ να μετατρέψω την εικόνα σε PDF απευθείας;** Ναι — αποθηκεύστε το αποτέλεσμα OCR ως PDF για να έχετε ένα αναζητήσιμο έγγραφο PDF.  
- **Υποστηρίζεται το PNG;** Απόλυτα· μπορείτε να **εξάγετε κείμενο από PNG** εικόνες με το ίδιο API.

## Τι είναι το OCR και γιατί να αποθηκεύσετε τα αποτελέσματα ως έγγραφα;

Η Οπτική Αναγνώριση Χαρακτήρων (OCR) μετατρέπει το τυπωμένο ή χειρόγραφο κείμενο μέσα σε εικόνες σε μηχανικά αναγνώσιμες συμβολοσειρές. Η αποθήκευση αυτών των συμβολοσειρών ως έγγραφα σας επιτρέπει να:

* Δημιουργήσετε αναζητήσιμα PDF για συμμόρφωση.  
* Συμπληρώσετε λογιστικά φύλλα (XLSX) για ανάλυση δεδομένων.  
* Δημιουργήσετε επεξεργάσιμες αναφορές (DOCX).  
* Αρχειοθετήσετε αρχεία απλού κειμένου (TXT) για γρήγορη αναζήτηση.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

- Το Aspose.OCR για .NET εγκατεστημένο. Μπορείτε να το κατεβάσετε **[εδώ](https://releases.aspose.com/ocr/net/)**.  
- Έναν φάκελο στον υπολογιστή σας που θα περιέχει τις εικόνες προέλευσης και τα έγγραφα εξόδου. Ενημερώστε τη μεταβλητή `dataDir` στον κώδικα ώστε να δείχνει σε αυτόν το φάκελο.

## Εισαγωγή Namespaces

Χρειαζόμαστε μερικά namespaces του .NET για πρόσβαση σε I/O αρχείων και τις κλάσεις Aspose OCR.

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

### Βήμα 1: Αρχικοποίηση Aspose.OCR

Ορίστε τη διαδρομή προς τον εργασιακό σας φάκελο και δημιουργήστε μια παρουσία του μηχανήματος OCR.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

### Βήμα 2: Αναγνώριση εικόνας

Περάστε το αρχείο εικόνας (π.χ., PNG) στον αναγνωριστή. Εδώ είναι που **αναγνωρίζουμε εικόνες κειμένου** και τις μετατρέπουμε σε ένα `RecognitionResult`.

```csharp
// Recognize image
RecognitionResult result = api.RecognizeImage(dataDir + "sample.png", new RecognitionSettings { });
```

### Βήμα 3: Αποθήκευση αποτελέσματος σε διαφορετικές μορφές

Τώρα εξάγουμε το αναγνωρισμένο κείμενο. Επιλέξτε τη μορφή που ταιριάζει στη ροή εργασίας σας — είτε χρειάζεστε **μετατροπή εικόνας σε PDF**, **εξαγωγή κειμένου από PNG**, είτε δημιουργία λογιστικού φύλλου.

```csharp
// Save the result in your preferred format
result.Save(RunExamples.GetDataDir_OCR() + "sample.docx", SaveFormat.Docx);
result.Save(RunExamples.GetDataDir_OCR() + "sample.txt", SaveFormat.Text);
result.Save(RunExamples.GetDataDir_OCR() + "sample.pdf", SaveFormat.Pdf);
result.Save(RunExamples.GetDataDir_OCR() + "sample.xlsx", SaveFormat.Xlsx);
```

### Βήμα 4: Εμφάνιση μηνύματος επιτυχίας

Ένα απλό μήνυμα στην κονσόλα επιβεβαιώνει ότι η διαδικασία ολοκληρώθηκε χωρίς σφάλματα.

```csharp
Console.WriteLine("SaveResultAsDocument executed successfully");
```

## Κοινά προβλήματα & Συμβουλές

- **Διαδρομές αρχείων:** Χρησιμοποιείτε πάντα απόλυτες διαδρομές ή βεβαιωθείτε ότι το `dataDir` τελειώνει με διαχωριστικό διαδρομής (`\` ή `/`).  
- **Ποιότητα εικόνας:** Οι εικόνες υψηλότερης ανάλυσης βελτιώνουν την ακρίβεια· σκεφτείτε προεπεξεργασία (ευθυγράμμιση, αποθόρυβο) για καλύτερα αποτελέσματα.  
- **Λειτουργία άδειας:** Σε λειτουργία αξιολόγησης η έξοδος μπορεί να περιέχει υδατογράφημα· εφαρμόστε έγκυρη άδεια για να το αφαιρέσετε.
