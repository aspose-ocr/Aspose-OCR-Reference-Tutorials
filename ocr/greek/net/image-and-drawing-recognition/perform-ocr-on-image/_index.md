# Πώς να OCR Εικόνας – Εκτέλεση OCR σε Εικόνα στην Αναγνώριση Εικόνας OCR

## Εισαγωγή

Στις σύγχρονες εφαρμογές, **πώς να OCR εικόνα** είναι μια συχνή ερώτηση για προγραμματιστές που χρειάζονται να μετατρέψουν σαρωμένα έγγραφα, στιγμιότυπα οθόνης ή φωτογραφίες σε αναζητήσιμο, επεξεργάσιμο κείμενο. Το Aspose.OCR για .NET σας προσφέρει ένα ισχυρό, εύχρηστο API που σας επιτρέπει να **εξάγετε κείμενο από εικόνα**, **μετατρέψετε εικόνα σε κείμενο** και **αναγνωρίσετε κείμενο εικόνας** με λίγες μόνο γραμμές κώδικα. Σε αυτό το tutorial θα περάσουμε από όλη τη διαδικασία — από τη ρύθμιση της βιβλιοθήκης μέχρι την εμφάνιση του αναγνωρισμένου κειμένου — ώστε να ενσωματώσετε δυνατότητες OCR στα έργα C# σε λίγα λεπτά.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη πρέπει να χρησιμοποιήσω;** Aspose.OCR για .NET
- **Μπορώ να επεξεργαστώ PNG, JPEG και TIFF;** Ναι, υποστηρίζονται όλες οι κοινές μορφές εικόνας
- **Απαιτείται άδεια για παραγωγική χρήση;** Ναι, απαιτείται εμπορική άδεια για παραγωγική χρήση
- **Ποιες εκδόσεις .NET είναι συμβατές;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6
- **Πόσο διαρκεί μια βασική κλήση OCR;** Συνήθως κάτω από ένα δευτερόλεπτο για μια τυπική εικόνα

## Προαπαιτούμενα

Πριν βυθιστείτε στον κώδικα, βεβαιωθείτε ότι έχετε:

1. **Aspose.OCR για .NET Library** – Κατεβάστε και εγκαταστήστε τη από το [download link](https://releases.aspose.com/ocr/net/).  
2. **Περιβάλλον Ανάπτυξης** – Οποιοδήποτε IDE συμβατό με .NET (Visual Studio, Rider, VS Code κ.λπ.).  
3. **Δείγμα εικόνας** – Για αυτόν τον οδηγό θα χρησιμοποιήσουμε το `sample.png` τοποθετημένο σε φάκελο της επιλογής σας.

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

## Πώς να OCR Εικόνας χρησιμοποιώντας Aspose.OCR για .NET

Παρακάτω παρουσιάζεται η ολοκληρωμένη ροή εργασίας, χωρισμένη σε σαφή, αριθμημένα βήματα. Κάθε βήμα περιλαμβάνει μια σύντομη εξήγηση ακολουθούμενη από τον ακριβή κώδικα που πρέπει να αντιγράψετε.

### Βήμα 1: Καθορισμός Καταλόγου Εγγράφου

```csharp
string dataDir = "Your Document Directory";
```

Αντικαταστήστε το `"Your Document Directory"` με την απόλυτη ή σχετική διαδρομή που περιέχει το `sample.png`. Αυτό ενημερώνει το API πού να ψάξει για την εικόνα που θέλετε να επεξεργαστείτε.

### Βήμα 2: Αρχικοποίηση Aspose.OCR

```csharp
AsposeOcr api = new AsposeOcr();
```

Δημιουργώντας μια παρουσία του `AsposeOcr` αποκτάτε πρόσβαση σε όλες τις μεθόδους OCR, όπως η `RecognizeImage`.

### Βήμα 3: Αναγνώριση Εικόνας

```csharp
string result = api.RecognizeImage(dataDir + "sample.png");
```

Η μέθοδος `RecognizeImage` διαβάζει το αρχείο εικόνας και επιστρέφει το εξαγόμενο κείμενο ως συμβολοσειρά. Εδώ πραγματοποιείται η κύρια εργασία — **αναγνώριση κειμένου εικόνας**.

### Βήμα 4: Εμφάνιση Αναγνωρισμένου Κειμένου

```csharp
Console.WriteLine(result);
```

Μπορείτε είτε να εκτυπώσετε το αποτέλεσμα στην κονσόλα, να το γράψετε σε αρχείο, ή να το περάσετε σε άλλο στοιχείο για περαιτέρω επεξεργασία.

### Βήμα 5: Ολοκλήρωση Διαδικασίας

```csharp
Console.WriteLine("PerformOCROnImage executed successfully");
```

Ένα απλό μήνυμα επιβεβαίωσης σας βοηθά να βεβαιωθείτε ότι η κλήση OCR ολοκληρώθηκε χωρίς εξαιρέσεις.

- **Ευρεία Υποστήριξη Μορφών** – Διαχειρίζεται PNG, JPEG, BMP, TIFF και άλλα, καθιστώντας εύκολη τη **μετατροπή εικόνας σε κείμενο** ανεξαρτήτως πηγής.
