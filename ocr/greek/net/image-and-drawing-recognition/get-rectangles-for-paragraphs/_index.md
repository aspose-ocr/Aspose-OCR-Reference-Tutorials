# Πώς να Εξάγετε Ορθογώνια για Παραγράφους στην OCR Αναγνώριση Εικόνας

## Εισαγωγή

Καλώς ήρθατε στον ολοκληρωμένο οδηγό μας για **πώς να εξάγετε ορθογώνια** για παραγράφους στην OCR αναγνώριση εικόνας με το Aspose.OCR για .NET. Αν θέλετε να βελτιώσετε τη ροή επεξεργασίας εγγράφων, να εντοπίσετε τα όρια των παραγράφων και να αυτοματοποιήσετε την εξαγωγή δεδομένων, βρίσκεστε στο σωστό μέρος. Θα περάσουμε από κάθε βήμα — από τη ρύθμιση του περιβάλλοντος μέχρι την εκτύπωση των συντεταγμένων των ορθογωνίων — ώστε να μπορείτε να αρχίσετε να χρησιμοποιείτε τα αποτελέσματα OCR αμέσως.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “εξαγωγή ορθογωνίων”;** Επιστρέφει τα πλαίσια οριοθέτησης (x, y, width, height) των εντοπισμένων περιοχών κειμένου.  
- **Ποια μέθοδος API παρέχει τα ορθογώνια;** `AsposeOcr.GetRectangles` με `AreasType.PARAGRAPHS`.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να επεξεργαστώ πολλαπλές εικόνες ταυτόχρονα;** Ναι — κάντε βρόχο στη λίστα εικόνων σας και καλέστε `GetRectangles` για κάθε αρχείο.  
- **Ποιοι τύποι αρχείων υποστηρίζονται;** PNG, JPEG, TIFF, BMP και πολλοί άλλοι.

## Τι σημαίνει “πώς να εξάγετε ορθογώνια” στην OCR;
Στην ορολογία της OCR, η εξαγωγή ορθογωνίων σημαίνει την ταυτοποίηση των γεωμετρικών ορίων που περικλείουν κάθε παράγραφο ή γραμμή κειμένου μέσα σε μια εικόνα. Αυτές οι συντεταγμένες σας επιτρέπουν να επισημάνετε, να περικόψετε ή να αναλύσετε περαιτέρω συγκεκριμένα τμήματα ενός σαρωμένου εγγράφου.

## Γιατί να εξάγετε τις συντεταγμένες παραγράφων;
- **Ακριβής μετα-επεξεργασία** – μπορείτε να περάσετε κάθε ορθογώνιο σε επόμενες διαδικασίες (π.χ., μετάφραση, διαγραφή).  
- **Βελτιωμένο UI/UX** – επιδείξτε πλαίσια οριοθέτησης πάνω στην αρχική εικόνα για να δείξετε στους χρήστες πού βρέθηκε το κείμενο.  
- **Αυτοματοποίηση σε παρτίδες** – εντοπίστε και απομονώστε γρήγορα παραγράφους σε μεγάλα σύνολα εγγράφων.

## Προαπαιτούμενα

- Βασικές γνώσεις C# και .NET.  
- Περιβάλλον ανάπτυξης με εγκατεστημένο το Aspose.OCR για .NET – μπορείτε να το κατεβάσετε [εδώ](https://releases.aspose.com/ocr/net/).  
- Εξοικείωση με έννοιες επεξεργασίας εικόνας και τη σημασία της OCR για την εξαγωγή κειμένου από σαρωμένα αρχεία.

## Εισαγωγή Ονομάτων Χώρων

Στο αρχείο C# σας, εισάγετε τα απαιτούμενα ονόματα χώρων ώστε οι κλάσεις OCR να είναι διαθέσιμες:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Ορίστε το φάκελο που περιέχει τις εικόνες που θέλετε να αναλύσετε:

```csharp
string dataDir = "Your Document Directory";
```

## Βήμα 2: Αρχικοποιήστε το Παράδειγμα AsposeOcr

Δημιουργήστε ένα αντικείμενο `AsposeOcr` – αυτό σας δίνει πρόσβαση σε όλες τις λειτουργίες OCR:

```csharp
AsposeOcr api = new AsposeOcr();
```

## Βήμα 3: Καθορίστε τη Διαδρομή της Εικόνας

Δείξτε στο σύστημα το ακριβές αρχείο εικόνας που θέλετε να επεξεργαστείτε:

```csharp
string fullPath = dataDir + "sample.png";
```

## Βήμα 4: Αναγνωρίστε την Εικόνα και Λάβετε τα Ορθογώνια Παραγράφων

Καλέστε τη μέθοδο `GetRectangles`. Ορίζοντας το `detect_areas` σε `true` λέτε στη μηχανή να επιστρέψει **παράγραφο** ορθογώνια:

```csharp
List<Rectangle> rectangles = api.GetRectangles(fullPath, AreasType.PARAGRAPHS, true);
```

## Βήμα 5: Εκτυπώστε τα Αποτελέσματα

Εμφανίστε τις συντεταγμένες ώστε να δείτε τα **συντεταγμένα εξαγόμενων παραγράφων** που εντοπίστηκαν:

```csharp
Console.WriteLine("Areas coordinates:");
rectangles.ForEach(a => Console.WriteLine($"x:{a.X} y:{a.Y} width:{a.Width} height:{a.Height}"));
```

## Συνηθισμένα Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| Δεν επιστρέφονται ορθογώνια | Η ποιότητα της εικόνας είναι πολύ χαμηλή ή το `AreasType` είναι λανθασμένο | Βεβαιωθείτε ότι η εικόνα είναι καθαρή και χρησιμοποιήστε `AreasType.PARAGRAPHS`. |
| Οι συντεταγμένες είναι εκτός ευθυγράμμισης | Ασυμφωνία κλίμακας DPI | Ορίστε το σωστό DPI κατά τη φόρτωση της εικόνας ή χρησιμοποιήστε `api.Config.Dpi`. |
| Εξαίρεση άδειας | Εκτέλεση χωρίς έγκυρη άδεια σε παραγωγή | Εφαρμόστε προσωρινή ή μόνιμη άδεια μέσω `api.SetLicense`. |
