# Πώς να εκτελέσετε εξαγωγή κειμένου από εικόνα από ροή χρησιμοποιώντας το Aspose OCR

Καλώς ήρθατε στον κόσμο της **εξαγωγής κειμένου από εικόνα** με το **Aspose.OCR for .NET**. Σε αυτό το σεμινάριο θα δείτε πώς να διαβάσετε μια ροή εικόνας, να εκτελέσετε OCR σε αρχείο PNG και να μεταφέρετε το αναγνωρισμένο κείμενο στην εφαρμογή C# σας. Είτε δημιουργείτε μια αλυσίδα επεξεργασίας εγγράφων, ένα εργαλείο αυτοματοποίησης εισαγωγής δεδομένων, είτε απλώς πειραματίζεστε με OCR, τα παρακάτω βήματα θα σας μετατρέψουν μια ακατέργαστη εικόνα σε αναζητήσιμο κείμενο σε λίγα λεπτά.

## Γρήγορες Απαντήσεις
- **Τι δείχνει αυτό το σεμινάριο;** Εξαγωγή κειμένου από εικόνα που παρέχεται ως ροή χρησιμοποιώντας το Aspose OCR.  
- **Ποια κύρια λέξη-κλειδί στοχεύεται;** *image text extraction* (χρησιμοποιείται σε όλο τον οδηγό).  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται εμπορική άδεια για χρήση σε παραγωγή.  
- **Μπορώ να επεξεργαστώ αρχεία PNG απευθείας;** Ναι – το Aspose OCR διαχειρίζεται μορφές **ocr png file** χωρίς επιπλέον μετατροπή.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Τι είναι η εξαγωγή κειμένου από εικόνα;
Η εξαγωγή κειμένου από εικόνα (επίσης γνωστή ως OCR) μετατρέπει τους οπτικούς χαρακτήρες σε μια εικόνα σε επεξεργάσιμο, αναζητήσιμο κείμενο. Με το Aspose OCR μπορείτε να παρέχετε ένα `MemoryStream` που περιέχει οποιαδήποτε υποστηριζόμενη εικόνα (PNG, JPEG, BMP κ.λπ.) και να λάβετε τη αναγνωρισμένη συμβολοσειρά με μία κλήση.

## Γιατί να επιλέξετε το Aspose OCR για εξαγωγή κειμένου από εικόνα;
- **Ευρεία υποστήριξη γλωσσών** – λειτουργεί με δεκάδες γλώσσες αμέσως.  
- **Απλό API** – με λίγες γραμμές C# μετατρέπει ένα **image to memorystream** σε αναγνώσιμο κείμενο.  
- **Υψηλή ακρίβεια** – προχωρημένοι αλγόριθμοι διαχειρίζονται θορυβώδεις σάρωση και PNG χαμηλής ανάλυσης.  
- **Διαπλατφορμική** – λειτουργεί σε Windows, Linux και macOS με .NET Core.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

- Aspose.OCR for .NET εγκατεστημένο (κατεβάστε από το [Aspose.OCR for .NET Documentation](https://reference.aspose.com/ocr/net/)).  
- Ένα δείγμα αρχείου εικόνας (π.χ., **sample.png**) τοποθετημένο σε φάκελο που μπορείτε να αναφέρετε από τον κώδικα.

## Εισαγωγή Namespaces

Προσθέστε τα απαιτούμενα namespaces στο αρχείο C# σας:

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

## Οδηγός βήμα‑βήμα

### Βήμα 1: Ορισμός καταλόγου εγγράφου
```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";
```
Αντικαταστήστε το **"Your Document Directory"** με τον πραγματικό φάκελο που περιέχει το *sample.png*.

### Βήμα 2: Αρχικοποίηση της μηχανής Aspose OCR
```csharp
// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```
Δημιουργώντας ένα αντικείμενο `AsposeOcr` αποκτάτε πρόσβαση σε όλες τις μεθόδους OCR.

### Βήμα 3: Ανάγνωση ροής εικόνας και αναγνώριση κειμένου
```csharp
// Recognize image
using (MemoryStream ms = new MemoryStream())
using (FileStream file = new FileStream(dataDir + "sample.png", FileMode.Open, FileAccess.Read))
{
    file.CopyTo(ms);
    result = api.RecognizeImage(ms);
}
```
Εδώ ανοίγουμε το **sample.png**, αντιγράφουμε τα byte του σε ένα `MemoryStream` και περνάμε αυτή τη ροή στη `RecognizeImage`. Αυτό δείχνει το **image stream ocr** και το πρότυπο **read image stream c#** σε μία ροή.

### Βήμα 4: Εμφάνιση του αναγνωρισμένου κειμένου
```csharp
// Display the recognized text
Console.WriteLine(result);
```
Το αποτέλεσμα OCR εκτυπώνεται στην κονσόλα· μπορείτε επίσης να το αποθηκεύσετε σε βάση δεδομένων ή αρχείο.

### Βήμα 5: Επιβεβαίωση επιτυχούς εκτέλεσης
```csharp
Console.WriteLine("RecognizeImageFromStream executed successfully");
```
Μια απλή επιβεβαίωση σας ενημερώνει ότι η διαδικασία ολοκληρώθηκε χωρίς εξαιρέσεις.
