# Πώς να Διορθώσετε την Κλίση Εικόνας σε C# – Πλήρες Μάθημα Aspose OCR

Έχετε αναρωτηθεί ποτέ **πώς να διορθώσετε την κλίση εικόνας** που προέρχεται από έναν σαρωτή που είναι λίγο στραβός; Δεν είστε μόνοι. Σε πολλά πραγματικά έργα η πηγή της εικόνας είναι κλινόμενη με μερικές μοίρες, περιέχει θόρυβο τύπου αλατιού‑πιπεριού, και πρέπει ακόμη να διαβαστεί ως απλό κείμενο.  

Τα καλά νέα; Με το Aspose OCR μπορείτε να διορθώσετε την περιστροφή, να αφαιρέσετε τον θόρυβο, και στη συνέχεια **να αναγνωρίσετε κείμενο από εικόνα**—όλα σε λίγες γραμμές κώδικα. Σε αυτό το μάθημα θα περάσουμε από όλη τη διαδικασία, θα εξηγήσουμε γιατί κάθε φίλτρο είναι σημαντικό, και θα σας δώσουμε ένα έτοιμο πρόγραμμα C#.

> *Συμβουλή:* Αν δεν έχετε χρησιμοποιήσει ποτέ το Aspose OCR, κατεβάστε μια δωρεάν δοκιμή από την ιστοσελίδα του Aspose· το API λειτουργεί αμέσως με .NET 6+.

---

## Τι Θα Χρειαστείτε

- **.NET 6 SDK** (ή οποιαδήποτε πρόσφατη έκδοση .NET) – ο κώδικας μεταγλωττίζεται με Visual Studio, Rider ή το `dotnet` CLI.  
- **Πακέτο NuGet Aspose.OCR** – εγκαταστήστε το με `dotnet add package Aspose.OCR`.  
- Μια **δείγμα εικόνας** που είναι ελαφρώς κλινόμενη και περιέχει θόρυβο (π.χ., `skewed.png`).  
- Βασικές γνώσεις C# – δεν χρειάζεται να είστε ειδικός, απλώς άνετοι με τις δηλώσεις `using` και την κονσόλα.

Αυτό είναι όλο. Χωρίς επιπλέον μηχανές OCR, χωρίς εγγενή DLLs, μόνο μια αναφορά NuGet.

---

## Πώς να Διορθώσετε την Κλίση Εικόνας – Επισκόπηση Βήμα‑προς‑Βήμα

Παρακάτω χωρίζουμε τη διαδικασία σε λογικά βήματα. Κάθε βήμα έχει σαφή επικεφαλίδα, ένα απόσπασμα κώδικα, και μια σύντομη παράγραφο “γιατί” ώστε να κατανοήσετε τη λογική πίσω από την κλήση.

![παράδειγμα διόρθωσης κλίσης εικόνας](https://example.com/deskew-demo.png "πώς να διορθώσετε την κλίση εικόνας")

---

### Βήμα 1: Ρύθμιση του Μηχανήματος OCR

Πρώτα δημιουργούμε ένα αντικείμενο `OcrEngine`. Το μπλοκ `using` εγγυάται σωστή απελευθέρωση πόρων, που απελευθερώνει τις εγγενείς μνήμες μόλις τελειώσουμε.

```csharp
using Aspose.OCR;
using Aspose.OCR.Filters;
using System;

class Preprocess
{
    static void Main()
    {
        // Step 1 – instantiate the OCR engine
        using (OcrEngine ocrEngine = new OcrEngine())
        {
            // The rest of the pipeline goes here...
        }
    }
}
```

*Γιατί είναι σημαντικό:* Το `OcrEngine` είναι η καρδιά του Aspose OCR. Κρατάει την εικόνα, την αλυσίδα φίλτρων και τις ρυθμίσεις αναγνώρισης. Η χρήση του σε `using` αποτρέπει διαρροές μνήμης, ειδικά όταν επεξεργαζόμαστε δεκάδες σελίδες σε μια παρτίδα.

---

### Βήμα 2: Φόρτωση της Σαρωμένης Εικόνας

Φορτώνουμε το αρχείο με `ImageStream.FromFile`. Η διαδρομή μπορεί να είναι απόλυτη ή σχετική με τον φάκελο εργασίας του εκτελέσιμου.

```csharp
ocrEngine.Image = ImageStream.FromFile("YOUR_DIRECTORY/skewed.png");
```

*Γιατί είναι σημαντικό:* Η μηχανή λειτουργεί πάνω σε ροή εικόνας στη μνήμη. Η σωστή διαδρομή είναι το μόνο σημείο όπου μπορεί να εμφανιστεί `FileNotFoundException`, γι’ αυτό ελέγξτε τη δομή των φακέλων πριν τρέξετε.

---

### Βήμα 3: Προσθήκη Φίλτρων Προεπεξεργασίας

Εδώ απαντάμε στο **πώς να αφαιρέσετε θόρυβο** και **πώς να διορθώσετε την περιστροφή της εικόνας**. Δύο ενσωματωμένα φίλτρα κάνουν το σκληρό έργο:

```csharp
// DeskewFilter corrects small rotation (usually up to ±5°)
ocrEngine.Filters.Add(new DeskewFilter());

// DenoiseFilter removes salt‑and‑pepper noise
ocrEngine.Filters.Add(new DenoiseFilter());
```

*Γιατί αυτά τα φίλτρα;*  
- **DeskewFilter** αναλύει τις βάσεις του κειμένου, υπολογίζει τη γωνία κλίσης, και περιστρέφει το bitmap πίσω στην οριζόντια θέση. Χωρίς αυτό, η μηχανή OCR μπορεί να ερμηνεύσει λανθασμένα χαρακτήρες (π.χ. “l” vs. “i”).  
- **DenoiseFilter** εφαρμόζει αλγόριθμο βασισμένο σε μεσαίο φίλτρο που εξομαλύνει τα απομονωμένα μαύρα/λευκά pixel—ακριβώς αυτό που σημαίνει “αφαίρεση θορύβου αλατιού‑πιπεριού” στην επεξεργασία εικόνας. Αυτό βελτιώνει δραματικά τις βαθμολογίες εμπιστοσύνης.

Αν έχετε μια πολύ θολή σάρωση, μπορείτε επίσης να προσθέσετε ένα `SharpenFilter` πριν, αλλά αυτό είναι προαιρετικό.

---

### Βήμα 4: Εκτέλεση της Αναγνώρισης OCR

Τώρα ζητάμε από το Aspose OCR να κάνει τη δουλειά του. Η μέθοδος `Recognize` επιστρέφει boolean που υποδεικνύει επιτυχία.

```csharp
if (ocrEngine.Recognize())
{
    // Step 5 – output the cleaned text
    Console.WriteLine("Cleaned text:\n" + ocrEngine.Text);
}
else
{
    Console.Error.WriteLine("OCR failed – check the image and filter configuration.");
}
```

*Γιατί ελέγχουμε το αποτέλεσμα:* Περιστασιακά η μηχανή δεν μπορεί να βρει κείμενο (π.χ., κενή σελίδα). Ο χειρισμός της περίπτωσης `false` αποτρέπει σιωπηλή αποτυχία που θα ήταν δύσκολο να εντοπιστεί αργότερα.

---

### Βήμα 5: Επαλήθευση του Αποτελέσματος

Όταν τρέξετε το πρόγραμμα, θα δείτε κάτι σαν:

```
Cleaned text:
The quick brown fox jumps over the lazy dog.
```

Αν το κείμενο εξακολουθεί να είναι ακατάληπτο, σκεφτείτε:

- **Αύξηση ανοχής deskew** – `new DeskewFilter(0.1)` επιτρέπει στο φίλτρο να αποδεχθεί μεγαλύτερες γωνίες.  
- **Προσθήκη `BinarizeFilter`** πριν το denoise για μετατροπή της εικόνας σε καθαρό μαύρο‑άσπρο.  
- **Έλεγχος DPI** – σαρώσεις χαμηλής ανάλυσης (< 150 dpi) συχνά χρειάζονται upscale πριν το OCR.

## Πώς να Αφαιρέσετε Θόρυβο – Προχωρημένες Επιλογές

Το βασικό `DenoiseFilter` λειτουργεί καλά για τυπικές κηλίδες σαρωτών, αλλά μερικές φορές αντιμετωπίζετε **αφαίρεση θορύβου αλατιού‑πιπεριού** σε παλιές σαρές φιλμ. Σε αυτές τις περιπτώσεις:

```csharp
// Apply a stronger median filter (kernel size 3x3)
ocrEngine.Filters.Add(new DenoiseFilter { KernelSize = 3 });
```

Ή συνδυάστε ένα **GaussianBlurFilter** πριν το denoise:

```csharp
ocrEngine.Filters.Add(new GaussianBlurFilter { Sigma = 0.8 });
ocrEngine.Filters.Add(new DenoiseFilter());
```

Αυτές οι προσαρμογές ανταλλάσσουν λίγο από τη σαφήνεια για μια πιο καθαρή δυαδική εικόνα, κάτι που συνήθως αυξάνει τη βαθμολογία εμπιστοσύνης OCR κατά 5‑10 %.

---

## Αναγνώριση Κειμένου από Εικόνα – Συμβουλές Μετά‑Επεξεργασίας

Αφού έχετε το `ocrEngine.Text`, ίσως θέλετε να:

- **Αφαιρέσετε κενά** – `ocrEngine.Text = ocrEngine.Text.Trim();`  
- **Κανονικοποιήσετε τα τέλη γραμμής** – `ocrEngine.Text = ocrEngine.Text.Replace("\r\n", "\n");`  
- **Τρέξετε ορθογραφικό έλεγχο** χρησιμοποιώντας `System.Globalization` ή βιβλιοθήκη τρίτου μέρους αν η γλώσσα προέλευσης δεν είναι Αγγλικά.

Όλα αυτά τα βήματα κάνουν το εξαγόμενο κείμενο έτοιμο για επεξεργασία, όπως η ενσωμάτωση σε ευρετήριο αναζήτησης ή φόρμα εισαγωγής δεδομένων.

## Ακραίες Περιπτώσεις & Συνηθισμένα Πιθανά Σφάλματα

| Κατάσταση | Τι να Προσέξετε | Προτεινόμενη Λύση |
|-----------|-------------------|-------------------|
| Εικόνα περιστρεφόμενη > 10° | Το `DeskewFilter` σταματά στο προεπιλεγμένο όριο | Δώστε προσαρμοσμένη μέγιστη γωνία: `new DeskewFilter { MaxAngle = 15 }` |
| Πολύ σκοτεινό φόντο | Τα φίλτρα μπορεί να ερμηνεύσουν το φόντο ως κείμενο | Προσθέστε `InvertColorsFilter` ή αυξήστε την αντίθεση |
| Πολυ‑σελίδα PDF | Το `OcrEngine` δουλεύει σε μία εικόνα τη φορά | Επαναλάβετε για κάθε σελίδα, δημιουργώντας νέο `OcrEngine` σε κάθε επανάληψη |
| Μη‑λατινικό αλφάβητο | Η προεπιλεγμένη γλώσσα είναι Αγγλικά | Ορίστε `ocrEngine.Language = OcrLanguage.Thai;` (ή οποιαδήποτε υποστηριζόμενη γλώσσα) |

Κρατώντας αυτά στο μυαλό θα αποφύγετε το κλασικό “Γιατί το OCR μου είναι κενό;” εφιάλτης.

## Πλήρες Παράδειγμα Λειτουργίας

Αντιγράψτε τον παρακάτω κώδικα σε ένα νέο κονσολικό έργο (`dotnet new console -n OcrDemo`). Επαναφέρετε το πακέτο Aspose OCR, αντικαταστήστε το `YOUR_DIRECTORY/skewed.png` με τη διαδρομή της δοκιμαστικής σας εικόνας, και τρέξτε.

```csharp
using Aspose.OCR;
using Aspose.OCR.Filters;
using System;

class Preprocess
{
    static void Main()
    {
        // Initialize OCR engine
        using (OcrEngine ocrEngine = new OcrEngine())
        {
            // Load the image that may be slightly rotated
            ocrEngine.Image = ImageStream.FromFile("YOUR_DIRECTORY/skewed.png");

            // Add preprocessing filters
            //   - DeskewFilter corrects small rotation
            //   - DenoiseFilter removes salt‑and‑pepper noise
            ocrEngine.Filters.Add(new DeskewFilter());
            ocrEngine.Filters.Add(new DenoiseFilter());

            // Perform OCR recognition
            if (ocrEngine.Recognize())
            {
                // Output the cleaned text
                Console.WriteLine("Cleaned text:\n" + ocrEngine.Text);
            }
            else
            {
                Console.Error.WriteLine("Recognition failed – verify the image and filters.");
            }
        }
    }
}
```

Η εκτέλεση αυτού του προγράμματος εκτυπώνει το καθαρισμένο, διορθωμένο κείμενο στην κονσόλα. Αυτή είναι ολόκληρη η ροή **πώς να διορθώσετε την κλίση εικόνας** σε λιγότερο από 50 γραμμές κώδικα.
