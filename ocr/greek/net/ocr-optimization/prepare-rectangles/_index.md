# Προετοιμασία Ορθογωνίων στην Αναγνώριση Εικόνας OCR

## Εισαγωγή

Η Οπτική Αναγνώριση Χαρακτήρων (OCR) είναι απαραίτητη για τη μετατροπή οπτικού περιεχομένου σε αναζητήσιμο, επεξεργάσιμο κείμενο. Σε αυτό το εκπαιδευτικό υλικό θα **εξάγετε κείμενο από εικόνα** προετοιμάζοντας προσαρμοσμένα ορθογώνια που εστιάζουν τη μηχανή OCR σε συγκεκριμένες περιοχές. Χρησιμοποιώντας το Aspose.OCR για .NET, θα περάσουμε από κάθε βήμα — από τη ρύθμιση του έργου σας μέχρι την ανάκτηση του αναγνωρισμένου κειμένου — ώστε να ενσωματώσετε ισχυρή λειτουργία εικόνα‑σε‑κείμενο στις εφαρμογές .NET.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “εξαγωγή κειμένου από εικόνα”;** Σημαίνει τη μετατροπή των οπτικών χαρακτήρων σε μια εικόνα σε συμβολοσειρές που διαβάζει η μηχανή.  
- **Ποια βιβλιοθήκη βοηθά σε αυτό στο .NET;** Aspose.OCR for .NET.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται άδεια για παραγωγή.  
- **Μπορώ να στοχεύσω συγκεκριμένες περιοχές;** Ναι, ορίζοντας ορθογώνια που περιορίζουν το πεδίο του OCR.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Τι είναι η “εξαγωγή κειμένου από εικόνα” με ορθογώνια;

Όταν ορίζετε ορθογώνιες ζώνες σε μια εικόνα, η μηχανή OCR επεξεργάζεται μόνο αυτές τις ζώνες. Αυτό βελτιώνει την ακρίβεια, μειώνει το χρόνο επεξεργασίας και σας επιτρέπει να αγνοήσετε θορυβώδη φόντα ή άσχετες ενότητες.

## Γιατί να προετοιμάσετε ορθογώνια πριν το OCR;

- **Εστίαση στο σχετικό περιεχόμενο:** Παράλειψη κεφαλίδων, υποσέλιδων ή διακοσμητικών γραφικών.  
- **Βελτίωση απόδοσης:** Μικρότερες περιοχές σημαίνουν ταχύτερη αναγνώριση.  
- **Βελτίωση ακρίβειας:** Λιγότερος οπτικός θόρυβος οδηγεί σε καθαρότερα αποτελέσματα.

## Προαπαιτούμενα

- Γνώση της C# και ανάπτυξης .NET.  
- Βιβλιοθήκη Aspose.OCR για .NET εγκατεστημένη – μπορείτε να την κατεβάσετε **[εδώ](https://releases.aspose.com/ocr/net/)**.  
- Ένα δείγμα εικόνας (π.χ., `sample.png`) που περιέχει το κείμενο που θέλετε να εξάγετε.

## Εισαγωγή Χώρων Ονομάτων

Πρώτα, φέρετε τους απαιτούμενους χώρους ονομάτων στο πεδίο ορατότητας:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

## Βήμα 1: Ρύθμιση Καταλόγου Εγγράφου

Καθορίστε πού βρίσκονται τα αρχεία εικόνας και δημιουργήστε μια παρουσία της μηχανής OCR.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## Πώς να εξάγετε κείμενο από εικόνα χρησιμοποιώντας πολλαπλά ορθογώνια

### Βήμα 2: Αναγνώριση Εικόνας με Πολλαπλά Ορθογώνια

#### 2.1 Ορισμός των ορθογωνίων

Δημιουργήστε μια λίστα από αντικείμενα `Rectangle` που περιγράφουν τις περιοχές που θέλετε η μηχανή OCR να σαρώσει.

```csharp
List<Rectangle> rects = new List<Rectangle>()
{
    new Rectangle(138, 352, 2033, 537),
    new Rectangle(147, 890, 2033, 1157),
    new Rectangle(923, 2045, 465, 102),
    new Rectangle(104, 2147, 2076, 819)
};
```

#### 2.2 Εκτέλεση OCR αναγνώρισης

Περάστε τη διαδρομή της εικόνας και τη λίστα ορθογωνίων στη μέθοδο `RecognizeImage`. Η μέθοδος επιστρέφει μια συλλογή συμβολοσειρών — κάθε στοιχείο αντιστοιχεί σε ένα ορθογώνιο.

```csharp
// first case
List<string> listResult = api.RecognizeImage(dataDir + "sample.png", rects);

// Display the recognized text
foreach (string s in listResult)
{
    Console.WriteLine(s);
}
```

### Βήμα 3: Αναγνώριση Εικόνας με Ρυθμίσεις Αναγνώρισης (Εναλλακτική Προσέγγιση)

#### 3.1 Ορισμός ρυθμίσεων αναγνώρισης

```csharp
RecognitionResult result = api.RecognizeImage(dataDir + "sample.png", new RecognitionSettings
{
    RecognitionAreas = rects
});
```

#### 3.2 Εμφάνιση αναγνωρισμένου κειμένου

```csharp
// Display the recognized text
foreach (string s in result.RecognitionAreasText)
{
    Console.WriteLine(s);
}
```

## Συνηθισμένα Προβλήματα & Συμβουλές

- **Λανθασμένες συντεταγμένες ορθογωνίου:** Βεβαιωθείτε ότι οι τιμές `X`, `Y`, `Width` και `Height` αντιστοιχούν σωστά στην περιοχή που θέλετε.  
- **Ποιότητα εικόνας:** Οι εικόνες χαμηλής ανάλυσης μπορεί να δώσουν φτωχά αποτελέσματα OCR· σκεφτείτε προεπεξεργασία (π.χ., δυαδικοποίηση).  
- **Κενά αποτελέσματα:** Επαληθεύστε ότι τα ορθογώνια περιέχουν πραγματικά κείμενο· διαφορετικά η μηχανή επιστρέφει κενές συμβολοσειρές.
