# Απόσπαση Κειμένου από Εικόνα με Ορθογώνια – Οδηγός OCR

## Εισαγωγή

Η Οπτική Αναγνώριση Χαρακτήρων (OCR) σας επιτρέπει να **αποσπάτε κείμενο από εικόνα** ώστε τα αρχεία να γίνονται αναζητήσιμα και επεξεργάσιμα. Σε αυτό το tutorial θα σας δείξουμε πώς να βελτιώσετε την ακρίβεια του OCR προετοιμάζοντας προσαρμοσμένα ορθογώνια που εστιάζουν τη μηχανή στις ακριβείς ζώνες που χρειάζεστε. Χρησιμοποιώντας το Aspose.OCR για .NET, θα δείτε τη πλήρη ροή εργασίας — από τη ρύθμιση του έργου μέχρι την ανάκτηση των αναγνωρισμένων συμβολοσειρών — ώστε να ενσωματώσετε αξιόπιστη μετατροπή εικόνας‑σε‑κείμενο σε οποιαδήποτε εφαρμογή .NET.

## Σύντομες Απαντήσεις
- **Τι σημαίνει “extract text from image”;** Μετατρέπει τους οπτικούς χαρακτήρες σε μια εικόνα σε συμβολοσειρές που μπορεί να διαβάσει η μηχανή.  
- **Ποια βιβλιοθήκη διαχειρίζεται αυτό στο .NET;** Το Aspose.OCR για .NET παρέχει μια πλήρη μηχανή OCR.  
- **Χρειάζομαι άδεια για παραγωγή;** Μια δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για την ανάπτυξη.  
- **Μπορώ να περιορίσω το OCR σε συγκεκριμένες ζώνες;** Ναι — ορίστε ορθογώνια για να στοχεύσετε μόνο τις περιοχές που περιέχουν χρήσιμο κείμενο.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Τι είναι το “extract text from image” με ορθογώνια;
Η διαδικασία `extract text from image` διαβάζει χαρακτήρες μέσα σε καθορισμένες ορθογώνιες ζώνες, αγνοώντας τα υπόλοιπα. Περιορίζοντας το OCR σε αυτές τις ζώνες κερδίζετε μεγαλύτερη ακρίβεια, ταχύτερη επεξεργασία και λιγότερη προσπάθεια μετα‑επεξεργασίας. Αυτή η προσέγγιση απομονώνει το κείμενο που σας ενδιαφέρει ενώ απορρίπτει τα γραφικά φόντου, τα διακοσμητικά στοιχεία και άλλος οπτικός θόρυβος που μπορεί να μπερδέσει τη μηχανή OCR.

## Γιατί να προετοιμάσετε ορθογώνια πριν το OCR;
Η προετοιμασία ορθογωνίων σας επιτρέπει να συγκεντρώσετε τη μηχανή OCR στα πιο σχετικα μέρη μιας εικόνας, βελτιώνοντας τόσο την ταχύτητα όσο και την ακρίβεια. Συνεπώς, περιορίζοντας την περιοχή ανάλυσης μειώνετε την ποσότητα των δεδομένων που πρέπει να εξετάσει η μηχανή, οδηγώντας σε πιο γρήγορα αποτελέσματα και λιγότερες λανθασμένες αναγνώσεις που προκαλούνται από περιττό οπτικό θόρυβο.

- **Εστίαση στο σχετικό περιεχόμενο:** Παράλειψη κεφαλίδων, υποσέλιδων ή διακοσμητικών γραφικών που θα μπερδέψουν τη μηχανή.  
- **Βελτίωση απόδοσης:** Οι μικρότερες περιοχές απαιτούν λιγότερους υπολογισμούς, μειώνοντας το χρόνο εκτέλεσης έως και 40 % σε μεγάλες σαρώσεις.  
- **Βελτίωση ακρίβειας:** Η μείωση του οπτικού θορύβου αυξάνει τα ποσοστά αναγνώρισης χαρακτήρων από ~85 % σε >95 % σε θορυβώδη έγγραφα.

## Γιατί αυτό είναι σημαντικό για πραγματικά έργα
Πολλά επιχειρηματικά έγγραφα — αποδείξεις, τιμολόγια, ταυτότητες — συνδυάζουν κείμενο με λογότυπα ή barcode. Σχεδιάζοντας ορθογώνια γύρω από τα πεδία που πραγματικά χρειάζεστε, εξάγετε μόνο τα πολύτιμα δεδομένα, μειώνοντας το επακόλουθο καθαρισμό και αυξάνοντας την αξιοπιστία των αυτοματοποιημένων ροών εργασίας. Αυτή η στοχευμένη εξαγωγή μειώνει την προσπάθεια μετα‑επεξεργασίας και βοηθά στη συμμόρφωση με τα πρότυπα διαχείρισης δεδομένων.

## Συνηθισμένες περιπτώσεις χρήσης

- **Έλεγχοι συμμόρφωσης:** Απομόνωση νομικών ρήσεων ή κανονιστικών δηλώσεων για επαλήθευση.

// The path to the documents directory.
string dataDir = "Your Document Directory";

## Πώς να αποσπάσετε κείμενο από εικόνα χρησιμοποιώντας πολλαπλά ορθογώνια
Φορτώστε την εικόνα μία φορά, στη συνέχεια δώστε μια λίστα ορθογωνίων στη μηχανή OCR. Κάθε ορθογώνιο ορίζει μια περιοχή ενδιαφέροντος, και η μηχανή επιστρέφει ξεχωριστή συμβολοσειρά για κάθε περιοχή, επιτρέποντάς σας να χειριστείτε τα πεδία ξεχωριστά και να συνδυάσετε τα αποτελέσματα όπως χρειάζεται.

### Ορισμός των ορθογωνίων

Τα αντικείμενα `Rectangle` περιγράφουν τις συντεταγμένες X‑Y και το μέγεθος κάθε ζώνης που θέλετε να σαρώσετε.  

```csharp
List<Rectangle> rects = new List<Rectangle>()
{
    new Rectangle(138, 352, 2033, 537),
    new Rectangle(147, 890, 2033, 1157),
    new Rectangle(923, 2045, 465, 102),
    new Rectangle(104, 2147, 2076, 819)
};
```

### Εκτέλεση αναγνώρισης OCR

Η μέθοδος `RecognizeImage` επεξεργάζεται την εικόνα χρησιμοποιώντας τη λίστα ορθογωνίων και επιστρέφει το αναγνωρισμένο κείμενο για κάθε περιοχή.  

```csharp
// first case
List<string> listResult = api.RecognizeImage(dataDir + "sample.png", rects);

// Display the recognized text
foreach (string s in listResult)
{
    Console.WriteLine(s);
}
```

## Πώς να αποσπάσετε κείμενο από εικόνα χρησιμοποιώντας RecognitionSettings (Εναλλακτική Προσέγγιση)
Αν προτιμάτε κλήση βασισμένη σε ρυθμίσεις, μπορείτε να πετύχετε το ίδιο αποτέλεσμα με το `RecognitionSettings`. Αυτό το αντικείμενο σας επιτρέπει να ομαδοποιήσετε τους ορισμούς ορθογωνίων μαζί με τη γλώσσα, το DPI και άλλες επιλογές OCR, παρέχοντας μια καθαρή κλήση API με μία παράμετρο.

### Ορισμός ρυθμίσεων αναγνώρισης

Το `RecognitionSettings` σας επιτρέπει να ομαδοποιήσετε τη λίστα ορθογωνίων και πρόσθετες επιλογές (π.χ., γλώσσα, DPI) σε ένα ενιαίο αντικείμενο.  

```csharp
RecognitionResult result = api.RecognizeImage(dataDir + "sample.png", new RecognitionSettings
{
    RecognitionAreas = rects
});
```

### Εμφάνιση αναγνωρισμένου κειμένου

Επανάληψη πάνω στις επιστρεφόμενες συμβολοσειρές και εμφάνιση του κειμένου κάθε περιοχής.  

```csharp
// Display the recognized text
foreach (string s in result.RecognitionAreasText)
{
    Console.WriteLine(s);
}
```

- **Ποιότητα εικόνας:** Οι εικόνες χαμηλής ανάλυσης ή έντονα συμπιεσμένες μειώνουν τα αποτελέσματα του OCR· εξετάστε προεπεξεργασία όπως η δυαδικοποίηση.  
- **Κενά αποτελέσματα:** Βεβαιωθείτε ότι τα ορθογώνια περιέχουν πραγματικά αναγνώσιμο κείμενο· διαφορετικά η μηχανή επιστρέφει κενές συμβολοσειρές.
