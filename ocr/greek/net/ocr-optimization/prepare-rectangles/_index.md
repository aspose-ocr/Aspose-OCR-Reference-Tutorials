# Προετοιμασία Ορθογωνίων στην Αναγνώριση Εικόνας OCR

## Εισαγωγή

Η Optical Character Recognition (OCR) είναι απαραίτητη για τη μετατροπή οπτικού περιεχομένου σε αναζητήσιμο, επεξεργάσιμο κείμενο. Σε αυτό το tutorial θα **εξαγάγετε κείμενο από εικόνα** δημιουργώντας προσαρμοσμένα ορθογώνια που εστιάζουν τη μηχανή OCR σε συγκεκριμένες περιοχές. Χρησιμοποιώντας το Aspose.OCR for .NET, θα περάσουμε από κάθε βήμα — από τη ρύθμιση του έργου σας μέχρι την ανάκτηση του αναγνωρισμένου κειμένου — ώστε να ενσωματώσετε ισχυρή λειτουργία εικόνα‑σε‑κείμενο στις εφαρμογές .NET σας.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “εξαγωγή κειμένου από εικόνα”;** Σημαίνει τη μετατροπή των οπτικών χαρακτήρων σε μια εικόνα σε συμβολοσειρές που διαβάζονται από μηχανή.  
- **Ποια βιβλιοθήκη βοηθά με αυτό στο .NET;** Aspose.OCR for .NET.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται άδεια για παραγωγή.  
- **Μπορώ να στοχεύσω συγκεκριμένες περιοχές;** Ναι, ορίζοντας ορθογώνια που περιορίζουν το εύρος του OCR.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Τι είναι η “εξαγωγή κειμένου από εικόνα” με ορθογώνια;
Όταν ορίζετε ορθογώνιες ζώνες σε μια εικόνα, η μηχανή OCR επεξεργάζεται μόνο αυτές τις ζώνες. Αυτό βελτιώνει την ακρίβεια, μειώνει το χρόνο επεξεργασίας και σας επιτρέπει να αγνοήσετε θορυβώδεις φόντους ή άσχετα τμήματα.

## Γιατί να προετοιμάσετε ορθογώνια πριν το OCR;
- **Εστίαση στο σχετικό περιεχόμενο:** Παράλειψη κεφαλίδων, υποσέλιδων ή διακοσμητικών γραφικών.  
- **Βελτίωση απόδοσης:** Μικρότερες περιοχές σημαίνουν ταχύτερη αναγνώριση.  
- **Βελτίωση ακρίβειας:** Λιγότερος οπτικός θόρυβος οδηγεί σε πιο καθαρά αποτελέσματα.

## Γιατί αυτό έχει σημασία για πραγματικά έργα
Πολλά επιχειρηματικά έγγραφα — αποδείξεις, τιμολόγια, ταυτότητες — έχουν μικτές διατάξεις όπου μόνο ορισμένα τμήματα περιέχουν πολύτιμο κείμενο. Χρησιμοποιώντας ορθογώνια, μπορείτε να εξάγετε μόνο τα απαραίτητα πεδία, μειώνοντας δραστικά την επεξεργασία μετά την εξαγωγή και αυξάνοντας τη συνολική αξιοπιστία της αυτοματοποιημένης ροής εργασίας σας.

## Συνηθισμένες περιπτώσεις χρήσης
- **Αυτοματοποίηση εισαγωγής δεδομένων:** Ανάκτηση συγκεκριμένων πεδίων από σαρωμένες φόρμες.  
- **Έλεγχοι συμμόρφωσης:** Απομόνωση και επαλήθευση νομικών τμημάτων κειμένου.  
- **Καταλογοποίηση περιεχομένου:** Καταλογοποίηση μόνο της επικεφαλίδας ή της λεζάντας μιας εικόνας για μηχανές αναζήτησης.

## Εισαγωγή Χώρων Ονομάτων

Πρώτα, φέρτε τους απαιτούμενους χώρους ονομάτων στο πεδίο ορατότητας:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;

```

## Βήμα 1: Ρύθμιση του Καταλόγου Εγγράφων σας

Καθορίστε πού βρίσκονται τα αρχεία εικόνας σας και δημιουργήστε μια παρουσία της μηχανής OCR.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## Πώς να εξάγετε κείμενο από εικόνα χρησιμοποιώντας πολλαπλά ορθογώνια

### Βήμα 2: Αναγνώριση Εικόνας με Πολλαπλά Ορθογώνια

#### 2.1 Ορισμός των ορθογωνίων

Δημιουργήστε μια λίστα από αντικείμενα `Rectangle` που περιγράφουν τις περιοχές που θέλετε η μηχανή OCR να σαρώσει.

```csharp
List<Rectangle> rects = new List<Rectangle>()
{
    new Rectangle(138, 352, 2033, 537),
    new Rectangle(147, 890, 2033, 1157),
    new Rectangle(923, 2045, 465, 102),
    new Rectangle(104, 2147, 2076, 819)
};
```

#### 2.2 Εκτέλεση OCR αναγνώρισης

Περάστε τη διαδρομή της εικόνας και τη λίστα ορθογωνίων στη μέθοδο `RecognizeImage`. Η μέθοδος επιστρέφει μια συλλογή συμβολοσειρών — κάθε στοιχείο αντιστοιχεί σε ένα ορθογώνιο.

```csharp
// first case
List<string> listResult = api.RecognizeImage(dataDir + "sample.png", rects);

// Display the recognized text
foreach (string s in listResult)
{
    Console.WriteLine(s);
}
```

### Βήμα 3: Αναγνώριση Εικόνας με Ρυθμίσεις Αναγνώρισης (Εναλλακτική Προσέγγιση)

#### 3.1 Ορισμός ρυθμίσεων αναγνώρισης

```csharp
RecognitionResult result = api.RecognizeImage(dataDir + "sample.png", new RecognitionSettings
{
    RecognitionAreas = rects
});
```

#### 3.2 Εμφάνιση αναγνωρισμένου κειμένου

```csharp
// Display the recognized text
foreach (string s in result.RecognitionAreasText)
{
    Console.WriteLine(s);
}
```

## Συχνά Προβλήματα & Συμβουλές

- **Λανθασμένες συντεταγμένες ορθογωνίου:** Βεβαιωθείτε ότι οι τιμές `X`, `Y`, `Width` και `Height` αντιστοιχούν σωστά στην περιοχή που θέλετε.  
- **Ποιότητα εικόνας:** Εικόνες χαμηλής ανάλυσης μπορεί να δώσουν φτωχά αποτελέσματα OCR· σκεφτείτε προεπεξεργασία (π.χ., δυαδικοποίηση).  
- **Κενά αποτελέσματα:** Επαληθεύστε ότι τα ορθογώνια περιέχουν κείμενο· διαφορετικά η μηχανή επιστρέφει κενές συμβολοσειρές.

| Συμπτωμα | Πιθανή Αιτία | Διόρθωση |

| Παραμορφωμένοι χαρακτήρες | Κακή αντίθεση ή θόρυβος | Εφαρμόστε καθαρισμό εικόνας (γκρι κλίμακα, κατώφλι) πριν το OCR |
