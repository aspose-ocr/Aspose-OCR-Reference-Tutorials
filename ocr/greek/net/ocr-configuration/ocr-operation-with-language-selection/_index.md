# Πώς να εκτελέσετε OCR με επιλογή γλώσσας στο Aspose.OCR

## Εισαγωγή

Αν χρειάζεστε **πώς να εκτελέσετε OCR** σε εικόνες και να εξάγετε κείμενο από αρχεία εικόνας σε μια εφαρμογή .NET, το Aspose.OCR για .NET παρέχει μια γρήγορη, ακριβή και γλωσσικά‑συνειδητή λύση. Σε αυτό το tutorial θα περάσουμε από ένα πραγματικό παράδειγμα που δείχνει την αναγνώριση εικόνας OCR με επιλογή γλώσσας, ώστε να μπορείτε να εξάγετε πολύγλωσσο κείμενο από εικόνες με λίγες μόνο γραμμές κώδικα.

## Γρήγορες Απαντήσεις
- **Τι κάνει το Aspose.OCR;** Αναγνωρίζει τυπωμένο και χειρόγραφο κείμενο σε εικόνες και επιστρέφει το εξαγόμενο κείμενο.  
- **Μπορώ να επιλέξω τη γλώσσα;** Ναι – μπορείτε να καθορίσετε οποιαδήποτε υποστηριζόμενη γλώσσα όπως Αγγλικά, Γερμανικά, Ισπανικά, Κινέζικα κ.λπ.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται άδεια για χρήση σε παραγωγή.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **Είναι η διόρθωση κλίσης αυτόματη;** Μπορείτε να ενεργοποιήσετε το `AutoSkew` και να ρυθμίσετε λεπτομερώς τη ρύθμιση `SkewAngle`.

## Γιατί να επιλέξετε το Aspose.OCR για εργασίες OCR;

- **Υψηλή ακρίβεια** σε πολλαπλές γραμματοσειρές και ποιότητες εικόνας.  
- **Ενσωματωμένη επιλογή γλώσσας** εξαλείφει την ανάγκη για εξωτερικά πακέτα γλώσσας.  
- **Απλό API** που ενσωματώνεται καθαρά σε υπάρχοντα έργα C#.  
- **Χωρίς εξωτερικές εξαρτήσεις** – όλα εκτελούνται τοπικά, διασφαλίζοντας την ασφάλεια των δεδομένων σας.

## Βήμα 1: Αρχικοποίηση Aspose.OCR

Ξεκινήστε αρχικοποιώντας μια παρουσία της κλάσης Aspose.OCR. Αυτό θέτει τη βάση για τη χρήση των δυνατοτήτων OCR στην εφαρμογή σας.

```csharp
// ExStart:1
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## Βήμα 2: Καθορισμός Διαδρομής Εικόνας

Στη συνέχεια, ορίστε τη διαδρομή προς την εικόνα στην οποία θέλετε να εκτελέσετε OCR. Βεβαιωθείτε ότι η εικόνα είναι προσβάσιμη από την εφαρμογή σας.

```csharp
// Image Path
string fullPath = dataDir + "sample.png";
```

## Βήμα 3: Αναγνώριση Εικόνας με Επιλογή Γλώσσας

Τώρα έρχεται η κύρια λειτουργία OCR. Χρησιμοποιήστε τη βιβλιοθήκη Aspose.OCR για να αναγνωρίσετε κείμενο από την καθορισμένη εικόνα. Ρυθμίστε τις ρυθμίσεις αναγνώρισης, συμπεριλαμβανομένης της επιλογής γλώσσας.

```csharp
// Recognize image           
RecognitionResult result = api.RecognizeImage(fullPath, new RecognitionSettings
{
    DetectAreas = true,
    RecognizeSingleLine = false,
    AutoSkew = true,
    SkewAngle = 0.2F,
    Language = Language.Eng, // Choose the language: none, eng, deu, por, spa, fra, ita, cze, dan, dum, est, fin, lav, lit, nor, pol, rum, srp_hrv, slk, slv, swe, chi
});
```

## Βήμα 4: Εκτύπωση και Εμφάνιση Αποτελεσμάτων

Μετά τη λειτουργία OCR, εκτυπώστε και εμφανίστε τα αποτελέσματα, συμπεριλαμβανομένου του αναγνωρισμένου κειμένου, των περιοχών, των προειδοποιήσεων και της αναπαράστασης JSON.

```csharp
// Print result
Console.WriteLine($"Text:\n {result.RecognitionText}");
Console.WriteLine("Areas:");
result.RecognitionAreasText.ForEach(a => Console.WriteLine($"{a}"));
Console.WriteLine("Warnings:");
result.Warnings.ForEach(w => Console.WriteLine($"{w}"));
Console.WriteLine($"JSON: {result.GetJson()}");
// ExEnd:1
```

## Συνηθισμένα Προβλήματα και Συμβουλές

- **Λανθασμένη επιλογή γλώσσας** – Εάν η έξοδος φαίνεται ακατάληπτη, ελέγξτε ξανά ότι η ιδιότητα `Language` ταιριάζει με τη γλώσσα της πηγαίας εικόνας.  
- **Κλίση εικόνων** – Ενεργοποιήστε το `AutoSkew` ή ρυθμίστε χειροκίνητα το `SkewAngle` για καλύτερη ακρίβεια σε κεκλιμένες σάρωσες.  
- **Μεγάλα αρχεία** – Επεξεργαστείτε μεγάλες εικόνες σε κομμάτια ή μειώστε την ανάλυση πριν τις περάσετε στο `RecognizeImage` για εξοικονόμηση μνήμης.
