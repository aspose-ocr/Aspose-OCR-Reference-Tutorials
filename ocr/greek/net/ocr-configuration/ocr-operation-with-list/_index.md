# Πώς να εκτελέσετε Batch OCR σε εικόνες με λίστα στο Aspose.OCR για .NET

## Εισαγωγή

Καλώς ήρθατε στο εκτενές μας tutorial σχετικά με **how to batch OCR** πολλαπλές εικόνες χρησιμοποιώντας το Aspose.OCR για .NET. Η Optical Character Recognition (OCR) μετατρέπει σαρωμένα έγγραφα χαρτιού, PDF ή αρχεία εικόνας σε επεξεργάσιμο, αναζητήσιμο κείμενο. Σε αυτόν τον οδηγό θα μάθετε πώς να **extract text from images**, να διαβάζετε κείμενο JPEG και να επεξεργάζεστε πολλά αρχεία σε μία κλήση — ιδανικό για σενάρια όπου χρειάζεται να **scan document to text** γρήγορα και αξιόπιστα.

## Γρήγορες Απαντήσεις
- **Τι κάνει το “multiple image OCR”;** Σας επιτρέπει να αναγνωρίζετε κείμενο από μια λίστα αρχείων εικόνας σε μία κλήση API.  
- **Ποιοι τύποι αρχείων υποστηρίζονται;** JPEG, PNG, BMP, TIFF, GIF και πολλά άλλα.  
- **Χρειάζομαι άδεια;** Απαιτείται προσωρινή άδεια για παραγωγή· μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση.  
- **Μπορώ να προσαρμόσω την αναγνώριση;** Ναι—χρησιμοποιήστε το `RecognitionSettings` για να ρυθμίσετε τη γλώσσα, την ανάλυση και την προεπεξεργασία.  
- **Πόσες εικόνες μπορώ να επεξεργαστώ ταυτόχρονα;** Πρακτικά οποιονδήποτε αριθμό· το API μεταδίδει κάθε αρχείο, έτσι η χρήση μνήμης παραμένει χαμηλή.

## Τι είναι το batch OCR και γιατί είναι σημαντικό;

**Batch OCR** (ή “how to batch OCR”) είναι η δυνατότητα να παρέχετε μια συλλογή διαδρομών εικόνων στο Aspose.OCR και να λαμβάνετε το αναγνωρισμένο κείμενο για κάθε εικόνα σε μία λειτουργία. Αυτή η προσέγγιση μειώνει τις δικτυακές μεταφορές, εξοικονομεί χρόνο ανάπτυξης και καθιστά εύκολη την ενσωμάτωση του OCR σε αυτοματοποιημένες γραμμές επεξεργασίας εγγράφων όπως η διαχείριση τιμολογίων, η αρχειοθέτηση ή η αυτοματοποίηση εισαγωγής δεδομένων.

## Γιατί να χρησιμοποιήσετε το Aspose.OCR για επεξεργασία εικόνων σε batch;

- **Υψηλή ακρίβεια** σε θορυβώδεις σαρώσεις και JPEG χαμηλής ανάλυσης.  
- **Ενσωματωμένη ανίχνευση γλώσσας** για πολυγλωσσικά έγγραφα.  
- **Πλήρης υποστήριξη .NET** – λειτουργεί με .NET Framework, .NET Core και .NET 5/6+.  
- **Χωρίς εξωτερικές εξαρτήσεις**—η βιβλιοθήκη διαχειρίζεται τη φόρτωση εικόνας, την προεπεξεργασία και την εξαγωγή κειμένου εσωτερικά.  
- **Επιλογές OCR image preprocessing** σας επιτρέπουν να βελτιώσετε τα αποτελέσματα για σαρώσεις χαμηλής ποιότητας.

## Προαπαιτούμενα

Πριν βουτήξουμε στον κώδικα, βεβαιωθείτε ότι έχετε τα παρακάτω προαπαιτούμενα:

1. Βιβλιοθήκη Aspose.OCR για .NET: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.OCR. Μπορείτε να τη κατεβάσετε από τη [Aspose.OCR for .NET download page](https://releases.aspose.com/ocr/net/).

2. Κατάλογος εγγράφων: Δημιουργήστε έναν φάκελο όπου αποθηκεύονται τα έγγραφα και οι εικόνες για την αναγνώριση OCR.

Τώρα που έχετε τα απαραίτητα, ας ξεκινήσουμε με τον οδηγό βήμα‑βήμα.

## Εισαγωγή Namespaces

Στο έργο C# σας, συμπεριλάβετε τα απαραίτητα namespaces για να χρησιμοποιήσετε το Aspose.OCR για .NET:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

## Οδηγός βήμα‑βήμα

### Βήμα 1: Ρυθμίστε τον φάκελο εγγράφων σας

Ξεκινήστε αρχικοποιώντας τη διαδρομή προς τον φάκελο εγγράφων σας και δημιουργώντας μια παρουσία `AsposeOcr`:

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

> **Pro tip:** Κρατήστε τα αρχεία εικόνας σε υποφάκελο (π.χ., `dataDir/ocr`) για να διατηρήσετε το έργο οργανωμένο.

### Βήμα 2: Καθορίστε τις διαδρομές εικόνων

Ορίστε τη λίστα των αρχείων εικόνας που θέλετε να επεξεργαστείτε. Μπορείτε να συνδυάσετε JPEG, PNG, BMP ή οποιαδήποτε υποστηριζόμενη μορφή:

```csharp
List<string> imagePaths = new List<string>
{
    dataDir + "0001460985.Jpeg",
    dataDir + "sample.png"
};
```

> **Why this matters:** Η παροχή ενός `List<string>` σας επιτρέπει να **batch OCR** χωρίς να γράψετε βρόχο μόνοι σας—το API κάνει τη σκληρή δουλειά.

### Βήμα 3: Εκτελέστε την αναγνώριση εικόνας OCR

Κλήστε το `RecognizeMultipleImages` με προαιρετικό `RecognitionSettings`. Εδώ μπορείτε να εφαρμόσετε **ocr image preprocessing** όπως διόρθωση κλίσης ή μείωση θορύβου:

```csharp
RecognitionResult[] result = api.RecognizeMultipleImages(imagePaths, new RecognitionSettings
{
   //default or custom settings
});
```

> **How to extract text with custom settings:** Εάν χρειάζεστε συγκεκριμένη γλώσσα ή υψηλότερο DPI, ορίστε το `RecognitionSettings.Language` και το `RecognitionSettings.Dpi`.

### Βήμα 4: Εμφάνιση αποτελεσμάτων αναγνώρισης

Διατρέξτε τα αποτελέσματα και εκτυπώστε το αναγνωρισμένο κείμενο για κάθε εικόνα:

```csharp
for (int i = 0; i < result.Length; i++)
{
	 Console.WriteLine($"Image: {i}\n Result:\n {result[i].RecognitionText}");
}
```

Τώρα θα πρέπει να δείτε το εξαγόμενο κείμενο για κάθε αρχείο να εκτυπώνεται στην κονσόλα, δείχνοντας πώς να **extract text from images** μαζικά.

## Κοινά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| Δεν επιστράφηκε κείμενο | Η ποιότητα της εικόνας είναι πολύ χαμηλή | Αυξήστε το DPI ή χρησιμοποιήστε `RecognitionSettings` για να ενεργοποιήσετε την προεπεξεργασία εικόνας |
| Εντοπίστηκε λανθασμένη γλώσσα | Η προεπιλεγμένη γλώσσα είναι Αγγλικά | Ορίστε το `RecognitionSettings.Language` στον κατάλληλο κωδικό γλώσσας |
| Έλλειψη μνήμης για μεγάλα batches | Φόρτωση πολλών εικόνων υψηλής ανάλυσης ταυτόχρονα | Επεξεργαστείτε τις εικόνες σε μικρότερα batches ή μεταδώστε τις χρησιμοποιώντας το `RecognizeMultipleImages` που ήδη διαχειρίζεται τη ροή |
