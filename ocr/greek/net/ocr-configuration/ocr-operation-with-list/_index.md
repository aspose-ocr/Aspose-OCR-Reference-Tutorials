# OCR πολλαπλών εικόνων με λίστα στο Aspose.OCR για .NET

## Εισαγωγή

Καλώς ήρθατε στο εκτενές μας tutorial για **multiple image ocr** χρησιμοποιώντας το Aspose.OCR για .NET. Η Οπτική Αναγνώριση Χαρακτήρων (OCR) μετατρέπει σαρωμένα έγγραφα χαρτιού, PDF ή αρχεία εικόνας σε επεξεργάσιμο, αναζητήσιμο κείμενο. Σε αυτόν τον οδηγό θα μάθετε πώς να εξάγετε κείμενο από εικόνες, να διαβάζετε κείμενο JPEG και να επεξεργάζεστε πολλά αρχεία με μία κλήση — ιδανικό για σενάρια όπου χρειάζεται να **scan document to text** γρήγορα και αξιόπιστα.

## Γρήγορες Απαντήσεις
- **Τι κάνει το “multiple image ocr”;** Σας επιτρέπει να αναγνωρίζετε κείμενο από μια λίστα αρχείων εικόνας με μία μόνο κλήση API.  
- **Ποιοι μορφές υποστηρίζονται;** JPEG, PNG, BMP, TIFF, GIF και πολλές άλλες.  
- **Χρειάζομαι άδεια;** Απαιτείται προσωρινή άδεια για παραγωγή· μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση.  
- **Μπορώ να προσαρμόσω την αναγνώριση;** Ναι—χρησιμοποιήστε το `RecognitionSettings` για να ρυθμίσετε τη γλώσσα, την ανάλυση και την προεπεξεργασία.  
- **Πόσες εικόνες μπορώ να επεξεργαστώ ταυτόχρονα;** Πρακτικά οποιονδήποτε αριθμό· το API μεταδίδει (stream) κάθε αρχείο, έτσι η χρήση μνήμης παραμένει χαμηλή.

## Τι είναι το multiple image ocr;
**multiple image ocr** είναι η δυνατότητα να παρέχετε μια συλλογή διαδρομών εικόνας στο Aspose.OCR και να λαμβάνετε το αναγνωρισμένο κείμενο για κάθε εικόνα σε μία λειτουργία. Αυτό εξοικονομεί χρόνο ανάπτυξης και μειώνει τις δικτυακές κλήσεις όταν εργάζεστε με παρτίδες σαρωμένων εγγράφων.

## Γιατί να χρησιμοποιήσετε το Aspose.OCR για επεξεργασία πολλαπλών εικόνων;
- **Υψηλή ακρίβεια** σε θορυβώδεις σαρώσεις και JPEG χαμηλής ανάλυσης.  
- **Ενσωματωμένη ανίχνευση γλώσσας** για πολυγλωσσικά έγγραφα.  
- **Πλήρης υποστήριξη .NET** – λειτουργεί με .NET Framework, .NET Core και .NET 5/6+.  
- **Χωρίς εξωτερικές εξαρτήσεις**—η βιβλιοθήκη διαχειρίζεται τη φόρτωση εικόνας, την προεπεξεργασία και την εξαγωγή κειμένου εσωτερικά.

## Προαπαιτούμενα

Πριν βουτήξουμε στον κώδικα, βεβαιωθείτε ότι έχετε τα παρακάτω προαπαιτούμενα:

1. Βιβλιοθήκη Aspose.OCR για .NET: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.OCR. Μπορείτε να τη κατεβάσετε από τη [Aspose.OCR for .NET download page](https://releases.aspose.com/ocr/net/).

2. Κατάλογος εγγράφων: Δημιουργήστε έναν φάκελο όπου θα αποθηκεύονται τα έγγραφα και οι εικόνες για την αναγνώριση OCR.

Τώρα που έχετε τα απαραίτητα, ας ξεκινήσουμε με τον οδηγό βήμα‑βήμα.

## Εισαγωγή Namespaces

Στο έργο C# σας, συμπεριλάβετε τα απαραίτητα namespaces για να χρησιμοποιήσετε το Aspose.OCR για .NET:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

## Βήμα 1: Ρυθμίστε τον Κατάλογο Εγγράφων σας

Ξεκινήστε αρχικοποιώντας τη διαδρομή προς τον κατάλογο εγγράφων σας:

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## Βήμα 2: Καθορίστε τις Διαδρομές Εικόνων

Πριν την αναγνώριση, ορίστε τις διαδρομές των εικόνων που θέλετε να επεξεργαστείτε. Για παράδειγμα, μπορείτε να **extract text images** από αρχεία JPEG και PNG:

```csharp
List<string> imagePaths = new List<string>
{
    dataDir + "0001460985.Jpeg",
    dataDir + "sample.png"
};
```

## Βήμα 3: Εκτελέστε την Αναγνώριση Εικόνας OCR

Ξεκινήστε τη διαδικασία αναγνώρισης OCR με τις καθορισμένες εικόνες. Αυτό το βήμα δείχνει τη διαχείριση **ocr multiple files**:

```csharp
RecognitionResult[] result = api.RecognizeMultipleImages(imagePaths, new RecognitionSettings
{
   //default or custom settings
});
```

## Βήμα 4: Εμφανίστε τα Αποτελέσματα Αναγνώρισης

Εκτυπώστε τα αποτελέσματα αναγνώρισης για κάθε εικόνα. Εδώ θα δείτε το εξαγόμενο κείμενο από κάθε αρχείο, διαβάζοντας αποτελεσματικά **reading JPEG text** και άλλες μορφές:

```csharp
for (int i = 0; i < result.Length; i++)
{
	 Console.WriteLine($"Image: {i}\n Result:\n {result[i].RecognitionText}");
}
```

## Κοινά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| Δεν επιστράφηκε κείμενο | Πολύ χαμηλή ποιότητα εικόνας | Αυξήστε DPI ή χρησιμοποιήστε `RecognitionSettings` για ενεργοποίηση προεπεξεργασίας εικόνας |
| Λάθος γλώσσα εντοπίστηκε | Η προεπιλεγμένη γλώσσα είναι τα Αγγλικά | Ορίστε `RecognitionSettings.Language` στον κατάλληλο κωδικό γλώσσας |
| Έλλειψη μνήμης για μεγάλες παρτίδες | Φόρτωση πολλών εικόνων υψηλής ανάλυσης ταυτόχρονα | Επεξεργαστείτε τις εικόνες σε μικρότερες παρτίδες ή μεταδώστε τις χρησιμοποιώντας `RecognizeMultipleImages` που ήδη διαχειρίζεται το streaming |
