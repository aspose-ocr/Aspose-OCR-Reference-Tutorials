# Ανάκτηση Κειμένου από Εικόνες Χρησιμοποιώντας Λειτουργία OCR σε Φακέλους

## Εισαγωγή

Καλώς ήρθατε στον κόσμο της Οπτικής Αναγνώρισης Χαρακτήρων (OCR) με **Aspose.OCR for .NET**! Εάν χρειάζεστε να **ανακτήσετε κείμενο από εικόνες** μαζικά—π.χ. ολόκληρο φάκελο σαρωμένων εγγράφων—αυτό το tutorial σας καθοδηγεί βήμα‑βήμα σε μια πρακτική, πραγματική λύση. Θα καλύψουμε τα πάντα, από τη ρύθμιση του έργου μέχρι την εκτύπωση του αναγνωρισμένου κειμένου, ώστε να ενσωματώσετε γρήγορα OCR βασισμένο σε φάκελο στις εφαρμογές C#.

## Γρήγορες Απαντήσεις
- **Τι διδάσκει αυτό το tutorial;** Πώς να ανακτήσετε κείμενο από εικόνες που αποθηκεύονται σε φάκελο χρησιμοποιώντας Aspose.OCR.
- **Ποια γλώσσα & πλατφόρμα;** C# με .NET (Framework ή .NET Core).
- **Κύρια προαπαιτούμενα;** Βιβλιοθήκη Aspose.OCR for .NET (σύνδεσμος λήψης παρακάτω).
- **Πόσες γραμμές κώδικα;** Μόνο επτά σύντομα μπλοκ κώδικα.
- **Μπορώ να μετατρέψω εικόνες σε κείμενο;** Ναι—αυτό το παράδειγμα δείχνει ακριβώς αυτό.

## Τι σημαίνει “ανάκτηση κειμένου από εικόνες”;
Η ανάκτηση κειμένου από εικόνες σημαίνει τη χρήση τεχνολογίας OCR για την ανάγνωση χαρακτήρων ενσωματωμένων σε εικόνες, PDF ή σαρωμένα έγγραφα και τη μετατροπή τους σε επεξεργάσιμες, αναζητήσιμες συμβολοσειρές. Το Aspose.OCR προσφέρει μια ισχυρή μηχανή που υποστηρίζει πολλές μορφές εικόνας και γλώσσες.

## Γιατί να χρησιμοποιήσετε Aspose.OCR για OCR βασισμένο σε φάκελο;
- **Υψηλή ακρίβεια** με ενσωματωμένη ανίχνευση γλώσσας.  
- **Επεξεργασία παρτίδας** μέσω `RecognizeMultipleImages`, ιδανική για φακέλους.  
- **Απλό API** που εντάσσεται φυσικά σε έργα C#.  
- **Κλιμακούμενο** – λειτουργεί τόσο σε επιτραπέζιους όσο και σε διακομιστές.

## Προαπαιτούμενα

- Βασική εξοικείωση με C# και ανάπτυξη .NET.  
- Visual Studio (οποιαδήποτε πρόσφατη έκδοση).  
- Βιβλιοθήκη **Aspose.OCR for .NET** – κατεβάστε την [εδώ](https://releases.aspose.com/ocr/net/).  
- Κατανόηση των εννοιών OCR (προαιρετικό αλλά χρήσιμο).

## Εισαγωγή Namespaces

Προσθέστε τις απαιτούμενες οδηγίες `using` στην κορυφή του αρχείου C# ώστε ο μεταγλωττιστής να γνωρίζει πού βρίσκονται οι κλάσεις OCR.

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου
Καθορίστε το φάκελο που περιέχει τις εικόνες που θέλετε να επεξεργαστείτε.

```csharp
// ExStart:1   
// The path to the documents directory.
string dataDir = "Your Document Directory";
```

> **Συμβουλή:** Χρησιμοποιήστε απόλυτη διαδρομή ή `Path.Combine` για να αποφύγετε προβλήματα διαχωριστών διαδρομής σε διαφορετικά λειτουργικά συστήματα.

### Βήμα 2: Αρχικοποίηση Aspose.OCR
Δημιουργήστε μια παρουσία του μηχανισμού OCR.

```csharp
// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

### Βήμα 3: Καθορισμός Διαδρομής Εικόνας
Κατευθύνετε το API στον συγκεκριμένο υποφάκελο που περιέχει τα αρχεία εικόνας.

```csharp
// Image Path
string fullPath = dataDir + "OCR";
```

> **Γιατί είναι σημαντικό:** Η μέθοδος `RecognizeMultipleImages` αναμένει διαδρομή φακέλου, όχι μεμονωμένο αρχείο.

### Βήμα 4: Αναγνώριση Εικόνων
Εκτελέστε OCR σε κάθε εικόνα μέσα στο φάκελο. Μπορείτε να προσαρμόσετε το `RecognitionSettings` εάν χρειάζεστε ενδείξεις γλώσσας ή συγκεκριμένη προεπεξεργασία.

```csharp
// Recognize image           
RecognitionResult[] result = api.RecognizeMultipleImages(fullPath, new RecognitionSettings
{
    //default or custom
});
```

### Βήμα 5: Εκτύπωση Αποτελεσμάτων
Διατρέξτε τον πίνακα `RecognitionResult` που επιστρέφεται και εμφανίστε το ανακτημένο κείμενο.

```csharp
// Print result
for (int i = 0; i < result.Length; i++)
{
    Console.WriteLine($"Image: {i}\n Result:\n {result[i].RecognitionText}");
}
```

> **Κοινό λάθος:** Η παράλειψη ελέγχου του `result.Length` μπορεί να προκαλέσει `IndexOutOfRangeException` όταν ο φάκελος είναι κενός. Πάντα επικυρώστε το περιεχόμενο του φακέλου πρώτα.

### Βήμα 6: Μήνυμα Ολοκλήρωσης
Σημειώστε την επιτυχή εκτέλεση.

```csharp
// ExEnd:1
Console.WriteLine("OCROperationWithFolder executed successfully");
```

## Συχνά Προβλήματα & Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| Δεν εμφανίζεται έξοδος | Λάθος ή κενή διαδρομή φακέλου | Επαληθεύστε ότι το `fullPath` δείχνει στον σωστό κατάλογο και περιέχει υποστηριζόμενες μορφές εικόνας (PNG, JPEG, TIFF). |
| Παραμορφωμένοι χαρακτήρες | Λανθασμένες ρυθμίσεις γλώσσας | Περνάτε ένα ρυθμισμένο `RecognitionSettings` με `Language` ορισμένο στον κατάλληλο κωδικό ISO. |
| Καθυστέρηση απόδοσης σε πολλές εικόνες | Επεξεργασία διαδοχικά στο νήμα UI | Εκτελέστε OCR σε παρασκήνιο ή χρησιμοποιήστε async μοτίβα για να διατηρήσετε το UI ανταποκρινόμενο. |
