# Εξαγωγή Κειμένου από Εικόνες Χρησιμοποιώντας Λειτουργία OCR σε Φακέλους

## Εισαγωγή

Καλώς ήρθατε στον κόσμο της Οπτικής Αναγνώρισης Χαρακτήρων (OCR) με **Aspose.OCR for .NET**! Εάν χρειάζεστε να **εξάγετε κείμενο από εικόνες** μαζικά—π.χ. ολόκληρο φάκελο σαρωμένων εγγράφων—αυτό το σεμινάριο σας καθοδηγεί βήμα προς βήμα σε μια πρακτική, πραγματική λύση. Θα καλύψουμε τα πάντα, από τη ρύθμιση του έργου μέχρι την εκτύπωση του αναγνωρισμένου κειμένου, ώστε να ενσωματώσετε γρήγορα το OCR βάσει φακέλου στις εφαρμογές C#. Στο τέλος, θα δείτε επίσης πώς αυτή η προσέγγιση σας επιτρέπει να **μετατρέψετε εικόνες σε κείμενο**, **εξάγετε κείμενο από σαρωμένα έγγραφα**, και **διαβάζετε κείμενο εικόνας σε C#** με λίγες μόνο γραμμές κώδικα.

## Γρήγορες Απαντήσεις
- **Τι διδάσκει αυτό το σεμινάριο;** Πώς να εξάγετε κείμενο από εικόνες που αποθηκεύονται σε φάκελο χρησιμοποιώντας Aspose.OCR.  
- **Ποια γλώσσα & πλατφόρμα;** C# με .NET (Framework ή .NET Core).  
- **Κύριο προαπαιτούμενο;** Βιβλιοθήκη Aspose.OCR for .NET (σύνδεσμος λήψης παρακάτω).  
- **Πόσες γραμμές κώδικα;** Μόνο επτά σύντομες μπλοκ κώδικα.  
- **Μπορώ να μετατρέψω εικόνες σε κείμενο;** Ναι—αυτό το παράδειγμα δείχνει ακριβώς αυτό.

## Τι σημαίνει «εξαγωγή κειμένου από εικόνες»;
Η εξαγωγή κειμένου από εικόνες σημαίνει τη χρήση τεχνολογίας OCR για την ανάγνωση χαρακτήρων ενσωματωμένων σε εικόνες, PDF ή σαρωμένα έγγραφα και τη μετατροπή τους σε επεξεργάσιμες, αναζητήσιμες συμβολοσειρές. Το Aspose.OCR παρέχει μια ισχυρή μηχανή που υποστηρίζει πολλές μορφές εικόνας και γλώσσες.

## Γιατί να χρησιμοποιήσετε το Aspose.OCR για OCR βάσει φακέλων;
- **Υψηλή ακρίβεια** με ενσωματωμένη ανίχνευση γλώσσας.  
- **Επεξεργασία παρτίδας** μέσω `RecognizeMultipleImages`, ιδανική για φακέλους.  
- **Απλό API** που ενσωματώνεται φυσικά σε έργα C#.  
- **Κλιμακώσιμο** – λειτουργεί τόσο σε επιτραπέζιες όσο και σε διακομιστικές περιβάλλοντα.

## Κοινές Περιπτώσεις Χρήσης
- Ψηφιοποίηση βιβλιοθήκης σαρωμένων τιμολογίων ή αποδείξεων.  
- Μετατροπή αρχειοθετημένων αρχείων PNG/JPEG σε κείμενο αναζητήσιμο για ευρετηρίαση.  
- Αυτοματοποίηση εισαγωγής δεδομένων διαβάζοντας κείμενο από εικόνες ετικετών προϊόντων.  
- Δημιουργία λειτουργίας αναζήτησης εγγράφων που χρειάζεται **εξαγωγή κειμένου από σαρωμένα έγγραφα** σε πραγματικό χρόνο.

## Εισαγωγή Χώρων Ονομάτων

Προσθέστε τις απαιτούμενες οδηγίες `using` στην κορυφή του αρχείου C# ώστε ο μεταγλωττιστής να γνωρίζει πού βρίσκονται οι κλάσεις OCR.

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου
Ορίστε το φάκελο που περιέχει τις εικόνες που θέλετε να επεξεργαστείτε.

```csharp
// ExStart:1   
// The path to the documents directory.
string dataDir = "Your Document Directory";
```

> **Συμβουλή:** Χρησιμοποιήστε απόλυτη διαδρομή ή `Path.Combine` για να αποφύγετε προβλήματα διαχωριστών διαδρομής σε διαφορετικά λειτουργικά συστήματα.

### Βήμα 2: Αρχικοποίηση Aspose.OCR
Δημιουργήστε μια παρουσία της μηχανής OCR.

```csharp
// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

### Βήμα 3: Καθορισμός Διαδρομής Εικόνας
Κατευθύνετε το API στο συγκεκριμένο υποφάκελο που περιέχει τα αρχεία εικόνας.

```csharp
// Image Path
string fullPath = dataDir + "OCR";
```

> **Γιατί είναι σημαντικό:** Η μέθοδος `RecognizeMultipleImages` αναμένει διαδρομή φακέλου, όχι μεμονωμένο αρχείο.

### Βήμα 4: Αναγνώριση Εικόνων
Εκτελέστε OCR σε κάθε εικόνα μέσα στο φάκελο. Μπορείτε να προσαρμόσετε το `RecognitionSettings` εάν χρειάζεστε υποδείξεις γλώσσας ή συγκεκριμένη προεπεξεργασία.

```csharp
// Recognize image           
RecognitionResult[] result = api.RecognizeMultipleImages(fullPath, new RecognitionSettings
{
    //default or custom
});
```

### Βήμα 5: Εκτύπωση Αποτελεσμάτων
Διατρέξτε τον πίνακα `RecognitionResult` που επιστρέφεται και εμφανίστε το εξαγόμενο κείμενο.

```csharp
// Print result
for (int i = 0; i < result.Length; i++)
{
    Console.WriteLine($"Image: {i}\n Result:\n {result[i].RecognitionText}");
}
```

> **Συνηθισμένη παγίδα:** Η παράλειψη ελέγχου του `result.Length` μπορεί να προκαλέσει `IndexOutOfRangeException` όταν ο φάκελος είναι κενός. Πάντα επικυρώστε το περιεχόμενο του φακέλου πρώτα.

### Βήμα 6: Μήνυμα Ολοκλήρωσης
Σημειώστε την επιτυχή εκτέλεση.

```csharp
// ExEnd:1
Console.WriteLine("OCROperationWithFolder executed successfully");
```

## Συμβουλές και Καλές Πρακτικές

- **Μέγεθος παρτίδας:** Εάν επεξεργάζεστε χιλιάδες αρχεία, σκεφτείτε να χωρίσετε το φάκελο σε μικρότερες παρτίδες για προβλέψιμη χρήση μνήμης.  
- **Υποδείξεις γλώσσας:** Η παροχή του σωστού κώδικα γλώσσας στο `RecognitionSettings` βελτιώνει δραστικά την ακρίβεια, ειδικά για μη λατινικά αλφάβητα.  
- **Ασύγχρονη επεξεργασία:** Τυλίξτε την κλήση OCR σε `Task.Run` ή χρησιμοποιήστε async/await για να διατηρήσετε τα νήματα UI ανταποκρίσιμα.  
- **Επικύρωση αρχείων:** Πριν καλέσετε `RecognizeMultipleImages`, φιλτράρετε τον κατάλογο για υποστηριζόμενες επεκτάσεις (`.png`, `.jpg`, `.jpeg`, `.tif`, `.tiff`).  

## Συνηθισμένα Προβλήματα & Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| Δεν επιστρέφεται έξοδος | Λανθασμένη ή κενή διαδρομή φακέλου | Επαληθεύστε ότι το `fullPath` δείχνει στον σωστό φάκελο και περιέχει υποστηριζόμενες μορφές εικόνας (PNG, JPEG, TIFF). |
| Παραμορφωμένοι χαρακτήρες | Λανθασμένες ρυθμίσεις γλώσσας | Περνάτε ένα ρυθμισμένο `RecognitionSettings` με `Language` ορισμένο στον κατάλληλο κωδικό ISO. |
| Καθυστέρηση απόδοσης σε πολλές εικόνες | Επεξεργασία διαδοχικά στο νήμα UI | Εκτελέστε OCR σε παρασκήνιο ή χρησιμοποιήστε async μοτίβα για να διατηρήσετε το UI ανταποκρίσιμο. |
