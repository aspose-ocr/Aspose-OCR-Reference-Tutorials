# Πώς να Εκτελέσετε OCR σε Εικόνες Αρχείου με το Aspose.OCR για .NET

## Εισαγωγή

Σε αυτό το ολοκληρωμένο tutorial θα ανακαλύψετε **πώς να εκτελέσετε OCR** σε αρχειοθετημένα αρχεία εικόνας χρησιμοποιώντας τη βιβλιοθήκη Aspose.OCR για .NET. Είτε χρειάζεστε **μετατροπή εικόνων σε κείμενο** είτε **εξαγωγή κειμένου από αρχείο**, ο οδηγός βήμα‑βήμα παρακάτω θα σας καθοδηγήσει σε όλα—από τη ρύθμιση του περιβάλλοντος ανάπτυξης μέχρι την ανάκτηση του αναγνωρισμένου κειμένου από κάθε εικόνα μέσα σε ένα αρχείο ZIP.

## Γρήγορες Απαντήσεις
- **Τι καλύπτει το tutorial;** Εκτέλεση OCR σε εικόνες αρχείου (ZIP) με το Aspose.OCR για .NET.  
- **Ποια είναι η κύρια λέξη-κλειδί;** *how to perform ocr*.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγή.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **Μπορώ να προσαρμόσω τις ρυθμίσεις αναγνώρισης;** Ναι—χρησιμοποιήστε `RecognitionSettings` για να βελτιώσετε την ακρίβεια.

## Τι είναι το OCR και γιατί να το χρησιμοποιήσετε σε Αρχεία;

Η Οπτική Αναγνώριση Χαρακτήρων (OCR) μετατρέπει σαρωμένες εικόνες ή PDF σε αναζητήσιμο, επεξεργάσιμο κείμενο. Όταν οι εικόνες είναι ενσωματωμένες σε ένα αρχείο (π.χ., αρχείο ZIP), η εξαγωγή και η αναγνώριση κάθε εικόνας ταυτόχρονα εξοικονομεί χρόνο και μειώνει την πολυπλοκότητα του κώδικα. Η μέθοδος `RecognizeMultipleImages` του Aspose.OCR καθιστά αυτή τη διαδικασία απλή.

## Βήμα 1: Ρύθμιση του Καταλόγου Εγγράφων σας

Ξεκινήστε αρχικοποιώντας τη διαδρομή προς τον κατάλογο εγγράφων σας:

```csharp
// ExStart:1
// The path to the documents directory.
string dataDir = "Your Document Directory";
// ExEnd:1
```

> **Συμβουλή:** Χρησιμοποιήστε `Path.Combine` για διαχείριση διαδρομών δια‑πλατφόρμας.

## Βήμα 2: Αρχικοποίηση του Aspose.OCR

Δημιουργήστε μια παρουσία της κλάσης Aspose.OCR για να ξεκινήσετε τις λειτουργίες OCR:

```csharp
// ExStart:3
AsposeOcr api = new AsposeOcr();
// ExEnd:3
```

## Βήμα 3: Καθορισμός Διαδρομής Εικόνας

Ορίστε τη πλήρη διαδρομή προς την εικόνα του αρχείου σας (αρχείο ZIP που περιέχει τις εικόνες που θέλετε να διαβάσετε):

```csharp
// ExStart:4
string fullPath = dataDir + "OCR.zip";
// ExEnd:4
```

## Βήμα 4: Αναγνώριση Εικόνας

Εκτελέστε την αναγνώριση OCR στο καθορισμένο αρχείο χρησιμοποιώντας προεπιλεγμένες ή προσαρμοσμένες ρυθμίσεις. Αυτή η κλήση εξάγει αυτόματα κάθε εικόνα από το ZIP και εκτελεί OCR σε αυτήν:

```csharp
// ExStart:5
RecognitionResult[] result = api.RecognizeMultipleImages(fullPath, new RecognitionSettings
{
   //default or custom settings
});
// ExEnd:5
```

Μπορείτε να ρυθμίσετε το `RecognitionSettings` για να βελτιώσετε την ακρίβεια για συγκεκριμένες γλώσσες ή ποιότητες εικόνας.

## Βήμα 5: Εκτύπωση Αποτελεσμάτων

Διατρέξτε τα αποτελέσματα και εκτυπώστε το αναγνωρισμένο κείμενο για κάθε εικόνα μέσα στο αρχείο:

```csharp
// ExStart:6
for (int i = 0; i < result.Length; i++)
{
	 Console.WriteLine($"Image: {i}\n Result:\n {result[i].RecognitionText}");
}
// ExEnd:6
```

Η έξοδος εμφανίζει τον δείκτη κάθε εικόνας ακολουθούμενο από το εξαγόμενο κείμενο, μετατρέποντας αποτελεσματικά **τις εικόνες σε κείμενο** και **εξάγοντας κείμενο από αρχεία**.

## Συνηθισμένα Προβλήματα & Επίλυση

| Πρόβλημα | Αιτία | Λύση |
|----------|-------|------|
| Δεν επιστρέφεται κείμενο | Πολύ χαμηλή ποιότητα εικόνας | Προεπεξεργασία εικόνων (π.χ., δυαδικοποίηση) ή προσαρμογή του `RecognitionSettings.Dpi` |
| Εξαίρεση κατά την ανάγνωση ZIP | Μη έγκυρη διαδρομή αρχείου | Επαληθεύστε ότι το `fullPath` δείχνει σε έγκυρο αρχείο `.zip` και ότι η εφαρμογή έχει δικαιώματα ανάγνωσης |
| Η άδεια δεν εφαρμόστηκε | Απουσία ή μη φόρτωση του αρχείου άδειας | Καλέστε `License license = new License(); license.SetLicense("Aspose.OCR.lic");` πριν δημιουργήσετε την παρουσία `AsposeOcr` |

## Συχνές Ερωτήσεις

**Ε: Μπορώ να χρησιμοποιήσω το Aspose.OCR για .NET χωρίς άδεια;**  
Α: Ναι, υπάρχει δωρεάν δοκιμή για αξιολόγηση, αλλά απαιτείται έκδοση με άδεια για παραγωγικές εγκαταστάσεις.

**Ε: Υποστηρίζει η βιβλιοθήκη αρχεία ZIP με κωδικό πρόσβασης;**  
Α: Προς το παρόν, η `RecognizeMultipleImages` λειτουργεί με τυπικά αρχεία ZIP. Για κρυπτογραφημένα αρχεία, εξάγετε πρώτα τις εικόνες χρησιμοποιώντας βιβλιοθήκη τρίτου, μετά περάστε τον πίνακα εικόνων στη μηχανή OCR.

**Ε: Πώς μπορώ να βελτιώσω την ακρίβεια για χειρόγραφο κείμενο;**  
Α: Ενεργοποιήστε τη σημαία `RecognitionSettings.EnableHandwritingRecognition` και ορίστε υψηλότερη τιμή DPI (π.χ., 300).

**Ε: Υπάρχει τρόπος να λάβω βαθμολογίες εμπιστοσύνης για κάθε αναγνωρισμένη γραμμή;**  
Α: Κάθε `RecognitionResult` περιέχει την ιδιότητα `Confidence` που μπορείτε να καταγράψετε ή να χρησιμοποιήσετε για φιλτράρισμα αποτελεσμάτων χαμηλής εμπιστοσύνης.
