# εξαγωγή ρωσικού κειμένου & ορισμός διαδρομής πόρων σε C# – οδηγός Aspose OCR

Ποτέ χρειάστηκε να **εξάγετε ρωσικό κείμενο** από σαρωμένο διαβατήριο και δεν ήξερες από πού να ξεκινήσεις; Σε αυτό το tutorial θα σε καθοδηγήσουμε σε όλη τη διαδικασία—πώς να εξάγεις ρωσικό κείμενο χρησιμοποιώντας το Aspose OCR, πώς να ορίσεις τη διαδρομή πόρων και πώς να φορτώσεις σωστά την εικόνα ώστε να διαβάσεις τα δεδομένα του ρωσικού διαβατηρίου σε ελάχιστο χρόνο.

Θα δεις ένα πλήρες, εκτελέσιμο παράδειγμα, θα καταλάβεις γιατί κάθε γραμμή είναι σημαντική και θα πάρεις μερικές πρακτικές συμβουλές που σε σώζουν από τα συνηθισμένα λάθη. Χωρίς αόριστους συνδέσμους «δείτε τα docs»—απλώς μια αυτόνομη λύση που μπορείς να αντιγράψεις‑επικολλήσεις και να τρέξεις σήμερα.

## Τι θα χρειαστείς πριν βυθιστούμε

- **.NET 6.0** (ή οποιαδήποτε πρόσφατη έκδοση .NET· το API είναι σταθερό μεταξύ 5.x‑7.x)
- **Aspose.OCR for .NET** πακέτο NuGet (`Install-Package Aspose.OCR`)
- Ένας φάκελος στο δίσκο που περιέχει το μοντέλο ρωσικής γλώσσας που παρέχεται με το Aspose OCR (συνήθως `Resources\Russian` μετά την αποσυμπίεση του πακέτου)
- Μια εικόνα ρωσικού διαβατηρίου (π.χ. `russian_passport.jpg`) τοποθετημένη σε αυτόν το φάκελο

Αυτό είναι όλο. Καμία επιπλέον υπηρεσία, κανένα κλειδί cloud, μόνο τοπική ρύθμιση.

## εξαγωγή ρωσικού κειμένου – επισκόπηση βήμα‑βήμα

Παρακάτω είναι ένας γρήγορος χάρτης του τι θα πετύχουμε:

1. **Ορισμός της διαδρομής πόρων** ώστε η μηχανή να εντοπίζει το μοντέλο ρωσικής γλώσσας.  
2. **Δημιουργία ενός OcrEngine** και δήλωση ότι δουλεύουμε με Ρωσικά.  
3. **Φόρτωση της εικόνας του διαβατηρίου** με τη μέθοδο `Image.Load` του Aspose.  
4. **Εκτέλεση της αναγνώρισης OCR** και σύλληψη του αποτελέσματος.  
5. **Εκτύπωση του εξαγόμενου κειμένου** στην κονσόλα (ή χρήση του όπως χρειάζεται).

Κάθε βήμα είναι χωρισμένο σε δική του ενότητα, με κώδικα, εξηγήσεις και ένα πλαίσιο «Pro tip».

---

## ορισμός διαδρομής πόρων για το μοντέλο ρωσικής γλώσσας

Το Aspose OCR παρέχει τα αρχεία δεδομένων γλώσσας ξεχωριστά από το κεντρικό DLL. Αν δεν δείξεις στη βιβλιοθήκη το σωστό φάκελο, θα πάρεις εξαίρεση όπως *«Unable to find language resources»*. Η κλήση `ResourceManager.SetLocalResourcePath` λύνει το πρόβλημα.

```csharp
using Aspose.OCR;
using Aspose.OCR.Models;
using Aspose.OCR.Resources;

// 👉 Replace this with the absolute path on your machine
string resourceFolder = @"C:\AsposeOCR\Resources";

// Step 1: Tell Aspose where to find the language models
ResourceManager.SetLocalResourcePath(resourceFolder);
```

**Γιατί είναι σημαντικό:**  
Ο ορισμός της διαδρομής πόρων μία φορά στην αρχή κάνει caching των αρχείων γλώσσας για όλη τη διάρκεια της διαδικασίας, ώστε να μην πληρώνεις το κόστος I/O σε κάθε κλήση αναγνώρισης.  

**Pro tip:** Κράτησε τη διαδρομή σε αρχείο ρυθμίσεων (`appsettings.json`) αν σκοπεύεις να μεταφέρεις την εφαρμογή μεταξύ περιβαλλόντων. Έτσι αποφεύγεις το hard‑coding των διαδρομών.

---

## δημιουργία OCR engine και καθορισμός ρωσικής γλώσσας

Τώρα που η μηχανή ξέρει πού να ψάξει, δημιουργούμε το `OcrEngine` και ορίζουμε την ιδιότητα `Language` σε `Language.Russian`. Αυτό λέει στον αναγνώστη ποιο σύνολο χαρακτήρων και ποιες ηθικές να χρησιμοποιήσει.

```csharp
// Step 2: Initialize the OCR engine for Russian
OcrEngine ocrEngine = new OcrEngine
{
    Language = Language.Russian
};
```

**Γιατί είναι σημαντικό:**  
Το Aspose OCR υποστηρίζει πάνω από 30 γλώσσες, αλλά πρέπει ρητά να επιλέξεις μία. Η επιλογή λανθασμένης γλώσσας μπορεί να μειώσει δραματικά την ακρίβεια επειδή η μηχανή εφαρμόζει διαφορετικό λεξικό και λογική τμηματοποίησης.

---

## φόρτωση εικόνας ocr – ανάγνωση φωτογραφίας ρωσικού διαβατηρίου

Με τη μηχανή έτοιμη, το επόμενο βήμα είναι η φόρτωση της εικόνας του διαβατηρίου. Η `Image.Load` του Aspose λειτουργεί με τις περισσότερες μορφές raster (JPEG, PNG, BMP, TIFF).  

```csharp
// Step 3: Load the passport image you want to process
string imagePath = Path.Combine(resourceFolder, "russian_passport.jpg");
Image sourceImage = Image.Load(imagePath);
```

**Κοινή περίπτωση άκρης:** Αν η εικόνα σου είναι multi‑page TIFF, θα χρειαστεί να επιλέξεις το σωστό frame (`sourceImage.GetFrame(0)`). Για τα περισσότερα διαβατήρια αρκεί ένα μοναδικό JPEG.

---

## ανάγνωση ρωσικού διαβατηρίου και εξαγωγή κειμένου από εικόνα

Τώρα το πιο βαρέως έργο: εκτέλεση του `Recognize` και σύλληψη του κειμένου. Η μέθοδος επιστρέφει ένα `OcrResult` που περιέχει το απλό string, τα confidence scores και προαιρετικές πληροφορίες διάταξης.

```csharp
// Step 4: Perform OCR on the loaded image
OcrResult ocrResult = ocrEngine.Recognize(sourceImage);
```

**Γιατί ίσως θέλεις περισσότερα:**  
Αν χρειάζεσαι bounding boxes για κάθε λέξη (χρήσιμο για επισήμανση), κάλεσε `ocrEngine.Recognize(sourceImage, true)` και εξέτασε το `ocrResult.Regions`.

---

## έξοδος του εξαγόμενου κειμένου – επαλήθευση του αποτελέσματος

Τέλος, εκτύπωσε το αναγνωρισμένο string στην κονσόλα. Σε μια πραγματική εφαρμογή πιθανότατα θα το αποθηκεύσεις σε βάση δεδομένων ή θα το περάσεις σε ρουτίνα επικύρωσης.

```csharp
// Step 5: Print the recognized Russian text
Console.WriteLine("=== Extracted Russian Text ===");
Console.WriteLine(ocrResult.Text);
```

Όταν τρέξεις το πρόγραμμα, θα πρέπει να δεις κάτι σαν:

```
=== Extracted Russian Text ===
ПАСПОРТ РОССИЙСКОЙ ФЕДЕРАЦИИ
Серия 45 12 № 1234567
Дата выдачи: 12.03.2015
...
```

Αν η έξοδος φαίνεται χαοτική, έλεγξε ξανά ότι η εικόνα είναι υψηλής ανάλυσης (≥300 dpi) και ότι πράγματι έδειξες στο φάκελο του μοντέλου ρωσικής γλώσσας.

## πλήρες, έτοιμο‑για‑εκτέλεση παράδειγμα

Παρακάτω είναι ολόκληρο το πρόγραμμα ενσωματωμένο σε ένα μόνο `Program.cs`. Αντέγραψέ το, προσαρμόσε τη διαδρομή `resourceFolder` και πάτα **F5**.

```csharp
// Program.cs
using System;
using System.IO;
using Aspose.OCR;
using Aspose.OCR.Models;
using Aspose.OCR.Resources;

class Program
{
    static void Main()
    {
        // -------------------------------------------------
        // 1️⃣ Set the path to the language resources folder
        // -------------------------------------------------
        string resourceFolder = @"C:\AsposeOCR\Resources";
        ResourceManager.SetLocalResourcePath(resourceFolder);

        // -------------------------------------------------
        // 2️⃣ Create an OCR engine for Russian language
        // -------------------------------------------------
        OcrEngine ocrEngine = new OcrEngine
        {
            Language = Language.Russian
        };

        // -------------------------------------------------
        // 3️⃣ Load the passport image you want to process
        // -------------------------------------------------
        string imagePath = Path.Combine(resourceFolder, "russian_passport.jpg");
        Image sourceImage = Image.Load(imagePath);

        // -------------------------------------------------
        // 4️⃣ Run the OCR recognizer
        // -------------------------------------------------
        OcrResult ocrResult = ocrEngine.Recognize(sourceImage);

        // -------------------------------------------------
        // 5️⃣ Show the extracted text
        // -------------------------------------------------
        Console.WriteLine("=== Extracted Russian Text ===");
        Console.WriteLine(ocrResult.Text);
    }
}
```

**Αναμενόμενη έξοδος κονσόλας** (κομμένη για συντομία):

```
=== Extracted Russian Text ===
ПАСПОРТ РОССИЙСКОЙ ФЕДЕРАЦИИ
Серия 45 12 № 1234567
Дата рождения: 01.01.1990
...
```

Τρέξε το πρόγραμμα μερικές φορές με διαφορετικές σάρωση διαβατηρίων για να δεις πώς η μηχανή αντιμετωπίζει διαφορετικές συνθήκες φωτισμού. Θα καταλάβεις γρήγορα ποιες ποιότητες εικόνας δίνουν τα καλύτερα αποτελέσματα **εξαγωγής ρωσικού κειμένου**.

## λίστα ελέγχου αντιμετώπισης προβλημάτων – κοινές παγίδες

| Symptom | Likely cause | Fix |
|---------|--------------|-----|
| `Unable to find language resources` | Λάθος διαδρομή `resourceFolder` | Επαλήθευσε ότι ο φάκελος περιέχει τα αρχεία `Russian\*.dat` |
| Κενή έξοδος | Ανάλυση εικόνας πολύ χαμηλή (<300 dpi) | Χρησιμοποίησε σάρωση υψηλότερης ανάλυσης ή μεγενθύνε με `Image.Resize` |
| Κατεστραμμένα κυριλλικά (ερωτηματικά) | Κωδικοποίηση κονσόλας δεν είναι UTF‑8 | Πρόσθεσε `Console.OutputEncoding = System.Text.Encoding.UTF8;` στην αρχή |
| Χαμηλά confidence scores | Η εικόνα του διαβατηρίου έχει αντανάκλαση ή θολή | Προεπεξεργάσου με `Image.AdjustContrast` ή καθάρισε τη σάρωση |

## επόμενα βήματα – πέρα από τη βασική εξαγωγή

Τώρα που μπορείς να **εξάγεις ρωσικό κείμενο** και έχεις μάθει να **ορίζεις τη διαδρομή πόρων**, σκέψου τις εξής επεκτάσεις:

- **Batch processing** – βρόχος σε φάκελο με εικόνες διαβατηρίων, αποθήκευση κάθε αποτελέσματος σε CSV.  
- **Data validation** – χρήση regular expressions για εξαγωγή αριθμών διαβατηρίου, ημερομηνιών και ονομάτων από το ακατέργαστο OCR string.  
- **Hybrid approach** – συνδυασμός Aspose OCR με νευρωνικό μοντέλο για περιοχές δύσκολες στην ανάγνωση.  
- **Localization** – αλλαγή του `Language` σε `Language.English` ή `Language.Ukrainian` και επαναχρησιμοποίηση του ίδιου κώδικα.  

Κάθε μία από αυτές τις ιδέες βασίζεται στα ίδια βασικά βήματα που καλύψαμε: ορισμός διαδρομής πόρων, φόρτωση εικόνας και κλήση `Recognize`.
