## Frequently Asked Questions

**Q: Can I use Aspose.OCR for Java in a commercial application?**  
A: Yes. A valid Aspose license removes all evaluation restrictions and allows unlimited deployment in commercial products.

**Q: Which image formats are supported for text extraction?**  
A: JPEG, PNG, BMP, TIFF, GIF, and image‑based PDF pages are all supported out of the box.

**Q: How do I improve accuracy for multilingual documents?**  
A: Set the `Language` property to the appropriate ISO language code (e.g., `en`, `fr`, `zh`) before calling the OCR engine.

**Q: Is there a way to process only a specific region of an image?**  
A: Absolutely. Use the `DetectAreasMode` or manually define a rectangle to limit OCR to the area you need.

**Q: What should I do if the extracted text contains many spelling errors?**  
A: Post‑process the result with a spell‑checking library or use Aspose’s built‑in `CorrectSpelling` method (available in newer versions).
