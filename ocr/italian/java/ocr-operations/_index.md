## Cos'è “recognize pdf text”?
Il riconoscimento del testo PDF comporta la scansione di ogni pagina di un PDF, l'applicazione di algoritmi OCR e la creazione di uno strato ricercabile che mappa la rappresentazione visiva ai caratteri reali. Aspose.OCR per Java automatizza questo processo, offrendo risultati di alta qualità senza intervento manuale.
