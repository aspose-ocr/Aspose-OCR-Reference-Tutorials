# วิธีดึงข้อความจากไฟล์ png ด้วย OCR โดยไม่ใช้การตรวจจับพื้นที่ข้อความ

## Introduction

การจดจำอักขระด้วยแสง (OCR) ได้กลายเป็นเทคโนโลยีสำคัญสำหรับการแปลงข้อความภาพให้เป็นข้อมูลที่ค้นหาและแก้ไขได้ หากคุณกำลังสงสัย **วิธีใช้ OCR** ในโครงการ .NET คู่มือนี้จะแสดงขั้นตอนโดยละเอียดว่า **วิธีดึงข้อความจากไฟล์ png** โดยไม่ต้องพึ่งพาการตรวจจับพื้นที่ข้อความ เมื่อจบบทเรียนคุณจะสามารถดึงข้อความจากภาพ PNG ได้อย่างรวดเร็วโดยใช้ Aspose.OCR for .NET และคุณยังจะได้เห็นวิธี **แปลง pdf เป็นภาพสำหรับ ocr** เมื่อจำเป็นต้องทำงานกับแหล่งข้อมูล PDF

## Quick Answers
- **การที่ไม่มีการตรวจจับพื้นที่ข้อความ** หมายความว่าอะไร? เครื่องมือ OCR จะอ่านภาพทั้งหมดแทนการค้นหาบล็อกข้อความก่อน.  
- **ต้องใช้ไลบรารีใด?** Aspose.OCR for .NET (มีรุ่นทดลองฟรี).  
- **รูปแบบภาพที่รองรับ?** PNG, JPEG, BMP, GIF, TIFF, และอื่น ๆ.  
- **ต้องใช้ไลเซนส์สำหรับการพัฒนาหรือไม่?** ไลเซนส์ชั่วคราวหรือรุ่นทดลองใช้ได้สำหรับการทดสอบ; ต้องมีไลเซนส์เต็มสำหรับการใช้งานจริง.  
- **เวลาในการทำงานโดยทั่วไป?** ไม่กี่ร้อยมิลลิวินาทีสำหรับ PNG ขนาดมาตรฐาน 300 × 200 px.

## What is OCR Image Recognition?

OCR image recognition หมายถึงกระบวนการวิเคราะห์ภาพแรสเตอร์และแปลงอักขระที่ตรวจพบให้เป็นข้อความที่เครื่องคอมพิวเตอร์อ่านได้ ด้วย Aspose.OCR คุณสามารถทำการแปลงนี้โดยตรงในโค้ด C# ของคุณ ทำให้เหมาะสำหรับสถานการณ์เช่น การประมวลผลใบแจ้งหนี้ การจัดเก็บเอกสาร หรือการดึงคำอธิบายจากภาพหน้าจอ

## Why Use Aspose.OCR for .NET?

- **ไม่มีการพึ่งพาภายนอก** – ไลบรารี .NET แท้  
- **ความแม่นยำสูง** สำหรับข้อความพิมพ์และลายมือ  
- **API ที่ง่าย** ที่ทำให้คุณมุ่งเน้นที่ตรรกะธุรกิจแทนการเตรียมภาพ  
- **การควบคุมเต็มรูปแบบ** – คุณสามารถเปิดหรือปิดการตรวจจับพื้นที่ข้อความตามต้องการ  

## Prerequisites

ก่อนจะลงมือเขียนโค้ด โปรดตรวจสอบว่าคุณมีสิ่งต่อไปนี้:

1. **Aspose.OCR for .NET** – ดาวน์โหลดและติดตั้งไลบรารีจากเว็บไซต์อย่างเป็นทางการ [here](https://releases.aspose.com/ocr/net/).  
2. **ภาพตัวอย่าง** – ไฟล์ PNG (เช่น `sample.png`) ที่มีข้อความที่คุณต้องการดึง  
3. **สภาพแวดล้อมการพัฒนา .NET** – Visual Studio, Rider หรือ IDE ใด ๆ ที่รองรับ C#  

## Import Namespaces

ในแอปพลิเคชัน .NET ของคุณ ให้นำเข้า namespace ที่จำเป็นเพื่อเข้าถึงฟังก์ชันของ Aspose.OCR เพิ่มบรรทัดต่อไปนี้ที่ส่วนบนของไฟล์โค้ดของคุณ:

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

## Step 1: Set Document Directory

ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสาร

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";
```

แทนที่ `"Your Document Directory"` ด้วยเส้นทางโฟลเดอร์จริงที่มี `sample.png` อยู่

## Step 2: Initialize Aspose.OCR

ขั้นตอนที่ 2: เริ่มต้น Aspose.OCR

```csharp
// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## Step 3: Recognize Image

ขั้นตอนที่ 3: จดจำภาพ

```csharp
// Recognize image
string result = api.RecognizeImage(dataDir + "sample.png", false);
```

พารามิเตอร์ `false` บอกให้เอนจิน **ไม่ทำการตรวจจับพื้นที่ข้อความ** ดังนั้นมันจะประมวลผลภาพทั้งหมดในหนึ่งรอบ ซึ่งมีประโยชน์เมื่อการจัดวางภาพง่ายหรือเมื่อคุณต้องการจับอักขระทุกตัว

## Step 4: Display Recognized Text

ขั้นตอนที่ 4: แสดงข้อความที่จดจำได้

```csharp
// Display the recognized text
Console.WriteLine(result);
```

ข้อความที่ดึงออกมาจะแสดงในคอนโซล คุณสามารถบันทึก ค้นหา หรือส่งต่อไปยังขั้นตอนต่อไปได้

## Step 5: Finalize Execution

ขั้นตอนที่ 5: สรุปการทำงาน

```csharp
Console.WriteLine("RecognizeImageWithoutTextAreaDetection executed successfully");
```

ข้อความยืนยันที่เป็นมิตรจะแจ้งให้คุณทราบว่าการทำ OCR เสร็จสมบูรณ์โดยไม่มีข้อผิดพลาด

## How to extract text from png without text area detection

วิธีดึงข้อความจาก png โดยไม่ใช้การตรวจจับพื้นที่ข้อความ

เมื่อคุณปิดการตรวจจับพื้นที่ข้อความ เอนจิน OCR จะถือว่าบิตแมพทั้งหมดเป็นบล็อกข้อความเดียว วิธีนี้เหมาะที่สุดสำหรับ:

- ภาพหน้าจอที่ง่ายซึ่งข้อความครอบคลุมทั้งภาพ  
- ใบเสร็จหรือบัตรที่สแกนที่มีการจัดวางสม่ำเสมอ  
- สถานการณ์ที่คุณต้องการรับประกันว่า **ไม่มีอักขระใด ๆ ถูกพลาด** เนื่องจากเอนจินข้ามส่วนใดส่วนหนึ่ง  

## How to convert pdf to image for ocr

วิธีแปลง pdf เป็นภาพสำหรับ ocr

หากแหล่งข้อมูลของคุณเป็น PDF ขั้นตอนการทำงานทั่วไปคือ:

1. ใช้ตัวแปลง PDF‑เป็น‑ภาพ (เช่น Aspose.PDF) เพื่อแปลงแต่ละหน้าเป็น PNG หรือ JPEG.  
2. ส่งไฟล์ภาพที่ได้ไปยังเมธอด `RecognizeImage` ที่แสดงข้างต้น.  

กระบวนการสองขั้นตอนนี้ทำให้คุณสามารถใช้ตรรกะ OCR เดียวกันกับเนื้อหา PDF โดยไม่ต้องเปลี่ยนโค้ดใด ๆ

## Common Use Cases

กรณีการใช้งานทั่วไป

- **การประมวลผลเป็นชุดของใบเสร็จที่สแกน** ที่แต่ละใบเป็นภาพเดียว  
- **การป้อนข้อมูลอัตโนมัติ** จากภาพหน้าจอของแบบฟอร์มที่มีการจัดวางคงที่  
- **การดึงคำอธิบาย** จากภาพสินค้าเพื่อแคตาล็อกอีคอมเมิร์ซ  

## Troubleshooting & Tips

การแก้ไขปัญหาและเคล็ดลับ

- **ตรวจสอบให้แน่ใจว่าภาพชัดเจน** – PNG ที่ความละเอียดต่ำหรือบีบอัดมากอาจลดความแม่นยำ  
- **หากต้องการความเร็วสูงขึ้น** ให้พิจารณาเปิดการตรวจจับพื้นที่ข้อความ (ตั้งค่าพารามิเตอร์ที่สองเป็น `true`)  
- **สำหรับข้อความหลายภาษา** ให้ตั้งค่าคุณสมบัติ `Language` ของอินสแตนซ์ `AsposeOcr` ก่อนเรียก `RecognizeImage`
