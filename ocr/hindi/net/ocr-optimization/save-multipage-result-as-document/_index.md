# इमेज को PDF C# में कनवर्ट करें – मल्टीपेज OCR परिणाम सहेजें

## परिचय

इस ट्यूटोरियल में आप सीखेंगे कि **इमेज को PDF C# में कैसे कनवर्ट करें** Aspose.OCR for .NET का उपयोग करके और परिणामी मल्टीपेज OCR आउटपुट को एक दस्तावेज़ के रूप में कैसे सहेजें। चाहे आपको **स्कैन की गई इमेज को PDF में कनवर्ट करने** की आवश्यकता हो आर्काइविंग के लिए या **इमेज से टेक्स्ट निकालने C#** के लिए डेटा प्रोसेसिंग, यह गाइड आपको हर कदम के साथ वास्तविक उदाहरण और बेस्ट‑प्रैक्टिस टिप्स प्रदान करता है।

## त्वरित उत्तर
- **यह ट्यूटोरियल क्या कवर करता है?** Aspose.OCR का उपयोग करके C# में कई इमेज को PDF/Docx/Txt/Pdf/Xlsx में कनवर्ट करना।
- **कौन से फ़ॉर्मेट समर्थित हैं?** Docx, Text, Pdf, और Xlsx (आप सीधे PDF भी आउटपुट कर सकते हैं)।
- **क्या लाइसेंस की जरूरत है?** मूल्यांकन के लिए एक फ्री ट्रायल काम करता है; प्रोडक्शन के लिए स्थायी लाइसेंस आवश्यक है।
- **कौन से .NET संस्करण संगत हैं?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7।
- **क्या कनवर्ट करते समय टेक्स्ट निकाल सकता हूँ?** हाँ—सेव करने से पहले OCR परिणामों से टेक्स्ट निकालें।

## “इमेज को PDF C# में कनवर्ट करना” क्या है?

C# में इमेज को PDF में कनवर्ट करना मतलब प्रोग्रामेटिक रूप से एक या अधिक बिटमैप फ़ाइलें (PNG, JPEG, TIFF, आदि) लेकर एक PDF दस्तावेज़ बनाना है जो विज़ुअल लेआउट को बनाए रखता है और वैकल्पिक रूप से OCR के माध्यम से सर्चेबल टेक्स्ट एम्बेड करता है। Aspose.OCR इस प्रक्रिया को सरल और अत्यधिक कस्टमाइज़ेबल बनाता है।

## इस कार्य के लिए Aspose.OCR क्यों उपयोग करें?

- **उच्च सटीकता** वाला OCR इंजन जो कई भाषाओं को सपोर्ट करता है।
- **मल्टीपेज सपोर्ट** – एक ही कॉल में इमेज बैच को संभालें।
- **सीधे लोकप्रिय ऑफिस फ़ॉर्मेट में सेव** करने की सुविधा, अतिरिक्त कनवर्ज़न स्टेप्स की जरूरत नहीं।
- **पूर्ण .NET इंटीग्रेशन** – कोई नेटिव डिपेंडेंसी या एक्सटर्नल टूल नहीं।

## नेमस्पेस इम्पोर्ट करें

अपने C# फ़ाइल में आवश्यक नेमस्पेस जोड़ें:

```csharp
using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using Aspose.OCR;
```

ये इम्पोर्ट्स आपको कलेक्शन्स, फ़ाइल हैंडलिंग, LINQ, और Aspose OCR क्लासेज़ तक पहुँच प्रदान करते हैं।

## चरण 1: अपना डॉक्यूमेंट डायरेक्टरी सेट करें

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";
```

`"Your Document Directory"` को उस एब्सॉल्यूट या रिलेटिव पाथ से बदलें जहाँ आपके स्रोत इमेज स्थित हैं और जहाँ आप आउटपुट फ़ाइलें सेव करना चाहते हैं।

## चरण 2: Aspose.OCR को इनिशियलाइज़ करें

```csharp
// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

`AsposeOcr` ऑब्जेक्ट बनाकर आप सभी OCR ऑपरेशन्स तक पहुँच प्राप्त करते हैं, जिसमें **इमेज को PDF C# में कनवर्ट** वर्कफ़्लो भी शामिल है।

## चरण 3: इमेज को रिकग्नाइज़ करें

```csharp
// Recognize image
List<RecognitionResult> result = api.RecognizeMultipleImages(
    new List<string> { dataDir + "sample.png", dataDir + "sample_bad.png" },
    new RecognitionSettings { }
).ToList();
```

`RecognizeMultipleImages` मेथड सूची में प्रत्येक फ़ाइल को प्रोसेस करता है और `RecognitionResult` का कलेक्शन रिटर्न करता है। आप किसी भी संख्या में इमेज फीड कर सकते हैं, जो **स्कैन की गई इमेज को PDF में कनवर्ट** करने के परिदृश्य के लिए आदर्श है।

## चरण 4: पसंदीदा फ़ॉर्मेट में परिणाम सेव करें

```csharp
// Save the result in your preferred format
AsposeOcr.SaveMultipageDocument(RunExamples.GetDataDir_OCR()+"sample.docx", SaveFormat.Docx, result);
AsposeOcr.SaveMultipageDocument(RunExamples.GetDataDir_OCR() + "sample.txt", SaveFormat.Text, result);
AsposeOcr.SaveMultipageDocument(RunExamples.GetDataDir_OCR() + "sample.pdf", SaveFormat.Pdf, result);
AsposeOcr.SaveMultipageDocument(RunExamples.GetDataDir_OCR() + "sample.xlsx", SaveFormat.Xlsx, result);
```

अपनी डाउनस्ट्रीम वर्कफ़्लो के अनुसार फ़ॉर्मेट चुनें:

- **Docx** – सर्चेबल टेक्स्ट के साथ एडिटेबल Word दस्तावेज़।
- **Text** – तेज़ डेटा माइनिंग के लिए प्लेन‑टेक्स्ट एक्सट्रैक्शन (**इमेज से टेक्स्ट निकालने C#**)।
- **Pdf** – क्लासिक PDF आउटपुट, आर्काइविंग के लिए उपयुक्त।
- **Xlsx** – टेबलर डेटा के लिए स्प्रेडशीट प्रतिनिधित्व।

A2: बिल्कुल! Aspose.OCR विभिन्न इमेज फ़ॉर्मेट को सपोर्ट करता है, जिससे आपके OCR टास्क में लचीलापन मिलता है।

## अतिरिक्त अक्सर पूछे जाने वाले प्रश्न

**Q: क्या मैं OCR का उपयोग किए बिना इमेज को PDF C# में कनवर्ट कर सकता हूँ?**  
A: हाँ, आप Aspose.PDF या अन्य लाइब्रेरीज़ का उपयोग करके शुद्ध इमेज‑से‑PDF कनवर्ज़न कर सकते हैं, लेकिन OCR जोड़ने से सर्चेबल टेक्स्ट मिल जाता है।

**Q: कनवर्ज़न के बाद इमेज से टेक्स्ट कैसे निकालूँ C# में?**  
A: `RecognizeMultipleImages` द्वारा रिटर्न किया गया `result` लिस्ट में `Text` प्रॉपर्टी होती है जिसे आप `.txt` फ़ाइल में लिख सकते हैं या सीधे प्रोसेस कर सकते हैं।

**Q: क्या कस्टम पेज मार्जिन या ओरिएंटेशन सेट करना संभव है?**  
A: PDF या Docx में सेव करने से पहले आप Aspose.Words या Aspose.PDF API के ज़रिए डॉक्यूमेंट लेआउट को बदल सकते हैं, फिर `SaveMultipageDocument` कॉल करें।

**Q: अगर कोई इमेज पढ़ी नहीं जा सके तो क्या होगा?**  
A: OCR इंजन उस पेज के लिए एक खाली `RecognitionResult` रिटर्न करता है; आप `result[i].Text` को null या empty स्ट्रिंग के लिए चेक करके उपयुक्त हैंडल कर सकते हैं।

**Q: क्या API क्लाउड डिप्लॉयमेंट को सपोर्ट करता है?**  
A: हाँ, लाइब्रेरी किसी भी .NET रनटाइम पर काम करती है, जिसमें Azure Functions और AWS Lambda शामिल हैं, बशर्ते रनटाइम संस्करण आवश्यकताओं को पूरा करता हो।
