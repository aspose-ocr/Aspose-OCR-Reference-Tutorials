# Aspose.OCR for .NET में लिस्ट के साथ मल्टीपल इमेज OCR

## परिचय

Aspose.OCR for .NET का उपयोग करके **मल्टीपल इमेज OCR** पर हमारे विस्तृत ट्यूटोरियल में आपका स्वागत है। ऑप्टिकल कैरेक्टर रिकग्निशन (OCR) स्कैन किए गए कागज़ दस्तावेज़ों, PDFs, या इमेज फ़ाइलों को संपादन योग्य, खोज योग्य टेक्स्ट में बदलता है। इस गाइड में आप इमेज से टेक्स्ट निकालना, JPEG टेक्स्ट पढ़ना, और एक ही कॉल में कई फ़ाइलों को प्रोसेस करना सीखेंगे—जो तेज़ और भरोसेमंद **डॉक्यूमेंट को टेक्स्ट में स्कैन** करने की आवश्यकता वाले परिदृश्यों के लिए एकदम उपयुक्त है।

## त्वरित उत्तर
- **“मल्टीपल इमेज OCR” क्या करता है?** यह आपको एक ही API कॉल में इमेज फ़ाइलों की सूची से टेक्स्ट पहचानने की सुविधा देता है।  
- **कौन‑से फ़ॉर्मैट समर्थित हैं?** JPEG, PNG, BMP, TIFF, GIF और कई अन्य।  
- **क्या लाइसेंस की आवश्यकता है?** प्रोडक्शन के लिए एक टेम्पररी लाइसेंस आवश्यक है; मूल्यांकन के लिए फ्री ट्रायल काम करता है।  
- **क्या मैं पहचान को कस्टमाइज़ कर सकता हूँ?** हाँ—`RecognitionSettings` का उपयोग करके भाषा, रिज़ॉल्यूशन और प्री‑प्रोसेसिंग को समायोजित कर सकते हैं।  
- **एक साथ कितनी इमेज प्रोसेस कर सकता हूँ?** व्यावहारिक रूप से कोई भी संख्या; API प्रत्येक फ़ाइल को स्ट्रीम करता है, जिससे मेमोरी उपयोग कम रहता है।

## मल्टीपल इमेज OCR क्या है?
**मल्टीपल इमेज OCR** वह क्षमता है जिससे आप इमेज पाथ्स का संग्रह Aspose.OCR को दे सकते हैं और प्रत्येक इमेज के लिए पहचाना गया टेक्स्ट एक ही ऑपरेशन में प्राप्त कर सकते हैं। यह विकास समय बचाता है और स्कैन किए गए दस्तावेज़ों के बैच को संभालते समय नेटवर्क राउंड‑ट्रिप्स को घटाता है।

## Aspose.OCR को मल्टीपल इमेज प्रोसेसिंग के लिए क्यों चुनें?
- **शोरयुक्त स्कैन और लो‑रिज़ॉल्यूशन JPEGs पर उच्च सटीकता**।  
- **बिल्ट‑इन भाषा डिटेक्शन** मल्टी‑लिंगुअल दस्तावेज़ों के लिए।  
- **पूर्ण .NET समर्थन** – .NET Framework, .NET Core, और .NET 5/6+ के साथ काम करता है।  
- **कोई बाहरी डिपेंडेंसी नहीं**—लाइब्रेरी इमेज लोडिंग, प्री‑प्रोसेसिंग और टेक्स्ट एक्सट्रैक्शन को आंतरिक रूप से संभालती है।

## पूर्वापेक्षाएँ

कोड में डुबकी लगाने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ मौजूद हैं:

1. Aspose.OCR for .NET लाइब्रेरी: सुनिश्चित करें कि आपके पास Aspose.OCR लाइब्रेरी इंस्टॉल है। आप इसे [Aspose.OCR for .NET डाउनलोड पेज](https://releases.aspose.com/ocr/net/) से डाउनलोड कर सकते हैं।

2. डॉक्यूमेंट डायरेक्टरी: एक ऐसी डायरेक्टरी सेट अप करें जहाँ OCR पहचान के लिए आपके दस्तावेज़ और इमेज संग्रहीत हों।

अब जब आपके पास आवश्यक चीज़ें हैं, चलिए चरण‑दर‑चरण गाइड शुरू करते हैं।

## नेमस्पेस इम्पोर्ट करें

अपने C# प्रोजेक्ट में Aspose.OCR for .NET उपयोग करने के लिए आवश्यक नेमस्पेस शामिल करें:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

## चरण 1: अपने डॉक्यूमेंट डायरेक्टरी को सेट अप करें

डॉक्यूमेंट डायरेक्टरी का पाथ इनिशियलाइज़ करें:

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## चरण 2: इमेज पाथ्स निर्दिष्ट करें

पहचान से पहले, उन इमेज के पाथ्स को परिभाषित करें जिन्हें आप प्रोसेस करना चाहते हैं। उदाहरण के तौर पर, आप JPEG और PNG फ़ाइलों से **टेक्स्ट इमेज निकाल** सकते हैं:

```csharp
List<string> imagePaths = new List<string>
{
    dataDir + "0001460985.Jpeg",
    dataDir + "sample.png"
};
```

## चरण 3: OCR इमेज पहचान करें

निर्दिष्ट इमेज के साथ OCR पहचान प्रक्रिया शुरू करें। यह चरण **एक साथ कई फ़ाइलों का OCR** दर्शाता है:

```csharp
RecognitionResult[] result = api.RecognizeMultipleImages(imagePaths, new RecognitionSettings
{
   //default or custom settings
});
```

## चरण 4: पहचान परिणाम प्रदर्शित करें

प्रत्येक इमेज के लिए पहचान परिणाम प्रिंट करें। यहाँ आप प्रत्येक फ़ाइल से निकाला गया टेक्स्ट देखेंगे, प्रभावी रूप से **JPEG टेक्स्ट पढ़ना** और अन्य फ़ॉर्मैट्स को संभालते हुए:

```csharp
for (int i = 0; i < result.Length; i++)
{
	 Console.WriteLine($"Image: {i}\n Result:\n {result[i].RecognitionText}");
}
```

## सामान्य समस्याएँ और समाधान

| समस्या | कारण | समाधान |
|-------|-------|-----|
| कोई टेक्स्ट नहीं मिला | इमेज क्वालिटी बहुत कम | DPI बढ़ाएँ, या `RecognitionSettings` का उपयोग करके इमेज प्री‑प्रोसेसिंग सक्षम करें |
| गलत भाषा पहचानी गई | डिफ़ॉल्ट भाषा अंग्रेज़ी है | `RecognitionSettings.Language` को उपयुक्त भाषा कोड पर सेट करें |
| बड़े बैच के लिए मेमोरी समाप्त | एक साथ कई हाई‑रिज़ॉल्यूशन इमेज लोड करना | इमेज को छोटे बैच में प्रोसेस करें या `RecognizeMultipleImages` का उपयोग करें जो पहले से स्ट्रीमिंग संभालता है |

उ: हाँ, `RecognitionSettings` क्लास आपको प्रत्येक बैच के लिए भाषा, रिज़ॉल्यूशन और प्री‑प्रोसेसिंग जैसे OCR पैरामीटर को ट्यून करने की अनुमति देती है।
