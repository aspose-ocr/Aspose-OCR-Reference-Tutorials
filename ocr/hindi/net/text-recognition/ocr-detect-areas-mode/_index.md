# ocr दस्तावेज़ मोड – OCR इमेज पहचान में Detect Areas मोड

## परिचय

आधुनिक .NET विकास में, **ocr दस्तावेज़ मोड** वह तरीका है जब आपको छवियों के भीतर टेक्स्ट को पहचानने पर सटीक नियंत्रण चाहिए। Aspose.OCR for .NET विभिन्न पहचान रणनीतियों के बीच स्विच करना आसान बनाता है, जिससे आप **extract table text image** को रसीदें, चालान या बहु‑स्तंभ दस्तावेज़ जैसे जटिल लेआउट से निकाल सकते हैं। यह **aspose ocr tutorial c#** आपको Detect Areas Mode फीचर के माध्यम से ले जाएगा, प्रत्येक मोड कब उपयोग करना है समझाएगा, और एक तैयार‑चलाने योग्य कोड नमूना दिखाएगा।

## त्वरित उत्तर
- **ocr दस्तावेज़ मोड क्या है?** वह डिटेक्शन रणनीतियों (PHOTO, DOCUMENT, COMBINE) का सेट है जो Aspose.OCR को टेक्स्ट क्षेत्रों को खोजने के लिए बताता है।
- **टेबल के लिए कौन सा मोड सबसे अच्छा है?** `PHOTO` मोड टेबल टेक्स्ट इमेज और छोटे टेक्स्ट ब्लॉकों को निकालने में उत्कृष्ट है।
- **क्या विकास के लिए लाइसेंस चाहिए?** परीक्षण के लिए एक फ्री ट्रायल लाइसेंस पर्याप्त है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6 और बाद के संस्करण।
- **सेटअप में कितना समय लगता है?** आमतौर पर 10 मिनट से कम में नमूना कोड को एकीकृत और चलाया जा सकता है।

## ocr दस्तावेज़ मोड क्या है?
`ocr दस्तावेज़ मोड` वह कॉन्फ़िगरेशन है जो Aspose.OCR को टेक्स्ट पहचान करने से पहले छवि को कैसे विभाजित करना है बताता है। तीन बिल्ट‑इन मोड हैं:

- **PHOTO** – फ़ोटोग्राफ़, रसीदें, चालान और छोटे टेक्स्ट क्षेत्रों के लिए अनुकूलित (टेबल टेक्स्ट इमेज निकालने के लिए आदर्श)।
- **DOCUMENT** – बहु‑स्तंभ प्रिंटेड पृष्ठों और एम्बेडेड ग्राफ़िक्स वाले दस्तावेज़ों के लिए उपयुक्त।
- **COMBINE** – PHOTO और DOCUMENT के परिणामों को मिलाकर सबसे व्यापक कवरेज प्रदान करता है।

## Detect Areas मोड क्यों उपयोग करें?
सही डिटेक्शन मोड चुनने से फॉल्स पॉज़िटिव कम होते हैं, प्रोसेसिंग तेज़ होती है, और सटीकता बढ़ती है—विशेषकर जब आप टेबल जैसे संरचित डेटा से निपट रहे हों। अपनी छवि प्रकार के अनुसार मोड को अनुकूलित करके आप पोस्ट‑प्रोसेसिंग के बिना विश्वसनीय OCR परिणाम प्राप्त कर सकते हैं।

## पूर्वापेक्षाएँ

शुरू करने से पहले सुनिश्चित करें कि आपके पास है:

- **Aspose.OCR for .NET** – लाइब्रेरी को [Aspose.OCR for .NET documentation](https://reference.aspose.com/ocr/net/) से डाउनलोड और इंस्टॉल करें।
- **Document Directory** – आपके मशीन पर एक फ़ोल्डर जिसमें वह छवियाँ हों जिन्हें आप प्रोसेस करना चाहते हैं (उदाहरण के लिए `table.png`)।

## नेमस्पेसेस इम्पोर्ट करें

पहले, अपने C# प्रोजेक्ट में Aspose.OCR के साथ काम करने के लिए आवश्यक नेमस्पेसेस इम्पोर्ट करें।

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

## चरण 1: Aspose.OCR को इनिशियलाइज़ करें

OCR इंजन का एक इंस्टेंस बनाएं और उसे अपने डेटा फ़ोल्डर की ओर इंगित करें।

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## चरण 2: इमेज लोड करें और Detect Areas मोड चुनें

लक्षित इमेज लोड करें और उस डिटेक्शन रणनीति को निर्दिष्ट करें जो आपके परिदृश्य से मेल खाती है।

```csharp
// Recognize image
RecognitionResult result = api.RecognizeImage(dataDir + "table.png", new RecognitionSettings
{
    // Choose the Detect Areas Mode
    DetectAreasMode = DetectAreasMode.PHOTO
    // Other options: NONE, DOCUMENT, COMBINE
});
```

## चरण 3: पहचाने गए टेक्स्ट को प्राप्त करें और प्रदर्शित करें

OCR पूरा होने के बाद, आप निकाले गए टेक्स्ट तक पहुंच सकते हैं—आगे की प्रोसेसिंग या डेटाबेस में स्टोर करने के लिए उपयुक्त।

```csharp
// Display the recognized text
Console.WriteLine(result.RecognitionText);

Console.WriteLine("OCRDetectAreasMode executed successfully");
```
