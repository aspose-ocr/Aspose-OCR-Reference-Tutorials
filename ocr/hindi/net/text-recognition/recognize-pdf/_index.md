# Aspose.OCR के साथ .NET में PDF को OCR कैसे करें

## परिचय

यदि आप .NET वातावरण में **how to ocr pdf** फ़ाइलों को प्रोसेस करने का भरोसेमंद तरीका खोज रहे हैं, तो आप सही जगह पर आए हैं। इस ट्यूटोरियल में हम PDF से टेक्स्ट निकालने, PDF को टेक्स्ट में बदलने, और Aspose.OCR लाइब्रेरी का उपयोग करके C#‑स्टाइल में PDF टेक्स्ट पढ़ने की पूरी प्रक्रिया को चरण‑दर‑चरण देखेंगे। चाहे आपको एक पेज प्रोसेस करना हो या **ocr multi page pdf**, नीचे दिए गए कदम आपको एक ठोस, प्रोडक्शन‑रेडी समाधान प्रदान करेंगे।

## त्वरित उत्तर
- **कौन सा लाइब्रेरी उपयोग करूँ?** Aspose.OCR for .NET  
- **क्या मैं मल्टी‑पेज PDFs से टेक्स्ट निकाल सकता हूँ?** हाँ – `DocumentRecognitionSettings` में `StartPage` और `PagesNumber` सेट करें।  
- **क्या प्रोडक्शन के लिए लाइसेंस चाहिए?** एक कमर्शियल लाइसेंस आवश्यक है; एक फ्री ट्रायल उपलब्ध है।  
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **क्या OCR टेक्स्ट निकालने का सबसे अच्छा तरीका है?** स्कैन किए गए PDFs या PDFs के भीतर इमेज के लिए OCR आवश्यक है; नेटिव PDFs के लिए PDF पार्सर तेज़ हो सकता है।

## OCR क्या है और PDF के लिए इसका उपयोग क्यों करें?

ऑप्टिकल कैरेक्टर रिकग्निशन (OCR) इमेज में मौजूद टेक्स्ट—जैसे स्कैन किए गए पेज—को सर्चेबल, एडिटेबल कैरेक्टर्स में बदल देता है। जब PDF में स्कैन किए गए पेज होते हैं, तो पारंपरिक टेक्स्ट एक्सट्रैक्शन विफल हो जाता है, इसलिए OCR **extract text pdf** और **convert pdf to text** को विश्वसनीय रूप से करने की प्राथमिक तकनीक बन जाता है।

## Aspose.OCR for .NET को क्यों चुनें?

- **उच्च सटीकता** कई भाषाओं और फ़ॉन्ट्स पर।  
- **बिल्ट‑इन सपोर्ट** मल्टी‑पेज PDFs के लिए, जिससे आप प्रोसेस करने के लिए पेज रेंज निर्दिष्ट कर सकते हैं।  
- **सिंपल API** जो C# प्रोजेक्ट्स में सहजता से इंटीग्रेट होता है, जिससे **read pdf text c#** या **extract pdf text c#** आसान हो जाता है।

## नेमस्पेस आयात करें

अपने .NET एप्लिकेशन में OCR फ़ंक्शनैलिटी तक पहुँचने के लिए Aspose.OCR नेमस्पेस आयात करें:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

## चरण 1: Aspose.OCR को प्रारंभ करें

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

यहाँ हम उस फ़ोल्डर को परिभाषित करते हैं जहाँ हमारा PDF स्थित है और एक `AsposeOcr` ऑब्जेक्ट बनाते हैं जो पहचान (recognition) करेगा।

## चरण 2: PDF पथ प्रदान करें

```csharp
// Image Path
string fullPath = dataDir + "multi_page_1.pdf";
```

`multi_page_1.pdf` को उस PDF के नाम से बदलें जिसे आप प्रोसेस करना चाहते हैं। यह पाथ OCR इंजन द्वारा उपयोग किया जाएगा।

## चरण 3: PDF को पहचानें (OCR मल्टी पेज PDF)

```csharp
// Recognize image
List<RecognitionResult> results = api.RecognizePdf(fullPath, new DocumentRecognitionSettings { StartPage = 2, PagesNumber = 2 });
```

`RecognizePdf` मेथड निर्दिष्ट पेजों पर OCR चलाता है। किसी भी रेंज को टार्गेट करने के लिए `StartPage` और `PagesNumber` को समायोजित करें, जो विशेष रूप से **ocr multi page pdf** परिदृश्यों में उपयोगी है।

## चरण 4: परिणाम प्रिंट करें

```csharp
// Print result
int pageCounter = 0;
foreach (var result in results)
{
    PrintRecognitionResult(result, pageCounter++);
}
```

लूप प्रत्येक पेज के `RecognitionResult` पर इटररेट करता है और निकाले गए टेक्स्ट को प्रिंट करता है। आप `PrintRecognitionResult` को अपनी लॉजिक से बदलकर टेक्स्ट को डेटाबेस में स्टोर या फ़ाइल में लिख सकते हैं।
