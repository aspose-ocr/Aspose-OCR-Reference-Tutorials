# Aspose.OCR का उपयोग करके छवियों से टेक्स्ट निकालें – अनुमत अक्षर

## परिचय

छवियों से टेक्स्ट निकालना आधुनिक अनुप्रयोगों में एक सामान्य आवश्यकता है—चाहे आप इनवॉइस प्रोसेस कर रहे हों, रसीदें स्कैन कर रहे हों, या मुद्रित दस्तावेज़ों को डिजिटल बना रहे हों। **Aspose.OCR for Java** इस कार्य को सरल बनाता है, उच्च‑सटीकता पहचान और लचीले कॉन्फ़िगरेशन विकल्प प्रदान करता है जैसे कि अनुमत अक्षरों को निर्दिष्ट करना। इस ट्यूटोरियल में हम एक पूर्ण **aspose ocr java tutorial** के माध्यम से दिखाएंगे कि लाइब्रेरी कैसे सेट‑अप करें, OCR चलाएँ, और अपनी आवश्यकताओं के अनुसार कैरेक्टर सेट को सीमित करें।

## त्वरित उत्तर
- **Aspose.OCR क्या करता है?** यह छवियों से उच्च सटीकता के साथ टेक्स्ट निकालता है और कस्टम कैरेक्टर सेट का समर्थन करता है।  
- **क्या मुझे लाइसेंस चाहिए?** उत्पादन उपयोग के लिए एक अस्थायी या स्थायी लाइसेंस आवश्यक है।  
- **कौन सा JDK संस्करण समर्थित है?** नवीनतम JDK रिलीज़ पूरी तरह संगत हैं।  
- **क्या मैं पहचान किए गए अक्षरों को सीमित कर सकता हूँ?** हाँ—आउटपुट को प्रतिबंधित करने के लिए allowed‑characters API का उपयोग करें।  
- **सेट‑अप में कितना समय लगेगा?** बुनियादी कार्यान्वयन के लिए लगभग 10‑15 मिनट।

## “छवियों से टेक्स्ट निकालना” क्या है?
छवियों से टेक्स्ट निकालना वह प्रक्रिया है जिसमें दृश्य टेक्स्ट (जैसे मुद्रित या हाथ से लिखा) को मशीन‑पठनीय स्ट्रिंग्स में परिवर्तित किया जाता है। यह खोज, अनुक्रमण या डेटा विश्लेषण जैसे डाउनस्ट्रीम कार्यों को सक्षम बनाता है।

## चरण‑दर‑चरण गाइड

### चरण 1: अपना डॉक्यूमेंट डायरेक्टरी सेट करें

एक फ़ोल्डर परिभाषित करें जहाँ आप OCR‑प्रोसेस्ड परिणाम संग्रहीत करेंगे। यह पाथ बाद में इमेज फ़ाइल को लोकेट करने के लिए उपयोग किया जाएगा।

```java
String dataDir = "Your Document Directory";
```

### चरण 2: इमेज पाथ निर्दिष्ट करें

API को उस इमेज की ओर इंगित करें जिसे आप विश्लेषण करना चाहते हैं।

```java
String imagePath = dataDir + "0001460985.Jpeg";
```

### चरण 3: Aspose.OCR इंस्टेंस बनाएं

OCR इंजन को अपने लाइसेंस कुंजी के साथ इंस्टैंशिएट करें। कुंजी एक अस्थायी या स्थायी लाइसेंस स्ट्रिंग हो सकती है।

```java
AsposeOCR api = new AsposeOCR("YourLicenseKey");
```

### चरण 4: OCR पहचान चलाएँ

`RecognizeLine` मेथड को कॉल करके इमेज से टेक्स्ट की एक लाइन निकालें। परिणाम एक साधारण स्ट्रिंग होगा जिसे आप आगे प्रोसेस या स्टोर कर सकते हैं।

```java
try {
    String result = api.RecognizeLine(imagePath);
    System.out.println("File: " + imagePath);
    System.out.println("Result line: " + result);
} catch (IOException e) {
    e.printStackTrace();
}
```

> **प्रो टिप:** यदि आप आउटपुट को किसी विशिष्ट अक्षर सेट (जैसे केवल अंकों) तक सीमित करना चाहते हैं, तो `RecognizeLine` को कॉल करने से पहले `AsposeOCR` इंस्टेंस पर `setAllowedCharacters` मेथड का उपयोग करें। यह इंजन को परिभाषित सेट के बाहर के किसी भी अक्षर को अनदेखा करने के लिए मजबूर करता है।

**प्रश्न: क्या मैं Aspose.OCR में अनुमत अक्षर निर्दिष्ट कर सकता हूँ?**  
उत्तर: हाँ, आप `setAllowedCharacters` API का उपयोग करके कैरेक्टर सेट को कस्टमाइज़ कर सकते हैं। विस्तृत जानकारी के लिए आधिकारिक दस्तावेज़ देखें।

**प्रश्न: क्या Aspose.OCR नवीनतम JDK संस्करणों के साथ संगत है?**

**प्रश्न: क्या लाइन पहचान के अलावा अतिरिक्त OCR सुविधाएँ हैं?**  
उत्तर: हाँ, लाइब्रेरी ब्लॉक, पैराग्राफ, और पूर्ण‑पेज पहचान, साथ ही भाषा पैक और इमेज प्री‑प्रोसेसिंग विकल्पों का समर्थन करती है।
