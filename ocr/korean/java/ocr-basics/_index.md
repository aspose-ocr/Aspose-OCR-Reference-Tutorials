# 텍스트 이미지 추출 – OCR 기본

## 소개

이 포괄적인 가이드에서는 Aspose.OCR for Java를 사용하여 **텍스트 이미지를 추출하는 방법**을 배우게 되며, 스캔한 사진을 검색 가능하고 편집 가능한 콘텐츠로 변환합니다. 라이선스부터 기울기 보정까지 모든 과정을 안내해 드리며, **OCR 정확도를 향상**시키고 신뢰할 수 있는 문서 처리 파이프라인을 구축할 수 있습니다. 간단한 스캐너 앱을 만들든 엔터프라이즈급 솔루션을 구축하든, 이 단계들을 통해 OCR 과제를 자신 있게 해결할 수 있습니다.

## 빠른 답변
- **“텍스트 이미지 추출”이란 무엇인가요?** 그림 파일(PNG, JPEG, TIFF 등)에서 문자를 읽어 일반 텍스트로 변환하는 것을 의미합니다.  
- **Aspose.OCR을 사용하려면 라이선스가 필요합니까?** 예 – 유효한 라이선스를 사용하면 평가 워터마크가 제거되고 전체 성능을 사용할 수 있습니다.  
- **OCR 정확도를 높이려면 어떻게 해야 하나요?** 기울기 보정, 적절한 이미지 전처리, 최신 OCR 엔진 설정을 사용하십시오.  
- **Java에서 OCR 기울기 보정이 지원되나요?** 물론입니다 – Aspose.OCR은 기울기 각도를 계산하고 보정하는 내장 메서드를 제공합니다.  
- **여러 언어의 텍스트를 인식할 수 있나요?** 예 – API는 기본적으로 수십 개의 언어를 지원합니다.

## “텍스트 이미지 추출”이란?

이미지에서 텍스트를 추출한다는 것은 광학 문자 인식(OCR)을 적용하여 시각적인 문자를 기계가 읽을 수 있는 문자열로 변환하는 것을 의미합니다. Aspose.OCR for Java는 다양한 이미지 형식과 언어를 지원하며 이 변환을 효율적으로 처리합니다.

## 왜 Aspose.OCR for Java를 사용하나요?

- **높은 정확도** – 고급 알고리즘과 내장 기울기 보정으로 깨끗한 결과를 제공합니다.  
- **외부 종속성 없음** – 순수 Java 라이브러리로, 임베드하기 쉽습니다.  
- **포괄적인 문서** – 단계별 튜토리얼(아래와 같은) 덕분에 빠르게 진행할 수 있습니다.  
- **확장성** – 단일 페이지 스캔부터 대규모 배치 작업까지 모두 처리합니다.

## 전제 조건

- Java 8 이상 설치.  
- Maven 또는 Gradle을 사용한 종속성 관리.  
- 유효한 Aspose.OCR for Java 라이선스 파일(시험판은 Aspose 웹사이트에서 받을 수 있습니다).

### Aspose.OCR로 기울기 각도 계산 방법

기울어진 스캔은 OCR 품질을 크게 저하시킬 수 있습니다. 내장된 기울기 계산 메서드를 사용해 각도를 감지하고 인식 전에 이미지를 회전하십시오:

[Java에서 Aspose.OCR로 기울기 각도 계산](./calculate-skew-angle/)

> **왜 중요한가:** 기울기 보정(ocr skew correction)을 하면 인식률이 10‑20 % 향상되는 경우가 많습니다.

### OCR 텍스트 인식 – 텍스트 영역 사각형 얻기

기울기 보정 후에는 텍스트가 포함된 정확한 영역을 추출할 수 있습니다. 이는 추가 처리(예: 강조 표시 또는 삭제)를 위해 경계 상자 데이터가 필요할 때 유용합니다.

[Aspose.OCR에서 텍스트 영역 사각형 얻기](./get-rectangles-with-text-areas/)

> **사용 사례:** 텍스트 좌표를 PDF 주석으로 내보내거나 하위 자연어 파이프라인에 전달합니다.

## 일반적인 함정 및 해결책

| 문제 | 발생 원인 | 해결 방법 |
|-------|----------------|-----|
| 저해상도 이미지에서 낮은 정확도 | OCR 엔진이 픽셀화된 문자 처리에 어려움을 겪음 | OCR 전에 이미지를 확대하거나 샤프닝 필터를 적용하십시오. |
| 라이선스 인식 안 됨 | 라이선스 파일 경로가 잘못되었거나 로드되지 않음 | `License.setLicense("Aspose.OCR.lic")`가 클래스패스 리소스를 가리키는지 확인하십시오. |
| 기울기 각도가 0°로 반환됨 | 이미지가 이미 바로 서 있거나 전처리가 잘못됨 | 이미지에 명확한 기준선이 있는지 확인하고 시각적으로 검사하십시오. |

## 자주 묻는 질문

**Q: Aspose.OCR을 상용 제품에 사용할 수 있나요?**  
A: 예. 유효한 라이선스를 적용하면 제한 없이 모든 상용 애플리케이션에 라이브러리를 임베드할 수 있습니다.

**Q: 라이브러리가 손글씨 OCR을 지원하나요?**  
A: Aspose.OCR은 인쇄된 텍스트에 중점을 둡니다. 손글씨의 경우, Aspose와 함께 특화된 AI 서비스를 통합하는 것을 고려하십시오.

**Q: 노이즈가 많은 스캔에서 OCR 정확도를 어떻게 향상시킬 수 있나요?**  
A: 이미지를 전처리(이진화, 노이즈 제거)하고 항상 기울기 보정 단계를 수행하십시오. 이 조합이 최상의 결과를 제공합니다.

**Q: PDF에서 직접 텍스트를 추출할 수 있나요?**  
A: 먼저 PDF 페이지를 이미지로 변환(Aspose.PDF 또는 기타 PDF‑to‑image 도구 사용)한 뒤, 해당 이미지에 Aspose.OCR을 실행하십시오.

**Q: 기본적으로 지원되는 언어는 무엇인가요?**  
A: 영어, 스페인어, 중국어, 아라비아어 등을 포함해 30개 이상의 언어를 지원합니다. 언어 전환은 `ocrEngine.setLanguage(Language.English)`를 사용하십시오.
