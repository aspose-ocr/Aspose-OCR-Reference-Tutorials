**Q: ¿Cómo convierto un PDF escaneado en un PDF buscable?**  
A: Usa el tutorial `recognize-pdf` para aplicar OCR, luego guarda el documento con una capa de texto buscable.

**Q: ¿Puedo mejorar la precisión del OCR para un idioma específico?**  
A: Sí, habilita **ocr language selection** y establece el código de idioma apropiado antes del procesamiento.

**Q: ¿Qué pasa si solo necesito extraer texto de una región determinada de una imagen?**  
A: Aprovecha **Detect Areas Mode** para definir coordenadas y extraer texto de esa área específica.
