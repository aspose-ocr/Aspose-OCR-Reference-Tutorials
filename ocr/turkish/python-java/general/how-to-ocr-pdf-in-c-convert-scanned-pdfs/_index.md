# C# ile PDF OCR Nasıl Yapılır – Taralı PDF'leri Dönüştürme

Hiç **how to OCR PDF** yapmanız gerektiğinde, işbirliği etmeyi reddeden bir taralı dosyayla mı takıldınız? Tek başınız değilsiniz. Gerçek dünyadaki birçok projede—örneğin fatura dijitalleştirme veya tarihi raporları arşivleme—bir PDF içinde bir sürü görüntü bulursunuz ve bunları aranabilir hâle getirmenin tek yolu OCR çalıştırmaktır.  

İyi haber? Sadece birkaç C# satırıyla **convert scanned PDF**'i **create searchable PDF**'e dönüştürebilir, ince bir filigran ekleyebilir ve ham metni daha fazla işleme için çıkarabilirsiniz. Aşağıda tam olarak bunu yapan, çalıştırmaya hazır bir örnek göreceksiniz.

## Bu Eğitimde Neler Kapsanıyor

* `OcrEngine` sınıfını kullanarak **how to OCR PDF** nasıl yapılır.  
* Taralı bir PDF'i **create searchable PDF**'e dönüştürmek.  
* İlk sayfaya bir filigran eklemek (**add watermark pdf**).  
* Sonuçtan düz metin dizgilerini çekmek (**extract text from pdf**).  
* Yaygın tuzaklar—çok sayfalı belgelerle başa çıkma veya düşük çözünürlüklü taramalarla uğraşma gibi.

Harici dokümantasyon bağlantıları yok, yarım kalmış kod parçacıkları yok. İhtiyacınız olan her şey burada.

## Önkoşullar

* .NET 6.0 veya üzeri (kod .NET 5 ile de derlenir).  
* `OcrEngine` ve `ImageFormats` sağlayan OCR kütüphanesine referans (ör. `Aspose.OCR` veya benzeri bir paket).  
* Kontrol ettiğiniz bir klasöre yerleştirilmiş `input.pdf` adlı bir giriş PDF'i.  
* C# konsol uygulamalarıyla temel aşinalık.

Eğer bunlara sahipseniz, harika—hadi başlayalım.

## Adım 1: Projeyi Kurun ve Bağımlılıkları İçe Aktarın

Yeni bir konsol projesi oluşturun ve OCR paketini NuGet üzerinden ekleyin:

```bash
dotnet new console -n OcrPdfDemo
cd OcrPdfDemo
dotnet add package Aspose.OCR
```

Şimdi `Program.cs` dosyasını açın. En üstte, ihtiyacınız olan ad alanlarını ekleyin:

```csharp
using System;
using Aspose.OCR;          // OcrEngine lives here
using Aspose.OCR.Image;   // ImageFormats enum
```

*Neden önemli*: Doğru ad alanlarını içe aktarmak, derleyicinin `OcrEngine`'i bulmasını sağlar. Bu adımı atlamak `CS0246` hatasına yol açar ve derlemenizi durdurur.

## Adım 2: OCR Motorunu Başlatın

Motor, sürecin kalbidir. Onu bir kez örnekleyip her sayfa için yeniden kullanırsınız.

```csharp
// Step 2: Initialize the OCR engine
OcrEngine engine = new OcrEngine();
```

*Pro ipucu*: Ardışık olarak birçok PDF işleme planlıyorsanız, aynı `engine` örneğini yeniden kullanarak tekrarlanan lisans kontrollerinden kaçının.

## Adım 3: Kaynak PDF'i Yükleyin

Motorun hangi dosya üzerinde çalışacağını belirtin. `SetImageFromFile` yöntemi herhangi bir görüntü‑veya‑PDF kaynağını kabul eder.

```csharp
// Step 3: Load the scanned PDF you want to OCR
engine.SetImageFromFile(@"YOUR_DIRECTORY\input.pdf");
```

> **Bu adımın neden kritik olduğu** – Motor, her sayfanın bitmap temsiline ihtiyaç duyar. Bir PDF verdiğinizde, motor dahili olarak her sayfayı varsayılan DPI'ye (genellikle 300) göre rasterleştirir. Kaynak PDF zaten metin tabanlıysa, motor sadece değişiklik yapmadan geçirir, zaman kazandırır.

## Adım 4: OCR Tanımasını Çalıştırın

Şimdi asıl iş burada gerçekleşir. `Recognize` yöntemi her sayfayı tarar, glifleri çıkarır ve aranabilir bir katman oluşturur.

```csharp
// Step 4: Perform OCR on the loaded document
engine.Recognize();
```

*Yaygın soru*: *PDF 50 sayfa olursa ne olur?*  
`Recognize` varsayılan olarak tüm belgeyi işler. Bellek kısıtlı ortamlar için `engine.PageIndex` ve `engine.PageCount` ayarlarını sayfa‑sayfa çalışacak şekilde belirtebilirsiniz.

## Adım 5: Aranabilir PDF'i Kaydedin (Sayfa 1'de Filigran)

Sonucu PDF olarak kaydedeceğiz. İlk sayfa otomatik olarak bir filigran alacak çünkü OCR kütüphanesi varsayılan olarak “Generated by Aspose.OCR” üst katmanını ekler. Özel bir filigran ihtiyacınız varsa, kaydetmeden önce `engine.Watermark` özelliğini değiştirebilirsiniz.

```csharp
// Step 5: Save the OCR‑processed PDF with a watermark on page 1
engine.Save(@"YOUR_DIRECTORY\output.pdf", ImageFormats.Pdf);
```

Bu çağrıdan sonra, metni seçebileceğiniz, kopyalayabileceğiniz ve herhangi bir yerel PDF gibi arayabileceğiniz bir **create searchable PDF** elde edeceksiniz.

## Adım 6: Düz Metni Çıkarın (Opsiyonel ama Kullanışlı)

Eğer indeksleme veya daha ileri analiz için **extract text from pdf** yapmak isterseniz, motor size ham dizeye erişim sağlar.

```csharp
// Step 6: Pull the recognized text into a string
string extractedText = engine.Text;
Console.WriteLine("=== Extracted Text Start ===");
Console.WriteLine(extractedText);
Console.WriteLine("=== Extracted Text End ===");
```

`engine.Text` özelliği tüm sayfaların metnini satır sonlarını koruyarak birleştirir. Sayfa bazlı ayrıntıya ihtiyacınız varsa `\f` (form feed) ile bölümlendirebilirsiniz.

## Tam Çalışan Örnek

Aşağıda tam program yer alıyor. `Program.cs` içine kopyalayıp yapıştırın, `YOUR_DIRECTORY`'yi mutlak ya da göreli bir yol ile değiştirin ve `dotnet run` komutunu çalıştırın.

```csharp
using System;
using Aspose.OCR;
using Aspose.OCR.Image;

namespace OcrPdfDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            // 1️⃣ Initialize the OCR engine
            OcrEngine engine = new OcrEngine();

            // 2️⃣ Load the source PDF to be processed
            // Make sure the file exists; otherwise you'll get a FileNotFoundException.
            string inputPath = @"YOUR_DIRECTORY\input.pdf";
            engine.SetImageFromFile(inputPath);

            // 3️⃣ Perform OCR recognition on the loaded document
            engine.Recognize();

            // 4️⃣ Save the recognized document as a PDF (watermark appears on page 1)
            string outputPath = @"YOUR_DIRECTORY\output.pdf";
            engine.Save(outputPath, ImageFormats.Pdf);

            // 5️⃣ (Optional) Extract plain text for further use
            string extractedText = engine.Text;
            Console.WriteLine("=== Extracted Text Start ===");
            Console.WriteLine(extractedText);
            Console.WriteLine("=== Extracted Text End ===");

            Console.WriteLine($"OCR complete! Searchable PDF saved to: {outputPath}");
        }
    }
}
```

### Beklenen Çıktı

Programı çalıştırmak, çıkarılan metni konsola yazdırır ve `output.pdf` dosyasını oluşturur. PDF'i herhangi bir görüntüleyicide (Adobe Acrobat, Edge vb.) açın ve orijinal taramada yer alan bir kelimeyi aramayı deneyin—artık aranabilir olduğunu göreceksiniz. İlk sayfa varsayılan filigranı gösterir, **add watermark pdf** adımının başarılı olduğunu doğrular.

## Sık Sorulan Sorular & Özel Durumlar

| Soru | Cevap |
|----------|--------|
| **Tarama düşük çözünürlükte ise ne olur?** | Tanımadan önce DPI'yi artırın: `engine.ImageInfo.DpiX = engine.ImageInfo.DpiY = 600;` Bu, OCR motoruna daha fazla detay sağlar. |
| **Özel bir filigran seçebilir miyim?** | Evet. `Save`'den önce `engine.Watermark = "Confidential – Processed on " + DateTime.Now.ToShortDateString();` olarak ayarlayın. |
| **Şifre korumalı PDF'leri nasıl ele alırım?** | `SetImageFromFile`'den önce `engine.LoadPassword = "yourPassword";` kullanın. |
| **OCR'ı belirli sayfalara sınırlamanın bir yolu var mı?** | Sayfaları 3‑7 işlemek için `engine.PageIndex = 2;` ve `engine.PageCount = 5;` ayarlayın. |
| **Orijinal görüntüleri dokunulmaz tutmam gerekirse?** | OCR sonrası, orijinal PDF'i OCR katmanı ile bir PDF kütüphanesi (ör. `Aspose.PDF`) kullanarak birleştirerek görüntü kalitesini koruyabilirsiniz. |

## İpuçları & En İyi Uygulamalar

* **Pro ipucu:** Bunu bir UI uygulamasına entegre ediyorsanız OCR'ı arka plan iş parçacığında çalıştırın—aksi takdirde UI donacaktır.  
* **Dikkat edin:** Karışık raster ve vektör içeriğe sahip PDF'ler. Motor sadece raster sayfaları OCR'lar; vektör metin otomatik olarak seçilebilir kalır.  
* **Performans ayarı:** OCR dil verilerini önbelleğe alın (`engine.Language = OcrLanguage.English;`) böylece her belge için yeniden yüklemek zorunda kalmazsınız.

## Sonuç

Artık C# içinde **how to OCR PDF** için tam, uçtan uca bir çözümünüz var. `OcrEngine`'i başlatarak, taralı bir dosyayı yükleyerek, metni tanıyarak, bir **create searchable pdf** kaydederek, filigran ekleyerek ve isteğe bağlı olarak **extract text from pdf** yaparak, herhangi bir görüntü‑tabanlı PDF'i aranabilir, indekslenebilir bir varlığa dönüştürebilirsiniz.

Sonraki adımlar? Bu iş akışını bir belge yönetim sistemiyle zincirlemeyi deneyin veya farklı dillerle (ör. `engine.Language = OcrLanguage.French;`) deney yapın. Ayrıca bir klasördeki birden fazla dosyayı toplu işleme keşfedebilirsiniz—her seferinde yeni bir `engine` örneğiyle adımları döngüye alarak.

Kodlamaktan keyif alın ve PDF'leriniz her zaman aranabilir olsun!
