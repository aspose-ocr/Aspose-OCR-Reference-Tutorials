# استخراج النص من الصورة – التعرف على السطر باستخدام Aspose.OCR

## المقدمة

أصبح التعرف الضوئي على الحروف (OCR) هو الحل المفضل لتحويل صور النص إلى محتوى قابل للبحث والتحرير. إذا كنت تبحث عن **استخراج النص من الصورة** بسرعة وموثوقية، فإن Aspose.OCR لـ .NET يقدم واجهة برمجة تطبيقات قوية وسهلة الاستخدام للمطورين تعمل على كل من إطار .NET الكامل ومشاريع **ASP OCR .NET Core**. في هذا البرنامج التعليمي سنستعرض كل ما تحتاج معرفته للتعرف على السطور في صورة، تحويل تلك السطور إلى نص عادي، وعرض النتيجة — كل ذلك باستخدام كود C# نظيف وسهل المتابعة.

## إجابات سريعة
- **ماذا يفعل Aspose.OCR؟** يقرأ النص المطبوع أو المكتوب يدويًا من صيغ الصور ويعيد سلاسل نصية عادية.  
- **ما صيغ الصور المدعومة؟** PNG، JPEG، BMP، GIF، TIFF وغيرها.  
- **هل أحتاج إلى ترخيص للاختبار؟** النسخة التجريبية المجانية تعمل للتطوير؛ الترخيص مطلوب للإنتاج.  
- **هل يمكن تشغيله على .NET Core؟** نعم – المكتبة تدعم .NET Framework 4.5+، .NET Core 3.1+، .NET 5/6.  
- **كم يستغرق التعرف على سطر بسيط؟** عادةً أقل من ثانية لملف PNG قياسي.

## ما هو “استخراج النص من الصورة”؟

استخراج النص من صورة يعني استخدام تقنية OCR لتحليل البكسلات المرئية، تحديد الأحرف، وإخراجها كنص قابل للقراءة آليًا. يتيح ذلك سيناريوهات مثل رقمنة المستندات الممسوحة، أتمتة إدخال البيانات من الإيصالات، أو إنشاء أرشيفات قابلة للبحث.

## لماذا تستخدم Aspose.OCR لـ .NET؟

- **دقة عالية** عبر لغات وخطوط متعددة.  
- **لا توجد تبعيات خارجية** – كود مُدار بالكامل، سهل التكامل.  
- **دعم شامل للصيغ** – يعمل مع PNG، JPEG، BMP، وأكثر.  
- **واجهة برمجة تطبيقات بسيطة** – بضع أسطر من الكود تنقلك من الصورة إلى النص.  

### كيف يساعدك هذا في **تحويل PNG إلى نص**؟

نظرًا لأن Aspose.OCR يمكنه قراءة ملفات PNG مباشرةً، يمكنك تمرير صورة PNG ممسوحة إلى طريقة `RecognizeLine` والحصول على سلسلة نصية نظيفة دون أي خطوات تحويل وسيطة.

قبل أن نبدأ، تأكد من أن لديك:

## كيفية استخراج النص من الصورة باستخدام Aspose.OCR

فيما يلي تنفيذ خطوة بخطوة. كل كتلة كود لم تتغير عن البرنامج التعليمي الأصلي، مما يضمن بقاء المنطق دقيقًا.

### الخطوة 1: تهيئة Aspose.OCR

```csharp
// ExStart:1
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
// ExEnd:1
```

> **نصيحة احترافية:** استخدم مسارًا مطلقًا أو `Path.Combine` لتجنب مشاكل فواصل المسار عبر أنظمة التشغيل.

### الخطوة 2: التعرف على سطور الصورة

```csharp
// ExStart:3
// Recognize image
string result = api.RecognizeLine(dataDir + "sample_line.png");
// ExEnd:3
```

طريقة `RecognizeLine` تركز على سطر واحد من النص، مما يجعلها مثالية عندما تعرف تخطيط صورتك. كما أنها طريقة رائعة **لقراءة نص الصورة الممسوحة** عندما يحتوي المستند على سطر واحد من البيانات المهمة.

### الخطوة 3: عرض النص المعترف به

```csharp
// ExStart:4
// Display the recognized text
Console.WriteLine(result);
// ExEnd:4
```

تشغيل البرنامج سيطبع السطر المستخرج إلى وحدة التحكم، مؤكدًا أن عملية **استخراج النص من الصورة** نجحت.

### الخطوة 4: رسالة الانتهاء

```csharp
Console.WriteLine("RecognizeLine executed successfully");
```

ظهور هذه الرسالة يعني أن عملية OCR اكتملت دون أخطاء.

## كيف تحسن دقة OCR مع Aspose.OCR؟

- **استخدم صورًا عالية الدقة** (300 dpi أو أعلى).  
- **طبق معالجة مسبقة للصور** مثل التثنائي أو إزالة الضوضاء باستخدام `api.PreprocessImage`.  
- **اختر اللغة الصحيحة** باستخدام `api.Language = OcrLanguage.English;` (أو تعداد اللغة المناسب).  
- **قص الحدود** لإزالة الخلفية غير ذات الصلة.

| دقة منخفضة | صورة منخفضة الدقة | استخدم مصدرًا بدقة أعلى أو قم بمعالجة الصورة مسبقًا (مثل التثنائي). |
| صيغة غير مدعومة | الصورة غير موجودة في القائمة المدعومة | حوّل الصورة إلى PNG أو JPEG قبل استدعاء `RecognizeLine`. |
