# استخراج النص من الصورة – التعرف على السطر باستخدام Aspose.OCR

## المقدمة

أصبح التعرف الضوئي على الأحرف (OCR) هو الحل المفضل لتحويل صور النص إلى محتوى قابل للبحث والتحرير. إذا كنت تبحث عن **استخراج النص من ملفات الصورة** بسرعة وموثوقية، فإن Aspose.OCR لـ .NET يقدم واجهة برمجة تطبيقات قوية وسهلة الاستخدام للمطورين. في هذا الدرس سنستعرض كل ما تحتاج معرفته للتعرف على السطور في صورة، تحويل تلك السطور إلى نص عادي، وعرض النتيجة—كل ذلك باستخدام كود C# واضح وسهل المتابعة.

## إجابات سريعة
- **ماذا يفعل Aspose.OCR؟** يقرأ النص المطبوع أو المكتوب بخط اليد من صيغ الصور ويعيد سلاسل نصية عادية.  
- **ما هي صيغ الصور المدعومة؟** PNG، JPEG، BMP، GIF، TIFF وغيرها.  
- **هل أحتاج إلى ترخيص للاختبار؟** النسخة التجريبية المجانية تكفي للتطوير؛ الترخيص مطلوب للإنتاج.  
- **هل يمكن تشغيله على .NET Core؟** نعم – المكتبة تدعم .NET Framework 4.5+، .NET Core 3.1+، .NET 5/6.  
- **كم يستغرق التعرف على سطر بسيط؟** عادةً أقل من ثانية لملف PNG قياسي.

## ما هو “استخراج النص من الصورة”؟

استخراج النص من الصورة يعني استخدام تقنية OCR لتحليل البكسلات المرئية، تحديد الأحرف، وإخراجها كنص قابل للقراءة آليًا. يتيح ذلك سيناريوهات مثل رقمنة المستندات الممسوحة، أتمتة إدخال البيانات من الإيصالات، أو إنشاء أرشيفات قابلة للبحث.

## لماذا نستخدم Aspose.OCR لـ .NET؟

- **دقة عالية** عبر لغات وخطوط متعددة.  
- **بدون تبعيات خارجية** – كود مُدار بالكامل، سهل التكامل.  
- **دعم شامل للصيغ** – يعمل مع PNG، JPEG، BMP، وغيرها.  
- **واجهة برمجة تطبيقات بسيطة** – بضع أسطر من الكود تكفي لتحويل الصورة إلى نص.

## المتطلبات السابقة

قبل أن نبدأ، تأكد من وجود ما يلي:

- **بيئة التطوير** – Visual Studio 2022 (أو أي بيئة تطوير .NET تفضلها).  
- **Aspose.OCR لـ .NET** – حمّله من [رابط التحميل](https://releases.aspose.com/ocr/net/).  
- **دليل المستندات** – مجلد على جهازك يحتوي على صورة العينة (`sample_line.png`). استبدل “Your Document Directory” في الكود بالمسار الفعلي.

## استيراد المساحات الاسمية

في .NET، تُعطيك المساحات الاسمية الوصول إلى الفئات التي تحتاجها. أضف عبارات `using` هذه في أعلى ملف C# الخاص بك:

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

## كيفية استخراج النص من الصورة باستخدام Aspose.OCR

فيما يلي التنفيذ خطوة بخطوة. كل كتلة شفرة تبقى كما هي من الدرس الأصلي، لضمان بقاء المنطق دون تغيير.

### الخطوة 1: تهيئة Aspose.OCR

```csharp
// ExStart:1
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
// ExEnd:1
```

> **نصيحة احترافية:** استخدم مسارًا مطلقًا أو `Path.Combine` لتجنب مشاكل فواصل المسارات بين أنظمة التشغيل.

### الخطوة 2: التعرف على سطور الصورة

```csharp
// ExStart:3
// Recognize image
string result = api.RecognizeLine(dataDir + "sample_line.png");
// ExEnd:3
```

تركز طريقة `RecognizeLine` على سطر نص واحد، مما يجعلها مثالية عندما تعرف تخطيط صورتك.

### الخطوة 3: عرض النص المعترف به

```csharp
// ExStart:4
// Display the recognized text
Console.WriteLine(result);
// ExEnd:4
```

عند تشغيل البرنامج سيطبع السطر المستخرج على وحدة التحكم، مؤكدًا أن عملية **استخراج النص من الصورة** نجحت.

### الخطوة 4: رسالة الانتهاء

```csharp
Console.WriteLine("RecognizeLine executed successfully");
```

ظهور هذه الرسالة يعني أن عملية OCR انتهت دون أخطاء.

## المشكلات الشائعة والحلول

| المشكلة | السبب | الحل |
|-------|--------|-----|
| `FileNotFoundException` | مسار `dataDir` غير صحيح | تحقق من مسار المجلد وتأكد من وجود `sample_line.png`. |
| دقة منخفضة | صورة منخفضة الدقة | استخدم مصدرًا بدقة أعلى أو عالج الصورة مسبقًا (مثل التحويل إلى ثنائي). |
| صيغة غير مدعومة | الصورة ليست ضمن الصيغ المدعومة | حوّل الصورة إلى PNG أو JPEG قبل استدعاء `RecognizeLine`. |
