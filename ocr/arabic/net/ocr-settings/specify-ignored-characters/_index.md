# دعم لغة OCR: تحديد الأحرف المتجاهلة في التعرف على الصور

## المقدمة

دعم لغة OCR هو حجر الأساس لأتمتة المستندات الحديثة، حيث يتيح للتطبيقات قراءة النص من الصور عبر العديد من الأبجديات والرموز. في هذا الدرس ستتعلم كيفية إخبار **Aspose.OCR for .NET** بتجاهل أحرف معينة أثناء التعرف—حيلة أساسية عندما تحتاج إلى مخرجات أنظف أو تريد تصفية الضوضاء مثل أرقام الصفحات أو الرموز الزخرفية. بنهاية الدليل ستحصل على مقتطف جاهز للتنفيذ يوضح الميزة من البداية إلى النهاية.

## إجابات سريعة
- **ما معنى “الأحرف المتجاهلة”?** الأحرف التي يتخطاها محرك OCR أثناء بناء سلسلة النتيجة.  
- **لماذا نستخدمها؟** تحسين الدقة عندما تكون بعض الرموز غير ذات صلة بمنطق عملك.  
- **ما هي طريقة API التي تتعامل معها؟** `RecognitionSettings.IgnoredCharacters`.  
- **هل يمكن دمجها مع حزم اللغات؟** نعم—الأحرف المتجاهلة تعمل جنبًا إلى جنب مع أي لغة تقوم بتحميلها.  
- **هل يلزم ترخيص؟** يلزم وجود ترخيص مؤقت أو كامل للاستخدام في الإنتاج.

## لماذا تحديد الأحرف المتجاهلة؟

في العديد من السيناريوهات الواقعية—مثل معالجة الفواتير، الإيصالات، أو النماذج متعددة اللغات—قد تواجه أحرفًا متكررة لا تشكل جزءًا من البيانات ذات المعنى (مثل الشرطات المستخدمة كفواصل، أرقام الصفحات، أو الرموز الزخرفية). بإخبار محرك OCR بتخطي هذه الأحرف، تقلل من جهد ما بعد المعالجة وتحسن موثوقية التحليلات اللاحقة.

## دليل خطوة بخطوة

### الخطوة 1: إعداد دليل المستندات الخاص بك

ابدأ بتحديد الدليل الذي تُخزن فيه مستنداتك. استبدل `"Your Document Directory"` بالمسار الفعلي لمستنداتك.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";
```

### الخطوة 2: تهيئة Aspose.OCR

أنشئ نسخة من محرك OCR. هذا الكائن سيتعامل مع جميع استدعاءات التعرف اللاحقة.

```csharp
// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

### الخطوة 3: التعرف على الصورة مع الأحرف المتجاهلة

مرّر ملف الصورة مع كائن `RecognitionSettings` الذي يدرج الأحرف التي تريد تجاهلها. في هذا المثال نتجاهل الأحرف `a` و `b` و `1`.

```csharp
// Recognize image with specified ignored characters
RecognitionResult result = api.RecognizeImage(dataDir + "SpanishOCR.bmp", new RecognitionSettings
{
    IgnoredCharacters = "ab1"
});
```

### الخطوة 4: عرض النص المتعرف عليه

أخيرًا، اطبع النص المنقح إلى وحدة التحكم أو أي مخرج آخر تفضله.

```csharp
// Display the recognized text
Console.WriteLine(result.RecognitionText);
```

## المشكلات الشائعة والنصائح

- **مسار غير صحيح:** تأكد من أن `dataDir` ينتهي بفاصل مسار (`\` أو `/`) المناسب لنظام التشغيل الخاص بك.  
- **لغة غير مدعومة:** يجب أن يكون لدى محرك OCR حزمة اللغة للصورة المصدر؛ وإلا لن يتم تطبيق الأحرف المتجاهلة بشكل صحيح.  
- **أخطاء الترخيص:** إذا رأيت استثناءً متعلقًا بالترخيص، تحقق من أن ملف الترخيص المؤقت مُشار إليه بشكل صحيح في مشروعك.
