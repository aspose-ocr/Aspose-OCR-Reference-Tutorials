# استخراج النص من الصور باستخدام Aspose.OCR – الأحرف المسموح بها

## المقدمة

استخراج النص من الصور هو طلب شائع في التطبيقات الحديثة—سواء كنت تعالج الفواتير، أو تمسح الإيصالات، أو تقوم برقمنة المستندات المطبوعة. **Aspose.OCR for Java** يجعل هذه المهمة سهلة، حيث يقدم التعرف بدقة عالية وخيارات تكوين مرنة مثل تحديد الأحرف المسموح بها. في هذا البرنامج التعليمي سنستعرض **دورة Aspose OCR Java** كاملة تُظهر لك كيفية إعداد المكتبة، تشغيل OCR، وتقييد مجموعة الأحرف لتتناسب مع احتياجاتك.

## إجابات سريعة
- **ماذا يفعل Aspose.OCR؟** يستخرج النص من الصور بدقة عالية ويدعم مجموعات أحرف مخصصة.  
- **هل أحتاج إلى ترخيص؟** يلزم وجود ترخيص مؤقت أو دائم للاستخدام في بيئة الإنتاج.  
- **ما نسخة JDK المدعومة؟** إصدارات JDK الأخيرة متوافقة بالكامل.  
- **هل يمكنني تقييد الأحرف المعترف بها** نعم—استخدم واجهة برمجة تطبيقات allowed‑characters لتقيد المخرجات.  
- **كم يستغرق الإعداد؟** حوالي 10‑15 دقيقة لتطبيق أساسي.

## ما هو “استخراج النص من الصور”؟
استخراج النص من الصور يشير إلى عملية تحويل النص البصري (مثل المطبوعة أو المكتوبة بخط اليد) إلى سلاسل يمكن قراءتها آليًا. هذا يتيح مهامًا لاحقة مثل البحث، الفهرسة، أو تحليل البيانات.

## لماذا نستخدم Aspose.OCR for Java؟
- **دقة عالية** عبر لغات وخطوط متعددة.  
- **واجهة برمجة تطبيقات بسيطة** يمكن دمجها مع أي مشروع Java.  
- **قابلة للتخصيص** من حيث مجموعات الأحرف، حزم اللغات، ومعالجة الصور مسبقًا.  
- **بدون تبعيات خارجية**—المكتبة مكتفية ذاتيًا.

## استيراد الحزم

بعد تجهيز المتطلبات، استورد الحزم الضرورية إلى مشروع Java الخاص بك:

```java
package com.aspose.ocr.examples.OcrFeatures;

import com.aspose.ocr.AsposeOCR;
import com.aspose.ocr.examples.Utils;

import java.io.IOException;
```

## دليل خطوة بخطوة

### الخطوة 1: تعيين دليل المستندات الخاص بك

حدد مجلدًا ستخزن فيه نتائج OCR المعالجة. يُستخدم هذا المسار لاحقًا لتحديد موقع ملف الصورة.

```java
String dataDir = "Your Document Directory";
```

### الخطوة 2: تحديد مسار الصورة

وجه الـ API إلى الصورة التي تريد تحليلها.

```java
String imagePath = dataDir + "0001460985.Jpeg";
```

### الخطوة 3: إنشاء مثيل Aspose.OCR

أنشئ محرك OCR باستخدام مفتاح الترخيص الخاص بك. يمكن أن يكون المفتاح ترخيصًا مؤقتًا أو دائمًا.

```java
AsposeOCR api = new AsposeOCR("YourLicenseKey");
```

### الخطوة 4: تنفيذ التعرف على النص

استدعِ طريقة `RecognizeLine` لاستخراج سطر نص من الصورة. النتيجة هي سلسلة نصية عادية يمكنك معالجتها أو تخزينها لاحقًا.

```java
try {
    String result = api.RecognizeLine(imagePath);
    System.out.println("File: " + imagePath);
    System.out.println("Result line: " + result);
} catch (IOException e) {
    e.printStackTrace();
}
```

> **نصيحة احترافية:** إذا كنت بحاجة لتقييد المخرجات إلى مجموعة محددة من الأحرف (مثل الأرقام فقط)، استخدم طريقة `setAllowedCharacters` على مثيل `AsposeOCR` قبل استدعاء `RecognizeLine`. هذا يضمن أن المحرك سيتجاهل أي أحرف خارج المجموعة المحددة.

## المشكلات الشائعة والحلول

| المشكلة | السبب | الحل |
|-------|--------|-----|
| **لا توجد مخرجات أو سلسلة فارغة** | مسار الصورة غير صحيح أو تنسيق صورة غير مدعوم | تحقق من `imagePath` واستخدم تنسيقًا مدعومًا (JPEG, PNG, BMP) |

| **الترخيص غير مُطبق** | مفتاح الترخيص مفقود أو غير صالح | تأكد من صحة سلسلة الترخيص ووضعها في مُنشئ `AsposeOCR` |
