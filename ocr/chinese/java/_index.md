## Common Issues & Troubleshooting Tips
- **倾斜的图像** – 在识别前使用 `calculateSkewAngle()` 方法。  
- **低分辨率文件** – 提高 DPI 或进行图像预处理（例如二值化）。  
- **语言检测不正确** – 使用 `setLanguage(Language.English)` 明确设置语言。  

## Frequently Asked Questions

**问：我可以在商业 Java 应用中使用 Aspose OCR 吗？**  
答：可以，需要有效的商业许可证。提供免费试用供评估。

**问：如何从多页 PDF 中提取 image text java？**  
答：使用 `OcrEngine` 加载 PDF，遍历页面，并对每页调用 `recognize()`。

**问：库是否支持从右到左的语言，如阿拉伯语？**  
答：当然——只需设置相应的语言代码，引擎即可处理 RTL 文本。

**问：大批量处理时有哪些性能考虑？**  
答：异步处理图像并复用 `OcrEngine` 实例以降低开销。

**问：是否可以将 OCR 限制在图像的特定区域？**  
答：可以，在识别前启用 Detect Areas Mode 并定义区域坐标。
