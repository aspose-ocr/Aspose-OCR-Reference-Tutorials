# OCR 設定

## Introduction

.NET プロジェクトで画像から **how to extract OCR** データを取得したい場合、ここが最適な場所です。Aspose.OCR for .NET は OCR 画像認識をシンプルかつ高速に、かつ高度にカスタマイズ可能にします。本ガイドでは、アーカイブ、フォルダー、言語選択、リストベースの処理といった主要な構成シナリオを順に解説し、画像からテキストを自信を持って抽出できるようにします。

## Quick Answers
- **What does “how to extract OCR” mean?** OCR エンジンを使用して画像ベースのテキストを読み取り、編集可能な文字列に変換することを指します。  
- **Which library is recommended for .NET?** Aspose.OCR は外部依存なしで OCR 画像認識を行う豊富な API を提供します。  
- **Do I need a license?** 無料トライアルで評価は可能です。商用利用には商用ライセンスが必要です。  
- **Which .NET versions are supported?** .NET Framework 4.5+、.NET Core 3.1+、.NET 5/6/7+ をサポートしています。  
- **Can I process multiple languages?** はい。OCR 言語選択機能を使用して多言語ドキュメントに対応できます。

## What is “how to extract OCR” in .NET?
OCR の抽出とは、画像（または画像のコレクション）を Aspose.OCR に渡し、視覚コンテンツを解析して文字を識別し、認識されたテキストを返すことです。このプロセスは、スキャンした契約書のデジタル化、レシートの読み取り、PDF のインデックス作成などのシナリオを支えます。

## Why use Aspose.OCR for OCR image recognition?
- **High accuracy** 幅広いフォントや画像品質に対して高精度を実現。  
- **Built‑in language support** OCR 言語選択でリアルタイムに言語を切り替え可能。  
- **No external installations** Windows、Linux、macOS で追加インストール不要で動作。  
- **Flexible input sources** アーカイブ、フォルダー、ストリーム、インメモリリストなど柔軟に入力を受け取れます。

## Prerequisites
- .NET 開発環境（Visual Studio 2022 または VS Code）。  
- Aspose.OCR for .NET NuGet パッケージがインストール済み（`Install-Package Aspose.OCR`）。  
- 処理したい画像を含むサンプル画像またはアーカイブ。

### Common Use Cases
- **Extract text images** スキャンした請求書からテキストを抽出し、会計業務を自動化。  
- 大規模な文書アーカイブをインデックス化し、検索可能なリポジトリを構築。

### Tips & Pitfalls
- **Pro tip:** 画像を事前に前処理（デスキュー、二値化など）して精度を向上させましょう。  
- **Warning:** 大容量のアーカイブはメモリを大量に消費する可能性があります。バッチ処理を検討してください。

## Frequently Asked Questions

**Q: Can I use Aspose.OCR in a commercial .NET application?**  
A: はい、有効な Aspose ライセンスを取得すれば商用利用が完全にサポートされます。

**Q: Does OCR language selection work for right‑to‑left scripts?**  
A: 完全に対応しています。Aspose.OCR はアラビア語、ヘブライ語など RTL 言語をサポートしています。

**Q: How do I handle PDFs that contain images?**  
A: まず画像ページを抽出します（例: Aspose.PDF を使用）。その後、抽出した画像を Aspose.OCR に渡します。

**Q: What if the archive contains non‑image files?**  
A: API はサポート外のファイルを自動的にスキップします。必要に応じて拡張子でフィルタリングできます。

**Q: Is there a way to get confidence scores for each recognized word?**  
A: はい、`OcrResult` オブジェクトが各単語の信頼度スコアを提供し、プログラムから取得可能です。
