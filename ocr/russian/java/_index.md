## Frequently Asked Questions

**Q: Can I use Aspose.OCR for Java in a commercial application?**  
A: Да. Действительная лицензия Aspose снимает все ограничения оценки и позволяет неограниченно развёртывать продукт в коммерческих приложениях.

**Q: Which image formats are supported for text extraction?**  
A: JPEG, PNG, BMP, TIFF, GIF и страницы PDF, основанные на изображениях, поддерживаются «из коробки».

**Q: How do I improve accuracy for multilingual documents?**  
A: Установите свойство `Language` в соответствующий ISO‑код языка (например, `en`, `fr`, `zh`) перед вызовом OCR‑движка.

**Q: Is there a way to process only a specific region of an image?**  
A: Абсолютно. Используйте `DetectAreasMode` или вручную задайте прямоугольник, чтобы ограничить OCR нужной областью.

**Q: What should I do if the extracted text contains many spelling errors?**  
A: Выполните пост‑обработку результата с помощью библиотеки проверки орфографии или используйте встроенный метод Aspose `CorrectSpelling` (доступен в более новых версиях).
