# Как выполнить извлечение текста из изображения из потока с помощью Aspose OCR

Добро пожаловать в мир **извлечения текста из изображения** с **Aspose.OCR for .NET**. В этом руководстве вы увидите, как прочитать поток изображения, выполнить OCR для PNG‑файла и получить распознанный текст в ваше приложение C#. Независимо от того, создаёте ли вы конвейер обработки документов, инструмент автоматизации ввода данных или просто экспериментируете с OCR, приведённые ниже шаги помогут вам превратить необработанное изображение в поисковый текст за считанные минуты.

## Краткие ответы
- **Что демонстрирует это руководство?** Извлечение текста из изображения, предоставленного в виде потока, с помощью Aspose OCR.  
- **Какое ключевое слово является основным?** *image text extraction* (используется по всему руководству).  
- **Нужна ли лицензия для разработки?** Бесплатная пробная версия подходит для тестирования; для использования в продакшене требуется коммерческая лицензия.  
- **Можно ли обрабатывать PNG‑файлы напрямую?** Да — Aspose OCR обрабатывает форматы **ocr png file** без дополнительного преобразования.  
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Что такое извлечение текста из изображения?
Извлечение текста из изображения (также называемое OCR) преобразует визуальные символы на изображении в редактируемый, поисковый текст. С помощью Aspose OCR вы можете передать `MemoryStream`, содержащий любое поддерживаемое изображение (PNG, JPEG, BMP и т.д.), и получить распознанную строку одним вызовом.

## Почему стоит выбрать Aspose OCR для извлечения текста из изображения?
- **Широкая поддержка языков** – работает с десятками языков сразу.  
- **Простой API** – несколько строк C# преобразуют **image to memorystream** в читаемый текст.  
- **Высокая точность** – продвинутые алгоритмы обрабатывают шумные сканы и PNG с низким разрешением.  
- **Кроссплатформенный** – работает на Windows, Linux и macOS с .NET Core.

## Требования

Перед началом убедитесь, что у вас есть:

- Установлен Aspose.OCR for .NET (скачайте из [Aspose.OCR for .NET Documentation](https://reference.aspose.com/ocr/net/)).  
- Файл примера изображения (например, **sample.png**) размещён в папке, к которой вы можете обратиться из кода.

## Импорт пространств имён

Добавьте необходимые пространства имён в ваш файл C#:

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

## Пошаговое руководство

### Шаг 1: Установите каталог документов
Замените **"Your Document Directory"** на фактическую папку, содержащую *sample.png*.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";
```

### Шаг 2: Инициализируйте движок Aspose OCR
Создание объекта `AsposeOcr` предоставляет доступ ко всем методам OCR.

```csharp
// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

### Шаг 3: Прочитайте поток изображения и распознайте текст
Здесь мы открываем **sample.png**, копируем его байты в `MemoryStream` и передаём этот поток в `RecognizeImage`. Это демонстрирует **image stream ocr** и шаблон **read image stream c#** в едином процессе.

```csharp
// Recognize image
using (MemoryStream ms = new MemoryStream())
using (FileStream file = new FileStream(dataDir + "sample.png", FileMode.Open, FileAccess.Read))
{
    file.CopyTo(ms);
    result = api.RecognizeImage(ms);
}
```

### Шаг 4: Отобразите распознанный текст
Результат OCR выводится в консоль; вы также можете сохранить его в базе данных или файле.

```csharp
// Display the recognized text
Console.WriteLine(result);
```

### Шаг 5: Подтвердите успешное выполнение
Простое подтверждение сообщает, что процесс завершён без исключений.

```csharp
Console.WriteLine("RecognizeImageFromStream executed successfully");
```

| *Неподдерживаемый формат изображения* | Конвертируйте источник в PNG или JPEG перед вызовом `RecognizeImage`. |
