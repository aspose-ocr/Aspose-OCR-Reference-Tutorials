# Анализ макета OCR – Получение прямоугольников строк из изображений

## Introduction

В этом руководстве вы узнаете **как получить прямоугольники строк OCR** с помощью Aspose.OCR для .NET. К концу руководства вы сможете **распознавать строки текста на изображении** и **извлекать координаты строк** для каждой обнаруженной строки — идеально для последующей обработки, такой как **анализ макета OCR**, извлечение данных или пользовательская отрисовка.

## Quick Answers
- **Что означает “get OCR line rectangles”?** Возвращает ограничивающие рамки каждой строки текста, обнаруженной на изображении.  
- **Какой метод API используется?** `AsposeOcr.GetRectangles(..., AreasType.LINES, ...)`.  
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для разработки; для продакшна требуется коммерческая лицензия.  
- **Поддерживаемые форматы изображений?** PNG, JPEG, BMP, TIFF и другие.  
- **Можно ли запускать на .NET Core?** Да, Aspose.OCR полностью поддерживает .NET Core и .NET 5/6.

## Import Namespaces

Убедитесь, что необходимые пространства имён импортированы в ваш проект. Добавьте следующие строки в начало вашего C# файла:

```csharp
using System;
using System.Collections.Generic;

```

Теперь давайте разберём процесс получения прямоугольников строк в распознавании изображений OCR на простые‑для‑выполнения шаги.

## layout analysis ocr – Step‑by‑Step Guide

### Step 1: Set Up Your Document Directory

```csharp
// ExStart:3
string dataDir = "Your Document Directory";
// ExEnd:3
```

Замените `"Your Document Directory"` на фактический путь к папке, содержащей ваше примерное изображение.

### Step 2: Initialize Aspose.OCR

```csharp
// ExStart:4
AsposeOcr api = new AsposeOcr();
// ExEnd:4
```

Создайте экземпляр класса `AsposeOcr` для доступа к функционалу OCR.

### Step 3: Specify Image Path

```csharp
// ExStart:5
string fullPath = dataDir + "sample.png";
// ExEnd:5
```

Укажите полный путь к изображению, которое нужно распознать.

### Step 4: Recognize Image and Get Rectangles

```csharp
// ExStart:6
List<Rectangle> lines = api.GetRectangles(fullPath, AreasType.LINES, false);
// ExEnd:6
```

Метод `GetRectangles` возвращает список объектов `Rectangle`, каждый из которых представляет координаты обнаруженной строки текста. Это основа **получения прямоугольников строк OCR** и позволяет выполнять **анализ макета OCR**.

### Step 5: Print Result

```csharp
// ExStart:7
Console.WriteLine("Areas coordinates:");
lines.ForEach(a => Console.WriteLine($"x:{a.X} y:{a.Y} width:{a.Width} height:{a.Height}"));
// ExEnd:7
```

Выведите координаты обнаруженных областей в консоль. Вы увидите значения, которые позже можно использовать для **извлечения координат строк** для пользовательской обработки.

## Common Issues and Solutions

| Проблема | Решение |
|----------|---------|
| **Не возвращаются прямоугольники** | Убедитесь, что изображение содержит чёткий горизонтальный текст и указано `AreasType.LINES`. |
| **Некорректные координаты** | Проверьте DPI изображения; изображения с низким разрешением могут приводить к неточным границам. |
| **Узкое место производительности на больших изображениях** | Измените размер изображения до разумного разрешения перед вызовом `GetRectangles`. |
| **Исключение лицензии** | Используйте пробную лицензию для тестирования; примените полную лицензию для продакшна, чтобы избежать ограничений оценки. |

## Frequently Asked Questions

**Q: Можно ли извлечь отдельные слова вместо целых строк?**  
A: Да, используйте `AreasType.WORDS` с тем же методом `GetRectangles`, чтобы получить ограничивающие рамки уровня слов.

**Q: Поддерживает ли API многостраничные PDF?**  
A: Сначала преобразуйте каждую страницу PDF в изображение, затем вызовите `GetRectangles` для каждого изображения.

**Q: Как обрабатывать повернутый текст?**  
A: Включите опцию авто‑поворота в настройках OCR или предварительно поверните изображение перед обработкой.

**Q: Есть ли способ получить оценки уверенности для каждой строки?**  
A: После получения прямоугольников вызовите `api.RecognizeImage(...).Lines`, чтобы получить объекты строк, включающие значения уверенности.

**Q: Какие версии .NET совместимы?**  
A: Библиотека работает с .NET Framework 4.5+, .NET Core 3.1+, и .NET 5/6.
