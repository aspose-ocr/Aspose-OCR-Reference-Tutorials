# Конвертация изображений в PDF C# – Сохранение многостраничного результата OCR

## Введение

В этом руководстве вы узнаете, как **конвертировать изображения в PDF C#** с помощью Aspose.OCR для .NET и сохранить полученный многостраничный результат OCR в виде документа. Независимо от того, нужно ли вам **конвертировать отсканированные изображения в PDF** для архивирования или **извлекать текст из изображений C#** для обработки данных, это руководство проведёт вас через каждый шаг — с реальными примерами и рекомендациями по лучшим практикам.

## Быстрые ответы
- **Что покрывает это руководство?** Конвертация нескольких изображений в PDF/Docx/Txt/Pdf/Xlsx с использованием Aspose.OCR в C#.
- **Какие форматы поддерживаются?** Docx, Text, Pdf и Xlsx (можно также напрямую выводить PDF).
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для оценки; для продакшн‑использования требуется постоянная лицензия.

## Что такое «convert images to PDF C#»?

Конвертация изображений в PDF в C# означает программное преобразование одного или нескольких bitmap‑файлов (PNG, JPEG, TIFF и др.) в PDF‑документ, сохраняющий визуальное оформление и, при желании, включающий поисковый текст через OCR. Aspose.OCR делает этот процесс простым и высоко настраиваемым.

## Почему стоит использовать Aspose.OCR для этой задачи?

- **Высокая точность** OCR‑движка, поддерживающего множество языков.  
- **Поддержка многостраничных документов** — обработка пакетов изображений одним вызовом.  
- **Прямое сохранение** в популярные офисные форматы без дополнительных шагов конвертации.  
- **Полная интеграция с .NET** — без нативных зависимостей и внешних инструментов.

## Импорт пространств имён

Добавьте необходимые пространства имён в ваш файл C#:

```csharp
using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using Aspose.OCR;
```

Эти импорты предоставляют доступ к коллекциям, работе с файлами, LINQ и классам Aspose OCR.

## Шаг 1: Установите каталог документов

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";
```

Замените `"Your Document Directory"` на абсолютный или относительный путь к папке, где находятся исходные изображения, и куда вы хотите сохранять выходные файлы.

## Шаг 2: Инициализируйте Aspose.OCR

```csharp
// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

Создание объекта `AsposeOcr` даёт доступ ко всем операциям OCR, включая рабочий процесс **convert images to PDF C#**.

## Шаг 3: Распознайте изображения

```csharp
// Recognize image
List<RecognitionResult> result = api.RecognizeMultipleImages(
    new List<string> { dataDir + "sample.png", dataDir + "sample_bad.png" },
    new RecognitionSettings { }
).ToList();
```

Метод `RecognizeMultipleImages` обрабатывает каждый файл из списка и возвращает коллекцию `RecognitionResult`. Вы можете передать любое количество изображений, что идеально подходит для сценариев **convert scanned images to PDF**.

## Шаг 4: Сохраните результаты в нужных форматах

```csharp
// Save the result in your preferred format
AsposeOcr.SaveMultipageDocument(RunExamples.GetDataDir_OCR()+"sample.docx", SaveFormat.Docx, result);
AsposeOcr.SaveMultipageDocument(RunExamples.GetDataDir_OCR() + "sample.txt", SaveFormat.Text, result);
AsposeOcr.SaveMultipageDocument(RunExamples.GetDataDir_OCR() + "sample.pdf", SaveFormat.Pdf, result);
AsposeOcr.SaveMultipageDocument(RunExamples.GetDataDir_OCR() + "sample.xlsx", SaveFormat.Xlsx, result);
```

Выберите формат, который лучше всего подходит для вашего последующего рабочего процесса:

- **Docx** — редактируемый документ Word с поисковым текстом.  
- **Text** — извлечение простого текста для быстрой аналитики (**extract text from images C#**).  
- **Pdf** — классический PDF‑вывод, идеальный для архивирования.  
- **Xlsx** — представление в виде таблицы для табличных данных.

## Распространённые сценарии использования

- **Цифровой архив:** Конвертировать отсканированные бумажные контракты в поисковые PDF‑файлы.  
- **Автоматизация ввода данных:** Извлекать текст из чеков или счетов и загружать его в базу данных.  
- **Пакетная обработка:** Обрабатывать тысячи изображений за один запуск с минимальным объёмом кода.

## Устранение неполадок и рекомендации

- **Большие наборы изображений:** Обрабатывайте изображения небольшими партиями, чтобы избежать всплесков памяти.  
- **Качество изображений:** Убедитесь, что разрешение не менее 300 dpi для оптимальной точности OCR.  
- **Ошибки лицензии:** Проверьте, что файл лицензии корректно загружен перед вызовом методов OCR.

## Дополнительные часто задаваемые вопросы

**Q: Можно ли конвертировать изображения в PDF C# без использования OCR?**  
A: Да, вы можете использовать Aspose.PDF или другие библиотеки для чистой конвертации изображений в PDF, но OCR добавляет поисковый текст.

**Q: Как извлечь текст из изображений C# после конвертации?**  
A: Список `result`, возвращаемый `RecognizeMultipleImages`, содержит свойства `Text`, которые можно записать в файл `.txt` или обработать напрямую.

**Q: Можно ли задать пользовательские поля полей страницы или ориентацию?**  
A: При сохранении в PDF или Docx вы можете изменить макет документа через API Aspose.Words или Aspose.PDF перед вызовом `SaveMultipageDocument`.

**Q: Что происходит, если изображение не может быть прочитано?**  
A: OCR‑движок возвращает пустой `RecognitionResult` для этой страницы; вы можете проверить `result[i].Text` на `null` или пустую строку и обработать ситуацию соответственно.

**Q: Поддерживает ли API облачное развертывание?**  
A: Да, библиотека работает на любой среде .NET, включая Azure Functions и AWS Lambda, при условии соответствия требованиям версии runtime.
