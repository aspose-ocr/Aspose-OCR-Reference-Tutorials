# Как сохранить результат OCR в документ

## Введение

В этом руководстве вы узнаете **как сохранить вывод OCR**, используя Aspose.OCR for .NET. Мы пройдем процесс распознавания текста на изображении, а затем преобразуем этот текст в популярные форматы документов, такие как DOCX, TXT, PDF и XLSX. К концу вы сможете автоматизировать извлечение данных из картинок и сохранять их в виде поисковых, редактируемых файлов — идеально для архивирования, отчетности или последующей обработки.

## Быстрые ответы
- **Что означает “how to save ocr”?** Это относится к сохранению текста, распознанного из изображения, в файл формата DOCX, PDF и т.д.  
- **В какие форматы можно экспортировать?** Поддерживаются DOCX, TXT, PDF и XLSX «из коробки».  
- **Нужна ли лицензия?** Для оценки работает бесплатный пробный период; для использования в продакшене требуется коммерческая лицензия.  
- **Можно ли напрямую конвертировать изображение в PDF?** Да — сохраните результат OCR как PDF, получив поисковый PDF‑документ.  
- **Поддерживается ли PNG?** Абсолютно; вы можете **извлекать текст из PNG**‑изображений тем же API.

## Что такое OCR и почему сохранять результаты в документы?

Оптическое распознавание символов (OCR) преобразует печатный или рукописный текст внутри изображений в машинно‑читаемые строки. Сохранение этих строк в виде документов позволяет:

* Создавать поисковые PDF для соответствия требованиям.  
* Заполнять таблицы (XLSX) для анализа данных.  
* Генерировать редактируемые отчёты (DOCX).  
* Архивировать простые текстовые логи (TXT) для быстрого поиска.

## Предварительные требования

Прежде чем начать, убедитесь, что у вас есть:

- Установленный Aspose.OCR for .NET. Скачать его можно **[здесь](https://releases.aspose.com/ocr/net/)**.  
- Папка на вашем компьютере, в которой будут храниться исходные изображения и выходные документы. Обновите переменную `dataDir` в коде, указав путь к этой папке.

## Импорт пространств имён

Нужны несколько пространств имён .NET для работы с файловой системой и классами Aspose OCR.

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

### Шаг 1: Инициализация Aspose.OCR

Укажите путь к рабочему каталогу и создайте экземпляр OCR‑движка.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Initialize an instance of AsposeOcr
AsposeOcr api = new AsposeOcr();
```

### Шаг 2: Распознавание изображения

Передайте файл изображения (например, PNG) распознавателю. Здесь мы **распознаём текстовые изображения** и превращаем их в объект `RecognitionResult`.

```csharp
// Recognize image
RecognitionResult result = api.RecognizeImage(dataDir + "sample.png", new RecognitionSettings { });
```

### Шаг 3: Сохранение результата в разных форматах

Теперь экспортируем распознанный текст. Выберите формат, соответствующий вашему рабочему процессу — будь то **конвертация изображения в PDF**, **извлечение текста из PNG** или создание таблицы.

```csharp
// Save the result in your preferred format
result.Save(RunExamples.GetDataDir_OCR() + "sample.docx", SaveFormat.Docx);
result.Save(RunExamples.GetDataDir_OCR() + "sample.txt", SaveFormat.Text);
result.Save(RunExamples.GetDataDir_OCR() + "sample.pdf", SaveFormat.Pdf);
result.Save(RunExamples.GetDataDir_OCR() + "sample.xlsx", SaveFormat.Xlsx);
```

### Шаг 4: Вывод сообщения об успехе

Простое сообщение в консоли подтверждает, что процесс завершён без ошибок.

```csharp
Console.WriteLine("SaveResultAsDocument executed successfully");
```

## Распространённые ошибки и советы

- **Пути к файлам:** Всегда используйте абсолютные пути или убедитесь, что `dataDir` заканчивается разделителем пути (`\` или `/`).  
- **Качество изображения:** Изображения с более высоким разрешением повышают точность; рассмотрите предобработку (выравнивание, шумоподавление) для лучших результатов.  
- **Режим лицензии:** В режиме оценки вывод может содержать водяной знак; примените действующую лицензию, чтобы его убрать.
